Section 47-1

日別完了数の集計SQL

GitHub風のヒートマップを作成するため、過去1年分の日別タスク完了数を集計するSQLを実装します。

過去365日分の日別完了タスク数を集計

データ構造

日付をキー、完了数を値とするMapデータ

miscellaneous_services

HeatmapService

集計結果をヒートマップ描画用に変換

StatsMapper.xml（年間日別集計）
<select id="countDailyCompletedYearly" resultType="map">
    SELECT DATE(updated_at) AS completed_date,
           COUNT(*) AS cnt
    FROM todo
    WHERE user_id = #{userId}
      AND status = 'COMPLETED'
      AND updated_at >= DATE_SUB(CURDATE(), INTERVAL 365 DAY)
    GROUP BY DATE(updated_at)
    ORDER BY completed_date
</select>
HeatmapService.java
import org.springframework.stereotype.Service;
import java.time.LocalDate;
import java.util.*;

@Service
public class HeatmapService {

    private final StatsMapper statsMapper;

    public HeatmapService(StatsMapper statsMapper) {
        this.statsMapper = statsMapper;
    }

    /**
     * 過去365日分の日別完了数マップを取得
     * データがない日は0件として補完
     */
    public Map<String, Integer> getYearlyHeatmapData(Integer userId) {
        List<Map<String, Object>> rawData =
            statsMapper.countDailyCompletedYearly(userId);

        // SQLの結果をMapに変換
        Map<String, Integer> dbData = new LinkedHashMap<>();
        for (Map<String, Object> row : rawData) {
            dbData.put(
                row.get("completed_date").toString(),
                ((Number) row.get("cnt")).intValue()
            );
        }

        // 365日分を補完（データがない日は0）
        Map<String, Integer> result = new LinkedHashMap<>();
        LocalDate today = LocalDate.now();
        for (int i = 364; i >= 0; i--) {
            String dateStr = today.minusDays(i).toString();
            result.put(dateStr, dbData.getOrDefault(dateStr, 0));
        }

        return result;
    }
}
lightbulb日付の補完処理

SQLの結果にはタスクを完了した日だけが含まれます。完了0件の日もヒートマップに表示するため、Java側で365日分のデータを補完しています。

heatmap.js（SVGヒートマップ描画）
/**
 * GitHub風ヒートマップをSVGで描画
 */
function renderHeatmap(containerId, data) {
    const cellSize = 12;
    const cellGap = 3;
    const totalSize = cellSize + cellGap;
    const weeks = 53;
    const days = 7;
    const svgWidth = weeks * totalSize + 40;
    const svgHeight = days * totalSize + 30;

    const svg = document.createElementNS(
        'http://www.w3.org/2000/svg', 'svg');
    svg.setAttribute('viewBox', `0 0 ${svgWidth} ${svgHeight}`);
    svg.style.maxWidth = '100%';

    // 曜日ラベル
    const dayLabels = ['日', '月', '火', '水', '木', '金', '土'];
    [1, 3, 5].forEach(i => {
        const text = document.createElementNS(
            'http://www.w3.org/2000/svg', 'text');
        text.setAttribute('x', 0);
        text.setAttribute('y', 30 + i * totalSize + 9);
        text.setAttribute('font-size', '10');
        text.setAttribute('fill', '#94a3b8');
        text.textContent = dayLabels[i];
        svg.appendChild(text);
    });

    // データからセルを描画
    const entries = Object.entries(data);
    entries.forEach((entry, index) => {
        const [dateStr, count] = entry;
        const date = new Date(dateStr);
        const week = Math.floor(index / 7);
        const day = date.getDay();

        const rect = document.createElementNS(
            'http://www.w3.org/2000/svg', 'rect');
        rect.setAttribute('x', 30 + week * totalSize);
        rect.setAttribute('y', 20 + day * totalSize);
        rect.setAttribute('width', cellSize);
        rect.setAttribute('height', cellSize);
        rect.setAttribute('rx', 2);
        rect.setAttribute('fill', getHeatColor(count));
        rect.setAttribute('data-date', dateStr);
        rect.setAttribute('data-count', count);

        svg.appendChild(rect);
    });

    document.getElementById(containerId).appendChild(svg);
}

/**
 * 完了数に応じた色を返す
 */
function getHeatColor(count) {
    if (count === 0) return '#ebedf0';
    if (count <= 1) return '#9be9a8';
    if (count <= 3) return '#40c463';
    if (count <= 5) return '#30a14e';
    return '#216e39';
}
HeatmapController.java
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;
import com.fasterxml.jackson.databind.ObjectMapper;

@Controller
public class HeatmapController {

    private final HeatmapService heatmapService;
    private final ObjectMapper objectMapper;

    public HeatmapController(HeatmapService heatmapService,
                              ObjectMapper objectMapper) {
        this.heatmapService = heatmapService;
        this.objectMapper = objectMapper;
    }

    @GetMapping("/heatmap")
    public String heatmap(
            @AuthenticationPrincipal LoginUser loginUser,
            Model model) throws Exception {

        Map<String, Integer> data =
            heatmapService.getYearlyHeatmapData(
                loginUser.getUserId());

        // ThymeleafからJavaScriptへJSON形式で渡す
        model.addAttribute("heatmapData",
            objectMapper.writeValueAsString(data));
        return "heatmap";
    }
}
warningSVG名前空間

JavaScriptでSVG要素を動的生成する場合、document.createElementではなくdocument.createElementNS('http://www.w3.org/2000/svg', 'rect')を使用してください。名前空間を指定しないとSVG要素として認識されません。

heatmap.js（ツールチップ追加）
// ツールチップ要素を作成
const tooltip = document.createElement('div');
tooltip.className = 'heatmap-tooltip';
tooltip.style.display = 'none';
document.body.appendChild(tooltip);

// 各セルにイベントリスナーを追加
svg.querySelectorAll('rect[data-date]').forEach(rect => {
    rect.addEventListener('mouseenter', function(e) {
        const date = this.getAttribute('data-date');
        const count = this.getAttribute('data-count');
        tooltip.innerHTML =
            `<strong>${date}</strong><br>${count}件のタスクを完了`;
        tooltip.style.display = 'block';
    });

    rect.addEventListener('mousemove', function(e) {
        tooltip.style.left = (e.pageX + 12) + 'px';
        tooltip.style.top = (e.pageY - 40) + 'px';
    });

    rect.addEventListener('mouseleave', function() {
        tooltip.style.display = 'none';
    });
});

lightbulbpointer-events: noneの重要性

ツールチップにpointer-events: noneを設定することで、ツールチップがマウスイベントを阻害しません。これにより、セル間をスムーズに移動してもツールチップが正しく追従します。

# Step 1: 日別完了数の集計
過去365日分の日別タスク完了数を集計するSQLとServiceを作成してください。データのない日は0で補完してLinkedHashMapで返してください。

# Step 2: SVGヒートマップ描画
JavaScriptでSVGのrect要素を動的生成し、完了数に応じた5段階のグリーン系でGitHub風ヒートマップを描画してください。

# Step 3: ツールチップ
mouseenter/mouseleave/mousemoveイベントで、日付と完了件数のカスタムツールチップを表示してください。

# Step 4: 年間サマリーと凡例
年間合計完了数、1日の最大完了数、アクティブ日数のサマリーと、色の凡例（Less - More）を追加してください。
