Section 40-1

見積もり・実績カラム追加

タスクに見積もり時間と実績時間を記録するカラムを追加します。分単位で管理し、表示時に「○時間○分」形式に変換します。

見積もり時間

タスク作成時に想定作業時間を入力

timer

実績時間

タイマーや手動入力で実際の作業時間を記録

data_usage

分単位管理

INT型で分を保存し、表示時にフォーマット変換

schema.sql（時間カラム追加）
ALTER TABLE todo
    ADD COLUMN estimated_minutes INT DEFAULT NULL,
    ADD COLUMN actual_minutes INT DEFAULT 0;
Todo.java（時間フィールド追加）
public class Todo {
    // 既存フィールド...

    private Integer estimatedMinutes;  // 見積もり時間（分）
    private Integer actualMinutes;     // 実績時間（分）

    /**
     * 分を「○h ○m」形式に変換
     */
    public String formatMinutes(Integer minutes) {
        if (minutes == null || minutes == 0) return "-";
        int h = minutes / 60;
        int m = minutes % 60;
        if (h > 0 && m > 0) return h + "h " + m + "m";
        if (h > 0) return h + "h";
        return m + "m";
    }

    public String getEstimatedFormatted() {
        return formatMinutes(estimatedMinutes);
    }
    public String getActualFormatted() {
        return formatMinutes(actualMinutes);
    }

    // getter/setter
}
lightbulb分単位で保存する理由

INT型で分数を保存すると、集計（SUM/AVG）や比較が簡単です。表示時にJavaのメソッドで「○h ○m」形式にフォーマットすれば、データベース設計がシンプルに保てます。

Section 40-3

タイマー機能（JavaScript）

タスク詳細画面にストップウォッチ式タイマーを実装し、作業時間を自動計測します。開始/停止ボタンで操作し、停止時にサーバーに実績時間を送信します。

task-timer.js（ストップウォッチ）
class TaskTimer {
    constructor(taskId) {
        this.taskId = taskId;
        this.startTime = null;
        this.intervalId = null;
        this.display = document.getElementById('timer-display');
        this.btn = document.getElementById('timer-btn');
    }

    toggle() {
        if (this.intervalId) {
            this.stop();
        } else {
            this.start();
        }
    }

    start() {
        this.startTime = Date.now();
        this.btn.textContent = 'Stop';
        this.btn.classList.add('active');
        this.intervalId = setInterval(() => this.updateDisplay(), 1000);
    }

    stop() {
        clearInterval(this.intervalId);
        this.intervalId = null;
        this.btn.textContent = 'Start';
        this.btn.classList.remove('active');

        // 経過時間を分に変換してサーバーに送信
        const elapsed = Date.now() - this.startTime;
        const minutes = Math.round(elapsed / 60000);

        if (minutes > 0) {
            this.saveTime(minutes);
        }
    }

    updateDisplay() {
        const elapsed = Date.now() - this.startTime;
        const sec = Math.floor(elapsed / 1000);
        const h = Math.floor(sec / 3600);
        const m = Math.floor((sec % 3600) / 60);
        const s = sec % 60;
        this.display.textContent =
            `${String(h).padStart(2,'0')}:${String(m).padStart(2,'0')}:${String(s).padStart(2,'0')}`;
    }

    async saveTime(minutes) {
        try {
            await fetch(`/api/tasks/${this.taskId}/time`, {
                method: 'POST',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify({ minutes })
            });
            location.reload();
        } catch (err) {
            alert('時間の保存に失敗しました');
        }
    }
}

document.addEventListener('DOMContentLoaded', () => {
    const taskId = document.getElementById('task-id')?.value;
    if (taskId) {
        const timer = new TaskTimer(taskId);
        document.getElementById('timer-btn')
            ?.addEventListener('click', () => timer.toggle());
    }
});
TimeTrackingController.java（実績記録API）
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;
import java.util.Map;

@RestController
@RequestMapping("/api/tasks/{taskId}/time")
public class TimeTrackingController {

    private final TodoMapper todoMapper;

    public TimeTrackingController(TodoMapper todoMapper) {
        this.todoMapper = todoMapper;
    }

    @PostMapping
    public ResponseEntity<?> addTime(
            @PathVariable Integer taskId,
            @RequestBody Map<String, Integer> body) {

        Integer minutes = body.get("minutes");
        if (minutes == null || minutes <= 0) {
            return ResponseEntity.badRequest()
                .body(Map.of("error", "無効な時間です"));
        }

        todoMapper.addActualMinutes(taskId, minutes);
        return ResponseEntity.ok(Map.of(
            "message", minutes + "分を記録しました"));
    }
}
warningページ離脱時の対策

タイマー動作中にページを離脱すると計測中のデータが失われます。beforeunloadイベントで警告を表示するか、localStorageに開始時刻を保存して復帰できるようにしましょう。
