期限日と期限切れ警告

Section 9-1

期限日カラムの追加

ToDoに期限日を設定し、期限切れや期限間近を警告表示します。

完成イメージ

warning 期限切れ: 2024/01/10

schedule あと2日: 2024/01/20

calendar_today 2024/02/15

schema.sql 
ALTER TABLE todo ADD deadline DATE;
Todo.java（期限日追加） 
import java.time.LocalDate;
import java.time.temporal.ChronoUnit;

private LocalDate deadline;

public LocalDate getDeadline() {
    return deadline;
}

public void setDeadline(LocalDate deadline) {
    this.deadline = deadline;
}

// 期限切れ判定
public boolean isExpired() {
    return deadline != null && deadline.isBefore(LocalDate.now());
}

// 期限間近判定（3日以内）
public boolean isDeadlineSoon() {
    if (deadline == null) return false;
    long days = ChronoUnit.DAYS.between(LocalDate.now(), deadline);
    return days >= 0 && days <= 3;
}

// 残り日数取得
public long getDaysUntilDeadline() {
    if (deadline == null) return Long.MAX_VALUE;
    return ChronoUnit.DAYS.between(LocalDate.now(), deadline);
}
Section 9-2

日付入力フォーム

edit.html（日付入力） 
<tr>
    <th><label for="deadline">期限日</label></th>
    <td>
        <input type="date" id="deadline" th:field="*{deadline}"
               th:min="${#temporals.format(#temporals.createNow(), 'yyyy-MM-dd')}">
    </td>
</tr>
lightbulbmin属性

min属性で今日以降の日付のみ選択可能にできます。

Section 9-3

期限切れ警告の表示

index.html（期限表示） 
<td th:if="${todo.deadline != null}">
    <!-- 期限切れ -->
    <span th:if="${todo.expired}" class="deadline-expired">
        ⚠️ 期限切れ: <span th:text="${#temporals.format(todo.deadline, 'yyyy/MM/dd')}"></span>
    </span>
    <!-- 期限間近 -->
    <span th:if="${!todo.expired and todo.deadlineSoon}" class="deadline-soon">
        ⏰ あと<span th:text="${todo.daysUntilDeadline}"></span>日:
        <span th:text="${#temporals.format(todo.deadline, 'yyyy/MM/dd')}"></span>
    </span>
    <!-- 通常 -->
    <span th:if="${!todo.expired and !todo.deadlineSoon}" class="deadline-ok">
        📅 <span th:text="${#temporals.format(todo.deadline, 'yyyy/MM/dd')}"></span>
    </span>
</td>
<td th:unless="${todo.deadline != null}">-</td>

AIプログラミング

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、この機能を実装できます。

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリに期限日と期限切れ警告機能を追加してください。

【要件】
- ToDoに期限日（deadline）フィールドを追加
- 入力フォームに日付選択（input type="date"）を追加
- 一覧画面で期限日を表示
- 期限切れ（過去日）のToDoは赤色でハイライト
- 期限が近い（3日以内）ToDoは黄色でハイライト
- 期限日でのソートが可能

【技術スタック】
- Spring Boot 3.x
- Java LocalDate
- Thymeleaf (#temporals)
gps_fixed ステップ別プロンプト 
# Step 1: Entityとテーブルの修正
TodoエンティティにLocalDate型のdeadlineフィールドを追加し、データベースにDATE型カラムを追加してください。

# Step 2: フォームの修正
入力フォームにtype="date"のinputタグを追加し、日付選択できるようにしてください。

# Step 3: 期限切れ判定の実装
TodoエンティティにisOverdue()とisNearDeadline()メソッドを追加し、現在日付と比較してください。

# Step 4: 一覧画面の修正
期限切れの行にoverdue-row、期限が近い行にwarning-rowクラスを追加してスタイリングしてください。

表示フォーマットを指定すると、期待通りの見た目が得られます。
