CategoryCountDto.java（集計DTO） 
public class CategoryCountDto {

    private String categoryName;
    private Long count;

    public CategoryCountDto() {}

    public CategoryCountDto(String categoryName, Long count) {
        this.categoryName = categoryName;
        this.count = count;
    }

    // getter/setter
    public String getCategoryName() { return categoryName; }
    public void setCategoryName(String categoryName) { this.categoryName = categoryName; }
    public Long getCount() { return count; }
    public void setCount(Long count) { this.count = count; }
}
ChartMapper.java / ChartMapper.xml（集計SQL） 
// --- ChartMapper.java ---
@Mapper
public interface ChartMapper {
    List<CategoryCountDto> countByCategory();
    List<CategoryCountDto> countByPriority();
}

// --- ChartMapper.xml ---
<select id="countByCategory" resultType="CategoryCountDto">
    SELECT c.name AS categoryName, COUNT(t.id) AS count
    FROM todo t
    JOIN category c ON t.category_id = c.id
    GROUP BY c.name
    ORDER BY count DESC
</select>

<select id="countByPriority" resultType="CategoryCountDto">
    SELECT t.priority AS categoryName, COUNT(t.id) AS count
    FROM todo t
    GROUP BY t.priority
    ORDER BY FIELD(t.priority, 'HIGH', 'MEDIUM', 'LOW')
</select>
ChartApiController.java（集計APIエンドポイント） 
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;
import java.util.List;

@RestController
@RequestMapping("/api/charts")
public class ChartApiController {

    private final ChartMapper chartMapper;

    public ChartApiController(ChartMapper chartMapper) {
        this.chartMapper = chartMapper;
    }

    /**
     * カテゴリ別タスク数を取得
     */
    @GetMapping("/category-count")
    public List<CategoryCountDto> categoryCount() {
        return chartMapper.countByCategory();
    }

    /**
     * 優先度別タスク数を取得
     */
    @GetMapping("/priority-count")
    public List<CategoryCountDto> priorityCount() {
        return chartMapper.countByPriority();
    }
}

pie-chart.html（Thymeleafテンプレート） 
<!DOCTYPE html>
<html xmlns:th="http://www.thymeleaf.org">
<head>
    <title>カテゴリ別円グラフ</title>
    <link rel="stylesheet" th:href="@{/css/style.css}">
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
</head>
<body>
    <h1>タスク分布</h1>

    <div class="chart-container" style="max-width: 500px; margin: 0 auto;">
        <canvas id="categoryChart"></canvas>
    </div>

    <script>
    // APIからデータ取得してチャート描画
    fetch('/api/charts/category-count')
        .then(res => res.json())
        .then(data => {
            const labels = data.map(d => d.categoryName);
            const values = data.map(d => d.count);
            const total  = values.reduce((a, b) => a + b, 0);

            const colors = [
                '#3b82f6', '#22c55e', '#f59e0b',
                '#ef4444', '#a855f7', '#ec4899',
                '#06b6d4', '#84cc16'
            ];

            new Chart(document.getElementById('categoryChart'), {
                type: 'doughnut',
                data: {
                    labels: labels,
                    datasets: [{
                        data: values,
                        backgroundColor: colors.slice(0, labels.length),
                        borderWidth: 2,
                        borderColor: '#ffffff'
                    }]
                },
                options: {
                    responsive: true,
                    plugins: {
                        legend: { position: 'bottom' },
                        tooltip: {
                            callbacks: {
                                label: ctx => {
                                    const pct = ((ctx.parsed / total) * 100).toFixed(1);
                                    return ctx.label + ': ' + ctx.parsed + '件 (' + pct + '%)';
                                }
                            }
                        }
                    }
                },
                plugins: [{
                    // 中央にタスク総数を表示するカスタムプラグイン
                    id: 'centerText',
                    beforeDraw(chart) {
                        const { width, height, ctx } = chart;
                        ctx.save();
                        ctx.font = 'bold 24px "Noto Sans JP"';
                        ctx.textAlign = 'center';
                        ctx.textBaseline = 'middle';
                        ctx.fillStyle = '#374151';
                        ctx.fillText(total + '件', width / 2, height / 2);
                        ctx.restore();
                    }
                }]
            });
        });
    </script>
</body>
</html>

カスタムHTML凡例の実装 
// Chart.jsの組み込み凡例を無効化し、カスタムHTML凡例を生成
const chart = new Chart(document.getElementById('categoryChart'), {
    type: 'doughnut',
    data: { /* ... */ },
    options: {
        responsive: true,
        plugins: {
            legend: { display: false }  // 組み込み凡例を非表示
        }
    }
});

// カスタムHTML凡例を生成
function createCustomLegend(chart, containerId) {
    const container = document.getElementById(containerId);
    const data = chart.data;

    data.labels.forEach((label, index) => {
        const color = data.datasets[0].backgroundColor[index];
        const value = data.datasets[0].data[index];

        const item = document.createElement('div');
        item.className = 'legend-item';
        item.innerHTML =
            '<span class="legend-color" style="background:' + color + '"></span>' +
            '<span class="legend-label">' + label + '</span>' +
            '<span class="legend-value">' + value + '件</span>';

        // クリックで表示/非表示を切り替え
        item.addEventListener('click', () => {
            chart.toggleDataVisibility(index);
            chart.update();
            item.classList.toggle('legend-hidden');
        });

        // ホバーでセグメントをハイライト
        item.addEventListener('mouseenter', () => {
            chart.setActiveElements([{ datasetIndex: 0, index: index }]);
            chart.update();
        });
        item.addEventListener('mouseleave', () => {
            chart.setActiveElements([]);
            chart.update();
        });

        container.appendChild(item);
    });
}

createCustomLegend(chart, 'customLegend');

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリにカテゴリ別・優先度別の円グラフ表示機能を追加してください。

【要件】
- カテゴリ別タスク数をGROUP BYで集計するAPIを作成
- 優先度別（HIGH/MEDIUM/LOW）タスク数の集計APIも作成
- Chart.jsのドーナツチャートでカテゴリ分布を表示
- Chart.jsの円グラフで優先度分布を表示
- 2つのチャートを横並びにレイアウト
- ドーナツの中央にタスク総数を表示

【技術スタック】
- Spring Boot 3.x + MyBatis
- Chart.js（CDN）
- Thymeleaf
- fetch APIでデータ取得
gps_fixed ステップ別プロンプト 
# Step 1: 集計DTO・Mapperの作成
CategoryCountDto（categoryName, count）を作成し、カテゴリ別・優先度別にGROUP BYで集計するMyBatis SQLを実装してください。

# Step 2: REST APIコントローラー
/api/charts/category-count と /api/charts/priority-count の2つのエンドポイントをRestControllerで作成してください。

# Step 3: Chart.jsでドーナツチャート描画
fetch APIで集計データを取得し、Chart.jsのdoughnutタイプで描画してください。中央にタスク総数を表示するカスタムプラグインも追加してください。

# Step 4: カスタムHTML凡例
Chart.jsの組み込み凡例の代わりにHTML要素で凡例を自作してください。クリックで表示切替、ホバーでハイライトの機能を実装してください。

ユーザーが得たい情報の表示形式を具体的に指定すると実用的なチャートが得られます。
