Section 62-1

WYSIWYGライブラリ導入

タスクの説明欄にリッチテキスト（太字、リスト、リンクなど）を入力できるよう、Quill.jsエディタを導入します。CDNからの読み込みだけで簡単に使えるオープンソースのWYSIWYGエディタです。

Quillエディタのツールバー設定と初期化処理を実装します。必要な書式ボタンだけを表示し、フォーム送信時にHTMLコンテンツをhiddenフィールドにコピーします。

richtext-editor.js（エディタ初期化）
document.addEventListener('DOMContentLoaded', () => {
    // ツールバーの設定
    const toolbarOptions = [
        ['bold', 'italic', 'underline', 'strike'],
        [{ 'header': [1, 2, 3, false] }],
        [{ 'list': 'ordered' }, { 'list': 'bullet' }],
        [{ 'indent': '-1' }, { 'indent': '+1' }],
        ['link', 'code-block'],
        ['clean']  // 書式クリア
    ];

    // Quillエディタの初期化
    const quill = new Quill('#editor-container', {
        theme: 'snow',
        modules: {
            toolbar: toolbarOptions
        },
        placeholder: 'タスクの詳細を入力...'
    });

    // 既存データの読み込み（編集時）
    const hiddenField = document.getElementById('description-hidden');
    if (hiddenField.value) {
        quill.root.innerHTML = hiddenField.value;
    }

    // フォーム送信時にHTMLをhiddenフィールドにコピー
    const form = document.querySelector('form');
    form.addEventListener('submit', () => {
        const html = quill.root.innerHTML;
        // 空エディタの場合はnullを送信
        hiddenField.value = html === '<p><br></p>'
            ? '' : html;
    });

    // リアルタイムでhiddenフィールドを更新（任意）
    quill.on('text-change', () => {
        hiddenField.value = quill.root.innerHTML;
    });
});

Section 62-4

サニタイズ処理

ユーザーが入力したHTMLから危険なタグやスクリプトを除去するサニタイズ処理を実装します。jsoupライブラリのCleanerを使って安全なHTMLタグのみ許可します。

p, b, i, ul, ol, li, a, h1-h3, codeなどを許可

保存時処理

DBに保存する前にサニタイズを実行

build.gradle（jsoup依存関係）
dependencies {
    // HTMLサニタイズ
    implementation 'org.jsoup:jsoup:1.17.2'
}
HtmlSanitizer.java（サニタイズ処理）
import org.jsoup.Jsoup;
import org.jsoup.safety.Safelist;
import org.springframework.stereotype.Component;

@Component
public class HtmlSanitizer {

    // 許可するHTMLタグのホワイトリスト
    private static final Safelist WHITELIST = Safelist.relaxed()
        .addTags("pre", "code", "blockquote")
        .addAttributes("a", "href", "target", "rel")
        .addAttributes("pre", "class")
        .addAttributes("code", "class")
        .addProtocols("a", "href", "http", "https", "mailto")
        // scriptタグ、onclickなどのイベント属性は自動除去
        .removeTags("script", "style", "iframe", "object",
            "embed", "form", "input");

    /**
     * HTMLをサニタイズする
     * 許可されたタグ以外は全て除去される
     */
    public String sanitize(String html) {
        if (html == null || html.isBlank()) {
            return html;
        }
        return Jsoup.clean(html, WHITELIST);
    }
}
TodoService.java（保存前にサニタイズ）
@Service
public class TodoService {

    private final TodoMapper todoMapper;
    private final HtmlSanitizer sanitizer;

    public TodoService(TodoMapper todoMapper,
                        HtmlSanitizer sanitizer) {
        this.todoMapper = todoMapper;
        this.sanitizer = sanitizer;
    }

    public void createTodo(Todo todo) {
        // 説明文のHTMLをサニタイズ
        todo.setDescription(
            sanitizer.sanitize(todo.getDescription()));
        todoMapper.insert(todo);
    }

    public void updateTodo(Todo todo) {
        todo.setDescription(
            sanitizer.sanitize(todo.getDescription()));
        todoMapper.update(todo);
    }
}

lightbulbホワイトリスト方式が安全な理由

ブラックリスト方式（危険なタグを除外）は新しい攻撃手法に対応できません。
ホワイトリスト方式（安全なタグのみ許可）を使うことで、未知の攻撃にも対応できます。jsoupのSafelistはこのアプローチを採用しています。

edit_note 基本プロンプト
Spring Boot 3.x + ThymeleafのToDoアプリにリッチテキストエディタを追加してください。

【要件】
- Quill.js (CDN) をタスク説明欄に導入
- ツールバー: 太字、斜体、リスト、見出し、リンク、コードブロック
- フォーム送信時にHTMLをhiddenフィールドにコピーしてサーバーに送信
- jsoupでHTMLサニタイズ（XSS防止）してからDBに保存
- th:utextで詳細画面にリッチテキスト表示

【技術スタック】
- Quill.js 2.x（WYSIWYG）
- jsoup 1.17.x（HTMLサニタイズ）
- Thymeleaf th:utext
- Spring Boot 3.x + MyBatis
gps_fixed ステップ別プロンプト
# Step 1: Quill.js導入
Quill.jsをCDNから読み込み、タスクフォームのdescription欄にWYSIWYGエディタを配置してください。

# Step 2: エディタ初期化
ツールバーの設定、初期値の読み込み、フォーム送信時のHTMLコピー処理を実装してください。

# Step 3: HTMLの保存と表示
descriptionカラムをTEXT型に変更し、HTMLを保存。詳細画面ではth:utextで表示してください。

# Step 4: サニタイズ処理
jsoupを使って、保存前にHTMLをサニタイズするHtmlSanitizerクラスを作成してください。script/iframe/onclickなどを除去してください。
lightbulbプロンプトのコツ

「HTMLをDBに保存する前にjsoupでサニタイズし、XSSを防止してください」のように、セキュリティ要件を必ず含めましょう。リッチテキストはXSS攻撃の典型的な入口です。
