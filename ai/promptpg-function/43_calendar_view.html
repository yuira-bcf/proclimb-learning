CalendarEvent.java（イベントDTO） 
/**
 * FullCalendar用のイベントDTO
 * FullCalendarが要求するJSON形式に合わせる
 */
public class CalendarEvent {

    private String id;        // タスクID
    private String title;     // タスクタイトル
    private String start;     // 開始日（ISO 8601形式）
    private String end;       // 終了日（省略可）
    private String color;     // 背景色
    private String textColor; // 文字色
    private boolean allDay;   // 終日イベント

    public CalendarEvent(Todo todo) {
        this.id = String.valueOf(todo.getId());
        this.title = todo.getTitle();
        this.start = todo.getDeadline().toString();
        this.allDay = true;
        this.textColor = "#ffffff";

        // ステータス別に色分け
        if ("COMPLETED".equals(todo.getStatus())) {
            this.color = "#22c55e";  // 緑：完了
        } else if (todo.getDeadline().isBefore(
                java.time.LocalDate.now())) {
            this.color = "#ef4444";  // 赤：期限切れ
        } else {
            this.color = "#3b82f6";  // 青：未完了
        }
    }

    // getter/setter
}
CalendarApiController.java（REST API） 
import org.springframework.web.bind.annotation.*;
import org.springframework.security.core.annotation.AuthenticationPrincipal;
import java.util.List;
import java.util.stream.Collectors;

@RestController
@RequestMapping("/api/calendar")
public class CalendarApiController {

    private final TodoMapper todoMapper;

    public CalendarApiController(TodoMapper todoMapper) {
        this.todoMapper = todoMapper;
    }

    /**
     * FullCalendar用イベント一覧を返す
     * @param start FullCalendarが送信する表示期間の開始日
     * @param end   FullCalendarが送信する表示期間の終了日
     */
    @GetMapping("/events")
    public List<CalendarEvent> getEvents(
            @RequestParam String start,
            @RequestParam String end,
            @AuthenticationPrincipal LoginUser loginUser) {

        List<Todo> todos = todoMapper.findByUserIdAndDeadlineBetween(
            loginUser.getUserId(), start, end
        );

        return todos.stream()
            .filter(t -> t.getDeadline() != null)
            .map(CalendarEvent::new)
            .collect(Collectors.toList());
    }
}
TodoMapper.xml（期間指定検索） 
<select id="findByUserIdAndDeadlineBetween" resultType="Todo">
    SELECT id, title, status, deadline, category_id
    FROM todo
    WHERE user_id = #{userId}
      AND deadline >= #{start}
      AND deadline <  #{end}
    ORDER BY deadline ASC
</select>
warningstartとendパラメータ

FullCalendarは表示期間が変わるたびにstartとendパラメータを自動送信します。月表示の場合、前後の週も含まれるため実際の月の範囲より広くなります。

TodoController.java（日付パラメータ対応） 
/**
 * タスク新規作成フォーム
 * カレンダーからの遷移時はdateパラメータで期限を受け取る
 */
@GetMapping("/todo/new")
public String newTodoForm(
        @RequestParam(required = false) String date,
        Model model) {

    TodoForm form = new TodoForm();

    // カレンダーから日付が渡された場合、期限に設定
    if (date != null && !date.isEmpty()) {
        form.setDeadline(LocalDate.parse(date));
    }

    model.addAttribute("todoForm", form);
    return "todo/form";
}

/**
 * タスク作成処理
 * fromパラメータがcalendarの場合、カレンダーに戻る
 */
@PostMapping("/todo/new")
public String createTodo(
        @Valid @ModelAttribute TodoForm form,
        BindingResult result,
        @RequestParam(defaultValue = "list") String from) {

    if (result.hasErrors()) {
        return "todo/form";
    }

    todoService.create(form);

    if ("calendar".equals(from)) {
        return "redirect:/calendar";
    }
    return "redirect:/todo";
}

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリにカレンダービュー機能を追加してください。

【要件】
- FullCalendar v6でカレンダー表示（月/週/リスト切替）
- タスクの期限日をカレンダー上にイベントとして表示
- ステータス別に色分け（完了:緑、未完了:青、期限切れ:赤）
- 日付クリックでその日を期限としたタスク作成画面へ遷移
- イベントクリックでタスク編集画面へ遷移

【技術スタック】
- Spring Boot 3.x + MyBatis
- FullCalendar v6（CDN、日本語ロケール）
- REST API（JSON形式でイベントデータを返す）
gps_fixed ステップ別プロンプト 
# Step 1: FullCalendar導入
FullCalendar v6をCDNから導入し、日本語ロケールで月表示のカレンダーを表示するThymeleafテンプレートを作成してください。

# Step 2: イベントAPIの作成
/api/calendar/eventsエンドポイントを作成し、FullCalendarが要求する形式（id, title, start, color）でタスクデータをJSON返却してください。ステータスに応じて色を変えてください。

# Step 3: カレンダーイベント表示のカスタマイズ
eventDidMountで完了タスクに取り消し線を追加し、dayMaxEvents:3で表示件数を制限してください。

# Step 4: 日付クリックでタスク作成
日付クリック時に/todo/new?date=YYYY-MM-DDに遷移し、期限フィールドに日付を自動設定する機能を追加してください。
