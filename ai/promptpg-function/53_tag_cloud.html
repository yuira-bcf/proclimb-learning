<select id="countTagUsage" resultType="TagCountDto">
    SELECT t.name AS tagName, COUNT(tt.todo_id) AS usageCount
    FROM tag t
    JOIN todo_tag tt ON t.id = tt.tag_id
    GROUP BY t.name
    ORDER BY usageCount DESC
</select>
TagCountDto.java
public class TagCountDto {
    private String tagName;
    private Long usageCount;
    private int fontSize;  // 算出されたフォントサイズ(px)

    // getter/setter
    public String getTagName() { return tagName; }
    public void setTagName(String tagName) { this.tagName = tagName; }
    public Long getUsageCount() { return usageCount; }
    public void setUsageCount(Long usageCount) { this.usageCount = usageCount; }
    public int getFontSize() { return fontSize; }
    public void setFontSize(int fontSize) { this.fontSize = fontSize; }
}

Section 53-2

フォントサイズ算出ロジック

使用頻度に応じてフォントサイズを線形補間で算出します。最小12px〜最大48pxの範囲にマッピングします。

text_fields

線形補間

最小/最大の使用回数からフォントサイズを比例計算

tune

上下限クランプ

最小12px・最大48pxで範囲制限

functions

対数スケール（オプション）

頻度差が大きい場合にログ変換で調整

TagCloudService.java（フォントサイズ算出）
import org.springframework.stereotype.Service;
import java.util.Collections;
import java.util.List;

@Service
public class TagCloudService {

    private static final int MIN_FONT_SIZE = 12;
    private static final int MAX_FONT_SIZE = 48;

    private final TagCloudMapper tagCloudMapper;

    public TagCloudService(TagCloudMapper tagCloudMapper) {
        this.tagCloudMapper = tagCloudMapper;
    }

    /**
     * タグクラウド用データを取得（フォントサイズ算出済み）
     */
    public List<TagCountDto> getTagCloud() {
        List<TagCountDto> tags = tagCloudMapper.countTagUsage();
        if (tags.isEmpty()) return tags;

        long minCount = tags.stream()
            .mapToLong(TagCountDto::getUsageCount).min().orElse(0);
        long maxCount = tags.stream()
            .mapToLong(TagCountDto::getUsageCount).max().orElse(1);

        for (TagCountDto tag : tags) {
            int fontSize = calculateFontSize(
                tag.getUsageCount(), minCount, maxCount);
            tag.setFontSize(fontSize);
        }

        // シャッフルしてランダム配置にする
        Collections.shuffle(tags);
        return tags;
    }

    /**
     * 線形補間でフォントサイズを算出
     */
    private int calculateFontSize(long count, long min, long max) {
        if (max == min) return (MIN_FONT_SIZE + MAX_FONT_SIZE) / 2;

        double ratio = (double)(count - min) / (max - min);
        return (int)(MIN_FONT_SIZE + ratio * (MAX_FONT_SIZE - MIN_FONT_SIZE));
    }
}
lightbulb線形補間の計算式

fontSize = minSize + (count - min) / (max - min) * (maxSize - minSize)で、使用頻度を0.0〜1.0の比率に変換し、フォントサイズの範囲にマッピングします。

タグフィルタ（Controller + Mapper）
// --- TodoController.java に追加 ---
@GetMapping("/todo")
public String list(@RequestParam(required = false) String tag,
                   Model model) {
    if (tag != null && !tag.isEmpty()) {
        model.addAttribute("todos", todoMapper.findByTag(tag));
        model.addAttribute("activeTag", tag);
    } else {
        model.addAttribute("todos", todoMapper.findAll());
    }
    return "todo/list";
}

// --- TodoMapper.xml ---
<select id="findByTag" resultType="Todo">
    SELECT t.* FROM todo t
    JOIN todo_tag tt ON t.id = tt.todo_id
    JOIN tag tg ON tt.tag_id = tg.id
    WHERE tg.name = #{tagName}
    ORDER BY t.created_at DESC
</select>

edit_note 基本プロンプト
Spring Boot 3.xのToDoアプリにタグクラウド表示機能を追加してください。

【要件】
- タグの使用頻度をGROUP BYで集計するSQL
- 使用頻度に応じてフォントサイズを線形補間で算出（12px〜48px）
- Flexboxで自然に流れるクラウドレイアウト
- タグクリックでそのタグのToDoだけにフィルタリング
- 選択中タグの表示とフィルタ解除ボタン

【技術スタック】
- Spring Boot 3.x + MyBatis
- 多対多テーブル（todo_tag中間テーブル）
- Thymeleaf + CSS Flexbox
- th:styleで動的フォントサイズ
gps_fixed ステップ別プロンプト
# Step 1: タグ使用頻度集計
todo_tagテーブルをJOINしてタグ名ごとの使用回数をカウントするSQLとDTOを作成してください。

# Step 2: フォントサイズ算出
最小/最大の使用回数から線形補間でフォントサイズ（12px〜48px）を算出するサービスを実装してください。シャッフルでランダム配置にしてください。

# Step 3: タグクラウドHTML/CSS
Flexboxで折り返すクラウドレイアウトのThymeleafテンプレートを作成してください。th:styleで動的にfont-sizeを設定し、ホバーでスケールアップするCSSを追加してください。

# Step 4: タグクリック絞り込み
タグをクリックすると?tag=タグ名でToDoを絞り込む機能を実装してください。選択中タグのチップ表示と解除ボタンも追加してください。
lightbulbプロンプトのコツ

「フォントサイズの最小・最大を設定可能にしてください」のように、カスタマイズのポイントを具体的に指定すると再利用しやすいコードが得られます。
