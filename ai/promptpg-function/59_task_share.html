Section 59-1

共有トークン生成

タスクをログインなしで閲覧できる共有リンクを実現するために、ランダムなトークンを生成してDBに保存する仕組みを構築します。UUIDを使って推測困難なトークンを生成します。

推測不可能なランダムトークンをUUID.randomUUID()で生成

トークン、タスクID、有効期限を管理するテーブル

link

schema.sql（共有テーブル） 
CREATE TABLE task_shares (
    id INT AUTO_INCREMENT PRIMARY KEY,
    todo_id INT NOT NULL,
    token VARCHAR(36) NOT NULL UNIQUE,
    created_by INT NOT NULL,
    expires_at TIMESTAMP NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (todo_id) REFERENCES todo(id) ON DELETE CASCADE,
    FOREIGN KEY (created_by) REFERENCES users(id)
);
ShareService.java（トークン生成） 
import org.springframework.stereotype.Service;
import java.time.LocalDateTime;
import java.util.UUID;

@Service
public class ShareService {

    private final ShareMapper shareMapper;

    public ShareService(ShareMapper shareMapper) {
        this.shareMapper = shareMapper;
    }

    /**
     * 共有リンクを生成する
     * @param todoId  共有するタスクID
     * @param userId  共有を作成したユーザーID
     * @param hours   有効期限（時間）
     * @return 生成されたトークン
     */
    public String createShareLink(Integer todoId,
                                   Integer userId,
                                   int hours) {
        // 既存の共有を確認
        TaskShare existing = shareMapper.findByTodoId(todoId);
        if (existing != null) {
            // 有効期限を更新して既存トークンを返す
            shareMapper.updateExpiry(
                existing.getId(),
                LocalDateTime.now().plusHours(hours));
            return existing.getToken();
        }

        // 新規トークン生成
        TaskShare share = new TaskShare();
        share.setTodoId(todoId);
        share.setToken(UUID.randomUUID().toString());
        share.setCreatedBy(userId);
        share.setExpiresAt(
            LocalDateTime.now().plusHours(hours));

        shareMapper.insert(share);
        return share.getToken();
    }

    /**
     * 共有リンクを無効化する
     */
    public void revokeShare(Integer todoId) {
        shareMapper.deleteByTodoId(todoId);
    }
}
lightbulbトークンのセキュリティ

UUID.randomUUID()は暗号論的に安全な乱数を使用するため、総当たり攻撃に対して十分な強度があります。
さらにセキュリティが必要な場合はSecureRandomで64バイトのトークンを生成しBase64エンコードする方法もあります。

Section 59-2

公開ビューコントローラー

共有トークンを使って認証なしでタスクの詳細を表示するコントローラーを実装します。Spring Securityの設定でこのパスを認証対象外にする必要があります。

public

公開エンドポイント

/share/{token}で認証なしアクセス可能なビュー

verified

トークン検証

トークンの存在と有効期限を検証

visibility

読み取り専用

共有ビューは閲覧のみ、編集機能なし

ShareController.java（公開ビュー） 
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.*;
import java.time.LocalDateTime;

@Controller
@RequestMapping("/share")
public class ShareController {

    private final ShareMapper shareMapper;
    private final TodoMapper todoMapper;

    public ShareController(ShareMapper shareMapper,
                           TodoMapper todoMapper) {
        this.shareMapper = shareMapper;
        this.todoMapper = todoMapper;
    }

    /**
     * 共有リンクでタスクを表示（認証不要）
     */
    @GetMapping("/{token}")
    public String viewShared(@PathVariable String token,
                             Model model) {
        // トークンを検索
        TaskShare share = shareMapper.findByToken(token);

        if (share == null) {
            model.addAttribute("error", "共有リンクが見つかりません");
            return "share/error";
        }

        // 有効期限チェック
        if (share.getExpiresAt().isBefore(LocalDateTime.now())) {
            model.addAttribute("error",
                "この共有リンクは有効期限が切れています");
            return "share/error";
        }

        // タスク情報を取得
        Todo todo = todoMapper.findById(share.getTodoId());
        if (todo == null) {
            model.addAttribute("error", "タスクが見つかりません");
            return "share/error";
        }

        model.addAttribute("todo", todo);
        model.addAttribute("expiresAt", share.getExpiresAt());
        return "share/view";
    }
}

lightbulb共有ビューのセキュリティ

共有ビューには編集機能を含めないことが重要です。読み取り専用のテンプレートを用意し、フォームやボタンは表示しないようにします。
また、共有URLにはタスクIDではなくトークンを使い、URLの推測を不可能にしています。

Section 59-3

共有リンクの有効期限

共有リンクに有効期限を設定し、期限切れのリンクを自動的に無効化する機能を実装します。スケジューラで期限切れリンクを定期的にクリーンアップします。

schedule

有効期限設定

1時間、24時間、7日間などの期限オプション

auto_delete

自動クリーンアップ

@Scheduledで期限切れトークンを定期削除

block

手動無効化

ユーザーが任意のタイミングで共有を停止

ShareCleanupScheduler.java（期限切れ削除） 
import org.springframework.scheduling.annotation.Scheduled;
import org.springframework.stereotype.Component;

@Component
public class ShareCleanupScheduler {

    private final ShareMapper shareMapper;

    public ShareCleanupScheduler(ShareMapper shareMapper) {
        this.shareMapper = shareMapper;
    }

    /**
     * 毎日午前3時に期限切れの共有リンクを削除
     */
    @Scheduled(cron = "0 0 3 * * *")
    public void cleanupExpiredShares() {
        int deleted = shareMapper.deleteExpired();
        if (deleted > 0) {
            System.out.println(
                "期限切れ共有リンクを" + deleted + "件削除しました");
        }
    }
}
ShareMapper.xml（SQL） 
<mapper namespace="com.example.todo.mapper.ShareMapper">

    <insert id="insert" useGeneratedKeys="true" keyProperty="id">
        INSERT INTO task_shares (todo_id, token, created_by, expires_at)
        VALUES (#{todoId}, #{token}, #{createdBy}, #{expiresAt})
    </insert>

    <select id="findByToken" resultType="TaskShare">
        SELECT * FROM task_shares WHERE token = #{token}
    </select>

    <select id="findByTodoId" resultType="TaskShare">
        SELECT * FROM task_shares
        WHERE todo_id = #{todoId}
          AND expires_at > NOW()
        LIMIT 1
    </select>

    <delete id="deleteExpired">
        DELETE FROM task_shares WHERE expires_at < NOW()
    </delete>

    <delete id="deleteByTodoId">
        DELETE FROM task_shares WHERE todo_id = #{todoId}
    </delete>

</mapper>
lightbulb@EnableSchedulingの設定

@Scheduledを使うには、メインのアプリケーションクラスに@EnableSchedulingアノテーションを付ける必要があります。
これにより、Spring Bootのタスクスケジューラが有効になります。

share.js（共有リンク操作） 
const todoId = /*[[${todo.id}]]*/ 0;

async function generateShareLink() {
    const hours = document.getElementById('share-expiry').value;

    const response = await fetch(`/api/share/${todoId}`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ hours: parseInt(hours) })
    });

    const data = await response.json();
    const shareUrl = window.location.origin
        + '/share/' + data.token;

    document.getElementById('share-url').value = shareUrl;
    document.getElementById('share-expiry-info').textContent =
        `有効期限: ${data.expiresAt}`;
    document.getElementById('share-link-area')
        .style.display = 'block';
}

function copyShareUrl() {
    const urlInput = document.getElementById('share-url');
    navigator.clipboard.writeText(urlInput.value)
        .then(() => alert('URLをコピーしました'));
}

async function revokeShare() {
    if (!confirm('共有を停止しますか？')) return;

    await fetch(`/api/share/${todoId}`, { method: 'DELETE' });
    document.getElementById('share-link-area')
        .style.display = 'none';
    alert('共有を停止しました');
}

function openShareDialog() {
    document.getElementById('share-dialog')
        .style.display = 'flex';
}

function closeShareDialog() {
    document.getElementById('share-dialog')
        .style.display = 'none';
}

edit_note 基本プロンプト
Spring Boot 3.xのToDoアプリにタスク共有リンク機能を追加してください。

【要件】
- UUIDベースの共有トークンを生成してDBに保存
- /share/{token}で認証なしでタスクを閲覧可能（読み取り専用）
- 有効期限（1h/24h/7日）を設定可能
- @Scheduledで期限切れトークンの自動クリーンアップ
- 共有リンクの生成・コピー・無効化UI

【技術スタック】
- Spring Boot 3.x + MyBatis
- Spring Security（/share/**をpermitAll）
- UUID.randomUUID()
- JavaScript Clipboard API
gps_fixed ステップ別プロンプト
# Step 1: 共有テーブルとトークン生成
task_sharesテーブルを作成し、UUIDトークンを生成・保存するサービスを実装してください。

# Step 2: 公開ビュー
/share/{token}で認証なしでタスクを閲覧できるコントローラーとテンプレートを作成してください。SecurityConfigで/share/**をpermitAllにしてください。

# Step 3: 有効期限管理
共有リンクに有効期限を設定し、@Scheduledで期限切れトークンを毎日削除するスケジューラを作成してください。

# Step 4: 共有UI
タスク詳細画面に共有ダイアログを追加し、リンク生成・URLコピー・共有停止の操作をできるようにしてください。
