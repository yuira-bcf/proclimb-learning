Section 33-1

タグ検索REST API

入力文字列に部分一致するタグの候補をJSONで返すREST APIを実装します。LIKE検索で高速にサジェストを提供します。

search

部分一致検索

LIKE '%keyword%'でタグ名を検索

data_object

JSONレスポンス

タグID・名前のリストをJSON配列で返却

timer

結果制限

LIMIT 10で候補数を制限してパフォーマンス確保

TagApiController.java（タグ検索API） 
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;
import java.util.List;

@RestController
@RequestMapping("/api/tags")
public class TagApiController {

    private final TagMapper tagMapper;

    public TagApiController(TagMapper tagMapper) {
        this.tagMapper = tagMapper;
    }

    /**
     * タグ名で部分一致検索（サジェスト用）
     */
    @GetMapping("/search")
    public ResponseEntity<List<Tag>> search(
            @RequestParam String q) {
        if (q == null || q.trim().isEmpty()) {
            return ResponseEntity.ok(List.of());
        }
        List<Tag> tags = tagMapper.searchByName(q.trim());
        return ResponseEntity.ok(tags);
    }
}
TagMapper.xml（タグ検索SQL） 
<select id="searchByName" resultType="Tag">
    SELECT id, name
    FROM tags
    WHERE name LIKE CONCAT('%', #{keyword}, '%')
    ORDER BY name ASC
    LIMIT 10
</select>

lightbulbSQLインジェクション対策

MyBatisの#{keyword}はプリペアドステートメントとしてバインドされるため、SQLインジェクションは発生しません。
${keyword}（ドル記号）を使うと文字列がそのまま埋め込まれて危険なので、LIKE検索でも必ず#{}を使いましょう。

fetch API

tag-autocomplete.js 
document.addEventListener('DOMContentLoaded', () => {
    const input = document.getElementById('tagInput');
    const dropdown = document.getElementById('tagDropdown');
    let debounceTimer = null;

    // --- キー入力でサジェスト取得 ---
    input.addEventListener('input', () => {
        clearTimeout(debounceTimer);

        const query = input.value.trim();
        if (query.length < 1) {
            dropdown.style.display = 'none';
            return;
        }

        // 300msデバウンス
        debounceTimer = setTimeout(() => {
            fetchSuggestions(query);
        }, 300);
    });

    /** REST APIからタグ候補を取得 */
    async function fetchSuggestions(query) {
        try {
            const res = await fetch(
                `/api/tags/search?q=${encodeURIComponent(query)}`);
            const tags = await res.json();
            renderDropdown(tags);
        } catch (err) {
            console.error('タグ取得エラー:', err);
        }
    }

    /** サジェストドロップダウンを描画 */
    function renderDropdown(tags) {
        if (tags.length === 0) {
            dropdown.style.display = 'none';
            return;
        }
        dropdown.innerHTML = tags.map(tag =>
            `<div class="suggestion-item" data-id="${tag.id}"
                  data-name="${tag.name}">${tag.name}</div>`
        ).join('');
        dropdown.style.display = 'block';
    }

    // --- ドロップダウン外クリックで閉じる ---
    document.addEventListener('click', (e) => {
        if (!e.target.closest('.tag-input-wrapper')) {
            dropdown.style.display = 'none';
        }
    });
});
warningデバウンスの重要性

デバウンスなしだと「Spring」と入力する間に「S」「Sp」「Spr」...と6回のAPIリクエストが発生します。
300ms程度のデバウンスを入れることで、ユーザーが入力を止めてからリクエストを送るようにしましょう。

Section 33-4

選択タグのチップ表示

選択したタグをチップ（バッジ）として表示し、削除アイコンで個別に除去できるUIを実装します。hidden inputでフォーム送信に含めます。

label

チップ表示

選択済みタグをバッジ形式で並べて表示

cancel

削除機能

チップの×ボタンでタグを除去

input

hidden input連動

選択タグIDをhidden inputでフォーム送信

tag-autocomplete.js（チップ追加・削除） 
const selectedTagIds = new Set();

/** サジェスト候補をクリックしてタグ選択 */
dropdown.addEventListener('click', (e) => {
    const item = e.target.closest('.suggestion-item');
    if (!item) return;

    const tagId = item.dataset.id;
    const tagName = item.dataset.name;

    // 重複チェック
    if (selectedTagIds.has(tagId)) return;

    addTagChip(tagId, tagName);
    input.value = '';
    dropdown.style.display = 'none';
});

/** タグチップを追加 */
function addTagChip(tagId, tagName) {
    selectedTagIds.add(tagId);

    // チップ要素
    const chip = document.createElement('span');
    chip.className = 'tag-chip';
    chip.dataset.id = tagId;
    chip.innerHTML = `${tagName}
        <span class="material-icons remove-tag"
              onclick="removeTag('${tagId}')">close</span>`;

    document.getElementById('selectedTags').appendChild(chip);

    // hidden input（フォーム送信用）
    const hidden = document.createElement('input');
    hidden.type = 'hidden';
    hidden.name = 'tagIds';
    hidden.value = tagId;
    hidden.id = 'tagHidden_' + tagId;
    document.getElementById('tagHiddenInputs').appendChild(hidden);
}

/** タグを除去 */
function removeTag(tagId) {
    selectedTagIds.delete(tagId);

    // チップを削除
    const chip = document.querySelector(
        `.tag-chip[data-id="${tagId}"]`);
    if (chip) chip.remove();

    // hidden inputを削除
    const hidden = document.getElementById('tagHidden_' + tagId);
    if (hidden) hidden.remove();
}

AIプログラミング

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、この機能を実装できます。

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリにタグのオートコンプリート機能を追加してください。

【要件】
- tagsテーブルとtodo_tags中間テーブルを作成
- GET /api/tags/search?q=キーワード でタグ候補をJSON返却
- テキスト入力時にfetch APIでリアルタイムサジェスト（300msデバウンス）
- サジェストドロップダウンから選択するとチップ表示
- 選択タグはhidden inputでフォーム送信に含める

【技術スタック】
- Spring Boot 3.x + MyBatis
- @RestController + Thymeleaf
- Vanilla JavaScript（fetch API）
gps_fixed ステップ別プロンプト 
# Step 1: テーブル設計
tagsテーブル（id, name UNIQUE）とtodo_tags中間テーブル（todo_id, tag_id複合主キー）をCASCADE付きで作成してください。

# Step 2: タグ検索API
GET /api/tags/search?q=キーワードで部分一致検索し、最大10件のタグをJSON返却する@RestControllerを作成してください。

# Step 3: サジェストJS
テキスト入力時に300msデバウンスでfetch APIを呼び出し、結果をドロップダウンに表示するJavaScriptを作成してください。

# Step 4: チップUI
サジェストから選択したタグをチップ（バッジ）表示し、×ボタンで削除可能に。hidden inputでタグIDをフォーム送信できるようにしてください。
lightbulb プロンプトのコツ

「外部ライブラリ（Select2など）を使わずVanilla JavaScriptで実装してください」と指定すると、
依存関係なしのシンプルな実装が得られます。学習目的には自作がおすすめです。
