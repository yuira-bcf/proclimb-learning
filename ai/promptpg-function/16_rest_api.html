リクエストボディをJavaオブジェクトに変換

TodoApiController.java 
import org.springframework.web.bind.annotation.*;
import org.springframework.http.ResponseEntity;
import org.springframework.http.HttpStatus;

@RestController
@RequestMapping("/api/todo")
public class TodoApiController {

    private final TodoService todoService;

    public TodoApiController(TodoService todoService) {
        this.todoService = todoService;
    }

    /**
     * 一覧取得
     * GET /api/todo
     */
    @GetMapping
    public List<Todo> getAll() {
        return todoService.findAll();
    }

    /**
     * 1件取得
     * GET /api/todo/{id}
     */
    @GetMapping("/{id}")
    public ResponseEntity<Todo> getById(@PathVariable Integer id) {
        Todo todo = todoService.findById(id);
        if (todo == null) {
            return ResponseEntity.notFound().build();
        }
        return ResponseEntity.ok(todo);
    }

    /**
     * 新規作成
     * POST /api/todo
     */
    @PostMapping
    public ResponseEntity<Todo> create(@RequestBody Todo todo) {
        Todo created = todoService.save(todo);
        return ResponseEntity.status(HttpStatus.CREATED).body(created);
    }

    /**
     * 更新
     * PUT /api/todo/{id}
     */
    @PutMapping("/{id}")
    public ResponseEntity<Todo> update(@PathVariable Integer id,
                                         @RequestBody Todo todo) {
        todo.setId(id);
        Todo updated = todoService.update(todo);
        return ResponseEntity.ok(updated);
    }

    /**
     * 削除
     * DELETE /api/todo/{id}
     */
    @DeleteMapping("/{id}")
    public ResponseEntity<Void> delete(@PathVariable Integer id) {
        todoService.delete(id);
        return ResponseEntity.noContent().build();
    }
}

Section 15-3

レスポンスのカスタマイズ

ApiResponse.java（レスポンスラッパー） 
public class ApiResponse<T> {
    private boolean success;
    private String message;
    private T data;

    public static <T> ApiResponse<T> success(T data) {
        ApiResponse<T> response = new ApiResponse<>();
        response.success = true;
        response.data = data;
        return response;
    }

    public static <T> ApiResponse<T> error(String message) {
        ApiResponse<T> response = new ApiResponse<>();
        response.success = false;
        response.message = message;
        return response;
    }

    // getter/setter
}

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリをREST APIとして実装してください。

【要件】
- RESTful設計：GET/POST/PUT/DELETE
- エンドポイント：/api/todo
- JSONでリクエスト/レスポンス
- 適切なHTTPステータスコード（200, 201, 204, 400, 404）
- レスポンスは統一フォーマット（success, message, data）
- CORS設定（フロントエンドからのアクセス許可）

【技術スタック】
- Spring Boot 3.x (@RestController)
- Jackson (JSON変換)
- ResponseEntity
gps_fixed ステップ別プロンプト 
# Step 1: RestControllerの作成
TodoApiControllerを@RestControllerで作成し、/api/todoにマッピングしてください。

# Step 2: CRUD操作の実装
GET（一覧・1件）、POST（作成）、PUT（更新）、DELETE（削除）を実装し、適切なHTTPステータスを返してください。

# Step 3: レスポンスラッパーの作成
ApiResponse<T>クラスを作成し、success, message, dataフィールドを持つ統一レスポンス形式を実装してください。

# Step 4: CORS設定
@CrossOriginまたはWebMvcConfigurerでCORS設定を行い、フロントエンドからのアクセスを許可してください。
