Section 7-1

Enum型で優先度を定義

ToDoに優先度（高/中/低）を設定できる機能を追加します。

完成イメージ

高 中 低
Priority.java（Enum） 
package com.example.todo.enums;

public enum Priority {
    HIGH("高", "priority-high"),
    MEDIUM("中", "priority-medium"),
    LOW("低", "priority-low");

    private final String displayName;
    private final String cssClass;

    Priority(String displayName, String cssClass) {
        this.displayName = displayName;
        this.cssClass = cssClass;
    }

    public String getDisplayName() {
        return displayName;
    }

    public String getCssClass() {
        return cssClass;
    }
}
Todo.java（Entityに追加） 
@Enumerated(EnumType.STRING)  // JPA用
private Priority priority = Priority.MEDIUM;  // デフォルト値

// getter / setter
public Priority getPriority() {
    return priority;
}

public void setPriority(Priority priority) {
    this.priority = priority;
}
Section 7-2

selectタグでの選択

edit.html（selectタグ） 
<tr>
    <th><label for="priority">優先度</label></th>
    <td>
        <select id="priority" th:field="*{priority}">
            <option th:each="p : ${T(com.example.todo.enums.Priority).values()}"
                    th:value="${p}"
                    th:text="${p.displayName}"></option>
        </select>
    </td>
</tr>
lightbulbT()演算子

ThymeleafのT()はJavaクラスを参照する演算子です。
Enumの全値を.values()で取得できます。

smart_toy AI

AIプログラミング

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、この機能を実装できます。

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリに優先度設定機能を追加してください。

【要件】
- 優先度は「高」「中」「低」の3段階
- Enum型で優先度を定義
- 入力フォームでselectタグを使用して選択
- 一覧画面で優先度に応じた色分けバッジを表示
  - 高：赤系、中：黄系、低：緑系
- 優先度でのソートも可能

【技術スタック】
- Java Enum
- Spring Boot 3.x
- MyBatis
- Thymeleaf (th:each, th:selected)
gps_fixed ステップ別プロンプト 
# Step 1: Enumの作成
Priority.javaを作成し、HIGH/MEDIUM/LOWの3つの値を定義してください。表示名とCSSクラス名をプロパティとして持たせてください。

# Step 2: Entityとテーブルの修正
TodoエンティティにpriorityフィールドをPriority型で追加し、データベースにもカラムを追加してください。

# Step 3: フォームの修正
入力フォームにselectタグを追加し、th:eachでPriority.values()をループしてオプションを生成してください。

# Step 4: 一覧画面の修正
優先度に応じたCSSクラスを適用し、色分けバッジで表示してください。
