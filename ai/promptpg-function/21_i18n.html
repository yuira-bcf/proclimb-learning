Section 20-1

MessageSourceの設定

MessageSourceを使って、アプリケーションを多言語対応させます。

language

MessageSource

メッセージをロケールに応じて取得するインターフェース

sync

LocaleResolver

現在のロケールを決定・変更するインターフェース

WebConfig.java（国際化設定） 
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.servlet.LocaleResolver;
import org.springframework.web.servlet.config.annotation.InterceptorRegistry;
import org.springframework.web.servlet.config.annotation.WebMvcConfigurer;
import org.springframework.web.servlet.i18n.LocaleChangeInterceptor;
import org.springframework.web.servlet.i18n.SessionLocaleResolver;
import java.util.Locale;

@Configuration
public class WebConfig implements WebMvcConfigurer {

    /**
     * ロケールリゾルバー: セッションベースでロケールを保持
     */
    @Bean
    public LocaleResolver localeResolver() {
        SessionLocaleResolver resolver = new SessionLocaleResolver();
        resolver.setDefaultLocale(Locale.JAPANESE);  // デフォルトは日本語
        return resolver;
    }

    /**
     * ロケール変更インターセプター: URLパラメータで言語を切り替え
     * 例: ?lang=en で英語に切り替え
     */
    @Bean
    public LocaleChangeInterceptor localeChangeInterceptor() {
        LocaleChangeInterceptor interceptor = new LocaleChangeInterceptor();
        interceptor.setParamName("lang");
        return interceptor;
    }

    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        registry.addInterceptor(localeChangeInterceptor());
    }
}

<body>
    <h1 th:text="#{todo.list.title}">ToDo一覧</h1>

    <!-- 言語切り替えリンク -->
    <div class="language-switcher">
        <a th:href="@{''(lang='ja')}">日本語</a> |
        <a th:href="@{''(lang='en')}">English</a>
    </div>

    <table>
        <thead>
            <tr>
                <th th:text="#{todo.title}">タイトル</th>
                <th th:text="#{todo.status}">ステータス</th>
                <th th:text="#{todo.priority}">優先度</th>
            </tr>
        </thead>
        <tbody>
            <tr th:each="todo : ${todoList}">
                <td th:text="${todo.title}"></td>
                <td>
                    <!-- 条件によるメッセージ切り替え -->
                    <span th:if="${todo.completed}" th:text="#{status.completed}">完了</span>
                    <span th:unless="${todo.completed}" th:text="#{status.pending}">未完了</span>
                </td>
                <td th:text="#{${'priority.' + todo.priority.name().toLowerCase()}}"></td>
            </tr>
        </tbody>
    </table>

    <a th:href="@{/todo/new}" th:text="#{button.create}">新規作成</a>
</body>
</html>

Section 20-3

言語切り替え機能

コントローラーでMessageSourceを使用 
import org.springframework.context.MessageSource;
import org.springframework.context.i18n.LocaleContextHolder;

@Controller
@RequestMapping("/todo")
public class TodoController {

    private final TodoService todoService;
    private final MessageSource messageSource;

    public TodoController(TodoService todoService, MessageSource messageSource) {
        this.todoService = todoService;
        this.messageSource = messageSource;
    }

    @PostMapping
    public String create(@Valid TodoForm form, BindingResult result,
                         RedirectAttributes redirectAttributes) {
        if (result.hasErrors()) {
            return "todo/form";
        }

        todoService.create(form);

        // 現在のロケールに応じたメッセージを取得
        Locale locale = LocaleContextHolder.getLocale();
        String message = messageSource.getMessage(
            "message.create.success", null, locale);

        redirectAttributes.addFlashAttribute("message", message);
        return "redirect:/todo";
    }
}

LocaleResolverの種類

種類説明用途
SessionLocaleResolverセッションにロケールを保存ログイン後の言語設定
CookieLocaleResolverCookieにロケールを保存永続的な言語設定
AcceptHeaderLocaleResolverブラウザの設定を使用自動言語検出
FixedLocaleResolver固定ロケール単一言語アプリ
warning注意点
  • メッセージファイルはsrc/main/resources直下に配置
  • 日本語はUTF-8で保存（native2asciiは不要）
  • キーが見つからない場合はデフォルトファイル（messages.properties）を参照

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリに国際化（多言語対応）機能を追加してください。

【要件】
- MessageSourceを使用して日本語/英語の切り替えに対応
- SessionLocaleResolverでセッションベースのロケール管理
- LocaleChangeInterceptorでURLパラメータ(?lang=ja/?lang=en)による言語切り替え
- messages_ja.properties、messages_en.propertiesにメッセージを定義
- Thymeleafで#{key}形式でメッセージを参照
- バリデーションメッセージも多言語対応

【技術スタック】
- Spring Boot 3.x
- MessageSource
- Thymeleaf
- Jakarta Validation
