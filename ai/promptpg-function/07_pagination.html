Section 6-2

Pageableの使い方

TodoController.java 
import org.springframework.data.domain.Page;
import org.springframework.data.domain.Pageable;
import org.springframework.data.web.PageableDefault;

@GetMapping
public String index(@PageableDefault(size = 10, sort = "createdAt",
                                     direction = Sort.Direction.DESC) Pageable pageable,
                    Model model) {

    Page<Todo> todoPage = todoService.findAll(pageable);

    model.addAttribute("todoPage", todoPage);
    model.addAttribute("todoList", todoPage.getContent());

    return "todo/index";
}
TodoRepository.java（Spring Data JPA） 
import org.springframework.data.domain.Page;
import org.springframework.data.domain.Pageable;
import org.springframework.data.jpa.repository.JpaRepository;

public interface TodoRepository extends JpaRepository<Todo, Integer> {

    // Spring Data JPAが自動でページング対応
    Page<Todo> findAll(Pageable pageable);

    // 検索 + ページング
    Page<Todo> findByTitleContaining(String keyword, Pageable pageable);
}

Pageオブジェクトの主なメソッド

メソッド説明
getContent()現在ページのデータリスト
getTotalPages()総ページ数
getTotalElements()総データ件数
getNumber()現在のページ番号（0始まり）
hasNext()次のページがあるか
hasPrevious()前のページがあるか
Section 6-3

ページャーUIの実装

index.html（ページャー） 
<!-- ページャー -->
<nav class="pagination" th:if="${todoPage.totalPages > 1}">
    <!-- 前へ -->
    <a th:if="${todoPage.hasPrevious()}"
       th:href="@{/todo(page=${todoPage.number - 1})}">« 前へ</a>
    <span th:unless="${todoPage.hasPrevious()}" class="disabled">« 前へ</span>

    <!-- ページ番号 -->
    <th:block th:each="i : ${#numbers.sequence(0, todoPage.totalPages - 1)}">
        <span th:if="${i == todoPage.number}" class="active" th:text="${i + 1}"></span>
        <a th:unless="${i == todoPage.number}"
           th:href="@{/todo(page=${i})}" th:text="${i + 1}"></a>
    </th:block>

    <!-- 次へ -->
    <a th:if="${todoPage.hasNext()}"
       th:href="@{/todo(page=${todoPage.number + 1})}">次へ »</a>
    <span th:unless="${todoPage.hasNext()}" class="disabled">次へ »</span>
</nav>

<!-- 件数表示 -->
<p class="page-info">
    全 <span th:text="${todoPage.totalElements}"></span> 件中
    <span th:text="${todoPage.number * todoPage.size + 1}"></span> -
    <span th:text="${todoPage.number * todoPage.size + todoPage.numberOfElements}"></span> 件を表示
</p>
lightbulbページ番号の注意

Spring Dataのページ番号は0始まりです。
表示時は+1して1始まりにしましょう。

smart_toy AI

AIプログラミング

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、この機能を実装できます。

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリにページネーション機能を追加してください。

【要件】
- 1ページあたり10件表示
- 「前へ」「次へ」ボタンを配置
- ページ番号リンクを表示（1, 2, 3...）
- 現在のページをハイライト表示
- 総件数と表示範囲を表示（例：全50件中 1-10件）

【技術スタック】
- Spring Boot 3.x (Pageable, Page)
- MyBatis (LIMIT, OFFSET)
- Thymeleaf
gps_fixed ステップ別プロンプト 
# Step 1: Controllerの修正
TodoControllerに@PageableDefault(size = 10)を追加し、Page<Todo>を返すようにしてください。

# Step 2: Mapperの修正
TodoMapper.xmlにLIMITとOFFSETを使用したページング用SQLを追加してください。総件数取得用のCOUNTクエリも追加してください。

# Step 3: ページャーUIの実装
Thymeleafでページ番号リンクを生成し、現在のページにactiveクラスを付与してください。

# Step 4: 表示範囲の表示
「全{totalElements}件中 {start}-{end}件を表示」のような情報を追加してください。
lightbulb プロンプトのコツ

「検索条件とページネーションを組み合わせて、検索結果もページングできるようにしてください」のように、
既存機能との連携を指定すると、統合された実装が得られます。
