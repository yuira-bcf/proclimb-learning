Section 10-1

チェックボックスの実装

複数のToDoを選択して一括削除できる機能を追加します。

index.html（チェックボックス） 
<form th:action="@{/todo/bulk-delete}" method="post" id="bulkDeleteForm">
    <div class="action-bar">
        <button type="submit" class="btn btn-danger"
                onclick="return confirm('選択した項目を削除しますか？');">
            選択した項目を削除
        </button>
    </div>

    <table class="data-table">
        <thead>
            <tr>
                <th><input type="checkbox" id="selectAll" onclick="toggleAll(this)"></th>
                <th>No</th>
                <th>タイトル</th>
                <th>登録者</th>
            </tr>
        </thead>
        <tbody>
            <tr th:each="todo : ${todoList}">
                <td>
                    <input type="checkbox" name="ids" th:value="${todo.id}" class="item-checkbox">
                </td>
                <td th:text="${todo.id}"></td>
                <td th:text="${todo.title}"></td>
                <td th:text="${todo.author}"></td>
            </tr>
        </tbody>
    </table>
</form>
Section 10-2

配列パラメータの受け取り

TodoController.java 
/**
 * 一括削除
 */
@PostMapping("/bulk-delete")
public String bulkDelete(@RequestParam(required = false) List<Integer> ids,
                         RedirectAttributes redirectAttributes) {

    if (ids == null || ids.isEmpty()) {
        redirectAttributes.addFlashAttribute("error", "削除する項目を選択してください");
        return "redirect:/todo";
    }

    int count = todoService.deleteByIds(ids);

    redirectAttributes.addFlashAttribute("message", count + "件を削除しました");
    return "redirect:/todo";
}
TodoMapper.xml 
<delete id="deleteByIds">
    DELETE FROM todo
    WHERE id IN
    <foreach collection="ids" item="id" open="(" separator="," close=")">
        #{id}
    </foreach>
</delete>

Section 10-3

全選択/全解除機能

JavaScript（全選択） 
<script>
function toggleAll(source) {
    const checkboxes = document.querySelectorAll('.item-checkbox');
    checkboxes.forEach(checkbox => {
        checkbox.checked = source.checked;
    });
}

// 個別チェック時にヘッダーのチェック状態を更新
document.addEventListener('DOMContentLoaded', function() {
    const checkboxes = document.querySelectorAll('.item-checkbox');
    const selectAll = document.getElementById('selectAll');

    checkboxes.forEach(checkbox => {
        checkbox.addEventListener('change', function() {
            const allChecked = Array.from(checkboxes).every(cb => cb.checked);
            const someChecked = Array.from(checkboxes).some(cb => cb.checked);
            selectAll.checked = allChecked;
            selectAll.indeterminate = someChecked && !allChecked;
        });
    });
});
</script>
lightbulbindeterminate状態

一部だけ選択された状態をindeterminateプロパティで表現できます。
ブラウザで「−」マークが表示されます。

AIプログラミング

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、この機能を実装できます。

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリに一括削除機能を追加してください。

【要件】
- 各行にチェックボックスを追加
- 「全選択」チェックボックスで全てのチェックをON/OFF
- 「選択した項目を削除」ボタンを配置
- 削除前に確認ダイアログを表示
- 選択されたIDをリストで送信して一括削除

【技術スタック】
- Spring Boot 3.x (@RequestParam List)
- JavaScript (チェックボックス制御)
- MyBatis (IN句で一括削除)
- Thymeleaf
gps_fixed ステップ別プロンプト 
# Step 1: チェックボックスの追加
一覧テーブルの各行にチェックボックス（name="ids" value="${todo.id}"）を追加してください。

# Step 2: 全選択機能の実装
JavaScriptで「全選択」チェックボックスをクリックすると、全行のチェックボックスがON/OFFされる機能を追加してください。

# Step 3: Controllerの実装
POST /todo/bulk-deleteエンドポイントを作成し、@RequestParam List<Integer> idsで選択されたIDを受け取ってください。

# Step 4: Mapperの実装
MyBatisでDELETE文にIN句を使用し、複数IDを一括削除するSQLを作成してください。
