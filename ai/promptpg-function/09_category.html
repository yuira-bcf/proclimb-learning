Section 8-1

テーブル設計（1対多）

ToDoに複数のタグを紐付ける機能を追加します。

完成イメージ

仕事 重要 会議
schema.sql 
-- カテゴリ/タグテーブル
CREATE TABLE category (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) NOT NULL
);

-- ToDo-カテゴリ中間テーブル（多対多の場合）
CREATE TABLE todo_category (
    todo_id INT NOT NULL,
    category_id INT NOT NULL,
    PRIMARY KEY (todo_id, category_id),
    FOREIGN KEY (todo_id) REFERENCES todo(id),
    FOREIGN KEY (category_id) REFERENCES category(id)
);

-- 初期データ
INSERT INTO category (name) VALUES ('仕事'), ('プライベート'), ('重要'), ('会議');
Section 8-2

MyBatisでの結合

TodoMapper.xml（collection使用） 
<resultMap id="todoWithCategories" type="Todo">
    <id property="id" column="id"/>
    <result property="title" column="title"/>
    <result property="author" column="author"/>
    <!-- 1対多のカテゴリリスト -->
    <collection property="categories" ofType="Category">
        <id property="id" column="category_id"/>
        <result property="name" column="category_name"/>
    </collection>
</resultMap>

<select id="findAllWithCategories" resultMap="todoWithCategories">
    SELECT
        t.id, t.title, t.author,
        c.id AS category_id, c.name AS category_name
    FROM todo t
    LEFT JOIN todo_category tc ON t.id = tc.todo_id
    LEFT JOIN category c ON tc.category_id = c.id
    ORDER BY t.id
</select>
Todo.java（カテゴリリスト追加） 
private List<Category> categories = new ArrayList<>();

public List<Category> getCategories() {
    return categories;
}

public void setCategories(List<Category> categories) {
    this.categories = categories;
}

AIプログラミング

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、この機能を実装できます。

edit_note 基本プロンプト 
Spring Boot 3.xのToDoアプリにカテゴリ/タグ機能を追加してください。

【要件】
- カテゴリマスタテーブルを作成（id, name, color）
- ToDoにカテゴリを紐付け（多対1の関係）
- 入力フォームでカテゴリをselectで選択
- 一覧画面でカテゴリ名をバッジ表示（色付き）
- カテゴリでのフィルタリングが可能

【技術スタック】
- Spring Boot 3.x
- MyBatis (JOINクエリ)
- Thymeleaf
gps_fixed ステップ別プロンプト 
# Step 1: カテゴリテーブルの作成
categoryテーブル（id, name, color）を作成し、Categoryエンティティを定義してください。

# Step 2: リレーションの設定
todoテーブルにcategory_idを追加し、TodoエンティティにCategoryを関連付けてください。

# Step 3: カテゴリ選択の実装
フォームにカテゴリ選択用のselectタグを追加し、Controllerでカテゴリ一覧をModelに渡してください。

# Step 4: フィルタリングの実装
一覧画面にカテゴリフィルターを追加し、選択したカテゴリのToDoのみ表示できるようにしてください。
