Section 69-1

パーマリンク生成

タスクごとに一意のURLトークンを発行し、ログインなしでもタスクの内容を閲覧できるパーマリンクを生成します。

link

UUIDベースのトークン

推測困難なランダムURLトークンを生成

storage

share_tokenカラム

todoテーブルに共有トークン列を追加

toggle_on

公開/非公開の切替

ユーザーがタスクごとに共有設定を管理

schema.sql（共有トークン列追加）
ALTER TABLE todo ADD COLUMN share_token VARCHAR(36) UNIQUE;
ALTER TABLE todo ADD COLUMN is_public BOOLEAN DEFAULT FALSE;

-- インデックス追加（トークンで高速検索）
CREATE INDEX idx_todo_share_token ON todo(share_token);
ShareService.java（共有トークン管理）
import java.util.UUID;

@Service
public class ShareService {

    private final TodoMapper todoMapper;

    public ShareService(TodoMapper todoMapper) {
        this.todoMapper = todoMapper;
    }

    /**
     * 共有トークンを生成して公開設定にする
     */
    public String enableSharing(Integer todoId, Integer userId) {
        Todo todo = todoMapper.findById(todoId);

        // 所有者チェック
        if (!todo.getUserId().equals(userId)) {
            throw new IllegalArgumentException("権限がありません");
        }

        // トークンがなければ新規生成
        if (todo.getShareToken() == null) {
            String token = UUID.randomUUID().toString();
            todo.setShareToken(token);
        }
        todo.setPublic(true);
        todoMapper.updateShareSettings(todo);

        return todo.getShareToken();
    }

    /**
     * 共有を無効化
     */
    public void disableSharing(Integer todoId, Integer userId) {
        Todo todo = todoMapper.findById(todoId);
        if (!todo.getUserId().equals(userId)) {
            throw new IllegalArgumentException("権限がありません");
        }
        todo.setPublic(false);
        todoMapper.updateShareSettings(todo);
    }

    /**
     * 共有URLを取得
     */
    public String getShareUrl(String baseUrl, String token) {
        return baseUrl + "/share/" + token;
    }
}
lightbulbUUIDの安全性

UUID.randomUUID()は128ビットのランダム値で、推測はほぼ不可能です。連番IDと異なり、URLを知らない人がアクセスすることを防げます。

Section 69-2

公開タスクビュー

共有URLにアクセスした人がログインなしでタスクの内容を閲覧できる公開ビューを実装します。

public

認証不要のエンドポイント

/share/{token}はSpring Securityで認証をスキップ

visibility

読み取り専用ビュー

タスクの閲覧のみ、編集・削除操作は不可

block

非公開時の404

is_public=falseのタスクは404を返す

ShareController.java
@Controller
public class ShareController {

    private final TodoMapper todoMapper;

    public ShareController(TodoMapper todoMapper) {
        this.todoMapper = todoMapper;
    }

    /**
     * 公開タスクビュー（認証不要）
     */
    @GetMapping("/share/{token}")
    public String showSharedTask(@PathVariable String token, Model model) {
        Todo todo = todoMapper.findByShareToken(token);

        // タスクが存在しない or 非公開の場合は404
        if (todo == null || !todo.isPublic()) {
            throw new ResponseStatusException(
                HttpStatus.NOT_FOUND, "タスクが見つかりません");
        }

        model.addAttribute("todo", todo);
        model.addAttribute("shareUrl",
            "/share/" + todo.getShareToken());
        return "share/task-view";
    }
}

lightbulbSecurityConfigでの除外設定

/share/**パスをSpring SecurityのpermitAll()に追加して、認証なしでアクセスできるようにします。公開ビューでは読み取り専用の情報のみ表示し、編集機能は一切含めません。

ShareController.java（baseUrl設定）
@GetMapping("/share/{token}")
public String showSharedTask(@PathVariable String token,
                             HttpServletRequest request,
                             Model model) {
    Todo todo = todoMapper.findByShareToken(token);
    if (todo == null || !todo.isPublic()) {
        throw new ResponseStatusException(HttpStatus.NOT_FOUND);
    }

    // OGPで使うベースURLを動的に取得
    String baseUrl = request.getScheme() + "://" + request.getServerName();
    int port = request.getServerPort();
    if (port != 80 && port != 443) {
        baseUrl += ":" + port;
    }

    model.addAttribute("todo", todo);
    model.addAttribute("baseUrl", baseUrl);
    return "share/task-view";
}
lightbulbOGP画像の準備

og:imageにはデフォルトのOGP画像（1200x630px推奨）を設定します。画像がないとSNSでプレビューが表示されない場合があります。

クリップボードコピーUI

共有URLをワンクリックでクリップボードにコピーできるUIを実装します。

check_circle

コピー完了フィードバック

ボタンの見た目を一時的に変更して成功を通知

共有ダイアログ

URLコピー、SNS共有ボタンをまとめたダイアログ

JavaScript（クリップボードコピー）
/**
 * 共有URLをクリップボードにコピー
 */
async function copyShareUrl(todoId) {
    const url = window.location.origin + '/share/' + todoId;

    try {
        await navigator.clipboard.writeText(url);
        showCopyFeedback(true);
    } catch (err) {
        // フォールバック（古いブラウザ対応）
        const textArea = document.createElement('textarea');
        textArea.value = url;
        document.body.appendChild(textArea);
        textArea.select();
        document.execCommand('copy');
        document.body.removeChild(textArea);
        showCopyFeedback(true);
    }
}

/**
 * コピー完了のフィードバック表示
 */
function showCopyFeedback(success) {
    const btn = document.getElementById('copy-btn');
    const originalHtml = btn.innerHTML;

    if (success) {
        btn.innerHTML = '<span class="material-icons">check</span> コピー完了';
        btn.style.background = '#22c55e';
    }

    setTimeout(() => {
        btn.innerHTML = originalHtml;
        btn.style.background = '';
    }, 2000);
}

/**
 * 共有ダイアログを表示
 */
function showShareDialog(token) {
    const url = window.location.origin + '/share/' + token;
    document.getElementById('share-url-input').value = url;
    document.getElementById('share-dialog').style.display = 'flex';
}

lightbulbClipboard API の注意点

navigator.clipboard.writeText()はHTTPS環境でのみ動作します。開発環境（localhost）では動作しますが、HTTP本番環境では使えません。フォールバックとしてdocument.execCommand('copy')を用意しておきましょう。

edit_note 基本プロンプト
Spring Boot 3.xのToDoアプリにタスクURL共有機能を追加してください。

【要件】
- UUID形式のパーマリンクを生成（share_token列を追加）
- /share/{token}で認証なしにタスクを閲覧できる公開ビュー
- OGPメタタグでSNS共有時のリッチプレビューを表示
- ワンクリックでURLをクリップボードにコピー

【技術スタック】
- Spring Boot 3.x + MyBatis + Thymeleaf
- UUID（共有トークン）
- Open Graph Protocol（OGPメタタグ）
- JavaScript Clipboard API
gps_fixed ステップ別プロンプト
# Step 1: パーマリンク生成
todoテーブルにshare_token(UUID)列とis_public列を追加し、共有トークンの生成・有効化・無効化サービスを実装してください。

# Step 2: 公開タスクビュー
/share/{token}で認証なしにアクセスできるコントローラーとビューを作成してください。非公開タスクは404を返してください。

# Step 3: OGPメタタグ
公開ビューのheadにog:title, og:description, og:image, twitter:cardのメタタグを動的に設定してください。

# Step 4: クリップボードコピーUI
共有URLをワンクリックでコピーできるボタンとダイアログを実装してください。Clipboard APIとフォールバック（execCommand）を含めてください。

セキュアな共有機能が得られます。
