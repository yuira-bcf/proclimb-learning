メール送信の主要コンポーネント

  • JavaMailSender - Springのメール送信インターフェース
  • MimeMessageHelper - HTMLメール、添付ファイルのヘルパー
  • TemplateEngine - Thymeleafでメール本文を生成
  • @Async - 非同期でメールを送信

サンプルソースコード

application.properties 
# Gmail SMTP設定
spring.mail.host=smtp.gmail.com
spring.mail.port=587
spring.mail.username=${MAIL_USERNAME}
spring.mail.password=${MAIL_PASSWORD}
spring.mail.properties.mail.smtp.auth=true
spring.mail.properties.mail.smtp.starttls.enable=true

# 送信元アドレス
app.mail.from=noreply@example.com
MailService.java 
@Service
public class MailService {

    private final JavaMailSender mailSender;
    private final TemplateEngine templateEngine;

    @Value("${app.mail.from}")
    private String fromAddress;

    // HTMLメール送信（非同期）
    @Async
    public void sendTodoCreatedMail(String to, Todo todo) {
        MimeMessage message = mailSender.createMimeMessage();
        MimeMessageHelper helper = new MimeMessageHelper(message, true, "UTF-8");

        helper.setFrom(fromAddress);
        helper.setTo(to);
        helper.setSubject("【ToDo】新しいタスクが作成されました");

        // Thymeleafでメール本文を生成
        Context context = new Context();
        context.setVariable("todo", todo);
        String htmlContent = templateEngine.process("mail/todo-created", context);

        helper.setText(htmlContent, true);  // true = HTML形式
        mailSender.send(message);
    }

    // 添付ファイル付きメール送信
    public void sendWithAttachment(String to, String subject, File attachment) {
        MimeMessage message = mailSender.createMimeMessage();
        MimeMessageHelper helper = new MimeMessageHelper(message, true);

        helper.setTo(to);
        helper.setSubject(subject);
        helper.setText("添付ファイルをご確認ください");
        helper.addAttachment(attachment.getName(), attachment);

        mailSender.send(message);
    }
}

lightbulb コード解説

  • JavaMailSender - Spring Bootが自動設定するメール送信インターフェース
  • MimeMessageHelper - HTMLメール、添付ファイルを簡単に扱えるヘルパー
  • TemplateEngine - Thymeleafでメール本文を動的に生成
  • setText(content, true) - 第2引数trueでHTML形式を指定
  • @Async - メール送信を非同期化（画面の応答を待たせない）

【実装要件】
1. SMTP設定
   - application.propertiesにSMTPサーバー設定
   - Gmail SMTPを使用する例
   - 認証情報は環境変数から取得

2. メール送信サービス（MailService.java）
   - JavaMailSenderを使用
   - プレーンテキストメール送信
   - HTMLメール送信（Thymeleafテンプレート使用）
   - 添付ファイル付きメール送信
   - @Asyncで非同期送信

3. メールテンプレート
   - todo-created.html（ToDo作成通知）
   - deadline-reminder.html（期限リマインダー）
   - Thymeleafで動的にコンテンツを生成

4. 使用例
   - ToDo作成時に作成通知メールを送信
   - 期限が近いToDoのリマインダーを送信

5. スケジュール処理
   - @Scheduledで毎日期限チェック
   - 期限が1日前のToDoをリストアップしてメール送信
