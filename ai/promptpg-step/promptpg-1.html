Section 1-3

Hello World の表示

最初のControllerを作成し、ブラウザに「Hello! Spring Boot」と表示させます。

smart_toy AIプロンプト

Spring Bootで「Hello! Spring Boot」と表示するシンプルなControllerを作成してください。

要件:
- URLは /hello
- Thymeleafテンプレートを使用
- メッセージは動的にControllerから渡す

Controller、テンプレート、必要な設定をすべて含めてください。

生成される想定コード

HelloController.java 
package com.example.todo.controller;

import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;

@Controller
public class HelloController {

    @GetMapping("/hello")
    public String hello(Model model) {
        model.addAttribute("message", "Hello! Spring Boot");
        return "hello";
    }
}
lightbulb コード解説
  • @Controller - このクラスがMVCのControllerであることを示す
  • @GetMapping("/hello") - GETリクエストで /hello にアクセスした時に実行
  • Model model - ビューにデータを渡すためのオブジェクト
  • model.addAttribute() - テンプレートで使用するデータをセット
  • return "hello" - templates/hello.html を表示
templates/hello.html 
<!DOCTYPE html>
<html xmlns:th="http://www.thymeleaf.org">
<head>
    <meta charset="UTF-8">
    <title>Hello</title>
</head>
<body>
    <h1 th:text="${message}">メッセージ</h1>
</body>
</html>
lightbulb コード解説
  • xmlns:th - Thymeleafの名前空間を宣言
  • th:text="${message}" - Controllerから渡されたmessage変数を表示
  • 「メッセージ」の部分は実行時に動的に置き換わる

処理の流れ:
1. ブラウザが /hello にアクセス
2. @GetMapping("/hello") が付いたメソッドが呼ばれる
3. Model に "message" キーでデータをセット
4. return "hello" で hello.html テンプレートを指定
5. Thymeleaf が ${message} を実際の値に置換してHTMLを生成
check_circle 確認ポイント

アプリケーションを起動し、http://localhost:8080/hello にアクセスして「Hello! Spring Boot」が表示されれば成功です。
