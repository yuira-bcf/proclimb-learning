Section 8-1

左サイドバー：ファイル・ページ・レイヤー

  • メインメニュー
    • ホーム画面に戻ったり、各ツールなどの設定が可能です。
  • ファイル名
    • 現在のデザインファイル名（最初は「無題」）。クリックして名前変更できます。
  • ファイルの種類
    • 「ドラフト」または「チームのプロジェクト」のどちらを表示しているかが表記されます。
  • ファイルタブとアセットタブ
    • ファイルタブ：現在開いているデザインファイルのレイヤーが下に表示されます。
    • アセットタブ：コンポーネントやライブラリ（共通パーツ）を呼び出すときに使用します。
  • ページ
    • ページごとにデザインを分けられます（例：Page1、Page2など）。＋ボタンでページの追加が可能です。
  • レイヤー
    • 現在のページ内の要素（フレーム・図形・テキストなど）の一覧。
    • 階層構造で表示され、順番をドラッグで変更可能。
Section 8-2

中央：キャンバス（作業エリア）

  • 実際にデザインを作成する領域。
  • フレーム（画面サイズ）を作り、図形やテキスト、画像を配置します。
  • かなり広がるので、複数の画面デザインを並べてデザイン可能です。
  • スペースキー＋ドラッグでキャンバスを移動できます。
Section 8-3

右サイドバー：プロパティパネル

  • デザインタブ選択時：キャンバス上の要素の、各詳細設定が行えます。
  • プロトタイプ選択時：インタラクションやスクロール時の挙動の指定をしたりできます。
    • ※ 右パネルの詳細は後述します。
Section 8-4

下部ツールバー：ツールアイコン群

  1. 移動 -- オブジェクトを選択・移動します。
  2. フレーム -- 画面サイズを作成（スマホ・PCなど）。
  3. シェイプ -- 長方形、円、線、矢印などを作成。
  4. ペン -- 自由線やベクター図形を描きます。
  5. テキスト -- テキストを追加します。
  6. コメント -- 画像を挿入します。
  7. アクション -- プラグインなどを検索できます。
  8. モード -- 描画モードやデザインモード、Devモード（有料）を選べます。基本的にデザインモードのみ使用します。
Section 8-5

右上：共有・プレビューエリア

  • ▶ -- プレゼン（プロトタイプ再生）
  • 「共有」 -- 他のメンバーとファイルを共有（閲覧／編集権限の設定可能）
  • アイコン -- 現在ログインしているユーザーのアイコン
  • デザイン -- 色・余白・角丸・影など、オブジェクトの見た目を編集します。
  • プロトタイプ -- 画面同士をリンクして動作（クリック移動など）を設定します。

次から、実際にデザイン作業で多用する、下部ツールの各機能について説明します。

Section 8-6

移動

移動：V

オブジェクトを選択して移動します。デザインの基本操作モード。クリックで図形やテキストを選び、ドラッグで動かします。

手のひらツール：H

キャンバス全体をドラッグで移動（スクロール）します。オブジェクトは動かさず、作業領域を見渡したい時に使います。

拡大縮小：K

対象の要素を選択し、右パネルから大きさを拡大、縮小などの操作ができます。

Section 8-7

フレーム

フレーム：F

役割

Figmaで最も重要な「枠（コンテナ）」です。画面（アートボード）やコンポーネントを作るための基本単位。他の図形・テキスト・画像を中に入れてレイアウトをまとめることができます。

よく使う用途

  • スマホ・PC画面などを作る（例：iPhone 14のフレーム）
  • ボタンやカードなどの部品を作る
  • Auto Layout（自動整列）のベースにする

フレームを選択すると右サイドバーにプリセットが表示されます。ここではiPhone16をクリックします。中央キャンバスにアートボードが表示されます。また左サイドバーのレイヤーも追加されます。

セクション：Shift + S

役割

デザイン全体を整理するための枠です。複数のフレームや要素をまとめてグループ化できます。

よく使う用途

  • ページ単位で「見出し」「フッター」などをまとめる
  • プロジェクト内で画面ごとに整理する（例：「ログイン画面」「マイページ」）
  • コメントを管理・レビューしやすくする

スライス：S

役割

デザインの一部の画像共有、切り出し

良く使う用途

主にUIデザインをエンジニアに渡すときや、素材を切り出したいときに使われます。デザインの中から、必要な部分だけを画像ファイルとして書き出すための"切り取り枠" です。スクリーンショットと思っておいて問題ありません。

基本操作

  1. キャンバス上でドラッグして、書き出したい範囲を囲む
  2. 右側の「エクスポート」パネルに設定が表示される -- PNG / JPG / SVG / PDF などの形式を選択
  3. 出力内容が表示されます
  4. スライスオブジェクトはレイヤーに表示されます
Section 8-8

シェイプ

長方形（Rectangle）：R

基本の箱。ボタン/カード/背景など何でもこれで作成。Shiftで正方形、Alt(Option)で中心からドラッグ。角丸は右パネルのCorner radius、個別コーナーは「...」から。

線（Line）：L

点A→点Bを結ぶ直線。Shiftで 0°/45°/90° にスナップ。太さ/線端/破線は右パネルのStrokeで設定。

矢印（Arrow）：Shift + L

線に矢印の頭/尾が付いたもの。方向やタイプ（▲/▶など）はStroke > Arrowheadsで変更。アノテーションやフロー図に便利。

楕円（Ellipse）：O

円・楕円を作成。Shift + ドラッグで縦横比固定の正円、Alt(Option) + ドラッグで中心を起点とした円の作成ができます。作成した円の右にある〇をドラッグすることで、切り欠きやドーナツ形も作れます。

多角形（Polygon）

三角形〜n角形。作成後に右下の〇を↑/↓すると、辺の数を変更できます。角の丸みは、上の〇か、右サイドパネルで調整できます。

星（Star）

星形の図形。作成後に右の〇を↑/↓で頂点数の数、上の頂点内側の〇で、とがり具合を調整できます。

画像／動画の配置（Place Image/Video）：Shift + Cmd + K（Mac）／ Shift + Ctrl + K（Win）

画像や動画を挿入。置いた後はダブルクリックでクロップ、右パネルのFillで「塗り」としても適用可。

Section 8-9

ペン

ペンツール（Pen）：P

点（アンカーポイント）と線（パス）をつないで、自由なベクター図形を作成します。

使い方

クリックして点を打つと、直線が作成されます。ドラッグしながらクリックすると、曲線（ベジェ曲線）が描けます。最後の点を最初の点に戻すと閉じた図形になります。

編集

  • 作成後はダブルクリックでパス編集モードに入ります。
  • 点を動かす・追加・削除も自由にできます。
  • 右パネルで「塗り（Fill）」や「線（Stroke）」を設定可能。

※ペンツール（特にベジェ曲線）は慣れないと結構難しく、習得に時間を要します。

鉛筆

手描きのように自由に線を描くツールです。

使い方

キャンバス上でドラッグして線を描く。自動で滑らかな曲線（ベクター化）になります。

Section 8-10

テキスト

基本の使い方

  • ツールバーの 「T」 をクリック（またはキーボードで Tキー）。
  • キャンバス上でクリック → 文字を入力。
  • フォント・サイズ・色などを右側パネルで調整できます。

テキストの種類（2つの入力モード）

1. ワンクリック入力（ポイントテキスト）

キャンバスをクリックして入力開始。一行テキストに向く（タイトルやボタンなど）

2. ドラッグ入力（エリアテキスト）

ドラッグして枠を作って入力。段落文や長文に向く（説明文・本文など）

Section 8-11

コメント

デザイン上に コメントや指摘・相談メモを直接書き込む ためのツールです。共同作業時にとても便利で、Figmaの強みのひとつです。

使い方

  1. ツールバーの アイコンをクリック（または Cキー）
  2. キャンバス上でクリック → コメント入力欄が表示
  1. テキストを入力して「送信」
  1. コメントがピン（吹き出しマーク）として表示されます

デザインタブ

位置

  1. 配置 -- 親フレーム内で上・中央・下／左・中央・右に揃える
  2. 位置 -- 画面上の座標位置。数値を変えると正確に動かせます。
  3. 回転 -- オブジェクトを回転（0°〜360°）できます。
  4. 反転 -- 水平方向・垂直方向に反転。鏡像を作る時に便利。

サイズ

  1. W（Width） -- 幅を数値で指定。
  2. H（Height） -- 高さを数値で指定。
  3. 縦横比を固定するロック。

外見（Appearance）

  1. 不透明度 -- 透明度を0〜100％で調整。
  2. 角の半径 -- 角丸を作る。数字を増やすほど丸くなる。
  3. アイコン -- 角ごとに個別設定が可能（例：左上だけ丸く）。

塗り（Fill）

  1. カラーコード -- 例：#D9D9D9（16進数の色指定）
  2. ％ -- 色の透明度（Alpha値）
  3. ＋ -- 複数の塗りを重ねる（グラデーション・テクスチャなど）
  4. 塗りを一時的に非表示にする

塗りの右端の「ドロップ」アイコンをクリックすると、単色／線形グラデーション／放射状グラデーション／画像塗りが選べます。

線（Stroke）

  1. カラーコード -- 枠線の色を指定。
  2. ％ -- 色の透明度（Alpha値）
  3. 線を一時的に非表示にする
  4. 位置 -- 枠線を内側／中央／外側に描く。
  5. 太さ -- 枠線の太さ（px単位）。
  6. ＋ボタン -- 枠線を複数重ねる（ダブルラインなど）。
  7. 個々の線
  8. 線の設定。波線などの詳細

エフェクト

影・ぼかしなどの視覚効果を設定できます。

  1. ドロップシャドウ（Drop Shadow） -- 影をつける。距離・ぼかし・角度・色を設定可能。
    • インナーシャドウ（Inner Shadow） -- 内側に影をつける。
    • レイヤーブラー（Layer Blur） -- オブジェクト全体をぼかす。
    • 背景ブラー（Background Blur） -- 背景だけをぼかす。透明ガラス風表現に便利。

タイポグラフィ（Typography）パネル解説

フォント設定

  1. フォント名 -- 使用するフォントを選択します。
  2. スタイル -- フォントの太さ（ウェイト）や斜体（Italic）などを選択します。
  3. サイズ -- 文字の大きさ（px単位）を指定します。
  4. 行間（Line height） -- テキストの上下の余白。段落が読みやすくなります。数値入力（例：16）または「自動（Auto）」で設定。
  5. 文字間隔（Letter spacing） -- 各文字の間隔を%またはpxで調整。たとえば「2%」なら少し広め、「-2%」なら詰め気味になります。
  6. 左揃え／中央揃え／右揃え -- 段落の水平位置を設定します。
  7. 上揃え／中央揃え／下揃え -- テキストボックス内の縦位置を設定します。
POINT

px（ピクセル）とは

px（ピクセル）は、画面の中の「点」のようなもの。たとえばテレビやスマホの画面は、無数の小さな点でできています。デザインで「100pxの四角」といえば、点100個ぶんの幅をもつ四角という意味です。点の数が多いほど、より細かくてきれいに見えます。つまり、pxは「画面上でどれくらいの大きさか」を決めるための基本のものさしです。

制約

これは、フレームやオートレイアウトの中で、要素がどのように位置やサイズを保つかを指定するための重要な機能です。親フレーム（またはグループ）をリサイズしたときに、子要素（ボタンやテキストなど）が「どう動くか」を制御します。

たとえば：

  • 画面を広げてもボタンを中央に保つ
  • 右上に配置したアイコンを常に右上固定にする

といった設定ができます。

プログラマーが、受け取ったデザインがスマホ ⇔ タブレット ⇔ パソコンで表示されるときの、レスポンシブの動作やデザインの確認のためによく使われます。

  1. 左右（Horizontal） -- 横方向の固定方法
  2. 上下（Vertical） -- 縦方向の固定方法
  3. 真ん中の「＋」 は 中央揃え。

横方向（Horizontal）オプション

設定 説明
左 左端に固定。親フレームを広げても左の距離を維持。
右 右端に固定。右の距離を維持。
左＋右 左右両方に固定 → 幅が自動的に伸び縮みします。
中央 親の中央に位置を保つ。

縦方向（Vertical）オプション

設定 説明
上 上端に固定。
下 下端に固定。
上＋下 上下両方に固定 → 高さが伸び縮みします。
中央 縦の中央位置を保つ。
