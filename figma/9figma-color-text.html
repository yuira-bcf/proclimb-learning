Section 9-1

スタイルとは

スタイルは、色やフォントなどのデザイン属性を再利用可能な形で保存する機能です。CSSの変数のように、一箇所で定義して全体で使い回せます。

スタイルを使うメリット

sync

一括変更

スタイルを変更すると、適用されているすべての要素に反映されます。

rule

一貫性の確保

同じスタイルを使うことで、デザインの統一感が保たれます。

speed

効率化

毎回色コードやフォント設定を入力する手間が省けます。

スタイルの種類

スタイル種類 保存できる属性
Color styles 塗りつぶしの色、線の色
Text styles フォント、サイズ、行間など
Effect styles 影、ぼかしなど
Grid styles レイアウトグリッドの設定
Section 9-2

カラースタイルの作成

プロジェクトで使用する色をスタイルとして登録しましょう。

カラースタイル作成手順

1

図形を作成して色を設定

長方形ツール（R）で図形を作成し、Fillに登録したい色を設定します。

2

Fillセクションの4点アイコンをクリック

右パネルのFill横にある「Style」アイコン（4つの点）をクリックします。

3

「+」をクリック

スタイルパネルが開くので、右上の「+」をクリックします。

4

スタイル名を入力

わかりやすい名前を入力し、「Create style」をクリックします。

カラースタイルの命名規則

おすすめの命名パターン

  • Primary/100：プライマリカラーの薄いバリエーション
  • Primary/500：プライマリカラーの標準
  • Primary/900：プライマリカラーの濃いバリエーション
  • Text/Primary：本文テキストの色
  • Text/Secondary：サブテキストの色
  • Background/Light：明るい背景色
「/」で階層化

スタイル名に「/」を使うと、自動的にフォルダ分けされます。例：「Primary/500」と「Primary/700」は「Primary」フォルダにまとめられます。

ポートフォリオサイト用カラーパレット例

Primary/900

#1E293B

Accent/500

#3B82F6

Background

#F8FAFC

Text/Muted

#64748B

Section 9-3

テキストスタイルの作成

見出し、本文、キャプションなど、使用するテキストスタイルを定義しましょう。

テキストスタイル作成手順

1

テキストを作成

テキストツール（T）でテキストを入力します。

2

フォント設定を調整

フォント、サイズ、ウェイト、行間などを設定します。

3

Textセクションの4点アイコンをクリック

右パネルのText横にある「Style」アイコンをクリックします。

4

「+」でスタイル作成

名前を入力して「Create style」をクリックします。

テキストスタイルに含まれる属性

属性 説明
Font family フォントの種類（Noto Sans JPなど）
Font weight 太さ（Regular, Medium, Boldなど）
Font size 文字サイズ（px）
Line height 行間（px, %, Auto）
Letter spacing 文字間（px, %）
info テキストスタイルに色は含まれない

テキストスタイルには色情報は含まれません。テキストの色はカラースタイルを別途適用します。

推奨テキストスタイル例

Heading/H1

Noto Sans JP / Bold / 48px / Line 120%

Heading/H2

Noto Sans JP / Bold / 32px / Line 130%

Heading/H3

Noto Sans JP / SemiBold / 24px / Line 140%

Body/Regular - 本文テキストのサンプルです。読みやすさを重視して行間を広めに設定しています。

Noto Sans JP / Regular / 16px / Line 180%

Caption/Small - 補足テキスト

Noto Sans JP / Regular / 12px / Line 160%

Section 9-4

スタイルの適用と管理

作成したスタイルを他の要素に適用する方法と、スタイルの編集・削除について学びましょう。

スタイルの適用

1

要素を選択

スタイルを適用したい要素を選択します。

2

スタイルアイコンをクリック

Fill（色）またはText（テキスト）横の4点アイコンをクリックします。

3

スタイルを選択

一覧から適用したいスタイルをクリックします。

スタイルの編集

スタイルを編集すると、そのスタイルが適用されているすべての要素に変更が反映されます。

スタイル編集の方法

スタイル一覧でスタイル名を右クリック → 「Edit style」を選択します。色やフォント設定を変更して保存すると、すべてに反映されます。

実習

ポートフォリオ用スタイル作成

ポートフォリオサイトで使用するスタイルを作成しましょう。

作成するカラースタイル

  • Primary/Light：#F8FAFC（背景色）
  • Accent：#3B82F6（アクセントカラー）

作成するテキストスタイル

  • Heading/H1：Noto Sans JP Bold 48px
  • Heading/H2：Noto Sans JP Bold 32px
  • Heading/H3：Noto Sans JP SemiBold 24px
  • Body/Regular：Noto Sans JP Regular 16px
  • Caption：Noto Sans JP Regular 12px
