Section 5-2

ファイルブラウザ

ファイルブラウザでは、チームやプロジェクト内のファイルを一覧表示で確認できます。

表示切替

ファイルの表示形式は、グリッド表示とリスト表示を切り替えることができます。

グリッド表示

サムネイルが大きく表示され、デザインの内容を視覚的に確認しやすい

リスト表示

ファイル名、更新日時、編集者などの情報を一覧で確認できる

ファイルの操作

ファイルを右クリックすると、以下の操作が可能です。

edit

Rename

ファイル名を変更します。

content_copy

Duplicate

ファイルを複製します。

drive_file_move

Move to Project

別のプロジェクトに移動します。

delete

Delete

ファイルを削除します（ゴミ箱に移動）。

Section 5-3

ファイル作成

新しいデザインファイルを作成する方法を学びましょう。

新規ファイルの作成手順

1

「+ New design file」をクリック

ホーム画面右上、またはプロジェクト内の「+」ボタンをクリックします。

2

ファイルが開く

新しいデザインファイルがエディターで開きます。

3

ファイル名を設定

左上のファイル名をクリックして、わかりやすい名前に変更します。

ファイル作成のショートカット

ホーム画面で「Ctrl/Cmd + N」を押すと、新規ファイルを素早く作成できます。

FigJamファイルの作成

ホワイトボード機能を使いたい場合は「New FigJam file」を選択します。デザインファイルとFigJamファイルは用途が異なります。

Design file

UIデザイン、プロトタイプ作成用

FigJam file

ブレインストーミング、フローチャート用

画面の操作方法

Figmaのキャンバスを効率的に操作するための基本操作を覚えましょう。

基本操作

操作 Windows Mac
ズームイン Ctrl + スクロール上 / Ctrl + + Cmd + スクロール上 / Cmd + +
ズームアウト Ctrl + スクロール下 / Ctrl + - Cmd + スクロール下 / Cmd + -
パン（移動） スペース + ドラッグ スペース + ドラッグ
100%表示 Ctrl + 0 Cmd + 0
全体表示 Shift + 1 Shift + 1
選択範囲にフィット Shift + 2 Shift + 2
トラックパッドでの操作

Macのトラックパッドでは、2本指でピンチ操作するとズームイン/アウト、2本指でスワイプするとパン（移動）ができます。
