Section 8-1

制作実績セクションの概要

worksセクションは、自身の実績を示すセクションです。見てくれる方に、自分自身の経歴やスキルを知ってもらうためのセクションとなります。

worksセクションでは、カードと呼ばれるデザインを使用します。サンプル画像を使用しながら、Webサイトでよく使われるカードレイアウトの基本を学びます。

全体作業フロー
  1. タイトル文のインスタンス設置・調整
  2. タグコンポーネントを作成
  3. バリアント機能の設定
  4. 画像の配置と調整
  5. プロジェクトタイトル文の作成
  6. タグインスタンスの配置
  7. ④⑤⑥のオートレイアウト化と名称変更
  8. Card の複製とオートレイアウト化
  9. 画像の連続置換
  10. タグの切替
  11. View Moreボタンの設置
  12. 各要素のオートレイアウト化と微調整
Section 8-2

worksセクションの作成

① タイトル文のインスタンス設置・調整

  1. 左パネルの[アセット]タブから、[ title ]を選択
  2. ドラッグ＆ドロップで[ service ] の下、150pxのところに配置
  3. Alt(Option) + H (左右中央揃え) で位置を調整
  4. タイトル文と、その下のテキストを下記に変更

[ Service ] → [ Works ]

[ サービス ] → [制作実績]

② タグコンポーネントを作成

次に、制作実績のカテゴリーを示す「タグ」と呼ばれる要素を作成します。このタグはコンポーネント化し、「バリアント機能」というfigmaの機能を使うことで、「Webデザイン」「グラフィックデザイン」「UIデザイン」など複数のバリエーションをまとめて管理できるようになります。

「バリアント（Variant）」は、コンポーネントをより便利に使うための機能です。簡単に言うと、中身を自在に切り替えられるコンポーネントです。

  1. [ Compornents ] ページで、テキスト[グラフィックデザイン]を作成し、スタイルを設定します。

タグテキストのスタイル設定

  • テキストスタイル：text / sm
  • 配置：左揃え
  • カラースタイル：portfolio / black
  1. テキストを選択し、オートレイアウト化：Shift + A
  2. レイヤーに新たに作成されたフレームの名称を[ tag / グラフィック ]に変更
  3. [ tag/グラフィック]のオートレイアウトの各項目を、以下に設定

tagのオートレイアウト設定

  • フロー：水平
  • 配置：中央揃え
  • 間隔：0
  • 水平パディング：8
  • 垂直パディング：2
  • 角の半径：4
  1. [ tag /グラフィック] を選択し、[ 塗り ] → [ portfolio / gray ]に設定
  2. [ tag / グラフィック ] をコンポーネント化：Ctrl(Cmd) + Alt(Option) + K

ここまではこれまで同様のコンポーネント作成です。ここから、より機能を発展させた「バリアント」機能の設定をしていきます。

③ バリアント機能の設定

  1. tag/グラフィック を選択し、右パネル[ tag/グラフィ…]の右側にある[◇]をクリック（バリアントの追加）

※ バリアントの設定をすると、フレーム名が[ tag ]のみに自動的に変わります

  1. tagを選択し、下部の[ + ]をクリック（これで同じテキストが3つ、縦に並びます）
  2. 3つのテキストを上から[グラフィックデザイン] [ウェブデザイン] [ UIデザイン]に変更
  3. tagを選択し、右パネル[プロパティ] → [プロパティの編集]アイコンをクリック
  4. [ バリアントプロパティの編集 ]の名前、値を下記に設定

バリアントプロパティの設定

  • 名前：Category
  • 値：グラフィックデザイン
  • 値：UIデザイン
  • 値：ウェブデザイン
POINT

バリアントを使うと、1つのデザインコンポーネントの中で複数の状態（種類）を切り替えることができます。これにより、タグのような「同じデザインで内容だけ異なる要素」を効率よく管理できます。値の順番を入れ替える場合は、［バリアントプロパティの編集］パネルを開き、値の左端をドラッグして上下に並び替えます。

④〜⑦ カードの作成

  1. portfolio ページにて、[アセット]タブから「work6.png」 の画像を選択します：Ctrl(Cmd) + Shift + K
  2. title の下に適当に配置し、右パネル[サイズ] →[ W 300]に設定
  3. [ Work6 1 ]の下に、テキスト[プロジェクト名が入ります]を作成し、スタイルを設定

プロジェクト名のスタイル設定

  • テキストスタイル：title / sm
  • カラースタイル：portfolio / black
  1. 左パネル[アセット]タブから [ tag ] を選択
  2. テキストの下の適当な位置に、[ドラッグ＆ドロップ]で配置
  3. Work6, テキスト, tag の3つを選択し、オートレイアウト化：Shift + A
  4. レイヤーに新たに作成されたフレームの名称を[ card 1 ]に変更
  5. card 1 のオートレイアウトの各項目を、以下に設定

cardのオートレイアウト設定

  • フロー：垂直
  • サイズ：300
  • 配置：上揃え（左）
  • 間隔：24
  • 各パディング：0

⑧ Card の複製とオートレイアウト化

  1. card 1 を選択し、右側に複製：Shift + Alt(Option) +ドラッグ
  2. そのまま4回、連続でcard 1 を複製、card 1 を全部で6個作ります：Ctrl(Cmd) + Dを4回

※ フレームからはみ出た部分は見えなくなりますが、デフォルトの挙動です

  1. すべてのcrad を選択し、オートレイアウト化：Shift + A
  2. レイヤーに新たに作成されたフレームの名称を[ worksItem ]に変更
  3. worksItem のオートレイアウトの各項目を、以下に設定

worksItemのオートレイアウト設定

  • フロー：水平
  • 折り返し：あり
  • サイズ：1028
  • 配置：上揃え（左）
  • 間隔：64
  • 各パディング：0
  1. Alt(Option) + H で左右中央揃えにします。

⑨ 画像の連続置換

  1. Work1~5の、5つの画像を選択する：Ctrl(Cmd) + Shift + K

※ Ctrl(Cmd)を押しながら 画像を選択すると、複数選択が可能です

  1. 下記指定・画像に合わせて、連続クリックで各cardの画像を差し換え

画像の割り当て

  • card1：変更なし
  • card2：3つのスマホの画像（Work1）
  • card3：2つのスマホの画像（Work4）
  • card4：オレンジ画面のスマホの画像（Work3）
  • card5：波の画面のスマホ画像（Work5）
  • card6：デスクトップの画像（Work2）

⑩ タグの切替（バリアントの変更）

  1. card 1の[ tag ] を選択し、右パネル [ tag ]→[ Category ]の右側、[グラフィ……∨]をクリック
  2. ドロップダウンの中から[ UIデザイン ]を選択します。
  3. 1~2の手順を繰り返し、Card 2~6 でも同様に変更

各カードのタグ設定

  • Card2：UIデザイン
  • Card3：グラフィックデザイン（変更なし）
  • Card4：グラフィックデザイン（変更なし）
  • Card5：ウェブデザイン
  • Card6：ウェブデザイン

⑪⑫ ボタン設置と最終調整

  1. 左パネル、[アセット]タブから[ Button ] を選択
  2. Button インスタンスを[ worksItem ]の下、適当な位置にドラッグ＆ドロップで配置
  3. Button インスタンスのテキストを[ View More ]に変更
  4. [ worksItem ] [ Button ] の両方を選択し、オートレイアウト化：Shift + A
  5. レイヤーに新たに作成されたフレームの名称を[ worksContainer ]に変更
  6. worksContainer のオートレイアウトの各項目を、以下に設定

worksContainerのオートレイアウト設定

  • フロー：垂直
  • 配置：上揃え（中央）
  • 間隔：80
  • 各パディング：0
  1. [ worksContainer ] [ title ] の両方を選択し、オートレイアウト化：Shift + A
  2. レイヤーに新たに作成されたフレームの名称を[ works ]に変更
  3. works のオートレイアウトの各項目を、以下に設定

worksのオートレイアウト設定

  • フロー：垂直
  • 配置：上揃え（中央）
  • 間隔：64
  • 各パディング：0
  1. worksを左右中央揃えにします：Alt(Option) + H
完了

これでworksセクションが完成となります。次は最後のコンテンツセクションの作成です。

まとめ

第8章のまとめ

  • Worksセクションは制作実績を紹介するカードレイアウト
  • バリアント機能でタグなどの複数バリエーションを1つのコンポーネントで管理
  • 折り返し：ありで自動的にカードが次の行に折り返される
  • 画像の連続置換で効率的に画像を差し替え可能
  • バリアントは右パネルのドロップダウンから切り替え
  • View Moreボタンでさらなるコンテンツへの誘導を表現
