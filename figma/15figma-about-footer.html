第9章

About（私について）セクション

About セクションでは、人物面を表現します。ここでは円形のプロフィールアイコンと短い自己紹介文を作成しますが、一般的にはスキルの一覧表や経歴を記載したりもします。

1. Aboutセクションの概要

全体作業フロー
  1. タイトル文のインスタンス設置・調整
  2. アイコン画像の作成とトリミング
  3. プロフィールテキストの作成とスタイル設定
  4. 各要素のオートレイアウト化と微調整

2. About セクションの作成

① タイトル文のインスタンス設置・調整

  1. 左パネルの[アセット]タブから、[ title ]を選択
  2. [ works ] の下、150pxのところに、ドラッグ＆ドロップで配置
  3. Alt(Option) + H(左右中央揃え) で位置を調整
  4. タイトル文と、その下のテキストを下記に変更

[ Service ] → [ About ]

[ サービス ] → [ 私について ]

② アイコン画像の作成とトリミング

  1. title の下、適当な位置にシェイプ（楕円）を配置：サイズ [ W 200] [H 200 ]
  2. フォルダを開き、[ about.jpg ] を選択します：Ctrl(Cmd) + Shift + K
  3. 作成した楕円 をクリックして画像を配置
  4. 右サイドパネルの[塗り]の下[ 画像 ]をクリック
  5. [カスタム]タブ →[塗り] → [トリミング]を選択
  6. 方向キーやドラッグなどで、人物が正面中央になるように調整します。
  7. レイヤーの[ Ellpse ○] の名称を[ aboutIcon ]に変更

これでプロフィールアイコンが完成です。次はプロフィール分を作成していきます。

③ プロフィールテキストの作成とスタイル設定

  1. テキストを選択し[ aboutIcon ] の右側の適当な位置に[ Tanaka Aoi ]と入力して配置
  2. テキストに以下のスタイルとサイズ変更の設定をします

名前テキストのスタイル設定

  • テキストスタイル：text / lg
  • 色スタイル：portfolio / black
  1. 続けて以下のテキストを[ Tanaka Aoi] の下に作成し、スタイルとサイズ変更を設定します。

プロフィール文

企業やサービスの課題をデザインの力で解決するWebデザイナーです。UI/UXを重視し、ユーザーの行動心理を踏まえた情報設計と、目的達成につながるビジュアル表現を得意としています。企画段階から関わり、デザインだけでなく、実装・改善までを一貫して対応。デザインの見た目だけでなく、成果を生む体験を設計することを大切にしています。

  • テキストスタイル：text / md
  • サイズ変更：高さの自動調整
  • サイズ：W 480
  • 配置の編集：両端揃え
  1. 両方のテキストを選択し、オートレイアウト化：Shift + A

テキストのオートレイアウト設定

  • フロー：垂直
  • 配置：上揃え（左）
  • 間隔：16
  • 各パディング：0
  1. レイヤーに新たに作成されたフレームの名称を[ aboutText ] に変更

④ 各要素のオートレイアウト化と微調整

  1. [ aboutIcon ] [ aboutText ] の両方を選択し、オートレイアウト化：Shift + A
  2. レイヤーに新たに作成されたフレームの名称を[ aboutContainer ]に変更
  3. aboutContainerのオートレイアウトの各項目を、以下に設定

aboutContainerのオートレイアウト設定

  • フロー：水平
  • 配置：上揃え（中央）
  • 間隔：64
  • 各パディング：0

最後にタイトル文とcontainerを組合せ、背景色を付けてaboutセクションを完成させていきます。

  1. [ aboutContainer ] [ title ] の両方を選択し、オートレイアウト化：Shift + A
  2. レイヤーに新たに作成されたフレームの名称を[ about ]に変更
  3. about のオートレイアウトの各項目を、以下に設定

aboutのオートレイアウト設定

  • サイズ変更：W1440
  • フロー：垂直
  • 配置：上揃え(中央)
  • 間隔：64
  • 上パディング：80
  • 下パディング：120

※ パディングの右側の四角アイコンをクリックすると、各方向へのパディングが設定できます

  1. about に色スタイル[ portfolio / gray ]を設定
  2. about を左右中央揃えにします：Alt(Option) + H
完了

背景色をつけることでページ内の区切りが明確になり、見た目にメリハリが生まれます。これで、「About（私について）」セクションが完成です。残りは、ページの締めくくりとなるフッターの作成となります。

第10章

フッターの作成

フッターは著作権表記などを記載したり、改めてサイト内リンクやロゴが記載されたりします。今回は著作権表記のみしていきます。

1. フッターの概要

Webサイトの最下部に設置する「コピーライト」を作成します。ここでは 「© Tanaka Aoi」 といったシンプルな表記のフッターをデザインします。

全体作業フロー
  1. テキストを作成し、スタイルを設定
  2. オートレイアウト化と色スタイルの設定
  3. 要素の位置の微調整と名称変更
  4. 全体フレームの名称変更

2. フッターの作成

① テキストを作成し、スタイルを設定

  1. テキストを選択し[ about ] の下、適当な位置に[ ©Tanaka Aoi ]と入力して配置

※ © は[こぴーらいと]で変換できます。

  1. テキストに下記スタイルを設定します。

コピーライトテキストのスタイル設定

  • テキストスタイル：text / sm
  • カラースタイル：portfolio / white

※ 背景色と重なるため、一時的にテキストが見えなくなります

② オートレイアウト化と色スタイルの設定

  1. テキストを選択し、オートレイアウト化：Shift + A
  2. レイヤーに新たに作成されたフレームの名称を[ footer ]に変更
  3. footer のオートレイアウトの各項目を、以下に設定

footerのオートレイアウト設定

  • フロー：水平
  • サイズ変更：W1440
  • 配置：右揃え
  • 間隔：0
  • 右パディング：64
  1. footer の色スタイルに[ portfolio / lightblue ]を設定

③ 要素の位置の微調整と名称変更

  1. footer を選択し、[ about ]セクション下、[ 80px ]の位置になるよう調整
  2. Desktop - 1 を選択し、フッターの高さに合わせる
  3. レイヤーの[ Desktop - 1 ]を選択し、フレーム名称を[ design -PC- ]に変更
完成

これでポートフォリオサイトのデザインは完成です。テキストや画像は自由に編集できるので、自分のスタイルに合わせてカスタマイズしてみましょう。メインビジュアルの更新や、サムネイルの設定を行うと、より完成度の高いデザインになります。

まとめ

第9-10章のまとめ

  • Aboutセクションでは円形アイコンとプロフィール文で人物面を表現

  • 背景色でセクションの区切りを明確に
  • 上下パディングを別々に設定してセクションのバランスを調整
  • Footerは著作権表記などを記載するページ最下部の要素
  • ©は「こぴーらいと」で変換可能
  • 最後にフレーム名を「design -PC-」などわかりやすく変更
