6-3固定サイドバーレイアウト

ホーム / レイアウト設定 / 固定サイドバー

固定サイドバーレイアウト

サイドバーを画面に固定し、コンテンツをスクロールしてもナビゲーションが常に表示されるレイアウトです。

  • サイドバーが常に表示される
  • コンテンツスクロール時もナビゲーションにアクセス可能
  • 長いメニューはサイドバー内でスクロール
  • ページ遷移が素早くできる

6-3-1レイアウトの特徴

固定サイドバーは、多くの管理画面で採用されている定番のレイアウトです。

主なユースケース:

  • 多くのページを持つ管理画面
  • 頻繁にページ遷移するアプリケーション
  • 常にナビゲーションを表示したい場合

6-3-2実装のポイント

このレイアウトを実装する際の注意点:

  • position: fixed でサイドバーを固定
  • overflow-y: auto でサイドバー内スクロールを有効化
  • メインコンテンツにmargin-leftを設定
  • モバイルではスライドイン式に変更

下にスクロールしても、サイドバーは固定されたままです。
