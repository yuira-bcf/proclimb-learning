7-6-2 rocket_launch はじめに

help AdminPanelとは何ですか？ expand_more
AdminPanelは、Webアプリケーションの管理画面を簡単に構築できるダッシュボードテンプレートです。 豊富なUIコンポーネント、チャート、フォーム要素を備えており、すぐに使い始めることができます。 レスポンシブデザインにも対応しているため、スマートフォンやタブレットからでも快適に操作できます。
help 利用を開始するにはどうすればいいですか？ expand_more
以下の手順で簡単に始められます：
  • ダウンロードページからファイルをダウンロード
  • 解凍したファイルをWebサーバーに配置
  • ブラウザでアクセスして完了
詳細なセットアップガイドはドキュメントページをご覧ください。
help 対応しているブラウザは何ですか？ expand_more
最新版のChrome、Firefox、Safari、Edge、Operaに対応しています。 Internet Explorer 11はサポート対象外となります。 最良のパフォーマンスのため、常に最新バージョンのブラウザをご利用ください。

7-6-3 account_circle アカウント

help パスワードを忘れた場合はどうすればいいですか？ expand_more
ログイン画面の「パスワードを忘れた方」リンクをクリックしてください。 登録されているメールアドレスに、パスワードリセット用のリンクが送信されます。 リンクの有効期限は24時間です。
help アカウントを削除するにはどうすればいいですか？ expand_more
設定画面の「アカウント」セクションから削除できます。 アカウントを削除すると、すべてのデータが完全に削除され、復元できなくなります。 削除前に必要なデータをエクスポートしておくことをお勧めします。

7-6-4 payment 料金・支払い

help 利用可能な支払い方法は何ですか？ expand_more
以下の支払い方法に対応しています：
  • クレジットカード（Visa、MasterCard、JCB、American Express）
  • 銀行振込
  • PayPal
  • 請求書払い（法人のみ）
help 返金ポリシーはありますか？ expand_more
ご購入から30日以内であれば、理由を問わず全額返金いたします。 返金をご希望の場合は、サポートまでご連絡ください。 返金処理には5〜10営業日かかる場合があります。

7-6-5 build 技術サポート

help カスタマイズは可能ですか？ expand_more
はい、すべてのコンポーネントは自由にカスタマイズ可能です。 CSS変数を使用しているため、カラーテーマの変更も簡単に行えます。 :rootセクションの変数を変更するだけで、全体のテーマカラーを変更できます。
help バグを見つけた場合はどうすればいいですか？ expand_more
GitHubのIssueページから報告してください。 報告時には、以下の情報を含めていただけると対応がスムーズになります：
  • 発生した問題の詳細な説明
  • 再現手順
  • 使用しているブラウザとバージョン
  • スクリーンショット（あれば）
