Introduction

コンポーネントとは？

Bootstrapのコンポーネントとは、あらかじめデザインされた「UI部品」のことです。アラート、リスト、テーブルなど、Webサイトでよく使われるパーツがクラス名を付けるだけで美しく表示されます。

この章では、以下の基本コンポーネントを学びます。

  • アラート — ユーザーへの通知メッセージ
  • リストグループ — 項目の一覧表示
  • テーブル — データの表形式表示
  • プログレスバーとスピナー — 読み込みや進捗の表示

どれもクラス名を追加するだけで利用でき、自分でCSSを書く必要がありません。実務でも頻繁に使うコンポーネントばかりですので、しっかり覚えましょう。

Section 5-1

アラート

アラートの基本

アラート（Alert）は、ユーザーに重要な情報を伝えるためのコンポーネントです。成功メッセージ、エラー通知、注意喚起など、さまざまな場面で活用できます。

基本的な使い方は、alert クラスと色のバリエーションクラスを組み合わせるだけです。

HTML
<div class="alert alert-primary" role="alert">
  これはプライマリアラートです。
</div>

role="alert" はアクセシビリティのための属性です。スクリーンリーダーがこの要素をアラートとして認識するために付けましょう。

8つのカラーバリエーション

Bootstrapのアラートには8種類のカラーバリエーションが用意されています。それぞれ異なる意味合いを持つので、目的に応じて使い分けましょう。

クラス名 色 主な用途
alert-primary 青 重要な情報、メインのお知らせ
alert-secondary グレー 補足情報
alert-success 緑 成功メッセージ、完了通知
alert-danger 赤 エラー、危険な操作の警告
alert-warning 黄 注意喚起、警告メッセージ
alert-info 水色 ヒント、追加情報
alert-light 明るいグレー 軽い通知
alert-dark 濃いグレー 強調メッセージ
HTML
<div class="alert alert-success" role="alert">
  登録が完了しました！
</div>

<div class="alert alert-danger" role="alert">
  エラーが発生しました。入力内容を確認してください。
</div>

<div class="alert alert-warning" role="alert">
  パスワードの有効期限が近づいています。
</div>

<div class="alert alert-info" role="alert">
  新機能が追加されました。詳細はこちらをご覧ください。
</div>

色の使い分けのポイント

alert-success（緑）は「うまくいった」、alert-danger（赤）は「問題がある」、alert-warning（黄）は「注意が必要」という直感的な意味合いで使いましょう。ユーザーは色で情報の重要度を判断します。

閉じるボタン付きアラート（Dismissible Alert）

ユーザーがアラートを閉じられるようにするには、alert-dismissible クラスと fade show クラス、そして閉じるボタン（btn-close）を追加します。

HTML
<div class="alert alert-warning alert-dismissible fade show" role="alert">
  <strong>注意！</strong> セッションの有効期限が切れそうです。
  <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>
クラス・属性 役割
alert-dismissible 閉じるボタンの位置を調整するためのクラス
fade show 閉じるときのフェードアニメーションを有効にする
btn-close 閉じるボタン（×マーク）のスタイル
data-bs-dismiss="alert" クリック時にアラートを閉じるBootstrapのJS機能

JavaScriptが必要です

閉じるボタンの動作にはBootstrapのJavaScriptが必要です。CDNで bootstrap.bundle.min.js を読み込んでいることを確認しましょう。

アラート内のリンク

アラート内にリンクを入れたい場合は、alert-link クラスを使います。このクラスを使うと、リンクの色がアラートの配色に合ったものになります。

HTML
<div class="alert alert-primary" role="alert">
  システムのアップデートが利用可能です。
  <a href="#" class="alert-link">こちらから更新</a>してください。
</div>

<div class="alert alert-danger" role="alert">
  お支払いに問題があります。
  <a href="#" class="alert-link">お支払い情報を確認</a>してください。
</div>

実践例：会員登録フォームのメッセージ

実際のWebアプリケーションでよく見かけるアラートの使い方を紹介します。

HTML
<!-- 登録成功メッセージ -->
<div class="alert alert-success alert-dismissible fade show" role="alert">
  <strong>登録完了！</strong>
  ご登録いただきありがとうございます。確認メールを送信しました。
  <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>

<!-- 入力エラーメッセージ -->
<div class="alert alert-danger" role="alert">
  <strong>入力エラー：</strong>
  <ul class="mb-0">
    <li>メールアドレスの形式が正しくありません</li>
    <li>パスワードは8文字以上にしてください</li>
  </ul>
</div>

<!-- メンテナンス通知 -->
<div class="alert alert-info" role="alert">
  <strong>メンテナンスのお知らせ：</strong>
  2月20日(金) 02:00〜05:00にシステムメンテナンスを実施いたします。
  <a href="#" class="alert-link">詳細はこちら</a>
</div>
Section 5-2

リストグループ

リストグループの基本

リストグループ（List Group）は、項目を一覧で表示するためのコンポーネントです。サイドバーのメニュー、設定画面の項目、チャットの連絡先リストなど、さまざまな場面で使えます。

基本構造は、list-group を付けた <ul> の中に list-group-item を付けた <li> を並べるだけです。

HTML
<ul class="list-group">
  <li class="list-group-item">HTMLの基礎</li>
  <li class="list-group-item">CSSの基礎</li>
  <li class="list-group-item">JavaScriptの基礎</li>
  <li class="list-group-item">Bootstrapの基礎</li>
  <li class="list-group-item">レスポンシブデザイン</li>
</ul>

これだけで、ボーダーで区切られた美しいリストが完成します。

アクティブ状態と無効状態

リストの中で「現在選択中」の項目には active クラスを、操作できない項目には disabled クラスを追加します。

HTML
<ul class="list-group">
  <li class="list-group-item active" aria-current="true">
    HTML入門（学習中）
  </li>
  <li class="list-group-item">CSS入門</li>
  <li class="list-group-item">JavaScript入門</li>
  <li class="list-group-item disabled" aria-disabled="true">
    上級コース（未開放）
  </li>
</ul>
クラス 見た目 用途
active 背景がプライマリカラーに変化 現在選択中の項目を示す
disabled テキストが薄くなる 操作できない項目を示す

クリック可能なリスト（list-group-item-action）

リスト項目をクリック可能にするには、<a> タグまたは <button> タグに list-group-item-action クラスを追加します。ホバー時にハイライトされ、クリックできることがユーザーに伝わります。

HTML
<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action active" aria-current="true">
    ダッシュボード
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    プロフィール設定
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    通知設定
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    セキュリティ
  </a>
  <a href="#" class="list-group-item list-group-item-action disabled" tabindex="-1" aria-disabled="true">
    管理者設定（権限なし）
  </a>
</div>

ulとdivの使い分け

単なるリスト表示には <ul> + <li> を使います。クリック可能なリストには <div> + <a>（または<button>）の組み合わせを使いましょう。HTML的に正しい使い方です。

色付きリストアイテム

アラートと同様に、リストアイテムにもカラーバリエーションがあります。list-group-item-{color} クラスを使います。

HTML
<ul class="list-group">
  <li class="list-group-item list-group-item-primary">重要タスク</li>
  <li class="list-group-item list-group-item-success">完了タスク</li>
  <li class="list-group-item list-group-item-warning">注意が必要なタスク</li>
  <li class="list-group-item list-group-item-danger">期限超過タスク</li>
  <li class="list-group-item list-group-item-info">情報共有タスク</li>
</ul>

実践例：サイドバーメニュー（バッジ付き）

リストグループとバッジを組み合わせて、実用的なサイドバーメニューを作ってみましょう。d-flex と justify-content-between を使って、テキストとバッジを左右に配置します。

HTML
<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action d-flex justify-content-between align-items-center active">
    受信トレイ
    <span class="badge bg-light text-dark rounded-pill">14</span>
  </a>
  <a href="#" class="list-group-item list-group-item-action d-flex justify-content-between align-items-center">
    お気に入り
    <span class="badge bg-primary rounded-pill">3</span>
  </a>
  <a href="#" class="list-group-item list-group-item-action d-flex justify-content-between align-items-center">
    下書き
    <span class="badge bg-secondary rounded-pill">7</span>
  </a>
  <a href="#" class="list-group-item list-group-item-action d-flex justify-content-between align-items-center">
    送信済み
  </a>
  <a href="#" class="list-group-item list-group-item-action d-flex justify-content-between align-items-center">
    ゴミ箱
    <span class="badge bg-danger rounded-pill">2</span>
  </a>
</div>

Flexbox ユーティリティの活用

d-flex justify-content-between align-items-center はBootstrapのユーティリティクラスです。Flexboxを使ってテキストとバッジを横並びにし、両端に配置しています。第8章で詳しく学びます。

Section 5-3

テーブル

テーブルの基本

HTMLの <table> に table クラスを追加するだけで、見やすく整えられたテーブルになります。Bootstrapのテーブルは、行間のパディング、水平のボーダー、ヘッダーの強調表示が自動で適用されます。

HTML
<table class="table">
  <thead>
    <tr>
      <th>#</th>
      <th>名前</th>
      <th>メールアドレス</th>
      <th>登録日</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>田中太郎</td>
      <td>tanaka@example.com</td>
      <td>2025/01/15</td>
    </tr>
    <tr>
      <td>2</td>
      <td>佐藤花子</td>
      <td>sato@example.com</td>
      <td>2025/02/20</td>
    </tr>
    <tr>
      <td>3</td>
      <td>鈴木一郎</td>
      <td>suzuki@example.com</td>
      <td>2025/03/10</td>
    </tr>
  </tbody>
</table>

テーブルのバリエーション

Bootstrapには、テーブルの見た目を変更するためのクラスが複数用意されています。table クラスと組み合わせて使います。

クラス名 効果 説明
table-striped 縞模様 奇数行と偶数行で背景色が交互に変わる
table-bordered 全方向ボーダー すべてのセルに上下左右のボーダーが付く
table-hover ホバーハイライト 行にマウスを乗せると背景色が変わる
table-sm コンパクト表示 セルのパディングが半分になる
table-borderless ボーダーなし すべてのボーダーが消える
HTML
<!-- 縞模様 + ホバー + ボーダー の組み合わせ -->
<table class="table table-striped table-hover table-bordered">
  <thead>
    <tr>
      <th>商品名</th>
      <th>価格</th>
      <th>在庫</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>ノートPC</td>
      <td>¥98,000</td>
      <td>15</td>
    </tr>
    <tr>
      <td>マウス</td>
      <td>¥2,500</td>
      <td>120</td>
    </tr>
    <tr>
      <td>キーボード</td>
      <td>¥5,800</td>
      <td>45</td>
    </tr>
  </tbody>
</table>

クラスの組み合わせ

テーブルのバリエーションクラスは自由に組み合わせられます。table table-striped table-hover のように複数のクラスを同時に指定できます。最もよく使われる組み合わせは table-striped table-hover です。

レスポンシブテーブル

テーブルの列数が多い場合、スマートフォンでは横幅が足りなくなります。table-responsive クラスを付けた <div> でテーブルを囲むと、横スクロールが可能になります。

HTML
<div class="table-responsive">
  <table class="table">
    <thead>
      <tr>
        <th>ID</th>
        <th>商品名</th>
        <th>カテゴリ</th>
        <th>価格</th>
        <th>在庫数</th>
        <th>評価</th>
        <th>登録日</th>
        <th>操作</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>001</td>
        <td>ワイヤレスイヤホン</td>
        <td>オーディオ</td>
        <td>¥12,800</td>
        <td>50</td>
        <td>4.5</td>
        <td>2025/01/10</td>
        <td><a href="#">編集</a></td>
      </tr>
    </tbody>
  </table>
</div>

ブレイクポイント付きの table-responsive-{breakpoint} も使えます。例えば table-responsive-md は、md（768px）未満のときだけ横スクロールが有効になります。

セルの色分けとヘッダーのスタイル

特定のセルや行に色を付けたい場合は、table-{color} クラスを使います。また、ヘッダー行全体のスタイルを変更することもできます。

HTML
<table class="table table-hover">
  <!-- ダークヘッダー -->
  <thead class="table-dark">
    <tr>
      <th>科目</th>
      <th>点数</th>

      <td>95</td>
      <td>A</td>
    </tr>
    <tr>
      <td>英語</td>
      <td>78</td>
      <td>B</td>
    </tr>
    <tr class="table-danger">
      <td>国語</td>
      <td>42</td>
      <td>D</td>
    </tr>
    <tr class="table-warning">
      <td>理科</td>

クラス 適用先 効果
table-dark <thead> / <tr> 背景を黒、テキストを白にする
table-success <tr> / <td> 背景を薄い緑色にする
table-danger <tr> / <td> 背景を薄い赤色にする
table-warning <tr> / <td> 背景を薄い黄色にする
table-info <tr> / <td> 背景を薄い水色にする
table-primary <tr> / <td> 背景を薄い青色にする

テーブル内でバッジを活用

Section 5-4

プログレスバーとスピナー

プログレスバーの基本

プログレスバー（Progress Bar）は、処理の進行状況やスキルレベルなどを視覚的に表示するコンポーネントです。外側の progress と内側の progress-bar の2つの要素で構成されます。

HTML
<!-- 基本のプログレスバー -->
<div class="progress">
  <div class="progress-bar" role="progressbar" style="width: 75%"
       aria-valuenow="75" aria-valuemin="0" aria-valuemax="100">
    75%
  </div>
</div>
要素・属性 役割
progress 外側のコンテナ（背景のグレー部分）
progress-bar 内側のバー（色が付いた部分）
style="width: 75%" 進行度を幅で設定
aria-valuenow 現在の値（アクセシビリティ用）
aria-valuemin 最小値（通常0）
aria-valuemax 最大値（通常100）
HTML
<!-- さまざまな進行度 -->
<div class="progress mb-3">
  <div class="progress-bar" style="width: 25%">25%</div>
</div>
<div class="progress mb-3">
  <div class="progress-bar" style="width: 50%">50%</div>
</div>
<div class="progress mb-3">
  <div class="progress-bar" style="width: 75%">75%</div>
</div>
<div class="progress">
  <div class="progress-bar" style="width: 100%">100%</div>
</div>

プログレスバーのカラーバリエーション

プログレスバーの色は bg-{color} クラスで変更できます。

HTML
<div class="progress mb-3">
  <div class="progress-bar bg-success" style="width: 90%">HTML</div>
</div>
<div class="progress mb-3">
  <div class="progress-bar bg-info" style="width: 75%">CSS</div>
</div>
<div class="progress mb-3">
  <div class="progress-bar bg-warning" style="width: 50%">JavaScript</div>
</div>
<div class="progress">
  <div class="progress-bar bg-danger" style="width: 30%">React</div>
</div>

ストライプとアニメーション

progress-bar-striped でストライプ模様を追加し、progress-bar-animated でアニメーションを有効にできます。ファイルのアップロード中など、処理が進行中であることを示すのに効果的です。

HTML
<!-- ストライプ -->
<div class="progress mb-3">
  <div class="progress-bar progress-bar-striped" style="width: 60%">
    ストライプ
  </div>
</div>

<!-- ストライプ + アニメーション -->
<div class="progress mb-3">
  <div class="progress-bar progress-bar-striped progress-bar-animated" style="width: 80%">
    アップロード中...
  </div>
</div>

複数のバーを1つのプログレスに

1つの progress の中に複数の progress-bar を入れることで、積み上げ型のプログレスバーを作れます。

HTML
<!-- ストレージ使用量の表示例 -->
<p>ストレージ使用量</p>
<div class="progress">
  <div class="progress-bar bg-primary" style="width: 35%">写真 35%</div>
  <div class="progress-bar bg-success" style="width: 20%">動画 20%</div>
  <div class="progress-bar bg-warning" style="width: 15%">書類 15%</div>
</div>

スピナー

スピナー（Spinner）は、データの読み込み中やボタンの処理中であることを示す回転アニメーションです。2つのスタイルがあります。

クラス スタイル 説明
spinner-border ボーダー回転型 円形のボーダーが回転するスタイル
spinner-grow 拡大縮小型 点が拡大・縮小を繰り返すスタイル
HTML
<!-- ボーダー回転型スピナー -->
<div class="spinner-border" role="status">
  <span class="visually-hidden">読み込み中...</span>
</div>

<!-- 拡大縮小型スピナー -->
<div class="spinner-grow" role="status">
  <span class="visually-hidden">読み込み中...</span>
</div>

<!-- 色の変更 -->
<div class="spinner-border text-primary" role="status">
  <span class="visually-hidden">読み込み中...</span>
</div>
<div class="spinner-border text-success" role="status">
  <span class="visually-hidden">読み込み中...</span>
</div>
<div class="spinner-border text-danger" role="status">
  <span class="visually-hidden">読み込み中...</span>
</div>

<!-- 小さいスピナー -->
<div class="spinner-border spinner-border-sm" role="status">
  <span class="visually-hidden">読み込み中...</span>
</div>

visually-hiddenクラス

visually-hidden クラスは画面上には見えませんが、スクリーンリーダーには読み上げられます。スピナー自体には「読み込み中」という情報がないため、アクセシビリティのために必ずこのテキストを入れましょう。

実践例：ローディングボタン

ボタンの中にスピナーを入れて「処理中」の状態を表現します。これはフォーム送信時によく使われるパターンです。

HTML
<!-- 通常のボタン -->
<button type="button" class="btn btn-primary">
  送信する
</button>

<!-- ローディング中のボタン -->
<button type="button" class="btn btn-primary" disabled>
  <span class="spinner-border spinner-border-sm" role="status" aria-hidden="true"></span>
  送信中...
</button>

<!-- スピナーのみ（テキストなし） -->
<button type="button" class="btn btn-primary" disabled>
  <span class="spinner-border spinner-border-sm" role="status" aria-hidden="true"></span>
  <span class="visually-hidden">読み込み中...</span>
</button>

ボタンのdisabled属性

処理中のボタンには必ず disabled 属性を付けましょう。ユーザーが二重クリック（ダブルクリック）して同じ処理が重複実行されるのを防ぎます。
