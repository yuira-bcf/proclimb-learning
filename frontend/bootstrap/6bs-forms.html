Introduction

フォームはWebの基本

フォームはWebアプリケーションの中で最も重要なUI要素の一つです。ログイン画面、会員登録、お問い合わせ、検索ボックスなど、ユーザーからの入力を受け付けるあらゆる場面で使われます。

Bootstrapを使えば、複雑なCSSを書くことなく、美しく使いやすいフォームを簡単に作成できます。この章では以下の内容を学びます。

  • フォームコントロールの基本 — テキスト入力、テキストエリアの基本クラス
  • セレクト・チェックボックス・ラジオ — 選択系入力の美しい表示
  • フォームレイアウト — グリッドを使った水平フォーム、入力グループ
  • バリデーション表示 — 入力チェック結果の表示方法
Section 6-1

フォームコントロールの基本

form-controlクラス

Bootstrapのフォーム要素の中心となるのが form-control クラスです。<input>、<textarea>、<select> に付けることで、統一されたデザインのフォーム入力欄になります。

HTML
<div class="mb-3">
  <label for="userName" class="form-label">お名前</label>
  <input type="text" class="form-control" id="userName" placeholder="山田太郎">
</div>

<div class="mb-3">
  <label for="userEmail" class="form-label">メールアドレス</label>
  <input type="email" class="form-control" id="userEmail" placeholder="example@email.com">
</div>

<div class="mb-3">
  <label for="userPassword" class="form-label">パスワード</label>
  <input type="password" class="form-control" id="userPassword">
</div>
クラス 適用先 役割
form-control <input> / <textarea> 統一されたスタイルの入力欄にする
form-label <label> ラベルに適切な余白と太さを設定
mb-3 フォームグループを囲む <div> 下方向にマージンを追加して間隔を確保

labelとinputの紐付け

<label> の for 属性と <input> の id 属性を一致させましょう。これにより、ラベルをクリックすると入力欄にフォーカスが移ります。アクセシビリティの面でも重要です。

テキストエリア

複数行のテキスト入力には <textarea> を使います。同じく form-control クラスを付けるだけです。

HTML
<div class="mb-3">
  <label for="userMessage" class="form-label">お問い合わせ内容</label>
  <textarea class="form-control" id="userMessage" rows="5" placeholder="メッセージを入力してください"></textarea>
</div>

rows 属性で表示行数を指定できます。ユーザーが入力しやすい行数に設定しましょう。

フォームコントロールのサイズ

入力欄のサイズは form-control-lg（大）と form-control-sm（小）で変更できます。

HTML
<!-- 大きいサイズ -->
<input type="text" class="form-control form-control-lg mb-3" placeholder="大きい入力欄">

<!-- 通常サイズ -->
<input type="text" class="form-control mb-3" placeholder="通常の入力欄">

<!-- 小さいサイズ -->
<input type="text" class="form-control form-control-sm" placeholder="小さい入力欄">
クラス サイズ 使いどころ
form-control-lg 大 ヒーローセクションの検索ボックス、ログインフォームなど
（なし） 通常 一般的なフォーム
form-control-sm 小 テーブル内のフィルター、コンパクトなフォーム

読み取り専用と無効状態

編集できない入力欄を作るには、readonly 属性または disabled 属性を使います。

HTML
<!-- 読み取り専用（コピーは可能） -->
<div class="mb-3">
  <label for="readonlyInput" class="form-label">会員番号</label>
  <input type="text" class="form-control" id="readonlyInput" value="M-20250001" readonly>
</div>

<!-- 無効状態（操作不可） -->
<div class="mb-3">
  <label for="disabledInput" class="form-label">メールアドレス（変更不可）</label>
  <input type="email" class="form-control" id="disabledInput" value="user@example.com" disabled>
</div>

<!-- 読み取り専用のプレーンテキスト表示 -->
<div class="mb-3">
  <label for="plainText" class="form-label">登録メール</label>
  <input type="text" class="form-control-plaintext" id="plainText" value="user@example.com" readonly>
</div>
属性 見た目 コピー フォーム送信
readonly 通常のスタイルを維持 可能 値が送信される
disabled グレーアウトされる 不可 値が送信されない

ヘルプテキスト（form-text）

入力欄の下に補足説明を表示するには form-text クラスを使います。

HTML
<div class="mb-3">
  <label for="newPassword" class="form-label">新しいパスワード</label>
  <input type="password" class="form-control" id="newPassword" aria-describedby="passwordHelp">
  <div id="passwordHelp" class="form-text">
    パスワードは8文字以上で、英数字と記号を含めてください。
  </div>
</div>

aria-describedby属性

aria-describedby 属性で入力欄とヘルプテキストを紐付けることで、スクリーンリーダーがヘルプテキストを読み上げるようになります。アクセシビリティを意識した実装を心がけましょう。

Section 6-2

セレクト・チェックボックス・ラジオ

セレクトボックス（form-select）

ドロップダウン型の選択肢には form-select クラスを使います。通常の <select> タグにこのクラスを追加するだけで、美しいデザインになります。

HTML
<div class="mb-3">
  <label for="prefecture" class="form-label">都道府県</label>
  <select class="form-select" id="prefecture">
    <option selected>選択してください</option>
    <option value="tokyo">東京都</option>
    <option value="osaka">大阪府</option>
    <option value="aichi">愛知県</option>
    <option value="fukuoka">福岡県</option>
  </select>
</div>

サイズ変更は form-select-lg と form-select-sm で行えます。

HTML
<!-- 大きいセレクト -->
<select class="form-select form-select-lg mb-3">
  <option selected>大きいセレクトボックス</option>
</select>

<!-- 小さいセレクト -->
<select class="form-select form-select-sm">
  <option selected>小さいセレクトボックス</option>
</select>

チェックボックスとラジオボタン

チェックボックスとラジオボタンは、form-check で囲み、入力要素に form-check-input、ラベルに form-check-label を付けます。

HTML
<!-- チェックボックス -->
<h5>興味のある分野（複数選択可）</h5>
<div class="form-check">
  <input class="form-check-input" type="checkbox" id="checkHTML" value="html">
  <label class="form-check-label" for="checkHTML">HTML / CSS</label>
</div>
<div class="form-check">
  <input class="form-check-input" type="checkbox" id="checkJS" value="js">
  <label class="form-check-label" for="checkJS">JavaScript</label>
</div>
<div class="form-check">
  <input class="form-check-input" type="checkbox" id="checkJava" value="java" checked>
  <label class="form-check-label" for="checkJava">Java</label>
</div>

<!-- ラジオボタン -->
<h5 class="mt-4">経験レベル（1つ選択）</h5>
<div class="form-check">
  <input class="form-check-input" type="radio" name="level" id="levelBeginner" value="beginner" checked>
  <label class="form-check-label" for="levelBeginner">初心者</label>
</div>
<div class="form-check">
  <input class="form-check-input" type="radio" name="level" id="levelIntermediate" value="intermediate">
  <label class="form-check-label" for="levelIntermediate">中級者</label>
</div>
<div class="form-check">
  <input class="form-check-input" type="radio" name="level" id="levelAdvanced" value="advanced">
  <label class="form-check-label" for="levelAdvanced">上級者</label>
</div>

ラジオボタンのname属性

ラジオボタンでは、同じグループのすべてのボタンに同じ name 属性を設定します。これにより「1つだけ選択」の動作が実現されます。チェックボックスにはこの制約はありません。

スイッチトグル（form-switch）

ON/OFFの切り替えをスイッチスタイルで表示するには、form-check に form-switch クラスを追加します。

HTML
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" id="switchNotification" checked>
  <label class="form-check-label" for="switchNotification">通知を受け取る</label>
</div>
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" id="switchDarkMode">
  <label class="form-check-label" for="switchDarkMode">ダークモード</label>
</div>
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" id="switchNewsletter">
  <label class="form-check-label" for="switchNewsletter">ニュースレターを購読</label>
</div>

インラインチェックボックス・ラジオ

チェックボックスやラジオボタンを横並びにしたい場合は form-check-inline クラスを追加します。

HTML
<h5>性別</h5>
<div class="form-check form-check-inline">
  <input class="form-check-input" type="radio" name="gender" id="genderMale" value="male">
  <label class="form-check-label" for="genderMale">男性</label>
</div>
<div class="form-check form-check-inline">
  <input class="form-check-input" type="radio" name="gender" id="genderFemale" value="female">
  <label class="form-check-label" for="genderFemale">女性</label>
</div>
<div class="form-check form-check-inline">
  <input class="form-check-input" type="radio" name="gender" id="genderOther" value="other">
  <label class="form-check-label" for="genderOther">その他</label>
</div>
Section 6-3

フォームレイアウト

グリッドを使った水平フォーム

ラベルと入力欄を横並びにする「水平フォーム」は、Bootstrapのグリッドシステム（row/col）を使って実現します。

HTML
<form>
  <div class="row mb-3">
    <label for="hName" class="col-sm-3 col-form-label">お名前</label>
    <div class="col-sm-9">
      <input type="text" class="form-control" id="hName">
    </div>
  </div>

  <div class="row mb-3">
    <label for="hEmail" class="col-sm-3 col-form-label">メールアドレス</label>
    <div class="col-sm-9">
      <input type="email" class="form-control" id="hEmail">
    </div>
  </div>

  <div class="row mb-3">
    <label for="hPhone" class="col-sm-3 col-form-label">電話番号</label>
    <div class="col-sm-9">
      <input type="tel" class="form-control" id="hPhone">
    </div>
  </div>

  <div class="row">
    <div class="col-sm-9 offset-sm-3">
      <button type="submit" class="btn btn-primary">送信</button>
    </div>
  </div>
</form>
クラス 役割
row グリッドの行を作成
col-sm-3 sm以上で3カラム幅（ラベル用）
col-sm-9 sm以上で9カラム幅（入力欄用）
col-form-label ラベルの垂直位置を入力欄に合わせる
offset-sm-3 左側に3カラム分の余白を作る

col-form-labelを忘れずに

水平フォームのラベルには col-form-label クラスを付けましょう。これにより、ラベルのテキストが入力欄の高さと揃い、見た目がきれいになります。

入力グループ（input-group）

入力欄の前後にテキストやアイコン、ボタンを付加するには input-group を使います。メールアドレスの@マークや、通貨記号の表示によく使われます。

HTML
<!-- 前にテキストを付加 -->
<div class="input-group mb-3">
  <span class="input-group-text">@</span>
  <input type="text" class="form-control" placeholder="ユーザー名">
</div>

<!-- 後ろにテキストを付加 -->
<div class="input-group mb-3">
  <input type="text" class="form-control" placeholder="メールアドレス">
  <span class="input-group-text">@example.com</span>
</div>

<!-- 前後にテキストを付加 -->
<div class="input-group mb-3">
  <span class="input-group-text">¥</span>
  <input type="number" class="form-control" placeholder="金額">
  <span class="input-group-text">円</span>
</div>

<!-- ボタン付き入力グループ -->
<div class="input-group mb-3">
  <input type="text" class="form-control" placeholder="検索ワード">
  <button class="btn btn-primary" type="button">検索</button>
</div>

フローティングラベル（form-floating）

フローティングラベルは、入力欄にフォーカスしたり値を入力したりすると、ラベルが小さくなって上部に移動するモダンなデザインパターンです。

HTML
<div class="form-floating mb-3">
  <input type="text" class="form-control" id="floatingName" placeholder="お名前">
  <label for="floatingName">お名前</label>
</div>

<div class="form-floating mb-3">
  <input type="email" class="form-control" id="floatingEmail" placeholder="メールアドレス">
  <label for="floatingEmail">メールアドレス</label>
</div>

<div class="form-floating mb-3">
  <textarea class="form-control" id="floatingMessage" placeholder="メッセージ" style="height: 120px"></textarea>
  <label for="floatingMessage">メッセージ</label>
</div>

<div class="form-floating mb-3">
  <select class="form-select" id="floatingSelect">
    <option selected>選択してください</option>
    <option value="1">お問い合わせ</option>
    <option value="2">ご意見・ご要望</option>
    <option value="3">その他</option>
  </select>
  <label for="floatingSelect">カテゴリ</label>
</div>

フローティングラベルの注意点

フローティングラベルでは、inputの後にlabelを書く必要があります（通常とは逆順）。また、placeholder 属性が必須です（CSS のセレクタで使用されるため）。placeholderの値はラベルと同じテキストにするのが一般的です。

実践例：お問い合わせフォーム

ここまで学んだ内容を組み合わせて、実用的なお問い合わせフォームを作ってみましょう。

HTML
<form>
  <!-- 名前（2列） -->
  <div class="row mb-3">
    <div class="col-md-6">
      <label for="contactLastName" class="form-label">姓</label>
      <input type="text" class="form-control" id="contactLastName" placeholder="山田">
    </div>
    <div class="col-md-6">
      <label for="contactFirstName" class="form-label">名</label>
      <input type="text" class="form-control" id="contactFirstName" placeholder="太郎">
    </div>
  </div>

  <!-- メールアドレス -->
  <div class="mb-3">
    <label for="contactEmail" class="form-label">メールアドレス</label>
    <input type="email" class="form-control" id="contactEmail" placeholder="example@email.com">
  </div>

  <!-- 電話番号 -->
  <div class="mb-3">
    <label for="contactPhone" class="form-label">電話番号（任意）</label>
    <input type="tel" class="form-control" id="contactPhone" placeholder="090-1234-5678">
  </div>

  <!-- お問い合わせカテゴリ -->
  <div class="mb-3">
    <label for="contactCategory" class="form-label">お問い合わせ種別</label>
    <select class="form-select" id="contactCategory">
      <option selected>選択してください</option>
      <option value="general">一般的なお問い合わせ</option>
      <option value="support">技術サポート</option>
      <option value="billing">お支払いについて</option>
      <option value="other">その他</option>
    </select>
  </div>

  <!-- メッセージ -->
  <div class="mb-3">
    <label for="contactMessage" class="form-label">お問い合わせ内容</label>
    <textarea class="form-control" id="contactMessage" rows="5" placeholder="お問い合わせ内容を入力してください"></textarea>
  </div>

  <!-- 同意チェック -->
  <div class="form-check mb-3">
    <input class="form-check-input" type="checkbox" id="contactAgree">
    <label class="form-check-label" for="contactAgree">
      <a href="#">プライバシーポリシー</a>に同意します
    </label>
  </div>

  <!-- 送信ボタン -->
  <button type="submit" class="btn btn-primary">送信する</button>
</form>
Section 6-4

バリデーション表示

バリデーションとは

バリデーション（Validation）とは、ユーザーが入力した値が正しいかどうかを検証することです。例えば「メールアドレスの形式が正しいか」「必須項目が入力されているか」などのチェックを行い、結果をユーザーにフィードバックします。

Bootstrapでは、入力の成功/失敗に応じたスタイルとメッセージを簡単に表示できます。

is-valid と is-invalid

フォームコントロールに is-valid クラスを付けると緑色のスタイルが、is-invalid クラスを付けると赤色のスタイルが適用されます。

HTML
<!-- 入力が正しい場合 -->
<div class="mb-3">
  <label for="validName" class="form-label">お名前</label>
  <input type="text" class="form-control is-valid" id="validName" value="山田太郎">
  <div class="valid-feedback">
    入力OKです！
  </div>
</div>

<!-- 入力にエラーがある場合 -->
<div class="mb-3">
  <label for="invalidEmail" class="form-label">メールアドレス</label>
  <input type="email" class="form-control is-invalid" id="invalidEmail" value="abcdef">
  <div class="invalid-feedback">
    正しいメールアドレスの形式で入力してください。
  </div>
</div>
クラス 見た目 用途
is-valid 緑色のボーダー + チェックアイコン 入力が正しいことを示す
is-invalid 赤色のボーダー + ×アイコン 入力にエラーがあることを示す
valid-feedback 緑色のテキスト 成功メッセージを表示
invalid-feedback 赤色のテキスト エラーメッセージを表示

feedbackの表示条件

valid-feedback と invalid-feedback は、対応する入力欄に is-valid / is-invalid クラスが付いているときだけ表示されます。クラスがない場合は自動的に非表示になります。

セレクト・チェックボックスのバリデーション

バリデーションスタイルはセレクトボックスやチェックボックスにも適用できます。

HTML
<!-- セレクトボックスのバリデーション -->
<div class="mb-3">
  <label for="validSelect" class="form-label">カテゴリ</label>
  <select class="form-select is-valid" id="validSelect">
    <option selected value="web">Webデザイン</option>
  </select>
  <div class="valid-feedback">選択されています。</div>
</div>

<!-- チェックボックスのバリデーション -->
<div class="form-check mb-3">
  <input class="form-check-input is-invalid" type="checkbox" id="agreeCheck">
  <label class="form-check-label" for="agreeCheck">利用規約に同意します</label>
  <div class="invalid-feedback">同意が必要です。</div>
</div>

was-validatedによる一括バリデーション

フォーム送信時に一括でバリデーションを表示するには、<form> タグに was-validated クラスを追加します。HTML5のバリデーション属性（required など）と連動して、自動的にスタイルが適用されます。

HTML
<form class="was-validated">
  <div class="mb-3">
    <label for="wvName" class="form-label">お名前（必須）</label>
    <input type="text" class="form-control" id="wvName" required>
    <div class="invalid-feedback">お名前を入力してください。</div>
  </div>

  <div class="mb-3">
    <label for="wvEmail" class="form-label">メールアドレス（必須）</label>
    <input type="email" class="form-control" id="wvEmail" required>
    <div class="invalid-feedback">正しいメールアドレスを入力してください。</div>
  </div>

  <div class="mb-3">
    <label for="wvMessage" class="form-label">メッセージ（必須）</label>
    <textarea class="form-control" id="wvMessage" rows="3" required></textarea>
    <div class="invalid-feedback">メッセージを入力してください。</div>
  </div>

  <div class="form-check mb-3">
    <input class="form-check-input" type="checkbox" id="wvAgree" required>
    <label class="form-check-label" for="wvAgree">利用規約に同意します</label>
    <div class="invalid-feedback">続行するには同意が必要です。</div>
  </div>

  <button type="submit" class="btn btn-primary">送信する</button>
</form>

was-validatedの仕組み

was-validated クラスがフォームに付いていると、各入力欄の :valid / :invalid 擬似クラス（HTML5バリデーション結果）に応じて、Bootstrapのスタイルが自動適用されます。JavaScriptでフォーム送信時にこのクラスを追加するのが一般的なパターンです。

実践例：JavaScriptと連携したバリデーション

実際のWebサイトでは、JavaScriptでフォーム送信を制御し、バリデーションが通った場合にのみ送信する実装が一般的です。

HTML
<form id="registrationForm" novalidate>
  <div class="mb-3">
    <label for="regName" class="form-label">お名前</label>
    <input type="text" class="form-control" id="regName" required>
    <div class="valid-feedback">OK！</div>
    <div class="invalid-feedback">お名前を入力してください。</div>
  </div>

  <div class="mb-3">
    <label for="regEmail" class="form-label">メールアドレス</label>
    <input type="email" class="form-control" id="regEmail" required>
    <div class="valid-feedback">OK！</div>
    <div class="invalid-feedback">正しいメールアドレスを入力してください。</div>
  </div>

  <div class="mb-3">
    <label for="regPassword" class="form-label">パスワード</label>
    <input type="password" class="form-control" id="regPassword" minlength="8" required>
    <div class="valid-feedback">OK！</div>
    <div class="invalid-feedback">パスワードは8文字以上で入力してください。</div>
  </div>

  <button type="submit" class="btn btn-primary">登録する</button>

// フォーム送信時のバリデーション処理
const form = document.getElementById('registrationForm');

form.addEventListener('submit', function(event) {
  // HTML5のデフォルト送信を止める
  event.preventDefault();
  event.stopPropagation();

  // was-validatedクラスを追加してスタイルを適用
  form.classList.add('was-validated');

  // バリデーションが通っていれば送信処理
  if (form.checkValidity()) {
    alert('登録が完了しました！');
    // 実際にはここでサーバーへデータを送信
  }
});

novalidate属性

<form> に novalidate 属性を付けると、ブラウザ標準のバリデーション表示（吹き出し）が無効になります。代わりにBootstrapのスタイルでバリデーション結果を表示するため、このパターンでは必ず novalidate を追加しましょう。
