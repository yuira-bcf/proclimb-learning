Introduction

ナビゲーションバーの重要性

ナビゲーションバー（Navbar）は、Webサイトの「顔」とも言える最も重要なUI要素です。サイトのすべてのページに表示され、ユーザーがサイト内を移動するための道案内を担います。

Bootstrapのナビゲーションバーは、次のような機能をクラス名だけで実現できます。

  • レスポンシブ対応 — 画面幅に応じて自動でハンバーガーメニューに切り替わる
  • 配色バリエーション — ダーク、ライト、カスタムカラーの配色
  • 固定表示 — スクロールしても画面上部に固定
  • ドロップダウンメニュー — 複数階層のナビゲーション

この章では、基本構造から実践的なナビゲーションバーの構築まで、順を追って学んでいきます。

Section 7-1

Navbarの基本構造

最小構成のNavbar

BootstrapのNavbarは、いくつかのクラスを組み合わせて構築します。まずは最もシンプルな構成を見てみましょう。

HTML
<nav class="navbar navbar-expand-lg bg-body-tertiary">
  <div class="container">
    <a class="navbar-brand" href="#">MySite</a>

    <div class="collapse navbar-collapse">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">ホーム</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">サービス</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">料金</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">お問い合わせ</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
クラス 要素 役割
navbar <nav> ナビゲーションバーの基本スタイルを適用
navbar-expand-lg <nav> lg（992px）以上でリンクを横並び表示
bg-body-tertiary <nav> 背景色を設定（薄いグレー）
container <div> コンテンツの最大幅を制限し中央寄せ
navbar-brand <a> サイト名やロゴを表示する領域
navbar-collapse <div> レスポンシブ時に折りたたまれる領域
navbar-nav <ul> ナビゲーションリンクのリスト
nav-item <li> 個々のナビゲーション項目
nav-link <a> ナビゲーションリンクのスタイル

navbar-brandについて

navbar-brand はサイトのロゴやサイト名を表示するために使います。テキストだけでなく、<img> タグを入れてロゴ画像を表示することもできます。通常、サイトのトップページにリンクします。

activeなリンク

現在表示中のページのリンクには active クラスと aria-current="page" 属性を追加します。これにより、ユーザーが今どのページにいるか分かりやすくなります。

HTML
<!-- 現在のページ（太字で強調される） -->
<li class="nav-item">
  <a class="nav-link active" aria-current="page" href="#">ホーム</a>
</li>

<!-- 通常のリンク -->
<li class="nav-item">
  <a class="nav-link" href="#">サービス</a>
</li>

<!-- 無効なリンク -->
<li class="nav-item">
  <a class="nav-link disabled" aria-disabled="true">準備中</a>
</li>

navbar-expand のブレイクポイント

navbar-expand-{breakpoint} は、「どの画面幅からリンクを横並びにするか」を決めるクラスです。指定したブレイクポイント未満では、ナビゲーションリンクが折りたたまれます。

クラス 横並び開始幅 使いどころ
navbar-expand-sm 576px以上 リンクが少ない場合
navbar-expand-md 768px以上 タブレットサイズ以上で展開
navbar-expand-lg 992px以上 最も一般的（推奨）
navbar-expand-xl 1200px以上 リンクが多い場合
navbar-expand 常に横並び 折りたたみ不要の場合

一番よく使うのはlg

特に理由がなければ navbar-expand-lg を使いましょう。PC画面ではリンクが横並びに、タブレット以下ではハンバーガーメニューになる、最もバランスの良い設定です。

Section 7-2

ハンバーガーメニュー（レスポンシブ対応）

ハンバーガーメニューとは

ハンバーガーメニューとは、スマートフォンなどの狭い画面で、ナビゲーションリンクを三本線のアイコン（ハンバーガーアイコン）に折りたたむデザインパターンです。アイコンをタップするとメニューが展開されます。

Bootstrapでは navbar-toggler ボタンと collapse クラスの組み合わせで実現します。

HTML
<nav class="navbar navbar-expand-lg bg-body-tertiary">
  <div class="container">
    <!-- サイト名（常に表示） -->
    <a class="navbar-brand" href="#">MySite</a>

    <!-- ハンバーガーボタン（lg未満で表示） -->
    <button class="navbar-toggler" type="button"
            data-bs-toggle="collapse"
            data-bs-target="#navbarContent"
            aria-controls="navbarContent"
            aria-expanded="false"
            aria-label="メニューを開く">
      <span class="navbar-toggler-icon"></span>
    </button>

    <!-- 折りたたみ部分（lg以上で展開） -->
    <div class="collapse navbar-collapse" id="navbarContent">
      <ul class="navbar-nav me-auto mb-2 mb-lg-0">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">ホーム</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">サービス</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">料金</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">お問い合わせ</a>
        </li>
      </ul>
    </div>
  </div>
</nav>

トグルの仕組み

ハンバーガーメニューの開閉は、BootstrapのJavaScriptが自動で処理します。その仕組みは data-bs-* 属性で制御されています。

属性 値 役割
data-bs-toggle "collapse" クリック時にcollapseの開閉を行う
data-bs-target "#navbarContent" 開閉する要素をIDで指定
aria-controls "navbarContent" アクセシビリティ：制御対象の要素
aria-expanded "false" アクセシビリティ：現在の展開状態

IDの紐付けが重要

ハンバーガーボタンの data-bs-target="#navbarContent" と、折りたたみ部分の id="navbarContent" が一致していなければ動作しません。IDの値は自由ですが、必ず一致させましょう。

me-autoでリンクを左寄せに

上のコード例で使われている me-auto クラスは、「右側に自動マージンを追加する」という意味です。これにより、ナビゲーションリンクが左寄せになり、残りのスペースが右側に空きます。右側に検索フォームやボタンを配置したい場合に便利です。

HTML
<!-- me-auto: リンクを左寄せ -->
<ul class="navbar-nav me-auto">
  <!-- リンクが左に寄る -->
</ul>

<!-- ms-auto: リンクを右寄せ -->
<ul class="navbar-nav ms-auto">
  <!-- リンクが右に寄る -->
</ul>

mb-2 mb-lg-0 の意味

mb-2 mb-lg-0 は「通常は下マージン2、lg以上では下マージン0」という意味のレスポンシブユーティリティです。モバイル表示時にナビリンクと検索フォームの間に間隔を作り、PC表示時は不要なのでマージンを0にしています。

Section 7-3

配色とスタイル

配色の組み合わせ

ナビゲーションバーの配色は、テキスト色と背景色の組み合わせで決まります。Bootstrap 5ではいくつかの方法があります。

HTML
<!-- ダーク配色（黒背景 + 白テキスト） -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <div class="container">
    <a class="navbar-brand" href="#">Dark Navbar</a>
    <!-- ... -->
  </div>
</nav>

<!-- ライト配色（白背景 + 黒テキスト） -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container">
    <a class="navbar-brand" href="#">Light Navbar</a>
    <!-- ... -->
  </div>
</nav>

<!-- ブランドカラー（プライマリ背景 + 白テキスト） -->
<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
  <div class="container">
    <a class="navbar-brand" href="#">Primary Navbar</a>
    <!-- ... -->
  </div>
</nav>
テキスト色 背景色 見た目
navbar-dark bg-dark 黒背景に白テキスト
navbar-light bg-light 白背景に黒テキスト
navbar-dark bg-primary 青背景に白テキスト
navbar-dark bg-success 緑背景に白テキスト
navbar-dark bg-danger 赤背景に白テキスト

navbar-darkとnavbar-lightの使い分け

背景色が暗い場合は navbar-dark（テキストとアイコンが白）を、背景色が明るい場合は navbar-light（テキストとアイコンが黒）を使います。背景色とテキスト色のコントラストを確保することが大切です。

固定配置（fixed-top / sticky-top）

ナビゲーションバーをスクロールしても画面上部に固定表示するには、fixed-top または sticky-top クラスを使います。

HTML
<!-- ページ上部に固定（常に画面上部に表示） -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark fixed-top">
  <!-- ... -->
</nav>

<!-- スクロール時に固定（スクロール位置まで来たら固定） -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark sticky-top">
  <!-- ... -->
</nav>
クラス 動作 注意点
fixed-top 常に画面上部に固定 bodyに上パディングが必要
fixed-bottom 常に画面下部に固定 bodyに下パディングが必要
sticky-top スクロール到達後に固定 パディング調整は不要

fixed-topを使うときの重要な注意

fixed-top を使うと、ナビゲーションバーがページのフロー（通常の配置）から外れます。そのため、ナビバーの高さ分だけコンテンツが隠れてしまいます。<body> に上方向のパディングを追加しましょう。

CSS
/* fixed-topを使う場合、bodyに上パディングを追加 */
body {
  padding-top: 70px; /* ナビバーの高さに合わせて調整 */
}

シャドウの追加

ナビゲーションバーにシャドウ（影）を追加すると、コンテンツとの境界が分かりやすくなり、高級感のあるデザインになります。

HTML
<!-- シャドウ付きナビバー -->
<nav class="navbar navbar-expand-lg navbar-light bg-white shadow-sm">
  <div class="container">
    <a class="navbar-brand" href="#">MySite</a>
    <!-- ... -->
  </div>
</nav>

<!-- より強いシャドウ -->
<nav class="navbar navbar-expand-lg navbar-light bg-white shadow">
  <div class="container">
    <a class="navbar-brand" href="#">MySite</a>
    <!-- ... -->
  </div>
</nav>

shadow-sm（小さい影）、shadow（通常の影）、shadow-lg（大きい影）の3段階があります。ナビバーには shadow-sm が控えめでおすすめです。

Section 7-4

ドロップダウンと検索フォーム

ドロップダウンメニュー

ナビゲーションのリンクの中にサブメニュー（ドロップダウン）を含めることができます。リンクにマウスを乗せるか、クリックするとサブメニューが表示されます。

HTML
<li class="nav-item dropdown">
  <a class="nav-link dropdown-toggle" href="#" role="button"
     data-bs-toggle="dropdown" aria-expanded="false">
    サービス
  </a>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Webデザイン</a></li>
    <li><a class="dropdown-item" href="#">アプリ開発</a></li>
    <li><a class="dropdown-item" href="#">コンサルティング</a></li>
    <li><hr class="dropdown-divider"></li>
    <li><a class="dropdown-item" href="#">料金一覧</a></li>
  </ul>
</li>
クラス・属性 要素 役割
dropdown <li> ドロップダウンのコンテナ
dropdown-toggle <a> ▼マークを表示し、クリックで開閉
data-bs-toggle="dropdown" <a> JSでドロップダウンの動作を有効化
dropdown-menu <ul> サブメニューのコンテナ
dropdown-item <a> サブメニューの各リンク
dropdown-divider <hr> 区切り線

検索フォーム

ナビゲーションバーの右側に検索フォームを配置するのは、非常によくあるパターンです。d-flex クラスを使ってフォーム要素を横並びにします。

HTML
<!-- navbar-collapseの中の最後に配置 -->
<form class="d-flex" role="search">
  <input class="form-control me-2" type="search" placeholder="検索..." aria-label="検索">
  <button class="btn btn-outline-success" type="submit">検索</button>
</form>

me-2 は入力欄とボタンの間にスペースを作るためのマージンクラスです。d-flex で横並びにしているため、入力欄とボタンが横に並びます。

実践例：完成版ナビゲーションバー

ここまで学んだすべての要素を組み合わせた、実用的なナビゲーションバーを作ってみましょう。

HTML
<nav class="navbar navbar-expand-lg navbar-dark bg-dark shadow-sm">
  <div class="container">
    <!-- ブランド -->
    <a class="navbar-brand" href="#">
      TechCafe
    </a>

    <!-- ハンバーガーボタン -->
    <button class="navbar-toggler" type="button"
            data-bs-toggle="collapse"
            data-bs-target="#mainNavbar"
            aria-controls="mainNavbar"
            aria-expanded="false"
            aria-label="メニューを開く">
      <span class="navbar-toggler-icon"></span>
    </button>

    <!-- 折りたたみ部分 -->
    <div class="collapse navbar-collapse" id="mainNavbar">
      <!-- 左側リンク -->
      <ul class="navbar-nav me-auto mb-2 mb-lg-0">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">ホーム</a>
        </li>

        <!-- ドロップダウン -->
        <li class="nav-item dropdown">
          <a class="nav-link dropdown-toggle" href="#" role="button"
             data-bs-toggle="dropdown" aria-expanded="false">
            メニュー
          </a>
          <ul class="dropdown-menu">
            <li><a class="dropdown-item" href="#">ドリンク</a></li>
            <li><a class="dropdown-item" href="#">フード</a></li>
            <li><a class="dropdown-item" href="#">デザート</a></li>
            <li><hr class="dropdown-divider"></li>
            <li><a class="dropdown-item" href="#">季節限定メニュー</a></li>
          </ul>
        </li>

        <li class="nav-item">
          <a class="nav-link" href="#">店舗情報</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">お知らせ</a>
        </li>
      </ul>

      <!-- 右側：検索フォーム -->
      <form class="d-flex" role="search">
        <input class="form-control me-2" type="search" placeholder="メニューを検索..." aria-label="検索">
        <button class="btn btn-outline-light" type="submit">検索</button>
      </form>
    </div>
  </div>
</nav>

この実践例のポイント

この完成版ナビバーには次の機能が含まれています。(1) ブランド名の表示、(2) ハンバーガーメニューによるレスポンシブ対応、(3) ドロップダウンサブメニュー、(4) 右寄せの検索フォーム、(5) ダーク配色 + シャドウ。実際のWebサイト制作で使える、実用的な構成です。

ナビバーにボタンを配置

検索フォームの代わりに、ログインボタンやサインアップボタンを配置することもよくあります。

HTML
<!-- navbar-collapseの中の最後に配置 -->
<div class="d-flex gap-2">
  <a href="#" class="btn btn-outline-light">ログイン</a>
  <a href="#" class="btn btn-warning">新規登録</a>
</div>

gap-2 はFlexboxの子要素間にスペースを追加するクラスです。これにより、2つのボタンの間に適切な間隔が生まれます。
