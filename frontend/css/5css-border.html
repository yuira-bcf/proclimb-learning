Section 5-1

線を引くってどういうこと？

ボーダーは、要素のまわりに描かれる線のことです。
たとえばカードのフチを囲ったり、見出しを目立たせたり、区切り線を引いたりできます。

ボックスの構造（おさらい）

Section 5-2

線の太さを変えよう（border-width）

線の太さを変えるには border-width プロパティを使います。
たとえば「細い線」や「太い線」を指定できます。

CSS 
border-width: thin;  		/* 細い線 */
border-width: medium;  		/* 普通の線 */
border-width: thick;   		/* 太い線 */

上記のようにキーワードで指定もできますが、基本は数値＋単位で指定します。
複数の値を指定すると 上・右・下・左（時計回り） にそれぞれの太さを設定できます。

サンプル①：線の太さを比べよう

フォルダ構成

フォルダ構成 
Chapter5_2-2
 └── index.html
 └── style.css
HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="UTF-8" />
	<title>線の太さサンプル</title>
	<link rel="stylesheet" href="style.css" />
</head>
<body>
	<p class="thick">これは太い線です</p>
	<div class="mix">上1px・右4px・下8px・左12pxの線</div>
</body>
</html>
CSS 
body {
  font-family: sans-serif;
  background: #f6f8ff;
  padding: 20px;
}

p.thick {
  border-width: thick;
  border-style: solid;
  border-color: #4fa4ff;
  padding: 10px;
}

div.mix {
  border-width: 1px 4px 8px 12px; /* 上→右→下→左 */
  border-style: solid;
  border-color: #ff7a4f;
  padding: 10px;
  margin-top: 20px;
}

結果：1行目は"太い実線"、2行目は辺ごとに太さが違うカラフルボーダー！

Section 5-3

線のスタイルを変えよう（border-style）

線の種類を変えるには border-style を使います。
実線・点線・破線・二重線など、いろんな表情をつけられます。

CSS 
border-style: solid;   	/* 実線 */
border-style: dashed; 	/* 破線 */
border-style: dotted;  	/* 点線 */
border-style: double;  	/* 二重線 */

ほかにも groove / ridge / inset / outset などの立体的なスタイルもあります！

Section 5-4

線の色を変えよう（border-color）

線の色を変えるには border-color を使います。1色でも、辺ごとに違う色でもOKです！

CSS 
border-color: tomato;  				/* 名前で指定 */
border-color: #4fa4ff; 				/* 16進数カラー */
border-color: rgb(255, 180, 0); 			 /* RGB指定 */
border-color: #4fa4ff #ffb84f tomato #000;		/* 上→右→下→左 */

フォルダ構成

フォルダ構成

 └── index.html
 └── style.css

	<meta charset="UTF-8" />
	<title>線の色サンプル</title>

Section 5-5

一括で指定しよう（border）

実務で枠線のプロパティを指定するときは、ショートハンドと言われる記法を使います。
ショートハンドは値を連続で書くだけで、複数のプロパティを指定できる方法です。

ショートハンドの例

CSS 
border: 2px solid tomato;
　⇒ 線の太さが2px, 線の種類が直線, 線の色がtomato色

この順番で書くのが基本！
border: [太さ] [線の種類] [色];

フォルダ構成

フォルダ構成 
Chapter5_2-8
 └── index.html
 └── style.css
HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="UTF-8" />
	<title>一括指定のサンプル</title>
	<link rel="stylesheet" href="style.css" /></head>
<body>
	<p class="bottom">下だけ青い線</p>
	<div class="box">赤い点線ボーダー</div>
</body>
</html>
CSS 
body {
  font-family: sans-serif;
  background: #fffef9;
  padding: 20px;
}

.bottom {
  border-bottom: 2px solid #4fa4ff;
  padding: 10px;
}

.box {
  border: 5px dotted tomato;
  padding: 12px;
  margin-top: 20px;
}

結果：上の段落は「下だけ青い線」、下のボックスは「赤い点線ボーダー」。
