Introduction

Flexboxとは

ここまでは要素そのものに対するスタイルの当て方について学んできました。
その要素をどう配置するのか、が最後の決めるべき重要な点となります。

「Flexbox」とは Flexible Box Layout（フレキシブル・ボックス・レイアウト）の略で、
要素を箱（ボックス）に入れ、その中で配置を指定することで、柔軟に画面上に並べられるCSSの仕組みです。

これを使うことで、複数の要素を横並びにしたり、縦並びにしたり。
画面幅が変わったらそれに併せて折り返したりなど……
レスポンシブ設計には欠かせない概念となっています。

昔はこの横並びひとつ取っても、プロパティなどの設定が面倒くさかったのが、
いまでは基本は「display:flex」と書くだけで良くなりました。

Section 6-1

Flexboxの基本構造

Flexboxでは、親要素と子要素があって初めて機能します。

名前 意味
Flexコンテナ 親箱（全体を包む要素）
Flexアイテム 子箱（中に並ぶ要素）

HTMLで「親の中に子を入れる」形を作り、
CSSで「親に display: flex; をつける」と、子が"横並び"になります。

FlexBox適用前の例

フォルダ構成

フォルダ構成 
Chapter6_1-2
 └── index.html
 └── style.css
HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="UTF-8" />
	<title>Flexboxの基本</title>
	<link rel="stylesheet" href="style.css" />
</head>
<body>
	<div class="container">
	<div class="item">Item 1</div>
	<div class="item">Item 2</div>
	<div class="item">Item 3</div>
	<div class="item">Item 4</div>
	</div>
</body>
</html>
CSS 
.container {
  /* まだdisplay:flexはなし！ */

  background: #B7F7FF;
  margin: 10px;
  padding: 10px;
}
.item {
  background: #0bd;
  color: #fff;
  margin: 10px;
  padding: 10px;
}

現時点では、縦に積み重なって表示されます。

Section 6-2

横並び

.container に display: flex; をつけるだけで、子が横に並びます！

フォルダ構成

フォルダ構成 
Chapter6_1-3
 └── index.html
 └── style.css

▶index.html ※ 6_1-2 と同じ

CSS 
.container {
  display:flex;

  background: #B7F7FF;
  margin: 10px;
  padding: 10px;
}
.item {
  background: #0bd;
  color: #fff;
  margin: 10px;
  padding: 10px;
}

結果：これだけで、子要素がすべて横並びになります。

Section 6-3

並ぶ方向（flex-direction）

横並びにするだけでなく、さらに並ぶ方向なども指定することができます。
そのときにはflex-direction というプロパティを使います。

値 意味
row（初期値） 左→右に並ぶ
row-reverse 右→左に並ぶ
column 上→下に並ぶ
column-reverse 下→上に並ぶ

フォルダ構成

フォルダ構成 
Chapter6_1-4
 └── index.html
 └── style.css

▶index.html ※6_1-2 と同じ

CSS 
.container {
  display: flex;
  flex-direction: row-reverse; /* 右から左へ */

  background: #B7F7FF;
  margin: 10px;
  padding: 10px;
}
.item {
  background: #0bd;
  color: #fff;
  margin: 10px;
  padding: 10px;
}

結果：アイテムが右端から順に並びます。

Section 6-4

折り返しを設定しよう（flex-wrap）

要素の幅が決まっているとき、子要素が多いと親要素の幅を超えてしまう場合があります。
そんなときに便利なのがflex-wrap。これは親要素の幅を超えそうなとき、
自動的に折り返しをしてくれる機能です。

値 意味
nowrap 折り返さない（初期値）
wrap 折り返して下の行に続く
wrap-reverse 折り返して上の行に続く

フォルダ構成

フォルダ構成 
Chapter6_1-5
 └── index.html
 └── style.css

▶index.html ※6_1-2と同じ

CSS 
.container {
  display: flex;
  flex-wrap: wrap; /* 折り返しON！ */
}
.item {
  background: #0bd;
  color: #fff;
  margin: 10px;
  padding: 10px;
  width: 120px;
}

結果：ブラウザ幅を狭くすると、アイテムが折り返して2段に！

Section 6-5

水平方向の整列（justify-content）

通常だと、横並びの要素は左に詰まって並びます。
この並びの間隔を指定するのが、justify-contentというプロパティです。

値 意味
flex-start 左寄せ（初期値）
flex-end 右寄せ
center 中央寄せ
space-between 両端を空けて等間隔配置
space-around 要素のまわりを等間隔配置

フォルダ構成

フォルダ構成 
Chapter6_1-6
 └── index.html
 └── style.css

▶index.html ※6_1-2と同じ

CSS 
.container {
  display: flex;
  justify-content: space-between; /* 両端と間を均等配置 */

  background: #B7F7FF;
  margin: 10px;
  padding: 10px;
}
.item {
  background: #0bd;
  color: #fff;
  margin: 10px;
  padding: 10px;
  width: 120px;
}

結果：アイテムが両端にぴったり並び、間隔も等しく見えます！

値を変更して試してみよう

Section 6-6

垂直方向の整列（align-items）

親要素の高さに余裕があるとき、子要素を縦のどこの位置に配置するのかを指定できます。
それがalign-itemsプロパティです。

値 意味
stretch 高さを均等に伸ばす（初期値）
flex-start 上寄せ
flex-end 下寄せ
center 真ん中寄せ
baseline 文字のベースラインで揃える

フォルダ構成

フォルダ構成 
Chapter6_1-7
 └── index.html
 └── style.css

▶index.html ※6_1-2と同じ

CSS 
.container {
  display: flex;
  align-items: center; /* 真ん中寄せ */
  height: 100vh;       /* 画面の高さ分広げる */

  background: #B7F7FF;
  margin: 10px;
  padding: 10px;
}
.item {
  background: #0bd;
  color: #fff;
  margin: 10px;
  padding: 10px;
width: 120px;
}

結果：縦の中央にアイテムが並び、バランスのとれた美しい配置に。

値を変更して試してみよう

Section 6-7

複数行の整列（align-content）

横並びの時と同様に、縦並びのときも各要素の配置位置を指定できます。
その場合はalign-contentプロパティを指定します。
（※ flex-wrap: wrap; を指定しているときだけ有効）

値 意味
stretch 均等に広がる（初期値）
flex-start 上寄せ
flex-end 下寄せ
center 中央寄せ
space-between 上下端に配置して間を均等
space-around 各行の上下に等間隔

フォルダ構成

フォルダ構成 
Chapter6_1-8
 └── index.html
 └── style.css
CSS 
.container {
  display: flex;
  flex-wrap: wrap;
  align-content: space-between; /* 上下端＋等間隔配置 */
  height: 300px;

background: #B7F7FF;
  margin: 10px;
  padding: 10px;
}
.item {
  background: #0bd;
  color: #fff;
  margin: 10px;
  padding: 10px;
  width: 120px;
}

	</div>
</body>
</html>

まとめ

よく使う組み合わせ

用途 よく使うプロパティ例
横並びで中央に揃えたい display: flex; justify-content: center;
縦横のど真ん中に配置したい display: flex; justify-content: center; align-items: center;
行を複数段にしたい flex-wrap: wrap;
カードを均等に並べたい display:flex;justify-content: space-between; flex-wrap: wrap;
