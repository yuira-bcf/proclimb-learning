Section 3-1

要素の大きさや領域

要素には、自身が持つ領域があります。
その領域の中に、外余白(margin)、枠線(border)、内余白(padding)、要素が含まれています。
要素自体の幅と高さは「枠線」「内余白」「要素」で、大きさが決まっています。

そして、幅は「width(ウィドスまたはウィズス)」
高さは「height(ハイト)」と表現します。

これらをプロパティとして指定し、適切な単位を含んだ値を指定して、
大きさを調整します。

Section 3-2

Webで使う主な単位（ざっくり早見表）

単位 読み方 種別 どんな基準？ 例 特徴
% パーセント 相対 親要素のサイズ比 width: 60% 親の大きさが変われば、自分も比率に併せて伸縮
rem レム 相対 ルート（html）のフォントサイズ font-size: 1.25rem どこでも同じ基準＝扱いやすい◎
vw ビューポート幅の% 相対 ブラウザの画面幅基準 width: 50vw ブラウザの画面幅に応じて伸縮（横）
vh ビューポート高の% 相対 ブラウザの画面高さ基準 height: 50vh ブラウザの画面高さに応じて伸縮（縦）
px ピクセル 絶対 物理ピクセル（論理） border-radius: 8px 親や画面幅に左右されず、常に"固定"の大きさを維持する

まずは px（固定） と %（親に追従） を押さえればOK！
レスポンシブでは vw/vh も便利です。

Section 3-3

実務の指針（困ったらこれ）

用途 指針
テキストサイズ 基本は rem（例：本文 1rem = 16px）
コンテナ幅 PCは max-width: 960px など + width: 90% で余白確保
カードの余白 rem で文字1文字単位で余白を入れるとバランスが良い
全幅ヒーロー height: 60vh や min-height: 60vh が扱いやすい

豆知識

max-widthやmin-heightなどは、最大・最低の幅の指定。
%などで指定していると追従するが、最大・最低の大きさを超えたり割ったりしないよう、
その値で止めることが出来る。レスポンシブなどの設定に良く使う。

Section 3-4

サイズ指定デモ

フォルダ構成

フォルダ構成

HTML 
<!doctype html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="description" content="width/height と単位の基本デモ">
  <title>Café Latte｜サイズ指定デモ</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <header class="hero">
    <h1> Café Latte</h1>
    <p class="lead">幅と高さ、そして単位の"効き方"を体験しよう。</p>
  </header>

  <main class="content">
    <!-- 1) px 指定 -->
    <section>
      <h2>固定サイズ（px）</h2>
      <div class="box px">width: 500px / height: 300px</div>
    </section>

    <!-- 2) % 指定（親の幅に追従） -->
    <section>
      <h2>割合指定（%）</h2>
      <div class="parent">
        親: 600px
        <div class="child">子: width 50% → 300px</div>
      </div>
    </section>

    <!-- 3) rem / em の違い -->
    <section>
      <h2>rem と em の違い</h2>
      <div class="scale">
        <p class="rem">rem = ルート基準（常に同じ）</p>
        <div class="em-scope">
          <p class="em">em = 親のフォントサイズに連動</p>
        </div>
      </div>
    </section>

    <!-- 4) ビューポート単位 -->
    <section>
      <h2>ビューポート（vw / vh）</h2>
      <div class="vh">高さ: 40vh（画面の4割）</div>
    </section>
  </main>
</body>
</html>
CSS 
@charset "UTF-8";

/* ベース */
html { font-size: 100%; }            /* 1rem = 16px（既定） */
body {
  font-family: "Hiragino Sans", "Yu Gothic", sans-serif;
  color: #333;
  background: #fffdf8;
  margin: 40px 20px;
}
h1 { color: #6b3e26; margin: 0 0 8px; line-height: 1.25; }
h2 { color: #6b3e26; margin: 24px 0 10px; }

/* 1) px 指定の箱（固定） */
.box.px {
  width: 500px;
  height: 300px;
  background: #0bd;
  color: #fff;
  display: grid;
  place-items: center;
  border-radius: 8px;
}

/* 2) % 指定：親 600px → 子 50%（=300px） */
.parent {
  width: 600px;
  background: #eee;
  padding: 10px;
  border-radius: 6px;
}
.parent .child {
  width: 50%;              /* 親の半分 */
  background: #c5e7ff;
  padding: 10px;
  border-radius: 4px;
}

/* 3) rem / em の違い */
.scale { font-size: 16px; }
.rem { font-size: 1.25rem; }         /* 1.25 × 16 = 20px（どこでも同じ） */
.em-scope { font-size: 20px; }       /* 親を20pxに変更 */
.em { font-size: 1.25em; }           /* 1.25 × 20 = 25px（親に連動） */

/* 4) ビューポート単位 */
.vh {
  height: 40vh;                      /* 画面の縦の40% */
  background: #ffe4b5;
  display: grid;
  place-items: center;
  border: 1px dashed #e0a14a;
  border-radius: 8px;
}

/* ちょっと整える */
section { margin-bottom: 28px; }
.lead { margin: 6px 0 20px; color: #6b3e26; }
.content { display: grid; gap: 10px; }
まとめ

第3章のまとめ

用途 指針
固定したい → px
親に追従 → %
全体で一貫 → rem（テキストの黄金）
画面に合わせたい → vw / vh
高さは中身に合わせる → height: auto（既定）
