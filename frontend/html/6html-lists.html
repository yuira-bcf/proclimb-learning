箇条書き・番号付きリスト

Introduction

リストタグの基本

Webページで「箇条書き」や「順番付きの項目」を並べたいときは、リストタグを使います。

Section 6-1

箇条書きリストを作る <ul> ＋ <li> タグ

「・」「●」のようなマーク付きのリストを作るときは、
<ul>（Unordered List＝順番のないリスト）と
<li>（List Item＝リスト項目）を組み合わせて使用します。

基本の書き方

HTML 
<ul>
  <li>卵（鶏卵）</li>
  <li>うなじ（大とり）</li>
  <li>塩（少々）</li>
</ul>
  • <ul> タグ ⇒ リスト（項目）を束ねる枠や箱のイメージ
  • <li> タグ ⇒ 1つのリスト

<ul>タグの中の<li>には、ブラウザが自動で「・」を付けて表示してくれます（消すことも出来る）
画面には<li>タグに書かれた項目だけが、傍点が文頭に付いた状態で表示されます。
<ul>タグ自体は画面には表示されません。

タグ 意味 使い方
<ul> 順番のないリスト リスト全体を囲む（親要素）
<li> リストの項目 各項目を1行ずつ書く
Section 6-2

番号付きリスト

順番のあるリストにしたいときは <ul> の代わりに <ol>（Ordered List） を使います。

HTML 
<ol>
  <li>下ごしらえをする</li>
  <li>焼く</li>
  <li>味付けする</li>
</ol>

<ul>と違って、<li>タグの文頭に数字が順番に振られた形で表示されます。
これにより、検索ブラウザに「項番などの順番が明示された記事」であることを伝えます。

まとめ

第6章のまとめ

  • <ul> + <li> で箇条書きリストを作る
  • <ol> + <li> で番号付きリストを作る
  • <ul>/<ol>はリスト全体を囲む親要素
  • <li>は各項目を表す子要素
