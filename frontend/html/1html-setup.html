Section 1-1

Webページの仕組み

Webページは、HTML、CSS、JavaScriptという3つの技術で構成されています。これらがどのように連携してWebサイトを作り上げているのか、基本的な仕組みを理解しましょう。

1.1 Webページを構成する3つの技術

Webページは以下の3つの技術で構成されています：

HTML

HTML（構造）

Webページの骨格を作ります。テキスト、画像、リンクなどのコンテンツを配置し、ページの構造を定義します。人間でいう「骨格」にあたります。

CSS

CSS（装飾）

HTMLで作った構造に色、大きさ、配置などのスタイルを適用します。人間でいう「服装やメイク」にあたります。

JS

JavaScript（動作）

ページに動きやインタラクティブな機能を追加します。ボタンをクリックしたときの動作などを制御します。人間でいう「行動や振る舞い」にあたります。

Point まずはHTMLとCSSから

この教科書では、まずHTMLとCSSを学びます。JavaScriptは別の教材で学習しますが、HTMLとCSSをしっかり理解していれば、JavaScript学習もスムーズに進められます。

1.2 ブラウザの役割

ブラウザ（Chrome、Firefox、Safari、Edgeなど）は、HTMLファイルを読み込み、その内容を人間が見やすい形で画面に表示します。

menu_book 構文説明

HTML基本構造（DOCTYPE, html, head, body）

<!DOCTYPE html> <html lang="言語コード"> <head>...</head> <body>...</body> </html>

<!DOCTYPE html>はHTML5文書であることを宣言します。<html>は文書全体を囲み、lang属性で言語を指定します。<head>にはメタ情報、<body>には表示コンテンツを記述します。

metaタグ

<meta 属性名="値">

ページに関するメタ情報（文字コード、ビューポート設定、説明文など）を設定します。charset="UTF-8"で日本語を正しく表示、name="viewport"でスマホ対応を行います。終了タグは不要な空要素です。

linkタグ

<link rel="関係性" href="ファイルパス">

外部リソース（CSS、フォント、アイコンなど）を読み込みます。rel="stylesheet"でCSSファイル、rel="icon"でファビコンを指定します。これも空要素で終了タグは不要です。

3.2 各部分の解説

コード 意味
<!DOCTYPE html> このファイルがHTML5であることを宣言
<html lang="ja"> HTML文書の開始。日本語のページであることを示す
<head> ページの設定情報を記述する領域
<meta charset="UTF-8"> 文字コードをUTF-8に設定（日本語対応）
<meta name="viewport"...> スマートフォン対応の設定
<title> ブラウザのタブに表示されるタイトル
<body> 実際に画面に表示されるコンテンツ
<h1> 見出し（最も重要な見出し）
<p> 段落（パラグラフ）

4.2 ファイル・フォルダ命名規則

Good
  • 半角英数字を使用
  • 小文字のみ使用
  • スペースの代わりにハイフン（-）
  • 意味のある名前をつける

my-profile.html

Bad
  • 日本語を使用
  • スペースを含む
  • 大文字と小文字の混在
  • 特殊文字の使用

My Profile.html

Warning 注意

ファイル名に日本語やスペースを使用すると、サーバーにアップロードした際に問題が発生することがあります。必ず半角英数字とハイフンを使用しましょう。

Section 1-5

Emmet記法で効率化

Emmetは、HTMLやCSSを素早く入力できる省略記法です。VS Codeに標準で組み込まれており、コーディングの効率を大幅に向上させます。

5.1 基本的なEmmet記法

Emmet記法 展開結果
! または html:5 HTML5の基本テンプレート
div <div></div>
.container <div class="container"></div>
#header <div id="header"></div>
ul>li*5 <ul>の中に<li>を5つ
p{テキスト} <p>テキスト</p>
a[href="#"] <a href="#"></a>
Tips 使い方

VS CodeでEmmet記法を入力後、TabキーまたはEnterキーを押すと、HTMLに展開されます。

5.2 よく使う組み合わせ

chat プロンプト 
## 機能
Emmet記法でナビゲーションメニューを作成

## 実装要件
- navタグで囲まれたナビゲーション
- ul/li構造のメニューリスト
- 4つのメニュー項目を配置
- 各項目にリンクを設定

## 使用するEmmet記法
- > : 子要素の作成
- *4 : 要素の繰り返し
- [href="#"] : 属性の追加
- {テキスト$} : テキストと連番
ナビゲーションメニュー Emmet

        <li><a href="#">メニュー2</a></li>
        <li><a href="#">メニュー3</a></li>
        <li><a href="#">メニュー4</a></li>
    </ul>
</nav>
Exercise

1章 練習問題

Q1

Webページを構成する3つの技術として正しい組み合わせはどれですか？

  1. HTML、Python、CSS
  2. HTML、CSS、JavaScript
  3. PHP、CSS、JavaScript
  4. HTML、SQL、JavaScript
Q2

HTMLファイルの拡張子として正しいものはどれですか？

  1. .htm
  2. .html
  3. .css
  4. AとBの両方
Q3

Emmet記法で .box と入力してTabキーを押すと、何に展開されますか？

  1. <box></box>
  2. <div class="box"></div>
  3. <div id="box"></div>
  4. <span class="box"></span>
Answer

1章 解説・解答

Q1の解答: B

Webページは「HTML（構造）」「CSS（装飾）」「JavaScript（動作）」の3つの技術で構成されます。Pythonは主にサーバーサイド、SQLはデータベース、PHPはサーバーサイドの言語です。

Q2の解答: D

.htmlと.htmの両方がHTMLファイルの拡張子として有効です。現在は.htmlが主流ですが、.htmも古いシステムとの互換性のために使用されることがあります。

Q3の解答: B

Emmetでは、ドット（.）から始まる記法はクラス名を意味します。要素名を指定しない場合は、デフォルトでdiv要素になります。IDを指定する場合はシャープ（#）を使用します。
