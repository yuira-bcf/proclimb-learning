プロンプトUIレイアウトとは

プロンプトUIレイアウトは、完成したWebページのCSSを「レイヤー」に分解し、段階的に積み重ねてレイアウトを構築する学習手法です。最もシンプルな「ブロックの枠線だけ」の状態から始め、サイズ、余白、配置、装飾を1つずつ追加していくことで、CSSの各プロパティがレイアウトにどう影響するかを視覚的に理解できます。

この手法のメリット

  • CSSの各プロパティの役割を視覚的に理解できる
  • レイアウト構築の「順序」を体系的に学べる
  • AIへのプロンプト指示を正確に出せるようになる
  • デバッグ時にどのレイヤーに問題があるか特定しやすい

7つのCSSレイヤー分類

expand_more

Webページを構成するCSSプロパティは、以下の7つのレイヤーに分類できます。

1. アウトライン（デバッグ用）

outline - ブロック要素の境界を可視化。実線=主要ブロック、破線=内部コンテナ、点線=テキスト要素

2. リセット / ベース

margin: 0 padding: 0 box-sizing font-family font-size line-height color background

3. ブロックサイズ

width height max-width min-height aspect-ratio padding margin gap

4. レイアウト

display flex grid position align-items justify-content grid-template-columns overflow

5. ビジュアル装飾

background border border-radius box-shadow opacity transition transform

6. テキスト / タイポグラフィ

font-size font-weight color text-decoration text-align letter-spacing white-space

7. インタラクション（JavaScript）

ドロワーメニューの開閉、バナースライダー、タブ切り替え、カウンター操作など

プロンプトでの指示方法

expand_more

AIにCSSレイアウトを段階的に構築させる場合、以下のようにレイヤーを明示してプロンプトを書きます。

Step 1: 構造の指示（HTML）

## HTML構造 以下のセクション構成でHTMLを作成してください： - header: ロゴ + 検索ボタン + ハンバーガーメニュー - quick-access: 4つのクイックリンク（grid 4列） - main-banner: 横スクロールバナー × 3枚 + ドットインジケーター - important-info: 左ボーダー付き警告ボックス - section × 4: カテゴリ / キャンペーン / ニュース / サービスリンク - footer: SNSアイコン + リンクグループ × 3 + コピーライト - bottom-nav: 5つのタブ（モバイル用固定ナビ） - drawer-menu: 右からスライドするメニュー テキストは空で、class名のみ付与してください。

Step 2: レイアウトの指示（CSS）

## CSSレイアウト指示 以下のレイアウトプロパティを適用してください： header-top: display: flex, justify-content: space-between, align-items: center quick-access: display: grid, grid-template-columns: repeat(4, 1fr) banner-slider: display: flex, overflow-x: auto, scroll-snap-type: x mandatory category-grid: display: grid, grid-template-columns: repeat(4, 1fr) campaign-scroll: display: flex, overflow-x: auto, gap: 12px footer-links: display: grid, grid-template-columns: repeat(3, 1fr) bottom-nav: position: fixed, bottom: 0, display: grid, grid-template-columns: repeat(5, 1fr) ## ワイヤーフレーム配色 全てグレースケールで、border-radius: 0 を使用

Step 3: グレースケール → カラー変換

## カラー変換指示 ワイヤーフレームのグレースケールを以下のブランドカラーに変換してください： プライマリカラー: #e60039（ブランドレッド） グラデーション: linear-gradient(135deg, #e60039, #ff4d6d) 背景色: #f5f5f5（ライトグレー） テキスト: #333（ダークグレー） border-radius: 10px（角丸に変更） box-shadow: 0 2px 8px rgba(0,0,0,0.08) を追加
