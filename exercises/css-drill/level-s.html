デザインシステムのCSS設計

verified
CSSポイント

デザインシステムでは、CSS変数、@layer、BEM命名規則を組み合わせて、再利用可能でスケーラブルなスタイル基盤を構築します。トークン（色、スペーシング）→ コンポーネント → ユーティリティの3層構造が基本です。

assignment課題

以下の仕様に従って、小規模デザインシステムのCSS基盤を設計・実装してください。

description仕様
  • @layer tokens, base, components, utilities; の4層構造
  • トークン: カラーパレット（primary/secondary/neutral 各3段階）、スペーシング（4px基準 x8段階）、フォントサイズ（4段階）、角丸（3段階）
  • ベース: リセット + 基本タイポグラフィ + リンクスタイル
  • コンポーネント: Button（4バリエーション）、Card、Badge、Avatar を BEM で定義
  • ユーティリティ: テキスト配置、マージン、表示切替、Flexbox ヘルパー
  • ダークモード対応: [data-theme="dark"] でトークンを上書き

レガシーCSS のモダン移行設計

verified
CSSポイント

Float ベースの旧レイアウトを Grid/Flexbox にリファクタリングする設計課題です。既存の見た目を維持しながら、段階的にモダン CSS へ移行する手順を学びます。

assignment課題

以下のレガシーCSSを、見た目を変えずにモダンCSSへリファクタリングしてください。

/* レガシーCSS（Float ベース） */
.container { max-width: 1200px; margin: 0 auto; overflow: hidden; }
.sidebar { float: left; width: 250px; }
.main-content { margin-left: 270px; }
.card { float: left; width: 31.33%; margin: 1%; }
.clearfix::after { content: ""; display: table; clear: both; }
.col-6 { float: left; width: 50%; padding: 0 15px; }
.row { margin: 0 -15px; overflow: hidden; }
description仕様
  • Float → Flexbox/Grid に置き換え
  • clearfix ハックを除去
  • margin-left によるサイドバーオフセットを Flexbox に変更
  • パーセント幅の .card を Grid の auto-fill に変更
  • .col-6 + .row を Grid に変更

codeベースコード

<!-- HTML -->
<div class="container">
  <aside class="sidebar">サイドバー</aside>
  <main class="main-content">
    <div class="card-grid">
      <div class="card">Card 1</div>
      <div class="card">Card 2</div>
      <div class="card">Card 3</div>
    </div>
    <div class="row">
      <div class="col">Column 1</div>
      <div class="col">Column 2</div>
    </div>
  </main>
</div>

/* CSS（モダンに書き換えてください） */

  max-width: 1200px;
  margin: 0 auto;
  /* TODO: Float を Flexbox に変更 */
}

  /* TODO: Float ベースの .row + .col-6 を Grid に変更 */

lightbulbヒント

サイドバーレイアウトは display: flex + flex: 1 で置き換えます。カードグリッドは display: grid; grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)); で置き換えます。

  display: flex;
  gap: 20px;
}

  width: 250px;
  flex-shrink: 0;
}

  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));

  display: grid;
  grid-template-columns: repeat(2, 1fr);

解説
  • Float レイアウトの問題点: clearfix ハック、パーセント計算の煩雑さ、垂直中央揃えの困難さ。
  • Flexbox では flex: 1 でメインコンテンツが自動的に残り幅を埋め、margin-left の計算が不要です。
  • Grid の auto-fill + minmax() は、パーセント幅 + マージンの計算を完全に置き換えます。
  • 移行時は @supports を使って段階的に適用することも可能: @supports (display: grid) { ... }

assignment課題

description仕様
