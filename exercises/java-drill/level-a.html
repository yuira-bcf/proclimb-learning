型安全な設定マネージャ

verified
JavaSilverポイント

メソッドのオーバーロード（引数の型・個数による多重定義）はJavaSilverで頻出です。戻り値の型だけが異なるオーバーロードはコンパイルエラーになる点も問われます。

assignment課題

ジェネリクスを使わずに、String / int / double / boolean の設定値を型安全に管理する ConfigManager クラスを設計・実装してください。

  • 内部では HashMap<String, String> で全値を文字列として保持する
  • オーバーロードされた set メソッドで各型の値を受け取り、文字列に変換して格納する

  • 各 get メソッドにはデフォルト値を受け取る第2引数のオーバーロードも用意する
  • キーが存在しない場合、デフォルト値なしの get は IllegalArgumentException をスロー
  • 不正な型変換（例: "abc" を int に変換）は IllegalArgumentException をスロー

lightbulbヒント

Integer.parseInt() や Double.parseDouble() が NumberFormatException をスローするのを catch して IllegalArgumentException に変換すると良いでしょう。Boolean.parseBoolean() は例外をスローしない（"true" 以外は全て false になる）点に注意してください。

import java.util.HashMap;
import java.util.Map;

public class ConfigManager {
    private Map<String, String> store = new HashMap<>();

    public void set(String key, String value) {
        store.put(key, value);
    }

    public void set(String key, int value) {
        store.put(key, String.valueOf(value));
    }

    public void set(String key, double value) {
        store.put(key, String.valueOf(value));
    }

    public void set(String key, boolean value) {
        store.put(key, String.valueOf(value));
    }

    public String getString(String key) {
        if (!store.containsKey(key)) {
            throw new IllegalArgumentException("Key not found: " + key);
        }
        return store.get(key);
    }

    public String getString(String key, String defaultValue) {
        return store.containsKey(key) ? store.get(key) : defaultValue;
    }

    public int getInt(String key) {
        String value = getString(key);
        try {
            return Integer.parseInt(value);
        } catch (NumberFormatException e) {
            throw new IllegalArgumentException(
                "Cannot convert '" + value + "' to int for key: " + key);
        }
    }

    public int getInt(String key, int defaultValue) {
        if (!store.containsKey(key)) return defaultValue;
        return getInt(key);
    }

    public double getDouble(String key) {
        String value = getString(key);
        try {
            return Double.parseDouble(value);
        } catch (NumberFormatException e) {
            throw new IllegalArgumentException(
                "Cannot convert '" + value + "' to double for key: " + key);
        }
    }

    public double getDouble(String key, double defaultValue) {
        if (!store.containsKey(key)) return defaultValue;
        return getDouble(key);
    }

    public boolean getBoolean(String key) {
        String value = getString(key);
        if (!value.equalsIgnoreCase("true")
                && !value.equalsIgnoreCase("false")) {
            throw new IllegalArgumentException(
                "Cannot convert '" + value + "' to boolean for key: " + key);
        }
        return Boolean.parseBoolean(value);
    }

    public boolean getBoolean(String key, boolean defaultValue) {
        if (!store.containsKey(key)) return defaultValue;
        return getBoolean(key);
    }

    public static void main(String[] args) {
        ConfigManager config = new ConfigManager();

        config.set("appName", "MyApp");
        config.set("maxRetry", 3);
        config.set("timeout", 30.5);
        config.set("debug", true);

        System.out.println("appName = " + config.getString("appName"));
        System.out.println("maxRetry = " + config.getInt("maxRetry"));
        System.out.println("timeout = " + config.getDouble("timeout"));
        System.out.println("debug = " + config.getBoolean("debug"));
        System.out.println("missing key with default = "
            + config.getInt("missing", 10));

        try {
            config.getString("unknown");
        } catch (IllegalArgumentException e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}
解説
  • 全ての値を String で統一管理することで、内部構造をシンプルに保ちつつ、型安全なAPIをオーバーロードで提供する設計パターンです。
  • set メソッドのオーバーロードでは、引数の型（String / int / double / boolean）が異なるため正しくコンパイルされます。戻り値だけが異なるオーバーロードは不可能です。
  • Boolean.parseBoolean() は "true"（大文字小文字無視）以外は全て false を返す仕様のため、独自のバリデーション（"true" / "false" チェック）を追加しています。
  • デフォルト値付き get メソッドでは、キーの存在チェックを先に行い、存在しなければデフォルト値を返すことで例外を回避しています。
  • NumberFormatException を catch して IllegalArgumentException に変換するパターンは、例外の翻訳（Exception Translation）と呼ばれ、APIの一貫性を保つ設計手法です。

内部フィールドは private final long cents のみ。コンストラクタで yen * 100 + sen を計算して保持します。toString() では String.format("%,d.%02d", cents / 100, Math.abs(cents % 100)) のようなフォーマットが使えます。equals() では instanceof チェックが必須です。

check_circle解答・解説 
public class Money {
    private final long cents;

    public Money(long yen, int sen) {
        this.cents = yen * 100 + sen;
    }

    private Money(long cents) {
        this.cents = cents;
    }

    public static Money of(double amount) {
        return new Money(Math.round(amount * 100));
    }

    public Money add(Money other) {
        return new Money(this.cents + other.cents);
    }

    public Money subtract(Money other) {
        return new Money(this.cents - other.cents);
    }

    public Money multiply(int times) {
        return new Money(this.cents * times);
    }

    @Override
    public String toString() {
        long yen = cents / 100;
        long sen = Math.abs(cents % 100);
        return String.format("¥%,d.%02d", yen, sen);
    }

    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (!(obj instanceof Money)) return false;
        Money other = (Money) obj;
        return this.cents == other.cents;
    }

    @Override
    public int hashCode() {
        return Long.hashCode(cents);
    }

    public static void main(String[] args) {
        Money a = new Money(1234, 56);
        Money b = new Money(765, 44);

        System.out.println("a = " + a);
        System.out.println("b = " + b);
        System.out.println("a + b = " + a.add(b));
        System.out.println("a - b = " + a.subtract(b));
        System.out.println("b * 3 = " + b.multiply(3));

        Money aCopy = Money.of(1234.56);
        System.out.println("a.equals(a copy) = " + a.equals(aCopy));

        System.out.println("double誤差の例: 0.1+0.2 = " + (0.1 + 0.2));
        Money m1 = Money.of(0.1);
        Money m2 = Money.of(0.2);
        System.out.println("Money精度: " + m1 + " + " + m2
            + " = " + m1.add(m2));
    }
}
解説
  • 金額計算で double を直接使うと 0.1 + 0.2 = 0.30000000000000004 のような誤差が生じます。long（セント）で管理することで正確な計算が可能です。
  • 不変オブジェクト（Immutable Object）パターン: フィールドが private final で、全メソッドが新しいインスタンスを返します。String と同じ設計思想です。
  • equals() のオーバーライドでは、(1) 自己比較チェック this == obj、(2) instanceof による型チェック、(3) フィールド比較 の3ステップが定石です。
  • hashCode() は equals() と必ずペアでオーバーライドします。equals が true なら hashCode も同じ値を返す契約（一般契約）に違反すると HashMap 等で不具合が起きます。
  • プライベートコンストラクタ Money(long cents) を用意し、内部メソッドから効率的にインスタンスを生成しています。静的ファクトリメソッド of() との組み合わせは実務でも頻出のパターンです。

lightbulbヒント

StringBuilder を使い、1文字ずつ処理します。\\ を見つけたら次の文字をそのまま追加。** を見つけたら閉じ ** を前方検索して見つかれば <b>...</b> に変換。見つからなければそのまま出力。* 単体も同様に処理します。

check_circle解答・解説 
public class MarkdownConverter {

    public static String convert(String markdown) {
        StringBuilder sb = new StringBuilder();
        int i = 0;
        int len = markdown.length();

        while (i < len) {
            char c = markdown.charAt(i);

            // エスケープ処理
            if (c == '\\' && i + 1 < len) {
                sb.append(markdown.charAt(i + 1));
                i += 2;
                continue;
            }

            // バッククォート（code）
            if (c == '`') {
                int close = markdown.indexOf('`', i + 1);
                if (close != -1) {
                    sb.append("<code>");
                    sb.append(markdown, i + 1, close);
                    sb.append("</code>");
                    i = close + 1;
                    continue;
                }
            }

            // ** (bold)
            if (c == '*' && i + 1 < len
                    && markdown.charAt(i + 1) == '*') {
                int close = markdown.indexOf("**", i + 2);
                if (close != -1) {
                    sb.append("<b>");
                    // 内部を再帰的に変換
                    String inner = markdown.substring(i + 2, close);
                    sb.append(convert(inner));
                    sb.append("</b>");
                    i = close + 2;
                    continue;
                }
            }

            // * (italic)
            if (c == '*') {
                int close = markdown.indexOf('*', i + 1);
                if (close != -1) {
                    sb.append("<i>");
                    sb.append(markdown, i + 1, close);
                    sb.append("</i>");
                    i = close + 1;
                    continue;
                }
            }

            sb.append(c);
            i++;
        }
        return sb.toString();
    }

    public static void main(String[] args) {
        String[] tests = {
            "**hello**",
            "*world*",
            "`code`",
            "\\*escaped\\*",
            "**bold and *italic***",
            "*unclosed"
        };
        for (String md : tests) {
            System.out.println(md + " → " + convert(md));
        }
    }
}
解説
  • charAt() ループによるパーサー実装は、状態遷移を意識した基本的な文字列処理パターンです。正規表現より制御が明確で、エッジケースを把握しやすい利点があります。
  • StringBuilder は可変の文字列バッファです。ループ内で + による文字列連結を行うと毎回新しい String オブジェクトが生成されるため、StringBuilder の方が効率的です。
  • indexOf() で閉じ記号を前方検索し、見つからなければ（-1）そのまま出力するフォールバック処理がポイントです。
  • **bold and *italic*** のケースでは、** の内部を convert() で再帰的に変換することで、ネスト対応を実現しています。
  • JavaSilverでは String の不変性、StringBuilder と StringBuffer の違い（スレッドセーフ性）が出題されます。

assignment課題

ランレングス圧縮と展開を行う RunLengthCodec クラスを実装してください。

  • compress: 連続する同じ文字をカウントして圧縮する（例: "aaabbbcc" → "a3b3c2"）
  • 1文字のみの場合は数字を省略する（例: "abc" → "abc", "aab" → "a2b"）
  • expand: 圧縮文字列を元に戻す（例: "a3b3c2" → "aaabbbcc"）
  • 展開時、数字が2桁以上にも対応する（例: "a12" → "aaaaaaaaaaaa"）
  • 不正入力（数字で始まる、空文字列等）は IllegalArgumentException をスロー

圧縮: 前の文字と現在の文字を比較し、同じならカウントを増やし、異なれば文字とカウント（2以上の場合）を出力。展開: Character.isLetter() で文字を判定し、直後の数字列を Character.isDigit() で読み取ります。

check_circle解答・解説 
public class RunLengthCodec {

    public static String compress(String input) {
        if (input == null || input.isEmpty()) {
            throw new IllegalArgumentException("Input must not be empty");
        }
        if (Character.isDigit(input.charAt(0))) {
            throw new IllegalArgumentException(
                "Input must not start with a digit");
        }

        StringBuilder sb = new StringBuilder();
        int count = 1;

        for (int i = 1; i <= input.length(); i++) {
            if (i < input.length()
                    && input.charAt(i) == input.charAt(i - 1)) {
                count++;
            } else {
                sb.append(input.charAt(i - 1));
                if (count > 1) {
                    sb.append(count);
                }
                count = 1;
            }
        }
        return sb.toString();
    }

    public static String expand(String compressed) {
        if (compressed == null || compressed.isEmpty()) {
            throw new IllegalArgumentException("Input must not be empty");
        }
        if (Character.isDigit(compressed.charAt(0))) {
            throw new IllegalArgumentException(
                "Input must not start with a digit");
        }

        StringBuilder sb = new StringBuilder();
        int i = 0;

        while (i < compressed.length()) {
            char c = compressed.charAt(i);
            i++;

            // 直後の数字を読み取る
            StringBuilder numBuf = new StringBuilder();
            while (i < compressed.length()
                    && Character.isDigit(compressed.charAt(i))) {
                numBuf.append(compressed.charAt(i));
                i++;
            }

            int repeat = numBuf.length() > 0
                ? Integer.parseInt(numBuf.toString()) : 1;
            for (int r = 0; r < repeat; r++) {
                sb.append(c);
            }
        }
        return sb.toString();
    }

    public static void main(String[] args) {
        System.out.println("compress: aaabbbcc → "
            + compress("aaabbbcc"));
        System.out.println("compress: abc → "
            + compress("abc"));
        System.out.println("compress: aab → "
            + compress("aab"));

        System.out.println("expand: a3b3c2 → "
            + expand("a3b3c2"));
        System.out.println("expand: abc → "
            + expand("abc"));
        System.out.println("expand: a12 → "
            + expand("a12"));

        String original = "aaabbbcccdddd";
        String roundTrip = expand(compress(original));
        System.out.println("round-trip: " + original
            + " → " + roundTrip);

        try {
            compress("");
        } catch (IllegalArgumentException e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}
解説
  • Character.isDigit() と Character.isLetter() は文字分類のユーティリティメソッドです。JavaSilverでは Wrapper クラスのメソッドとして出題されます。
  • 圧縮のループでは i <= input.length() とすることで、最後の文字グループも確実に処理しています。境界条件の処理はバグの温床になりやすいポイントです。
  • 展開で2桁以上の数字に対応するため、StringBuilder で数字列を蓄積してから Integer.parseInt() で変換しています。
  • ラウンドトリップテスト（compress → expand → 元に戻る）は、エンコード・デコード系の実装で必ず検証すべきテストケースです。
  • 入力バリデーションを先頭で行い、不正入力を早期に弾く「早期リターン」パターンは、メソッドの見通しを良くする設計手法です。

内部フィールドは int[] data と int size。拡張は Arrays.copyOf(data, data.length * 2) が便利です。挿入・削除時の要素シフトには System.arraycopy() を使います。インデックスチェックを private メソッドに切り出すと DRY になります。

check_circle解答・解説 
import java.util.Arrays;

public class MyIntList {
    private int[] data;
    private int size;

    public MyIntList() {
        this.data = new int[4];
        this.size = 0;
    }

    private void ensureCapacity() {
        if (size == data.length) {
            data = Arrays.copyOf(data, data.length * 2);
        }
    }

    private void checkIndex(int index) {
        if (index < 0 || index >= size) {
            throw new IndexOutOfBoundsException(
                "Index: " + index + ", Size: " + size);
        }
    }

    private void checkIndexForAdd(int index) {
        if (index < 0 || index > size) {
            throw new IndexOutOfBoundsException(
                "Index: " + index + ", Size: " + size);
        }
    }

    public void add(int value) {
        ensureCapacity();
        data[size++] = value;
    }

    public void add(int index, int value) {
        checkIndexForAdd(index);
        ensureCapacity();
        System.arraycopy(data, index, data, index + 1,
            size - index);
        data[index] = value;
        size++;
    }

    public int get(int index) {
        checkIndex(index);
        return data[index];
    }

    public int remove(int index) {
        checkIndex(index);
        int removed = data[index];
        System.arraycopy(data, index + 1, data, index,
            size - index - 1);
        size--;
        return removed;
    }

    public int size() {
        return size;
    }

    @Override
    public String toString() {
        StringBuilder sb = new StringBuilder("[");
        for (int i = 0; i < size; i++) {
            if (i > 0) sb.append(", ");
            sb.append(data[i]);
        }
        sb.append("]");
        return sb.toString();
    }

import java.util.Arrays;

public class Statistics {
    private final int[] data;
    private final int[] sorted;

    public Statistics(int[] data) {
        if (data == null || data.length == 0) {
            throw new IllegalArgumentException(
                "Data must not be empty");
        }
        this.data = Arrays.copyOf(data, data.length);
        this.sorted = Arrays.copyOf(data, data.length);
        Arrays.sort(this.sorted);
    }

    public double average() {
        long sum = 0;
        for (int v : data) {
            sum += v;
        }
        return Math.round((double) sum / data.length * 100.0)
            / 100.0;
    }

    public double median() {
        int n = sorted.length;
        if (n % 2 == 1) {
            return sorted[n / 2];
        } else {
            double mid = (sorted[n / 2 - 1] + sorted[n / 2])
                / 2.0;
            return Math.round(mid * 100.0) / 100.0;
        }
    }

    public int mode() {
        int maxCount = 0;
        int modeValue = sorted[0];

        int currentCount = 1;
        for (int i = 1; i < sorted.length; i++) {
            if (sorted[i] == sorted[i - 1]) {
                currentCount++;
            } else {
                if (currentCount > maxCount) {
                    maxCount = currentCount;
                    modeValue = sorted[i - 1];
                }
                currentCount = 1;
            }
        }
        // 最後のグループをチェック
        if (currentCount > maxCount) {
            modeValue = sorted[sorted.length - 1];
        }
        return modeValue;
    }

    public double stddev() {
        double avg = average();
        double sumSq = 0;
        for (int v : data) {
            double diff = v - avg;
            sumSq += diff * diff;
        }
        double variance = sumSq / data.length;
        return Math.round(Math.sqrt(variance) * 100.0)
            / 100.0;
    }

    public String report() {
        StringBuilder sb = new StringBuilder();
        sb.append("=== 統計レポート ===\n");
        sb.append("データ数: ").append(data.length).append("\n");
        sb.append(String.format("平均値: %.2f%n", average()));
        sb.append(String.format("中央値: %.2f%n", median()));
        sb.append("最頻値: ").append(mode()).append("\n");
        sb.append(String.format("標準偏差: %.2f%n", stddev()));
        sb.append("===================");
        return sb.toString();
    }

        int[] values = {3, 1, 4, 1, 5, 9, 2, 6, 5, 3};

public class VendingMachine {

    enum State {
        IDLE, COIN_INSERTED, DISPENSING
    }

    private State state = State.IDLE;
    private int balance = 0;
    private String selectedItem = null;

    public void insertCoin(int amount) {
        if (state != State.IDLE
                && state != State.COIN_INSERTED) {
            throw new IllegalStateException(
                state + "状態ではコインを投入できません");
        }
        State prev = state;
        balance += amount;
        state = State.COIN_INSERTED;
        System.out.println("[" + prev + " → " + state + "] "
            + amount + "円投入（残高: " + balance + "円）");
    }

    public void selectItem(String item, int price) {
        if (state != State.COIN_INSERTED) {
            throw new IllegalStateException(
                state + "状態では商品を選択できません");
        }
        if (balance < price) {
            throw new IllegalStateException(
                "残高不足（残高: " + balance
                + "円, 価格: " + price + "円）");
        }
        State prev = state;
        selectedItem = item;
        balance -= price;
        state = State.DISPENSING;
        System.out.println("[" + prev + " → " + state + "] "
            + "「" + item + "」を選択（価格: " + price + "円）");
    }

    public void dispense() {
        if (state != State.DISPENSING) {
            throw new IllegalStateException(
                state + "状態では排出できません");
        }
        State prev = state;
        int change = balance;
        System.out.println("[" + prev + " → IDLE] "
            + selectedItem + " を排出。おつり: "
            + change + "円");
        balance = 0;
        selectedItem = null;
        state = State.IDLE;
    }

    public void cancel() {
        if (state != State.COIN_INSERTED) {
            throw new IllegalStateException(
                state + "状態ではキャンセルできません");
        }
        State prev = state;
        int refund = balance;
        System.out.println("[" + prev + " → IDLE] "
            + "キャンセル。" + refund + "円を返却");
        balance = 0;
        state = State.IDLE;
    }

    public static void main(String[] args) {
        VendingMachine vm = new VendingMachine();

        vm.insertCoin(100);
        vm.insertCoin(50);
        vm.selectItem("コーラ", 120);
        vm.dispense();

        System.out.println("--- キャンセルテスト ---");
        vm.insertCoin(200);
        vm.cancel();

        System.out.println("--- エラーテスト ---");
        try {
            vm.selectItem("水", 100);
        } catch (IllegalStateException e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}

  • 状態マシン（State Machine）パターンは、オブジェクトの振る舞いが内部状態に依存する場合の基本的な設計手法です。enum で状態を定義し、各メソッドで遷移の妥当性を検証します。

import java.util.ArrayList;
import java.util.HashMap;
import java.util.LinkedHashMap;
import java.util.List;
import java.util.Map;

public class CommandExecutor {
    private Map<String, Runnable> commands
        = new LinkedHashMap<>();
    private List<String> log = new ArrayList<>();
    private int executionCount = 0;

    public void register(String name, Runnable command) {
        commands.put(name, command);
    }

    public void execute(String name) {
        Runnable cmd = commands.get(name);
        if (cmd == null) {
            throw new IllegalArgumentException(
                "Unknown command: " + name);
        }
        System.out.print("[実行] " + name + " → ");
        cmd.run();
        log.add(name);
        executionCount++;
    }

    public void executeAll() {
        for (String name : commands.keySet()) {
            execute(name);
        }
    }

    public String undo() {
        if (log.isEmpty()) {
            throw new IllegalStateException(
                "No commands to undo");
        }
        String last = log.remove(log.size() - 1);
        executionCount--;
        return last;
    }

    public int getExecutionCount() {
        return executionCount;
    }

    public void printLog() {
        System.out.println("--- 実行ログ ---");
        for (int i = 0; i < log.size(); i++) {
            System.out.println((i + 1) + ": " + log.get(i));
        }
    }

    public static void main(String[] args) {
        CommandExecutor exec = new CommandExecutor();

        // カウンタ変数（ラムダでキャプチャ）
        int[] counter = {0};

        exec.register("hello", () -> {
            System.out.println("Hello, World!");
        });
        exec.register("greet", () -> {
            System.out.println("こんにちは！");
        });
        exec.register("count", () -> {
            counter[0]++;
            System.out.println("カウント: " + counter[0]);
        });

        exec.execute("hello");
        exec.execute("greet");
        exec.execute("count");
        exec.execute("count");

        exec.printLog();
        System.out.println("実行回数: "
            + exec.getExecutionCount());

        System.out.println("--- Undo ---");
        String undone = exec.undo();
        System.out.println("Undo: " + undone);
        System.out.println("実行回数: "
            + exec.getExecutionCount());

        System.out.println("--- 全実行 ---");
        exec.executeAll();
    }
}

  • Runnable は run() メソッドのみを持つ関数型インタフェースです。ラムダ式 () -> { ... } で簡潔に実装できます。
  • LinkedHashMap は挿入順序を保持する Map です。executeAll() で登録順に実行するために使用しています。通常の HashMap は順序を保証しません。
  • ラムダ式から外部変数をキャプチャする場合、変数は実質 final（effectively final）である必要があります。int[] counter = {0} のように配列にすると、配列参照は final のまま中身を変更できます。

import java.time.LocalDate;
import java.time.temporal.ChronoUnit;
import java.util.Objects;

public class DateRange {
    private final LocalDate start;
    private final LocalDate end;

    public DateRange(LocalDate start, LocalDate end) {
        if (start.isAfter(end)) {
            throw new IllegalArgumentException(
                "start must not be after end");
        }
        this.start = start;
        this.end = end;
    }

    public boolean overlap(DateRange other) {
        return !this.start.isAfter(other.end)
            && !other.start.isAfter(this.end);
    }

    public boolean contains(LocalDate date) {
        return !date.isBefore(start) && !date.isAfter(end);
    }

    public DateRange merge(DateRange other) {
        if (!overlap(other)) {
            throw new IllegalArgumentException(
                "Ranges do not overlap");
        }
        LocalDate newStart = start.isBefore(other.start)
            ? start : other.start;
        LocalDate newEnd = end.isAfter(other.end)
            ? end : other.end;
        return new DateRange(newStart, newEnd);
    }

    public long getDays() {
        return ChronoUnit.DAYS.between(start, end) + 1;
    }

    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (!(obj instanceof DateRange)) return false;
        DateRange other = (DateRange) obj;
        return this.start.equals(other.start)
            && this.end.equals(other.end);
    }

    @Override
    public int hashCode() {
        return Objects.hash(start, end);
    }

    @Override
    public String toString() {
        return "[" + start + " ~ " + end + "] ("
            + getDays() + "日間)";
    }

    public static void main(String[] args) {
        DateRange range1 = new DateRange(
            LocalDate.of(2025, 1, 1),
            LocalDate.of(2025, 1, 31));
        DateRange range2 = new DateRange(
            LocalDate.of(2025, 1, 20),

import java.util.Collections;
import java.util.LinkedHashMap;
import java.util.Map;

public class HttpRequest {
    private final String method;
    private final String url;
    private final Map<String, String> headers;
    private final String body;

    private HttpRequest(Builder builder) {
        this.method = builder.method;
        this.url = builder.url;
        this.headers = Collections.unmodifiableMap(
            new LinkedHashMap<>(builder.headers));
        this.body = builder.body;
    }

    public String getMethod() { return method; }
    public String getUrl() { return url; }
    public Map<String, String> getHeaders() {
        return headers;
    }
    public String getBody() { return body; }

    @Override
    public String toString() {
        StringBuilder sb = new StringBuilder();
        sb.append(method).append(" ").append(url)
          .append("\n");
        sb.append("Headers: ").append(headers)
          .append("\n");
        sb.append("Body: ")
          .append(body == null ? "(none)" : body);
        return sb.toString();
    }

    public static class Builder {
        private String method;
        private String url;
        private Map<String, String> headers
            = new LinkedHashMap<>();
        private String body;

        public Builder method(String method) {
            this.method = method;
            return this;
        }

        public Builder url(String url) {
            this.url = url;
            return this;
        }

        public Builder header(String key, String value) {
            this.headers.put(key, value);
            return this;
        }

        public Builder body(String body) {
            this.body = body;
            return this;
        }

        public HttpRequest build() {
            if (method == null || url == null) {
                throw new IllegalStateException(
                    "method and url are required");
            }
            return new HttpRequest(this);
        }
    }

import java.util.HashMap;
import java.util.List;
import java.util.Map;

interface FileConverter {
    String convert(String input);
    String getSupportedFormat();
}

class CsvToJsonConverter implements FileConverter {
    @Override
    public String getSupportedFormat() {
        return "CSV->JSON";
    }

    @Override
    public String convert(String input) {
        String[] lines = input.split("\n");
        if (lines.length < 2) {
            throw new IllegalArgumentException(
                "CSV must have header and data rows");
        }
        String[] headers = lines[0].split(",");
        StringBuilder sb = new StringBuilder("[");

        for (int i = 1; i < lines.length; i++) {
            if (i > 1) sb.append(",");
            String[] values = lines[i].split(",");
            sb.append("{");
            for (int j = 0; j < headers.length; j++) {
                if (j > 0) sb.append(",");
                String val = j < values.length
                    ? values[j].trim() : "";
                sb.append("\"").append(headers[j].trim())
                  .append("\":\"").append(val).append("\"");
            }
            sb.append("}");
        }
        sb.append("]");
        return sb.toString();
    }
}

class JsonToCsvConverter implements FileConverter {
    @Override
    public String getSupportedFormat() {
        return "JSON->CSV";
    }

    @Override
    public String convert(String input) {
        // 簡易JSON配列パーサー
        List<Map<String, String>> records
            = new ArrayList<>();
        List<String> keys = new ArrayList<>();

        String trimmed = input.trim();
        // "[{...},{...}]" → "{...},{...}" に分割
        trimmed = trimmed.substring(1,
            trimmed.length() - 1);
        String[] objects = trimmed.split("\\},\\{");

        for (String obj : objects) {
            obj = obj.replace("{", "").replace("}", "");
            Map<String, String> record
                = new HashMap<>();
            String[] pairs = obj.split(",(?=\")");
            for (String pair : pairs) {
                String[] kv = pair.split(":");
                String key = kv[0].trim()
                    .replace("\"", "");
                String value = kv[1].trim()
                    .replace("\"", "");
                record.put(key, value);
                if (!keys.contains(key)) {
                    keys.add(key);
                }
            }
            records.add(record);
        }

        StringBuilder sb = new StringBuilder();
        sb.append(String.join(",", keys));
        for (Map<String, String> record : records) {
            sb.append("\n");
            for (int i = 0; i < keys.size(); i++) {
                if (i > 0) sb.append(",");
                sb.append(record.getOrDefault(
                    keys.get(i), ""));
            }
        }
        return sb.toString();
    }
}

    private Map<String, FileConverter> converters
        = new HashMap<>();

import java.util.ArrayList;
import java.util.List;

interface Transaction extends AutoCloseable {
    void execute() throws Exception;
    void rollback() throws Exception;
}

class TransactionException extends Exception {
    public TransactionException(String message,
            Throwable cause) {
        super(message, cause);
    }
}

class SimpleTransaction implements Transaction {
    private final String name;
    private final boolean shouldFail;

    public SimpleTransaction(String name,
            boolean shouldFail) {
        this.name = name;
        this.shouldFail = shouldFail;
    }

    @Override
    public void execute() throws Exception {
        if (shouldFail) {
            System.out.println("[Execute] "
                + name + " failed!");
            throw new Exception(
                name + " service unavailable");
        }
        System.out.println("[Execute] DB Insert: "
            + name);
    }

    @Override
    public void rollback() throws Exception {
        System.out.println("[Rollback] DB Insert: "
            + name);
    }

    @Override
    public void close() throws Exception {
        // リソース解放（今回はログのみ）
    }

    public String getName() { return name; }
}

class TransactionManager {
    private List<Transaction> transactions
        = new ArrayList<>();

    public void add(Transaction tx) {
        transactions.add(tx);
    }

    public void executeAll()
            throws TransactionException {
        List<Transaction> executed = new ArrayList<>();

        for (Transaction tx : transactions) {
            try {
                tx.execute();
                executed.add(tx);
            } catch (Exception e) {
                // ロールバック（逆順）
                TransactionException txEx
                    = new TransactionException(
                        "Transaction failed at: "
                        + getTransactionName(tx), e);

                for (int i = executed.size() - 1;
                        i >= 0; i--) {
                    try {
                        executed.get(i).rollback();
                    } catch (Exception rbEx) {
                        txEx.addSuppressed(rbEx);
                    }
                }
                throw txEx;
            }
        }
    }

    private String getTransactionName(Transaction tx) {
        if (tx instanceof SimpleTransaction) {
            return ((SimpleTransaction) tx).getName();
        }
        return tx.getClass().getSimpleName();
    }
}

import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

class ReadException extends Exception {
    public ReadException(String msg) { super(msg); }
}

class ProcessException extends Exception {
    public ProcessException(String msg) { super(msg); }
}

class WriteException extends Exception {
    public WriteException(String msg) { super(msg); }
}

class PipelineException extends Exception {
    public PipelineException(String msg) {
        super(msg);
    }
}

interface PipelineStage<T, R> {
    R execute(T input) throws Exception;
}

class PipelineExecutor {
    private PipelineStage<String, List<String>> reader;
    private PipelineStage<List<String>,
        List<String>> processor;
    private PipelineStage<List<String>, String> writer;

    public void setReader(
            PipelineStage<String,
                List<String>> reader) {
        this.reader = reader;
    }

    public void setProcessor(
            PipelineStage<List<String>,
                List<String>> processor) {
        this.processor = processor;
    }

    public void setWriter(
            PipelineStage<List<String>,
                String> writer) {
        this.writer = writer;
    }

    public String run(String input)
            throws PipelineException {
        List<Exception> errors = new ArrayList<>();
        List<String> readResult = null;
        List<String> processResult = null;
        String writeResult = null;

        // Read段階
        try {
            readResult = reader.execute(input);
        } catch (Exception e) {
            errors.add(
                new ReadException(e.getMessage()));
        }

        // Process段階
        try {
            if (readResult != null) {
                processResult
                    = processor.execute(readResult);
            } else {
                processResult
                    = processor.execute(
                        new ArrayList<>());
            }
        } catch (Exception e) {
            errors.add(
                new ProcessException(e.getMessage()));
        }

        // Write段階
        try {
            if (processResult != null) {
                writeResult
                    = writer.execute(processResult);
            } else {
                System.out.println("[Write] "
                    + "前段階のデータなしでスキップ");
                throw new WriteException(
                    "No data to write");
            }
        } catch (WriteException e) {
            errors.add(e);
        } catch (Exception e) {
            errors.add(
                new WriteException(e.getMessage()));
        }

        if (!errors.isEmpty()) {
            PipelineException pex
                = new PipelineException(
                    "Pipeline failed");
            for (Exception e : errors) {
                pex.addSuppressed(e);
            }
            throw pex;
        }

        return writeResult;
    }
}

    public static void main(String[] args) {
        // 正常実行

            List<String> lines

import java.util.ArrayList;
import java.util.Comparator;
import java.util.List;
import java.util.function.Function;
import java.util.function.Predicate;

public class Query<T> {
    private List<T> source;
    private List<Predicate<T>> filters
        = new ArrayList<>();
    private Comparator<T> sorter;
    private int limitCount = -1;

    private Query(List<T> source) {
        this.source = new ArrayList<>(source);
    }

    public static <T> Query<T> from(List<T> source) {
        return new Query<>(source);
    }

    public Query<T> where(Predicate<T> predicate) {
        filters.add(predicate);
        return this;
    }

    public Query<T> orderBy(Comparator<T> comparator) {
        this.sorter = comparator;
        return this;
    }

    public <R> Query<R> select(
            Function<T, R> mapper) {
        List<T> filtered = applyFiltersAndSort();
        List<R> mapped = new ArrayList<>();
        for (T item : filtered) {
            mapped.add(mapper.apply(item));
        }
        return new Query<>(mapped);
    }

    public Query<T> limit(int n) {
        this.limitCount = n;
        return this;
    }

    public List<T> execute() {
        List<T> result = applyFiltersAndSort();
        if (limitCount > 0
                && result.size() > limitCount) {
            result = result.subList(0, limitCount);
        }
        return new ArrayList<>(result);
    }

    private List<T> applyFiltersAndSort() {
        List<T> result = new ArrayList<>();
        for (T item : source) {
            boolean match = true;
            for (Predicate<T> filter : filters) {
                if (!filter.test(item)) {
                    match = false;
                    break;
                }
            }
            if (match) {
                result.add(item);
            }
        }
        if (sorter != null) {
            result.sort(sorter);
        }
        return result;
    }

    public static void main(String[] args) {
        // 数値クエリ
        System.out.println("=== 数値クエリ ===");
        List<Integer> numbers
            = List.of(3, 1, 4, 1, 5, 9, 2, 6, 8, 100);
        List<Integer> result = Query.from(numbers)
            .where(n -> n % 2 == 0)
            .orderBy(Comparator.reverseOrder())
            .limit(3)
            .execute();
        System.out.println(
            "偶数のみ（降順、上位3件）: " + result);

        // 文字列クエリ
        System.out.println("=== 文字列クエリ ===");
        List<String> names
            = List.of("Alice", "Bob", "Charlie",
                "Dave");
        List<String> upper = Query.from(names)
            .where(s -> s.length() >= 5)
            .<String>select(String::toUpperCase)
            .execute();
        System.out.println(
            "5文字以上を大文字変換: " + upper);

        // チェーンクエリ（カスタムオブジェクト）
        System.out.println("=== チェーンクエリ ===");
        List<String> people = List.of(
            "Alice:25", "Bob:30",
            "Charlie:35", "Dave:20");
        List<String> over30 = Query.from(people)
            .where(p -> {
                int age = Integer.parseInt(
                    p.split(":")[1]);
                return age >= 30;
            })
            .orderBy((a, b) -> {
                int ageA = Integer.parseInt(
                    a.split(":")[1]);
                int ageB = Integer.parseInt(
                    b.split(":")[1]);
                return Integer.compare(ageB, ageA);
            })
            .<String>select(p -> p.split(":")[0])
            .execute();
        System.out.println(
            "30歳以上の名前（年齢降順）: " + over30);
        System.out.println(
            "元リスト不変: " + people);
    }
}

  • メソッドチェーンは各メソッドが return this（または新しいインスタンス）を返すことで実現します。Builder パターンと同じテクニックです。

import java.time.DayOfWeek;
import java.time.LocalDate;
import java.time.YearMonth;
import java.time.temporal.TemporalAdjusters;
import java.util.ArrayList;
import java.util.Comparator;
import java.util.List;

interface RecurrenceRule {
    LocalDate nextOccurrence(LocalDate current);
}

class DailyRule implements RecurrenceRule {
    @Override
    public LocalDate nextOccurrence(LocalDate current) {
        return current.plusDays(1);
    }
}

class WeeklyRule implements RecurrenceRule {
    private final DayOfWeek dayOfWeek;

    public WeeklyRule(DayOfWeek dayOfWeek) {
        this.dayOfWeek = dayOfWeek;
    }

    @Override
    public LocalDate nextOccurrence(LocalDate current) {
        return current.with(
            TemporalAdjusters.next(dayOfWeek));
    }
}

class MonthlyRule implements RecurrenceRule {
    private final int dayOfMonth;

    public MonthlyRule(int dayOfMonth) {
        this.dayOfMonth = dayOfMonth;
    }

    @Override
    public LocalDate nextOccurrence(LocalDate current) {
        LocalDate next = current.plusMonths(1);
        YearMonth ym = YearMonth.from(next);
        int day = Math.min(dayOfMonth,
            ym.lengthOfMonth());
        return ym.atDay(day);
    }
}

class RecurringEvent {
    private final String name;
    private final LocalDate start;
    private final RecurrenceRule rule;

    public RecurringEvent(String name,
            LocalDate start, RecurrenceRule rule) {
        this.name = name;
        this.start = start;
        this.rule = rule;
    }

    public String getName() { return name; }

    public List<LocalDate> occurrences(
            LocalDate from, LocalDate to) {
        List<LocalDate> result = new ArrayList<>();
        LocalDate current = start;

        // startがfromより前なら、fromに追いつくまで進める
        while (current.isBefore(from)) {
            current = rule.nextOccurrence(current);
        }

        // from〜toの範囲で収集
        while (!current.isAfter(to)) {
            result.add(current);
            current = rule.nextOccurrence(current);
        }
        return result;
    }
}

class EventScheduler {
    private List<RecurringEvent> events
        = new ArrayList<>();

    public void addEvent(RecurringEvent event) {
        events.add(event);
    }

    public List<String> getSchedule(
            LocalDate from, LocalDate to) {
        // 日付 → イベント名リストを構築
        List<String[]> entries = new ArrayList<>();
        for (RecurringEvent event : events) {
            for (LocalDate date
                    : event.occurrences(from, to)) {
                entries.add(new String[]{
                    date.toString(), event.getName()});
            }
        }

        // 日付でソート
        entries.sort(Comparator.comparing(
            e -> e[0]));

        // 同一日のイベントをまとめる
        List<String> result = new ArrayList<>();
        String currentDate = "";
        StringBuilder sb = new StringBuilder();

        for (String[] entry : entries) {
            if (!entry[0].equals(currentDate)) {
                if (sb.length() > 0) {
                    result.add(sb.toString());
                }
                currentDate = entry[0];
                sb = new StringBuilder();
                sb.append(entry[0]).append(": ")
                  .append(entry[1]);
            } else {
                sb.append(", ").append(entry[1]);
            }
        }
        if (sb.length() > 0) {
            result.add(sb.toString());
        }
        return result;
    }
}

        System.out.println("週次MTG(月曜): "
            + weekly.occurrences(
                LocalDate.of(2025, 4, 1),
                LocalDate.of(2025, 4, 30)));

        // 月次イベント（31日指定 → 月末調整）
        System.out.println("=== 月次イベント（6か月） ===");
        RecurringEvent monthly = new RecurringEvent(
            "月末レポート",
            LocalDate.of(2025, 1, 31),
            new MonthlyRule(31));
        System.out.println("月末レポート(31日): "
            + monthly.occurrences(
                LocalDate.of(2025, 1, 1),
                LocalDate.of(2025, 6, 30)));

        // 統合スケジュール
        System.out.println(
            "=== 統合スケジュール（1週間） ===");
        EventScheduler scheduler
            = new EventScheduler();
        scheduler.addEvent(new RecurringEvent(
            "朝会",
            LocalDate.of(2025, 4, 1),
            new DailyRule()));
        scheduler.addEvent(new RecurringEvent(
            "週次MTG",
            LocalDate.of(2025, 4, 1),
            new WeeklyRule(DayOfWeek.MONDAY)));

        List<String> schedule
            = scheduler.getSchedule(
                LocalDate.of(2025, 4, 1),
                LocalDate.of(2025, 4, 7));
        for (String line : schedule) {
            System.out.println(line);
        }
    }
}
解説
  • RecurrenceRule インタフェースでポリモーフィズムを実現し、繰り返しルールの追加（隔週、年次等）を容易にしています。新しいルールは RecurrenceRule を実装するだけです。
  • LocalDate は不変クラスで、plusDays() / plusWeeks() / plusMonths() は新しいインスタンスを返します。JavaSilverでは日時APIの不変性が重要なポイントです。
  • MonthlyRule で Math.min(dayOfMonth, ym.lengthOfMonth()) とすることで、31日指定時に2月は28日（29日）、4月は30日に自動調整しています。
  • TemporalAdjusters.next(dayOfWeek) は「次の指定曜日」を返す調整子です。JavaSilverでは TemporalAdjusters の各メソッド（firstDayOfMonth, lastDayOfMonth 等）が出題されます。
  • YearMonth クラスは「年月」を表し、lengthOfMonth() でその月の日数、atDay() で特定日の LocalDate を取得できます。うるう年の判定にも対応しています。
