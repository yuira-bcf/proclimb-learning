assignment課題

長さ（m / km / mile）、重さ（kg / lb）、温度（℃ / ℉）を相互変換する UnitConverter クラスを実装してください。

enum LengthUnit { M, KM, MILE }
enum WeightUnit { KG, LB }

public static double convertLength(double value, LengthUnit from, LengthUnit to)
public static double convertWeight(double value, WeightUnit from, WeightUnit to)
public static double celsiusToFahrenheit(double c)
public static double fahrenheitToCelsius(double f)
public static String format(double value)
  • convertLength: 一度メートルに変換してから目的単位へ変換する（1 km = 1000 m, 1 mile = 1609.344 m）
  • convertWeight: 一度kgに変換してから目的単位へ変換する（1 lb = 0.453592 kg）
  • celsiusToFahrenheit: F = C * 9.0 / 5.0 + 32
  • fahrenheitToCelsius: C = (F - 32) * 5.0 / 9.0
  • format: 小数第2位まで四捨五入した文字列を返す
terminal期待出力
1.0 MILE = 1.61 KM
10.0 KG = 22.05 LB
100.0°C = 212.00°F
32.0°F = 0.00°C
lightbulbヒント

まず全ての値を基準単位（長さならメートル、重さならkg）に変換し、そこから目的単位に変換する「ハブ方式」が効率的です。switch 文で enum を分岐し、String.format("%.2f", value) で小数第2位まで整形できます。

check_circle解答・解説 
public class UnitConverter {

    enum LengthUnit { M, KM, MILE }
    enum WeightUnit { KG, LB }

    public static double convertLength(double value, LengthUnit from, LengthUnit to) {
        // まずメートルに変換
        double inMeters = switch (from) {
            case M    -> value;
            case KM   -> value * 1000.0;
            case MILE -> value * 1609.344;
        };
        // メートルから目的単位に変換
        return switch (to) {
            case M    -> inMeters;
            case KM   -> inMeters / 1000.0;
            case MILE -> inMeters / 1609.344;
        };
    }

    public static double convertWeight(double value, WeightUnit from, WeightUnit to) {
        double inKg = switch (from) {
            case KG -> value;
            case LB -> value * 0.453592;
        };
        return switch (to) {
            case KG -> inKg;
            case LB -> inKg / 0.453592;
        };
    }

    public static double celsiusToFahrenheit(double c) {
        return c * 9.0 / 5.0 + 32;
    }

    public static double fahrenheitToCelsius(double f) {
        return (f - 32) * 5.0 / 9.0;
    }

    public static String format(double value) {
        return String.format("%.2f", value);
    }

    public static void main(String[] args) {
        double miles = convertLength(1.0, LengthUnit.MILE, LengthUnit.KM);
        System.out.println("1.0 MILE = " + format(miles) + " KM");

        double lbs = convertWeight(10.0, WeightUnit.KG, WeightUnit.LB);
        System.out.println("10.0 KG = " + format(lbs) + " LB");

        double f = celsiusToFahrenheit(100.0);
        System.out.println("100.0°C = " + format(f) + "°F");

        double c = fahrenheitToCelsius(32.0);
        System.out.println("32.0°F = " + format(c) + "°C");
    }
}
解説
  • 「ハブ方式」により、N種類の単位間変換を N×2 のケースで実装できます。直接変換だと N×(N-1) ケース必要になり、単位追加時のコストが大きくなります。
  • Java 14以降の switch式（アロー構文）を使うと、各ケースが値を返す式として記述でき、break や変数宣言が不要になります。
  • enum を switch で分岐する場合、全ケースを網羅していないとコンパイルエラーになります（switch式の場合）。これにより単位追加時の実装漏れを防げます。
  • String.format("%.2f", value) は小数第2位まで表示します。JavaSilverでは printf / format の書式指定子が出題されます。
  • double 同士の演算では浮動小数点誤差が生じる可能性があります。厳密な計算が必要な場合は BigDecimal を使用します。

"3+5*2" のような文字列式をパースして計算する ExprParser クラスを実装してください。

public static int evaluate(String expr)
  • 対応する演算子: +, -, *, /
  • 演算子の優先順位を正しく処理すること（* と / は + と - より先に計算）
  • 数値は1桁以上の正の整数（例: 3, 12, 100）
  • スペースは含まれない前提
  • ゼロ除算時は ArithmeticException をスローする
terminal期待出力
3+5*2 = 13
10-2*3+4 = 8
100/10/2 = 5
1+2+3+4+5 = 15
lightbulbヒント

2パス方式がシンプルです。第1パス: 式を数値と演算子に分解して ArrayList に格納。第2パス: まず * と / を処理し、次に + と - を処理します。Character.isDigit(ch) で数字判定ができます。

check_circle解答・解説 
import java.util.ArrayList;
import java.util.List;

public class ExprParser {

    public static int evaluate(String expr) {
        // 第1パス: トークン分解
        List<Integer> numbers = new ArrayList<>();
        List<Character> operators = new ArrayList<>();

        int num = 0;
        for (int i = 0; i < expr.length(); i++) {
            char ch = expr.charAt(i);
            if (Character.isDigit(ch)) {
                num = num * 10 + (ch - '0');
            } else {
                numbers.add(num);
                operators.add(ch);
                num = 0;
            }
        }
        numbers.add(num); // 最後の数値

        // 第2パス: * と / を先に処理
        for (int i = 0; i < operators.size(); i++) {
            char op = operators.get(i);
            if (op == '*' || op == '/') {
                int left = numbers.get(i);
                int right = numbers.get(i + 1);
                int result = (op == '*') ? left * right : left / right;
                numbers.set(i, result);
                numbers.remove(i + 1);
                operators.remove(i);
                i--; // インデックス調整
            }
        }

        // 第3パス: + と - を処理
        int result = numbers.get(0);
        for (int i = 0; i < operators.size(); i++) {
            char op = operators.get(i);
            int right = numbers.get(i + 1);
            result = (op == '+') ? result + right : result - right;
        }

        return result;
    }

    public static void main(String[] args) {
        String[] tests = {"3+5*2", "10-2*3+4", "100/10/2", "1+2+3+4+5"};
        for (String expr : tests) {
            System.out.println(expr + " = " + evaluate(expr));
        }
    }
}
解説
  • ch - '0' で文字を数値に変換できます。'3' - '0' は 51 - 48 = 3 です。char は内部的に整数値（Unicode）として扱われます。
  • num * 10 + (ch - '0') で複数桁の数値を構築します。"12" なら 0*10+1=1, 1*10+2=12 となります。
  • 2パス方式では、先に * と / を処理してリストを縮小してから + と - を処理することで、演算子の優先順位を実現しています。
  • ArrayList.remove(index) は要素を除去し、後続要素のインデックスを詰めます。そのため i-- でインデックスを調整する必要があります。
  • JavaSilverでは Character.isDigit(), Character.isLetter() 等のユーティリティメソッドが出題されます。

String.split("\n") で行分割、String.split(",") で列分割します。Collections.sort() に Comparator.comparing() を渡すとカスタムソートできます。年齢列は Integer.parseInt() で数値変換してからソートしましょう。

check_circle解答・解説 
import java.util.ArrayList;
import java.util.List;

public class CsvParser {

    public static List<String[]> parse(String csv) {
        List<String[]> rows = new ArrayList<>();
        String[] lines = csv.split("\n");
        for (String line : lines) {
            rows.add(line.split(","));
        }
        return rows;
    }

    public static List<String[]> filterByColumn(
            List<String[]> rows, int col, String value) {
        List<String[]> result = new ArrayList<>();
        result.add(rows.get(0)); // ヘッダー行
        for (int i = 1; i < rows.size(); i++) {
            if (rows.get(i)[col].equals(value)) {
                result.add(rows.get(i));
            }
        }
        return result;
    }

    public static void sortByColumn(List<String[]> rows, int col) {
        // ヘッダー行（index 0）を除いてソート
        List<String[]> dataRows = rows.subList(1, rows.size());
        dataRows.sort((a, b) -> {
            try {
                int numA = Integer.parseInt(a[col]);
                int numB = Integer.parseInt(b[col]);
                return Integer.compare(numA, numB);
            } catch (NumberFormatException e) {
                return a[col].compareTo(b[col]);
            }
        });
    }

    public static void printTable(List<String[]> rows) {
        for (String[] row : rows) {
            System.out.println(String.join("\t", row));
        }
    }

    public static void main(String[] args) {
        String csv = "名前,部署,年齢\n田中,開発,28\n佐藤,営業,35\n"
                   + "鈴木,開発,24\n高橋,営業,31\n伊藤,開発,30";

        List<String[]> all = parse(csv);
        List<String[]> devTeam = filterByColumn(all, 1, "開発");
        sortByColumn(devTeam, 2); // 年齢列でソート

        System.out.println("--- 開発部のメンバー（年齢順） ---");
        printTable(devTeam);
    }
}
解説
  • split(",") は正規表現を引数に取ります。split("\\.") のようにメタ文字はエスケープが必要です。JavaSilverでは split の正規表現引数が頻出です。
  • subList() は元リストのビューを返します。ビューへの変更は元リストに反映されるため、ヘッダー行を除いたソートが可能です。
  • ラムダ式 (a, b) -> ... は Comparator<String[]> の compare メソッドを実装しています。JavaSilverではラムダ式と関数型インターフェースの関係が問われます。
  • Integer.parseInt() は変換不能な文字列に対して NumberFormatException をスローします。try-catch で数値ソートと文字列ソートを自動判別しています。
  • String.join() は区切り文字で配列や Iterable を結合する便利メソッドです。

"Hello, ${name}!" 形式のテンプレート文字列を変数マップで置換する TemplateEngine クラスを実装してください。

public static String render(String template, Map<String, String> vars)
public static Map<String, String> createContext(String... keyValues)
  • render: テンプレート内の ${key} を vars マップの対応する値で置換する
  • マップに存在しないキーは ${key} のまま残す
  • ネスト ${${x}} は未対応でよい
  • createContext: "key1", "value1", "key2", "value2" のように交互に渡されたキーと値から Map を生成する
terminal期待出力
Hello, 田中さん! Welcome to Java.
Your score is 95 / 100.
Unknown key: ${missing}
lightbulbヒント

indexOf("${") で開始位置を見つけ、indexOf("}", start) で終了位置を見つけます。substring(start + 2, end) でキー名を抽出し、Map.getOrDefault(key, original) で置換します。StringBuilder を使って効率的に組み立てましょう。

check_circle解答・解説 
import java.util.HashMap;
import java.util.Map;

public class TemplateEngine {

    public static String render(String template, Map<String, String> vars) {
        StringBuilder sb = new StringBuilder();
        int i = 0;

        while (i < template.length()) {
            int start = template.indexOf("${", i);
            if (start == -1) {
                sb.append(template.substring(i));
                break;
            }
            int end = template.indexOf("}", start);
            if (end == -1) {
                sb.append(template.substring(i));
                break;
            }

            sb.append(template, i, start); // ${ の前まで追加
            String key = template.substring(start + 2, end);
            String original = template.substring(start, end + 1);
            sb.append(vars.getOrDefault(key, original));
            i = end + 1;
        }

        return sb.toString();
    }

    public static Map<String, String> createContext(String... keyValues) {
        Map<String, String> map = new HashMap<>();
        for (int i = 0; i < keyValues.length - 1; i += 2) {
            map.put(keyValues[i], keyValues[i + 1]);
        }
        return map;
    }

    public static void main(String[] args) {
        Map<String, String> vars = createContext(
            "name", "田中さん",
            "lang", "Java",
            "score", "95",
            "total", "100"
        );

        System.out.println(render("Hello, ${name}! Welcome to ${lang}.", vars));
        System.out.println(render("Your score is ${score} / ${total}.", vars));
        System.out.println(render("Unknown key: ${missing}", vars));
    }
}
解説
  • indexOf("${", i) の第2引数で検索開始位置を指定できます。これにより文字列を先頭から順にスキャンしながら全ての ${...} を見つけられます。
  • StringBuilder.append(CharSequence, start, end) は部分文字列を効率的に追加できます。substring() で新たな String を生成するよりメモリ効率が良いです。
  • Map.getOrDefault(key, default) はキーが存在しない場合にデフォルト値を返します。get() + null チェックより簡潔です。
  • 可変長引数 String... keyValues はメソッド内では String[] として扱えます。JavaSilverでは可変長引数の型と配列の互換性が問われます。
  • HashMap はキーの順序を保証しません。順序が必要な場合は LinkedHashMap を使います。

転置行列では元の [i][j] が新しい [j][i] になります。新しい配列のサイズは new int[a[0].length][a.length] です。System.out.printf("%4d", value) で右寄せ表示できます。

check_circle解答・解説 
public class Matrix {

    public static int[][] add(int[][] a, int[][] b) {
        int rows = a.length;
        int cols = a[0].length;
        int[][] result = new int[rows][cols];
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                result[i][j] = a[i][j] + b[i][j];
            }
        }
        return result;
    }

    public static int[][] transpose(int[][] a) {
        int rows = a.length;
        int cols = a[0].length;
        int[][] result = new int[cols][rows];
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                result[j][i] = a[i][j];
            }
        }
        return result;
    }

    public static int[][] scalarMultiply(int[][] a, int scalar) {
        int rows = a.length;
        int cols = a[0].length;
        int[][] result = new int[rows][cols];
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                result[i][j] = a[i][j] * scalar;
            }
        }
        return result;
    }

    public static void print(int[][] m) {
        for (int[] row : m) {
            for (int val : row) {
                System.out.printf("%4d", val);
            }
            System.out.println();
        }
    }

    public static void main(String[] args) {
        int[][] a = {{1, 2, 3}, {4, 5, 6}};
        int[][] b = {{2, 3, 4}, {5, 6, 7}};

        System.out.println("--- 加算 A + B ---");
        print(add(a, b));

        System.out.println("--- 転置 A ---");
        print(transpose(a));

        System.out.println("--- スカラー倍 A * 3 ---");
        print(scalarMultiply(a, 3));
    }
}
解説
  • 2次元配列 int[][] a は「int 配列の配列」です。a.length は行数、a[0].length は列数を返します。JavaSilverでは配列の .length がフィールドであること（メソッドではない）が問われます。
  • 転置行列では new int[cols][rows] と行列サイズが逆になります。result[j][i] = a[i][j] がポイントです。
  • 各メソッドは新しい配列を返す（非破壊的操作）設計にしています。元の配列を変更しないため安全に使い回せます。
  • 拡張 for 文（for-each）は2次元配列にも使えます。外側で int[] row を取り出し、内側で各 int val を取り出します。
  • printf("%4d", val) は最低4文字幅で右寄せ表示します。表形式の出力に便利です。

import java.util.Arrays;

public class SortComparison {

    static int compareCount;
    static int swapCount;

    public static void bubbleSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < n - 1 - i; j++) {
                compareCount++;
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                    swapCount++;
                }
            }
        }
    }

    public static void selectionSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            int minIdx = i;
            for (int j = i + 1; j < n; j++) {
                compareCount++;
                if (arr[j] < arr[minIdx]) {
                    minIdx = j;
                }
            }
            if (minIdx != i) {
                int temp = arr[i];
                arr[i] = arr[minIdx];
                arr[minIdx] = temp;
                swapCount++;
            }
        }
    }

    public static void resetCounters() {
        compareCount = 0;
        swapCount = 0;
    }

    public static String getStats() {
        return "比較: " + compareCount + "回, 交換: " + swapCount + "回";
    }

    public static void main(String[] args) {
        int[] original = {64, 34, 25, 12, 22, 11, 90};
        System.out.println("元の配列: " + Arrays.toString(original));

        int[] arr1 = Arrays.copyOf(original, original.length);
        resetCounters();
        bubbleSort(arr1);
        System.out.println("--- バブルソート ---");
        System.out.println("結果: " + Arrays.toString(arr1));
        System.out.println(getStats());

        int[] arr2 = Arrays.copyOf(original, original.length);
        resetCounters();
        selectionSort(arr2);
        System.out.println("--- 選択ソート ---");
        System.out.println("結果: " + Arrays.toString(arr2));
        System.out.println(getStats());
    }
}
解説
  • バブルソートの内側ループ上限 n - 1 - i は、i回目のパスで末尾i個が確定済みであるため走査不要だからです。
  • 選択ソートは比較回数がバブルソートと同じですが、交換回数が少ないのが特徴です。各パスで最大1回しか交換しないためです。
  • Arrays.copyOf(arr, arr.length) はシャローコピー（浅いコピー）を返します。int[] はプリミティブ型配列なので問題ありませんが、オブジェクト配列の場合は参照がコピーされるだけです。
  • Arrays.toString() は配列を [要素1, 要素2, ...] 形式の文字列に変換します。配列を直接 System.out.println() に渡すとハッシュコードが表示されてしまいます。
  • static フィールドはクラスに属するため、インスタンスを作成せずにアクセスできます。JavaSilverでは static メンバーのアクセス方法が問われます。

public class CalendarPrinter {

    public static boolean isLeapYear(int year) {
        return (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
    }

    public static int getDaysInMonth(int year, int month) {
        int[] days = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
        if (month == 2 && isLeapYear(year)) {
            return 29;
        }
        return days[month];
    }

    // Zellerの公式: 0=日, 1=月, ..., 6=土
    public static int getDayOfWeek(int year, int month, int day) {
        if (month <= 2) {
            month += 12;
            year--;
        }
        int h = (day + 13 * (month + 1) / 5
                + year + year / 4 - year / 100 + year / 400) % 7;
        // Zellerの公式: h=0が土曜 → 日曜始まりに変換
        return (h + 6) % 7;
    }

    public static void printCalendar(int year, int month) {
        System.out.printf("    %d年 %d月%n", year, month);
        System.out.println(" 日  月  火  水  木  金  土");

        int startDay = getDayOfWeek(year, month, 1);
        int daysInMonth = getDaysInMonth(year, month);

        // 1日の前にスペースを出力
        for (int i = 0; i < startDay; i++) {
            System.out.print("    ");
        }

        for (int day = 1; day <= daysInMonth; day++) {
            System.out.printf("%3d ", day);
            if ((startDay + day) % 7 == 0) {
                System.out.println();
            }
        }
        System.out.println();
    }

    public static void main(String[] args) {
        printCalendar(2026, 3);
    }
}

import java.util.HashMap;
import java.util.Map;

public class MiniInterpreter {

    private Map<String, Integer> variables = new HashMap<>();

    public String execute(String line) {
        line = line.trim();

        // 代入文: x=5
        if (line.contains("=") && !line.contains("==")) {
            String[] parts = line.split("=", 2);
            String varName = parts[0].trim();
            try {
                int value = resolveValue(parts[1].trim());
                variables.put(varName, value);
                return varName + " = " + value;
            } catch (Exception e) {
                return "ERROR: " + e.getMessage();
            }
        }

        // 演算式: x+3, x*y など
        char[] ops = {'+', '-', '*', '/'};
        for (char op : ops) {
            int idx = line.indexOf(op);
            // 先頭の '-' は負数の可能性があるのでスキップ
            if (idx > 0) {
                String left = line.substring(0, idx).trim();
                String right = line.substring(idx + 1).trim();
                try {
                    int lVal = resolveValue(left);
                    int rVal = resolveValue(right);
                    return String.valueOf(compute(lVal, op, rVal));
                } catch (ArithmeticException e) {
                    return "ERROR: division by zero";
                } catch (Exception e) {
                    return "ERROR: " + e.getMessage();
                }
            }
        }

        // 単一変数参照: x
        try {
            return String.valueOf(resolveValue(line));
        } catch (Exception e) {
            return "ERROR: " + e.getMessage();
        }
    }

    private int resolveValue(String token) {
        try {
            return Integer.parseInt(token);
        } catch (NumberFormatException e) {
            if (variables.containsKey(token)) {
                return variables.get(token);
            }
            throw new RuntimeException(
                "undefined variable '" + token + "'");
        }
    }

    private int compute(int left, char op, int right) {
        return switch (op) {
            case '+' -> left + right;
            case '-' -> left - right;
            case '*' -> left * right;
            case '/' -> left / right; // ゼロ除算は ArithmeticException
            default  -> throw new RuntimeException(
                "unknown operator: " + op);
        };
    }

    public static void main(String[] args) {
        MiniInterpreter interp = new MiniInterpreter();
        String[] commands = {
            "x=10", "y=3", "x+y", "x*y", "x/y", "x-y", "z", "x/0"
        };
        for (String cmd : commands) {
            System.out.println("> " + cmd);
            System.out.println(interp.execute(cmd));
        }
    }
}
解説
  • HashMap は変数名と値のペアを保持するのに最適です。containsKey() で存在チェック、put() で上書き代入ができます。
  • split("=", 2) の第2引数は分割数の上限です。"x=1+2" を ["x", "1+2"] と分割でき、値部分に = が含まれていても正しく動作します。
  • int 同士の除算でゼロ除算すると ArithmeticException がスローされます。double の場合は Infinity や NaN になり例外は発生しません。
  • Integer.parseInt() が失敗したら変数名とみなすパターンは、トークンの種類判定に try-catch を使う実践的なテクニックです。
  • switch 式の default ケースにより、未知の演算子に対する安全策を設けています。JavaSilverでは switch の default の位置と実行順序が問われます。

        this.amount = amount;

class BankAccount {
    private static int nextAccountNo = 1001;
    private final int accountNo;
    private int balance;
    private ArrayList<Transaction> history;

    public BankAccount(int initialBalance) {
        this.accountNo = nextAccountNo++;
        this.balance = initialBalance;
        this.history = new ArrayList<>();
    }

    public void deposit(int amount) {
        if (amount <= 0) {
            throw new IllegalArgumentException("入金額は正の値にしてください");
        }
        balance += amount;
        history.add(new Transaction("DEPOSIT", amount, balance));
        System.out.println("入金: " + amount + " → 残高: " + balance);
    }

    public void withdraw(int amount) {
        if (amount <= 0) {
            throw new IllegalArgumentException("出金額は正の値にしてください");
        }
        if (amount > balance) {
            throw new IllegalArgumentException("残高不足です");
        }
        balance -= amount;
        history.add(new Transaction("WITHDRAW", amount, balance));
        System.out.println("出金: " + amount + " → 残高: " + balance);
    }

    public int getBalance() {
        return balance;
    }

    public void printHistory() {
        System.out.println("--- 取引履歴 ---");
        for (Transaction t : history) {
            System.out.println(t);
        }
    }

    @Override
    public String toString() {
        return "口座番号: " + accountNo + " (残高: " + balance + ")";
    }
}

import java.util.ArrayList;

class Book {
    String title;
    String author;
    String isbn;

    Book(String title, String author, String isbn) {
        this.title = title;
        this.author = author;
        this.isbn = isbn;
    }

    @Override
    public String toString() {
        return title + " / " + author + " [" + isbn + "]";
    }
}

class LibraryCard {
    private static final int MAX_BORROW = 5;
    private String ownerName;
    private ArrayList<Book> borrowedBooks;

    LibraryCard(String ownerName) {
        this.ownerName = ownerName;
        this.borrowedBooks = new ArrayList<>();
    }

    public void borrowBook(Book book) {
        if (borrowedBooks.size() >= MAX_BORROW) {
            throw new IllegalStateException(
                "貸出上限(" + MAX_BORROW + "冊)に達しています");
        }
        borrowedBooks.add(book);
        System.out.println("貸出: " + book.title + " (ISBN: " + book.isbn + ")");
    }

    public boolean returnBook(String isbn) {
        for (int i = 0; i < borrowedBooks.size(); i++) {
            if (borrowedBooks.get(i).isbn.equals(isbn)) {
                Book removed = borrowedBooks.remove(i);
                System.out.println("返却: " + removed.title + " → OK");
                return true;
            }
        }
        return false;
    }

    public Book searchByTitle(String keyword) {
        for (Book b : borrowedBooks) {
            if (b.title.contains(keyword)) {
                return b;
            }
        }
        return null;
    }

    public ArrayList<Book> searchByAuthor(String author) {
        ArrayList<Book> result = new ArrayList<>();
        for (Book b : borrowedBooks) {
            if (b.author.equals(author)) {
                result.add(b);
            }
        }
        return result;
    }

    public void printBorrowedBooks() {
        System.out.println("--- 貸出一覧 (" + borrowedBooks.size()
            + "/" + MAX_BORROW + "冊) ---");
        for (int i = 0; i < borrowedBooks.size(); i++) {
            System.out.println((i + 1) + ". " + borrowedBooks.get(i));
        }
    }
}

解説
  • static final int MAX_BORROW = 5 は定数です。static final フィールドは慣例として大文字スネークケースで命名します。JavaSilverでは final の意味（再代入不可）が問われます。
  • searchByTitle と searchByAuthor は引数の型は同じ String ですが、メソッド名が異なるため別メソッドです。オーバーロードは「同名・異なる引数リスト」で成立する点に注意してください。

import java.util.ArrayList;
import java.util.Collections;

interface Shape extends Comparable<Shape> {
    double area();
    double perimeter();
    String getName();

    @Override
    default int compareTo(Shape other) {
        return Double.compare(this.area(), other.area());
    }
}

class Circle implements Shape {
    private double radius;

    Circle(double radius) {
        this.radius = radius;
    }

    @Override
    public double area() {
        return Math.PI * radius * radius;
    }

    @Override
    public double perimeter() {
        return 2 * Math.PI * radius;
    }

    @Override
    public String getName() {
        return "Circle";
    }
}

class Rectangle implements Shape {
    private double width;
    private double height;

    Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }

    @Override
    public double area() {
        return width * height;
    }

    @Override
    public double perimeter() {
        return 2 * (width + height);
    }

    @Override
    public String getName() {
        return "Rectangle";
    }
}

class Triangle implements Shape {
    private double a, b, c;

    Triangle(double a, double b, double c) {
        this.a = a;
        this.b = b;
        this.c = c;
    }

    @Override
    public double area() {
        double s = (a + b + c) / 2.0;
        return Math.sqrt(s * (s - a) * (s - b) * (s - c));
    }

    @Override
    public double perimeter() {
        return a + b + c;
    }

    @Override
    public String getName() {
        return "Triangle";
    }
}

    public static double totalArea(ArrayList<Shape> shapes) {
        double total = 0;
        for (Shape s : shapes) {
            total += s.area();
        }
        return total;
    }

    public static Shape largestShape(ArrayList<Shape> shapes) {
        Shape largest = shapes.get(0);
        for (Shape s : shapes) {
            if (s.area() > largest.area()) {
                largest = s;
            }
        }
        return largest;
    }

import java.util.ArrayList;
import java.util.HashMap;
import java.util.Map;

class ValidationException extends Exception {
    private String fieldName;
    private String errorMessage;

    ValidationException(String fieldName, String errorMessage) {
        super(errorMessage);
        this.fieldName = fieldName;
        this.errorMessage = errorMessage;
    }

    public String getFieldName() { return fieldName; }
    public String getErrorMessage() { return errorMessage; }
}

interface Validator {
    void validate(String fieldName, String value) throws ValidationException;
}

class StringValidator implements Validator {
    private int minLength;
    private int maxLength;

    StringValidator(int minLength, int maxLength) {
        this.minLength = minLength;
        this.maxLength = maxLength;
    }

    @Override
    public void validate(String fieldName, String value) throws ValidationException {
        if (value == null || value.isEmpty()) {
            throw new ValidationException(fieldName,
                "「" + fieldName + "」は必須です");
        }
        if (value.length() < minLength || value.length() > maxLength) {
            throw new ValidationException(fieldName,
                "「" + fieldName + "」は" + minLength + "~" + maxLength
                    + "文字で入力してください: " + value);
        }
    }
}

class NumberRangeValidator implements Validator {
    private int min;
    private int max;

    NumberRangeValidator(int min, int max) {
        this.min = min;
        this.max = max;
    }

    @Override
    public void validate(String fieldName, String value) throws ValidationException {
        int num;
        try {
            num = Integer.parseInt(value);
        } catch (NumberFormatException e) {
            throw new ValidationException(fieldName,
                "「" + fieldName + "」は数値で入力してください: " + value);
        }
        if (num < min || num > max) {
            throw new ValidationException(fieldName,
                "「" + fieldName + "」の値が範囲外です (" + min + "~" + max + "): " + value);
        }
    }
}

class EmailValidator implements Validator {
    @Override
    public void validate(String fieldName, String value) throws ValidationException {
        if (value == null || value.isEmpty()) {
            throw new ValidationException(fieldName,
                "「" + fieldName + "」は必須です");
        }
        int atIndex = value.indexOf('@');
        if (atIndex <= 0 || atIndex >= value.length() - 1) {
            throw new ValidationException(fieldName,
                "「" + fieldName + "」のメール形式が不正です: " + value);
        }
    }
}

class ValidationRunner {
    public static ArrayList<ValidationException> runAll(
            Map<String, String> fields,
            Map<String, Validator> validators) {
        ArrayList<ValidationException> errors = new ArrayList<>();
        for (Map.Entry<String, Validator> entry : validators.entrySet()) {
            String fieldName = entry.getKey();
            String value = fields.getOrDefault(fieldName, "");
            try {
                entry.getValue().validate(fieldName, value);
                System.out.println("[OK] " + fieldName + ": " + value);
            } catch (ValidationException e) {
                System.out.println("[NG] " + fieldName + ": " + e.getErrorMessage());
                errors.add(e);
            }
        }
        return errors;
    }
}

    public static void main(String[] args) {

import java.util.Random;

class RetryExhaustedException extends RuntimeException {
    private int attempts;

    RetryExhaustedException(int attempts) {
        super(attempts + "回の試行が全て失敗しました");
        this.attempts = attempts;
    }

    public int getAttempts() {
        return attempts;
    }
}

class UnstableTask implements Runnable {
    private String taskName;
    private double failRate;
    private Random random;

    UnstableTask(String taskName, double failRate, long seed) {
        this.taskName = taskName;
        this.failRate = failRate;
        this.random = new Random(seed);
    }

    @Override
    public void run() {
        if (random.nextDouble() < failRate) {
            throw new RuntimeException("ランダム障害が発生しました");
        }
    }

    public String getTaskName() { return taskName; }
    public double getFailRate() { return failRate; }
}

class RetryExecutor {
    private int maxRetries;

    RetryExecutor(int maxRetries) {
        this.maxRetries = maxRetries;
    }

    public void execute(Runnable task) throws RetryExhaustedException {
        for (int i = 1; i <= maxRetries; i++) {
            try {
                task.run();
                System.out.println("[試行 " + i + "/" + maxRetries + "] 成功!");
                return;
            } catch (RuntimeException e) {
                System.out.println("[試行 " + i + "/" + maxRetries + "] 失敗: " + e.getMessage());
            }
        }
        throw new RetryExhaustedException(maxRetries);
    }
}

    public static void main(String[] args) {
        RetryExecutor executor = new RetryExecutor(3);

        // タスク1: 失敗率50% — シード値42で2回目に成功する
        UnstableTask task1 = new UnstableTask("データ送信", 0.5, 42);

解説
  • extends RuntimeException で非検査例外を定義しています。非検査例外は throws 宣言が任意です。ただし今回は明示的に宣言することで呼び出し側に失敗の可能性を伝えています。
  • Runnable は void run() だけを持つ関数型インタフェースです。ラムダ式 () -> { ... } でも実装できます。JavaSilverでは関数型インタフェースの定義と使用が出題されます。
  • Random.nextDouble() は 0.0 以上 1.0 未満の乱数を返します。シード値を指定すると同じ乱数列が再現されるため、テストに有用です。
  • for ループ内の return はループと execute メソッド全体を終了します。成功時に即座に抜けるこのパターンは「早期リターン」と呼ばれ、コードの可読性を高めます。
  • try-catch で RuntimeException を捕捉していますが、本来 RuntimeException は捕捉しなくてもコンパイルできます。明示的に catch するのはリトライロジックの要件です。

import java.util.ArrayList;
import java.util.HashMap;

class Product {
    String name;
    String category;
    int price;
    int stock;

    Product(String name, String category, int price, int stock) {
        this.name = name;
        this.category = category;
        this.price = price;
        this.stock = stock;
    }

    @Override
    public String toString() {
        return name + " (" + category + ") " + price + "円 在庫:" + stock;
    }
}

class InventoryManager {
    private HashMap<String, ArrayList<Product>> inventory = new HashMap<>();

    public void addProduct(Product product) {
        inventory.computeIfAbsent(product.category, k -> new ArrayList<>())
                 .add(product);
    }

    public boolean removeProduct(String name) {
        for (ArrayList<Product> products : inventory.values()) {
            for (int i = 0; i < products.size(); i++) {
                if (products.get(i).name.equals(name)) {
                    products.remove(i);
                    return true;
                }
            }
        }
        return false;
    }

    public ArrayList<Product> searchByCategory(String category) {
        return inventory.getOrDefault(category, new ArrayList<>());
    }

    public ArrayList<Product> filterByPrice(int maxPrice) {
        ArrayList<Product> result = new ArrayList<>();
        for (ArrayList<Product> products : inventory.values()) {
            products.forEach(p -> {
                if (p.price <= maxPrice) {
                    result.add(p);
                }
            });
        }
        result.sort((a, b) -> Integer.compare(a.price, b.price));
        return result;
    }

    public ArrayList<Product> getOutOfStock() {
        ArrayList<Product> result = new ArrayList<>();
        inventory.values().forEach(products ->
            products.forEach(p -> {
                if (p.stock == 0) {
                    result.add(p);
                }
            })
        );
        return result;
    }

    public void printCategorySummary() {
        System.out.println("=== カテゴリ別サマリー ===");
        inventory.forEach((category, products) -> {
            int totalStock = 0;
            for (Product p : products) {
                totalStock += p.stock;
            }
            System.out.printf("%-6s : %d商品 / 在庫合計 %d個%n",
                category, products.size(), totalStock);
        });
    }
}

解説
  • computeIfAbsent(key, mappingFunction) はキーが存在しない場合にのみ関数を実行して値を登録します。if (!map.containsKey(key)) { map.put(key, new ArrayList<>()); } の1行版です。
  • HashMap の values() は全ての値（ここでは ArrayList<Product>）のコレクションを返します。ネストした for 文で全商品を走査できます。
  • forEach() と ラムダ式 p -> { ... } は拡張for文の代替です。Iterable インタフェースの default メソッドとして Java 8 で追加されました。
  • sort((a, b) -> Integer.compare(a.price, b.price)) はラムダ式で Comparator を指定するソートです。Integer.compare() は安全な int 比較メソッドです。
  • getOrDefault(key, defaultValue) はキーが存在しない場合にデフォルト値を返します。null チェックの代わりに使うことで NullPointerException を防げます。

import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
import java.util.ArrayList;

class Event {
    String title;
    LocalDateTime start;
    LocalDateTime end;

    Event(String title, LocalDateTime start, LocalDateTime end) {
        this.title = title;
        this.start = start;
        this.end = end;
    }

    public boolean overlapsWith(Event other) {
        return this.start.isBefore(other.end) && other.start.isBefore(this.end);
    }

    @Override
    public String toString() {
        DateTimeFormatter fmt = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm");
        return start.format(fmt) + " ~ " + end.format(fmt).substring(11) + "  " + title;
    }
}

class Scheduler {
    private ArrayList<Event> events = new ArrayList<>();

    public void addEvent(Event event) {
        for (Event existing : events) {
            if (existing.overlapsWith(event)) {
                throw new IllegalArgumentException(
                    "既存イベント「" + existing.title + "」と時間が重複しています");
            }
        }
        events.add(event);
    }

    public ArrayList<Event> getEventsOnDate(LocalDate date) {
        ArrayList<Event> result = new ArrayList<>();
        events.forEach(e -> {
            if (e.start.toLocalDate().equals(date)) {
                result.add(e);
            }
        });
        return result;
    }

    public ArrayList<Event> getEventsBetween(LocalDateTime from, LocalDateTime to) {
        ArrayList<Event> result = new ArrayList<>();
        events.forEach(e -> {
            if (!e.start.isBefore(from) && e.start.isBefore(to)) {
                result.add(e);
            }
        });
        result.sort((a, b) -> a.start.compareTo(b.start));
        return result;
    }

    public ArrayList<Event> getTodayEvents() {
        return getEventsOnDate(LocalDate.now());
    }

    public void printSchedule() {
        System.out.println("=== スケジュール一覧 ===");
        ArrayList<Event> sorted = new ArrayList<>(events);
        sorted.sort((a, b) -> a.start.compareTo(b.start));
        for (Event e : sorted) {
            System.out.println(e);
        }
    }
}

解説
  • LocalDateTime は不変（イミュータブル）クラスで、日付と時刻を保持します。of() ファクトリメソッドで生成し、isBefore() / isAfter() で比較します。JavaSilverでは日時APIの使い方が頻出です。
  • 時間帯の重複判定 s1.isBefore(e2) && s2.isBefore(e1) は区間の重なり検出の定番アルゴリズムです。「片方の開始が他方の終了より前」が両方成り立てば重複しています。
  • toLocalDate() で LocalDateTime から LocalDate を抽出できます。時刻部分を無視して日付だけで比較したい場合に便利です。
  • DateTimeFormatter.ofPattern() でカスタム書式を指定できます。yyyy-MM-dd HH:mm は「年-月-日 時:分」形式です。M は月、m は分なので大文字・小文字の区別に注意が必要です。
  • sort((a, b) -> a.start.compareTo(b.start)) はラムダ式で Comparator を指定しています。LocalDateTime は Comparable を実装しているため、compareTo() で自然順序（時系列順）の比較ができます。
