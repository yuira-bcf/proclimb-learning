通貨換算エンジン

verified
JavaSilverポイント

enum の定義・フィールド・コンストラクタ、double の精度、三項演算子の評価順序はJavaSilverで頻出です。enum は暗黙的に java.lang.Enum を継承し、values() や name() が使えます。

assignment課題

enum Currency と CurrencyConverter クラスを実装してください。

  • Currency は JPY, USD, EUR の3つの定数を持つ
  • 各定数は「対USDレート」を double フィールドで保持する（JPY=150.0, USD=1.0, EUR=0.92）
  • convert(double amount, Currency from, Currency to) メソッドで通貨変換を行う
  • 変換結果は小数第2位まで四捨五入する
  • 同一通貨の場合は三項演算子で即座に元の金額を返す
terminal期待出力
100.00 USD -> 15000.00 JPY
250.00 USD -> 230.00 EUR
10000.00 JPY -> 66.67 USD
lightbulbヒント

通貨変換の考え方: まず from 通貨の金額をUSDに変換し（amount / from.rate）、その後 to 通貨に変換します（* to.rate）。四捨五入は Math.round(value * 100.0) / 100.0 で実現できます。

check_circle解答・解説 
public class CurrencyConverter {

    enum Currency {
        JPY(150.0), USD(1.0), EUR(0.92);

        final double rate; // 対USDレート

        Currency(double rate) {
            this.rate = rate;
        }
    }

    public static double convert(double amount, Currency from, Currency to) {
        // 同一通貨ならそのまま返す
        return (from == to)
            ? amount
            : Math.round(amount / from.rate * to.rate * 100.0) / 100.0;
    }

    public static void main(String[] args) {
        System.out.printf("100.00 USD -> %.2f JPY%n",
            convert(100, Currency.USD, Currency.JPY));
        System.out.printf("250.00 USD -> %.2f EUR%n",
            convert(250, Currency.USD, Currency.EUR));
        System.out.printf("10000.00 JPY -> %.2f USD%n",
            convert(10000, Currency.JPY, Currency.USD));
    }
}
解説
  • enum にフィールドとコンストラクタを持たせることで、各定数に固有のデータを紐付けられます。
  • 三項演算子 (条件) ? 値A : 値B は if-else の簡潔な代替です。JavaSilverでは評価順序や型の一致が問われます。
  • double の丸め処理で Math.round() を使う際、* 100.0 → / 100.0 のパターンは定番です。
  • printf の %.2f は小数第2位まで表示します。%n はプラットフォーム依存の改行です。

ビット演算フラグ管理

verified
JavaSilverポイント

ビット演算子 &（AND）、|（OR）、^（XOR）の動作と、整数リテラルのビット表現はJavaSilverで出題されます。論理演算子 && / || との混同に注意。

assignment課題

Permission クラスを実装してください。UNIX風のファイル権限をビット演算で管理します。

  • 定数: READ = 1（001）, WRITE = 2（010）, EXEC = 4（100）
  • hasPermission(int flags, int perm): 指定権限を持つか判定（& を使用）
  • addPermission(int flags, int perm): 権限を追加（| を使用）
  • removePermission(int flags, int perm): 権限を削除（& と ~ を使用）
  • togglePermission(int flags, int perm): 権限を反転（^ を使用）
  • toString(int flags): "rwx" 形式で文字列化（無い権限は "-"）
terminal期待出力
初期権限: r-- (1)
WRITE追加: rw- (3)
EXEC追加: rwx (7)
WRITE削除: r-x (5)
EXEC反転: r-- (1)
READ判定: true
WRITE判定: false
lightbulbヒント

権限チェック: (flags & perm) == perm で判定します。権限削除は flags & ~perm です。~ はビット反転演算子で、例えば ~2（010）は ...11111101 になります。

check_circle解答・解説 
public class Permission {

    static final int READ  = 1; // 001
    static final int WRITE = 2; // 010
    static final int EXEC  = 4; // 100

    public static boolean hasPermission(int flags, int perm) {
        return (flags & perm) == perm;
    }

    public static int addPermission(int flags, int perm) {
        return flags | perm;
    }

    public static int removePermission(int flags, int perm) {
        return flags & ~perm;
    }

    public static int togglePermission(int flags, int perm) {
        return flags ^ perm;
    }

    public static String toString(int flags) {
        String r = hasPermission(flags, READ)  ? "r" : "-";
        String w = hasPermission(flags, WRITE) ? "w" : "-";
        String x = hasPermission(flags, EXEC)  ? "x" : "-";
        return r + w + x;
    }

    public static void main(String[] args) {
        int flags = READ; // 001
        System.out.println("初期権限: " + toString(flags) + " (" + flags + ")");

        flags = addPermission(flags, WRITE);
        System.out.println("WRITE追加: " + toString(flags) + " (" + flags + ")");

        flags = addPermission(flags, EXEC);
        System.out.println("EXEC追加: " + toString(flags) + " (" + flags + ")");

        flags = removePermission(flags, WRITE);
        System.out.println("WRITE削除: " + toString(flags) + " (" + flags + ")");

        flags = togglePermission(flags, EXEC);
        System.out.println("EXEC反転: " + toString(flags) + " (" + flags + ")");

        System.out.println("READ判定: " + hasPermission(flags, READ));
        System.out.println("WRITE判定: " + hasPermission(flags, WRITE));
    }
}
解説
  • &（AND）: 両方のビットが1の場合のみ1。権限チェックに使用します。
  • |（OR）: どちらかのビットが1なら1。権限追加に使用します。
  • ^（XOR）: ビットが異なる場合に1。トグル（反転）に使用します。
  • ~（NOT）: 全ビットを反転。& ~perm で特定ビットだけを0にできます。
  • JavaSilverでは &（ビット演算）と &&（論理演算・短絡評価）の違いが問われます。

"田中太郎,28,東京".split(",") で String[] が得られます。Integer.parseInt(parts[1]) で文字列を int に変換。StringBuilder の append() をチェーンして整形文字列を組み立てましょう。

check_circle解答・解説 
public class CsvParser {

    static class Person {
        String name;
        int age;
        String city;

        Person(String name, int age, String city) {
            this.name = name;
            this.age = age;
            this.city = city;
        }
    }

    public static Person parseLine(String line) {
        String[] parts = line.split(",");
        String name = parts[0].trim();
        int age = Integer.parseInt(parts[1].trim());
        String city = parts[2].trim();
        return new Person(name, age, city);
    }

    public static String toDisplayString(Person p, int index) {
        StringBuilder sb = new StringBuilder();
        sb.append("[").append(index).append("] ")
          .append(p.name)
          .append(" (").append(p.age).append("歳) - ")
          .append(p.city);
        return sb.toString();
    }

    public static void main(String[] args) {
        String[] csvData = {
            "田中太郎,28,東京",
            "鈴木花子,34,大阪",
            "佐藤次郎,22,名古屋"
        };

        Person[] people = new Person[csvData.length];
        for (int i = 0; i < csvData.length; i++) {
            people[i] = parseLine(csvData[i]);
        }

        System.out.println("--- 社員一覧 ---");
        int totalAge = 0;
        for (int i = 0; i < people.length; i++) {
            System.out.println(toDisplayString(people[i], i + 1));
            totalAge += people[i].age;
        }
        double avgAge = (double) totalAge / people.length;
        System.out.println("合計: " + people.length + "名, 平均年齢: " + avgAge + "歳");
    }
}
解説
  • split(",") は正規表現として解釈されます。split("\\.") のようにメタ文字のエスケープが必要な場合があります。
  • Integer.parseInt() は変換できない文字列に対して NumberFormatException を投げます。trim() で前後の空白を除去するのが安全です。
  • StringBuilder は可変文字列。String の + 連結はループ内で使うと毎回新しいオブジェクトが生成されるため、StringBuilder が推奨されます。

テンプレート文字列中の {{変数名}} を HashMap の値で置換するエンジンを実装してください。

lightbulbヒント

indexOf("{{") で開始位置を見つけ、indexOf("}}", start) で終了位置を取得します。substring(start + 2, end) で変数名を抽出し、replace("{{" + key + "}}", value) で置換する方法がシンプルです。

check_circle解答・解説 
import java.util.HashMap;

public class TemplateEngine {

    public static String render(String template, HashMap<String, String> vars) {
        StringBuilder result = new StringBuilder(template);
        int start;
        while ((start = result.indexOf("{{")) != -1) {
            int end = result.indexOf("}}", start);
            if (end == -1) break;
            String key = result.substring(start + 2, end);
            String value = vars.containsKey(key) ? vars.get(key) : "[未定義]";
            result.replace(start, end + 2, value);
        }
        return result.toString();
    }

    public static void main(String[] args) {
        HashMap<String, String> vars = new HashMap<>();
        vars.put("name", "田中太郎");
        vars.put("age", "28");
        vars.put("city", "東京");
        vars.put("orderNo", "A-1234");

        String t1 = "Hello {{name}}, you are {{age}} years old.";
        System.out.println(render(t1, vars));

        String t2 = "お住まい: {{city}} / 職業: {{job}}";
        System.out.println(render(t2, vars));

        System.out.println("--- メール本文 ---");
        String t3 = "{{name}}様、ご注文番号{{orderNo}}の商品を発送しました。";
        System.out.println(render(t3, vars));
    }
}
解説
  • indexOf() は見つからない場合に -1 を返します。ループの終了条件に活用しています。
  • StringBuilder.replace(start, end, str) でインデックス範囲を直接置換できます。置換後に文字列長が変わるため、毎回 indexOf をやり直すのがポイントです。
  • containsKey() で存在チェックをしてから get() する、または getOrDefault() を使う方法もあります。
  • JavaSilverでは substring(begin, end) の end が「含まれない」ことが問われます（半開区間）。

加算は result[i][j] = a[i][j] + b[i][j] で各要素を足します。転置は result[j][i] = m[i][j] で行と列を入れ替えます。転置行列のサイズは new int[m[0].length][m.length] です。

check_circle解答・解説 
public class Matrix {

    public static int[][] add(int[][] a, int[][] b) {
        int rows = a.length;
        int cols = a[0].length;
        int[][] result = new int[rows][cols];
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                result[i][j] = a[i][j] + b[i][j];
            }
        }
        return result;
    }

    public static int[][] transpose(int[][] m) {
        int rows = m.length;
        int cols = m[0].length;
        int[][] result = new int[cols][rows];
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                result[j][i] = m[i][j];
            }
        }
        return result;
    }

    public static void print(int[][] m, String label) {
        System.out.println("[" + label + "]");
        for (int[] row : m) {
            StringBuilder sb = new StringBuilder();
            for (int val : row) {
                sb.append(String.format("%3d", val));
            }
            System.out.println(sb.toString());
        }
    }

    public static void main(String[] args) {
        int[][] a = {{1, 2, 3}, {4, 5, 6}};
        int[][] b = {{7, 8, 9}, {10, 11, 12}};

        print(a, "行列A");
        print(b, "行列B");
        print(add(a, b), "A + B");
        print(transpose(a), "Aの転置");
    }
}
解説
  • a.length で行数、a[0].length で列数を取得します。JavaSilverでは length がフィールド（配列）かメソッド（String）かの違いが問われます。
  • 転置では結果配列のサイズが [cols][rows] になる点に注意。元の行数が結果の列数になります。
  • 拡張for文（for-each）は配列の全要素を走査する場合に便利ですが、インデックスが必要な場合は通常のfor文を使います。
  • 新しい配列を生成して返すことで、元の配列が変更されない「非破壊的」な設計になっています。

public class ImageFilter {

    public static int[][] invert(int[][] img) {
        int rows = img.length;
        int cols = img[0].length;
        int[][] result = new int[rows][cols];
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                result[i][j] = 255 - img[i][j];
            }
        }
        return result;
    }

    public static int[][] blur(int[][] img) {
        int rows = img.length;
        int cols = img[0].length;
        int[][] result = new int[rows][cols];
        int[] di = {0, -1, 1, 0, 0};
        int[] dj = {0, 0, 0, -1, 1};

        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                int sum = 0;
                int count = 0;
                for (int d = 0; d < 5; d++) {
                    int ni = i + di[d];
                    int nj = j + dj[d];
                    if (ni >= 0 && ni < rows && nj >= 0 && nj < cols) {
                        sum += img[ni][nj];
                        count++;
                    }
                }
                result[i][j] = sum / count;
            }
        }
        return result;
    }

    public static void print(int[][] img, String label) {
        System.out.println("[" + label + "]");
        for (int[] row : img) {
            StringBuilder sb = new StringBuilder();
            for (int val : row) {
                sb.append(String.format("%4d", val));
            }
            System.out.println(sb.toString());
        }
    }

    public static void main(String[] args) {
        int[][] img = {
            {  0,  50, 100},
            {150, 200, 250},
            { 80, 120, 160}
        };

        print(img, "元画像");
        print(invert(img), "反転");
        print(blur(img), "ぼかし");
    }
}
解説
  • 方向配列 di[], dj[] を使うと、4方向 + 自身の走査をループ1つで書けます。この「方向配列パターン」は探索問題で頻出です。
  • 境界チェック ni >= 0 && ni < rows を怠ると ArrayIndexOutOfBoundsException が発生します。JavaSilverでは配列の範囲外アクセスが頻出テーマです。
  • sum / count は int 同士の除算なので小数部は切り捨てられます。これは画素値の場合は問題ありませんが、精度が必要な場合は (double) sum / count とキャストが必要です。

assignment課題

四則演算式の文字列を解析・計算する簡易電卓を実装してください。

  • 入力: "3+5*2" のような文字列（スペースなし、整数のみ）
  • 文字列を1文字ずつ走査して、数値と演算子に分解する
  • 演算子の優先順位を考慮: * と / を先に計算し、次に + と - を計算する
  • 複数桁の数値（例: "12+3"）にも対応する
terminal期待出力
3+5*2 = 13
10-2*3+4 = 8
100/5+3*2 = 26
1+2+3+4+5 = 15
lightbulbヒント

2パスで計算します。(1) 文字列を走査して数値リストと演算子リストに分解。(2) まず * / / を処理して結果を数値リストに戻す。(3) 残った + / - を左から順に計算。Character.isDigit(ch) で数字かどうかを判定できます。

check_circle解答・解説 
import java.util.ArrayList;

public class SimpleCalculator {

    public static int calculate(String expr) {
        // (1) 数値と演算子に分解
        ArrayList<Integer> nums = new ArrayList<>();
        ArrayList<Character> ops = new ArrayList<>();
        int num = 0;
        for (int i = 0; i < expr.length(); i++) {
            char ch = expr.charAt(i);
            if (Character.isDigit(ch)) {
                num = num * 10 + (ch - '0');
            } else {
                nums.add(num);
                ops.add(ch);
                num = 0;
            }
        }
        nums.add(num); // 最後の数値

        // (2) * と / を先に計算
        for (int i = 0; i < ops.size(); i++) {
            char op = ops.get(i);
            if (op == '*' || op == '/') {
                int left = nums.get(i);
                int right = nums.get(i + 1);
                int result = (op == '*') ? left * right : left / right;
                nums.set(i, result);
                nums.remove(i + 1);
                ops.remove(i);
                i--; // インデックス調整
            }
        }

        // (3) + と - を計算
        int result = nums.get(0);
        for (int i = 0; i < ops.size(); i++) {
            char op = ops.get(i);
            int right = nums.get(i + 1);
            result = (op == '+') ? result + right : result - right;
        }
        return result;
    }

    public static void main(String[] args) {

check_circle解答・解説 
public class MazeSolver {

    public static boolean solve(char[][] maze, int r, int c) {
        // 範囲外チェック
        if (r < 0 || r >= maze.length || c < 0 || c >= maze[0].length) {
            return false;
        }
        // 壁または訪問済み
        if (maze[r][c] == '#' || maze[r][c] == '*') {
            return false;
        }
        // ゴール到達
        if (maze[r][c] == 'G') {
            return true;
        }
        // 現在地を訪問済みにする
        maze[r][c] = '*';

        // 上下左右を再帰探索
        if (solve(maze, r - 1, c)) return true; // 上
        if (solve(maze, r + 1, c)) return true; // 下
        if (solve(maze, r, c - 1)) return true; // 左
        if (solve(maze, r, c + 1)) return true; // 右

        // どの方向もダメならバックトラック
        maze[r][c] = '.';
        return false;
    }

    public static void printMaze(char[][] maze) {
        for (char[] row : maze) {
            StringBuilder sb = new StringBuilder();
            for (int i = 0; i < row.length; i++) {
                if (i > 0) sb.append(' ');
                sb.append(row[i]);
            }
            System.out.println(sb.toString());
        }
    }

    public static void main(String[] args) {
        char[][] maze = {
            {'#','#','#','#','#','#','#'},
            {'#','S','.','.','#','.','#'},
            {'#','#','#','.','#','.','#'},
            {'#','.','.','.','.','.','#'},
            {'#','.','#','#','#','.','#'},
            {'#','.','.','.','.','G','#'},
            {'#','#','#','#','#','#','#'}
        };

        System.out.println("[迷路探索開始]");
        printMaze(maze);

        // スタート位置を探す
        int startR = -1, startC = -1;
        for (int i = 0; i < maze.length; i++) {
            for (int j = 0; j < maze[i].length; j++) {
                if (maze[i][j] == 'S') {
                    startR = i;
                    startC = j;
                }
            }
        }

        boolean found = solve(maze, startR, startC);
        if (found) {
            System.out.println("結果: ゴールに到達しました!");
        } else {
            System.out.println("結果: ゴールに到達できません");
        }
        printMaze(maze);
    }
}
解説
  • 再帰の終了条件を3つ設定: (1) 範囲外、(2) 壁/訪問済み、(3) ゴール到達。終了条件が不十分だと StackOverflowError になります。
  • バックトラック: 全方向が失敗した場合に '*' を '.' に戻します。これにより最終的に正しい経路のみが '*' で残ります。
  • この探索は「深さ優先探索（DFS）」と呼ばれます。再帰はJVM内部でスタックフレームを使うため、本質的にスタックベースの探索です。
  • char[][] は参照型なので、メソッド内で配列を変更すると呼び出し元にも影響します。JavaSilverでは「値渡し」と「参照渡し」の理解が問われます。

import java.util.ArrayList;

public class LibrarySystem {

    static class Book {
        private String title;
        private String author;
        private boolean isAvailable;

        public Book(String title, String author) {
            this.title = title;
            this.author = author;
            this.isAvailable = true;
        }

        public String getTitle()   { return title; }
        public String getAuthor()  { return author; }
        public boolean isAvailable() { return isAvailable; }
        public void setAvailable(boolean available) {
            this.isAvailable = available;
        }

        @Override
        public String toString() {
            String status = isAvailable ? "貸出可" : "貸出中";
            return "[" + status + "] " + title + " / " + author;
        }
    }

    static class Library {
        private ArrayList<Book> books = new ArrayList<>();

        public void addBook(Book book) {
            books.add(book);
        }

        public ArrayList<Book> search(String keyword) {
            ArrayList<Book> result = new ArrayList<>();
            for (Book book : books) {
                if (book.getTitle().contains(keyword)
                        || book.getAuthor().contains(keyword)) {
                    result.add(book);
                }
            }
            return result;
        }

        public boolean checkout(String title) {
            for (Book book : books) {
                if (book.getTitle().equals(title)) {
                    if (!book.isAvailable()) {
                        System.out.println("貸出失敗: "
                            + title + " は貸出中です");
                        return false;
                    }
                    book.setAvailable(false);
                    System.out.println("貸出完了: " + title);
                    return true;
                }
            }
            System.out.println("貸出失敗: " + title + " が見つかりません");
            return false;
        }

        public boolean returnBook(String title) {
            for (Book book : books) {
                if (book.getTitle().equals(title)) {
                    if (book.isAvailable()) {
                        System.out.println("返却失敗: "
                            + title + " は貸出中ではありません");
                        return false;
                    }
                    book.setAvailable(true);
                    System.out.println("返却完了: " + title);
                    return true;
                }
            }
            System.out.println("返却失敗: " + title + " が見つかりません");
            return false;
        }

        public void printAll() {
            for (Book book : books) {
                System.out.println(book);
            }
        }
    }

    public static void main(String[] args) {
        Library lib = new Library();
        lib.addBook(new Book("Java入門", "山田太郎"));
        lib.addBook(new Book("Pythonデータ分析", "鈴木花子"));

        System.out.println("=== 最終状態 ===");
        lib.printAll();
    }

import java.util.ArrayList;

public class VendingMachineApp {

    static class Product {
        String name;
        int price;
        int stock;

        Product(String name, int price, int stock) {
            this.name = name;
            this.price = price;
            this.stock = stock;
        }
    }

    static class VendingMachine {
        static int totalSales = 0; // 全販売機の総売上
        private ArrayList<Product> products = new ArrayList<>();

        public void addProduct(Product p) {
            products.add(p);
        }

        public void showProducts() {
            System.out.println("=== 商品一覧 ===");
            for (int i = 0; i < products.size(); i++) {
                Product p = products.get(i);
                System.out.printf("%d. %-8s ¥%d (在庫:%d)%n",
                    i + 1, p.name, p.price, p.stock);
            }
        }

        public int buy(String name, int money) {
            for (Product p : products) {
                if (p.name.equals(name)) {
                    if (p.stock <= 0) {
                        System.out.println("在庫切れ: " + name);
                        return money; // 全額返金
                    }
                    if (money < p.price) {
                        System.out.println("金額不足: " + name
                            + "は¥" + p.price
                            + "です（投入: ¥" + money + "）");
                        return money; // 全額返金
                    }
                    p.stock--;
                    totalSales += p.price;
                    int change = money - p.price;
                    System.out.println("購入成功: " + name
                        + " / お釣り: ¥" + change);
                    return change;
                }
            }
            System.out.println("商品が見つかりません: " + name);
            return money;
        }
    }

    public static void main(String[] args) {
        VendingMachine vm = new VendingMachine();
        vm.addProduct(new Product("コーラ", 150, 3));
        vm.addProduct(new Product("お茶", 130, 5));
        vm.addProduct(new Product("コーヒー", 170, 2));

        vm.showProducts();

        System.out.println("--- 購入: コーラ (投入: ¥200) ---");
        vm.buy("コーラ", 200);

        System.out.println("--- 購入: コーヒー (投入: ¥100) ---");
        vm.buy("コーヒー", 100);

        System.out.println("--- 購入: コーラ x3 ---");
        vm.buy("コーラ", 150);
        vm.buy("コーラ", 150);
        vm.buy("コーラ", 150); // 在庫切れ

        System.out.println("=== 総売上: ¥"
            + VendingMachine.totalSales + " ===");
    }
}

import java.util.ArrayList;

public class ZooSystem {

    // 抽象クラス
    static abstract class Animal {
        protected String name;

        Animal(String name) {
            this.name = name;
        }

        abstract String speak();

        void introduce() {
            String className = getClass().getSimpleName();
            System.out.println("[" + className + "] "
                + name + ": " + speak());
        }
    }

    // インタフェース
    interface Swimmable {
        void swim();
    }

    interface Flyable {
        void fly();
    }

    // 具象クラス
    static class Dog extends Animal implements Swimmable {
        Dog(String name) { super(name); }

        @Override
        String speak() { return "ワンワン!"; }

        @Override
        public void swim() {
            System.out.println(name + "がスイスイ泳いでいます");
        }
    }

    static class Cat extends Animal {
        Cat(String name) { super(name); }

        @Override
        String speak() { return "ニャー!"; }
    }

    static class Bird extends Animal implements Flyable {
        Bird(String name) { super(name); }

        @Override
        String speak() { return "チュンチュン!"; }

        @Override
        public void fly() {
            System.out.println(name + "が空を飛んでいます");
        }
    }

    public static void main(String[] args) {
        ArrayList<Animal> zoo = new ArrayList<>();
        zoo.add(new Dog("ポチ"));
        zoo.add(new Cat("タマ"));
        zoo.add(new Bird("ピーちゃん"));

        // ポリモーフィズム: 全動物を一括操作
        System.out.println("=== 動物園の仲間たち ===");
        for (Animal a : zoo) {
            a.introduce();
        }

        // instanceof で能力を判定
        System.out.println("--- 泳げる動物 ---");
        for (Animal a : zoo) {
            if (a instanceof Swimmable) {
                ((Swimmable) a).swim();
            }
        }

        System.out.println("--- 飛べる動物 ---");
        for (Animal a : zoo) {
            if (a instanceof Flyable) {
                ((Flyable) a).fly();
            }
        }
    }
}
解説
  • abstract class はインスタンス化できませんが、コンストラクタ・具象メソッド・フィールドを持てます。インタフェースは（Java 8以降の default メソッドを除き）実装を持てません。
  • クラスは単一継承（extends は1つ）ですが、インタフェースは複数実装（implements A, B）が可能です。
  • instanceof で型チェック後にキャストするのが安全なパターンです。チェックなしのキャストは ClassCastException の原因になります。
  • @Override アノテーションを付けると、親クラスのメソッドを正しくオーバーライドしていない場合にコンパイルエラーになり、ミスを防げます。

import java.util.ArrayList;
import java.util.Comparator;

public class ShapeSystem {

    static abstract class Shape {
        abstract double area();
        abstract double perimeter();
    }

    interface Drawable {
        void draw();
    }

    interface Resizable {
        void resize(double factor);
    }

    static class Circle extends Shape
            implements Drawable, Resizable {
        private double radius;

        Circle(double radius) { this.radius = radius; }

        @Override
        double area() { return Math.PI * radius * radius; }

        @Override
        double perimeter() { return 2 * Math.PI * radius; }

        @Override
        public void draw() {
            System.out.printf("○ 円を描画 (半径:%.1f)%n", radius);
        }

        @Override
        public void resize(double factor) {
            radius *= factor;
        }
    }

    static class Rectangle extends Shape
            implements Drawable, Resizable {
        private double width;
        private double height;

        Rectangle(double width, double height) {
            this.width = width;
            this.height = height;
        }

        @Override
        double area() { return width * height; }

        @Override
        double perimeter() { return 2 * (width + height); }

        @Override
        public void draw() {
            System.out.printf(
                "□ 長方形を描画 (幅:%.1f, 高さ:%.1f)%n",
                width, height);
        }

        @Override
        public void resize(double factor) {
            width *= factor;
            height *= factor;
        }
    }

    static class Triangle extends Shape implements Drawable {
        private double base;
        private double height;

        Triangle(double base, double height) {
            this.base = base;
            this.height = height;
        }

        @Override
        double area() { return base * height / 2; }

        @Override
        double perimeter() {
            // 二等辺三角形として斜辺を計算
            double side = Math.sqrt(
                (base / 2) * (base / 2) + height * height);
            return base + side * 2;
        }

        @Override
        public void draw() {
            System.out.printf(
                "△ 三角形を描画 (底辺:%.1f, 高さ:%.1f)%n",
                base, height);
        }
    }

    static void printShape(Shape s) {
        if (s instanceof Drawable) {
            ((Drawable) s).draw();
        }
        System.out.printf("  面積: %.2f / 周長: %.2f%n",
            s.area(), s.perimeter());
    }

    public static void main(String[] args) {
        ArrayList<Shape> shapes = new ArrayList<>();
        Circle circle = new Circle(3.0);
        Rectangle rect = new Rectangle(5.0, 3.0);
        Triangle tri = new Triangle(6.0, 4.0);
        shapes.add(circle);
        shapes.add(rect);
        shapes.add(tri);

        // 面積昇順でソートして一覧表示
        System.out.println("=== 図形一覧 ===");
        shapes.sort(Comparator.comparingDouble(Shape::area));
        for (Shape s : shapes) {
            printShape(s);
        }

        // Resizable な図形だけリサイズ
        System.out.println("--- リサイズ (x2.0) ---");
        for (Shape s : shapes) {
            if (s instanceof Resizable) {
                ((Resizable) s).resize(2.0);
                ((Drawable) s).draw();
                System.out.printf("  面積: %.2f / 周長: %.2f%n",
                    s.area(), s.perimeter());
            }
        }

        // 再ソート
        System.out.println("--- 面積の昇順でソート ---");
        shapes.sort(Comparator.comparingDouble(Shape::area));
        for (Shape s : shapes) {
            if (s instanceof Drawable) {
                ((Drawable) s).draw();
            }
        }
    }
}
解説
  • abstract class に2つの抽象メソッドを定義し、各サブクラスで実装を強制しています。実装を忘れるとコンパイルエラーになります。
  • Circle と Rectangle は Drawable と Resizable の2つのインタフェースを実装しています。Triangle は Drawable のみで、リサイズ機能を持ちません。

public class ExceptionHierarchy {

    // カスタム例外階層
    static class AppException extends Exception {
        private String errorCode;

        AppException(String message, String errorCode) {
            super(message);
            this.errorCode = errorCode;
        }

        AppException(String message, String errorCode,
                     Throwable cause) {
            super(message, cause);
            this.errorCode = errorCode;
        }

        String getErrorCode() { return errorCode; }
    }

    static class ValidationException extends AppException {
        ValidationException(String message, String code) {
            super(message, code);
        }

        ValidationException(String message, String code,
                            Throwable cause) {
            super(message, code, cause);
        }
    }

    static class BusinessException extends AppException {
        BusinessException(String message, String code) {
            super(message, code);
        }
    }

    // バリデーション
    static int validateAge(String input)
            throws ValidationException {
        int age;
        try {
            age = Integer.parseInt(input);
        } catch (NumberFormatException e) {
            throw new ValidationException(
                "年齢は数値で入力してください", "V001", e);
        }
        if (age < 0 || age > 150) {
            throw new ValidationException(
                "年齢は0〜150の範囲で入力してください", "V002");
        }
        return age;
    }

    static void registerUser(String name, int age)
            throws ValidationException, BusinessException {
        if (name == null || name.isEmpty()) {
            throw new ValidationException(
                "名前を入力してください", "V003");
        }
        if (age < 18) {
            throw new BusinessException(
                "18歳未満は登録できません", "B001");
        }
        System.out.println("登録成功: "
            + name + " (" + age + "歳)");
    }

    static void handleError(AppException e) {
        String type = (e instanceof ValidationException)
            ? "バリデーションエラー"
            : "ビジネスエラー";
        System.out.println("[" + e.getErrorCode() + "] "
            + type + ": " + e.getMessage());
        if (e.getCause() != null) {
            System.out.println("  原因: "
                + e.getCause().getMessage());
        }
    }

    public static void main(String[] args) {
        // テスト1: 正常
        System.out.println("--- テスト1: 正常登録 ---");
        try {
            int age = validateAge("25");
            registerUser("田中太郎", age);
        } catch (AppException e) {
            handleError(e);
        }

        // テスト2: 数値変換エラー
        System.out.println("--- テスト2: 数値変換エラー ---");
        try {
            int age = validateAge("abc");
            registerUser("鈴木", age);
        } catch (AppException e) {
            handleError(e);
        }

        // テスト3: 範囲外
        System.out.println("--- テスト3: 範囲外エラー ---");
        try {
            int age = validateAge("200");
            registerUser("佐藤", age);
        } catch (AppException e) {
            handleError(e);
        }

        // テスト4: ビジネスエラー
        System.out.println("--- テスト4: ビジネスエラー ---");
        try {
            int age = validateAge("15");
            registerUser("山田", age);
        } catch (AppException e) {
            handleError(e);
        }

        // テスト5: 名前空
        System.out.println("--- テスト5: 名前空エラー ---");
        try {
            int age = validateAge("30");
            registerUser("", age);
        } catch (AppException e) {
            handleError(e);
        }
    }
}
解説
  • AppException extends Exception は検査例外です。呼び出し元で try-catch または throws 宣言が必須です。RuntimeException を継承すると非検査例外になり、明示的な処理が不要になります。

  • throw は例外を投げる文、throws はメソッド宣言で例外を伝搬することを示すキーワードです。JavaSilverでよく混同が問われます。

public class ResourcePipeline {

    static class ResourceA implements AutoCloseable {
        ResourceA() {
            System.out.println("ResourceA: オープン");
        }

        void process() {
            System.out.println("ResourceA: 処理実行");
        }

        @Override
        public void close() {
            System.out.println("ResourceA: クローズ");
        }
    }

    static class ResourceB implements AutoCloseable {
        private boolean failOnClose;

        ResourceB(boolean failOnClose) {
            this.failOnClose = failOnClose;
            String mode = failOnClose
                ? " (closeで例外発生モード)" : "";
            System.out.println("ResourceB: オープン" + mode);
        }

        void process() {
            System.out.println("ResourceB: 処理実行");
        }

        @Override
        public void close() throws Exception {
            if (failOnClose) {
                System.out.println(
                    "ResourceB: クローズ → 例外発生!");
                throw new Exception(
                    "ResourceBのクローズに失敗");
            }
            System.out.println("ResourceB: クローズ");
        }
    }

    public static void main(String[] args) {
        // 正常ケース
        System.out.println("=== 正常ケース ===");
        try (ResourceA a = new ResourceA();
             ResourceB b = new ResourceB(false)) {
            a.process();
            b.process();
        } catch (Exception e) {
            System.out.println("キャッチ: " + e.getMessage());
        } finally {
            System.out.println("finally ブロック実行");
        }

        // close例外ケース
        System.out.println("=== close例外ケース ===");
        try (ResourceA a = new ResourceA();
             ResourceB b = new ResourceB(true)) {
            System.out.println("処理中...");
        } catch (Exception e) {
            System.out.println("キャッチ: " + e.getMessage());
            Throwable[] suppressed = e.getSuppressed();
            if (suppressed.length > 0) {
                for (Throwable t : suppressed) {
                    System.out.println(
                        "  抑制された例外: " + t.getMessage());
                }
            } else {
                System.out.println("  抑制された例外: なし");
            }
        } finally {
            System.out.println("finally ブロック実行");
        }
    }
}
解説
  • try-with-resources では、リソースは宣言の逆順で close() されます。ResourceA → ResourceB の順で宣言すると、ResourceB → ResourceA の順でクローズされます。
  • finally ブロックは close() の後に実行されます。実行順序: try本体 → close（逆順） → catch → finally。
  • try本体で例外が発生し、close() でも例外が発生した場合、close() の例外は「抑制された例外」として getSuppressed() で取得できます。
  • AutoCloseable の close() は Exception をスローできます。Closeable（AutoCloseable のサブインタフェース）は IOException のみです。

computeIfAbsent(name, k -> new ArrayList<>()) を使うと、キーが未登録の場合に自動で新しいリストを作成できます。平均計算では int の合計を double にキャストしてからリストサイズで割ります。ソートでは ArrayList に Map.Entry を詰めてから sort() するか、エントリのリストを作成してソートします。

import java.util.ArrayList;
import java.util.Comparator;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

public class GradeManager {

    private HashMap<String, List<Integer>> grades
        = new HashMap<>();

    public void addScore(String name, int score) {
        grades.computeIfAbsent(name, k -> new ArrayList<>())
              .add(score);
    }

    public double getAverage(String name) {
        List<Integer> scores = grades.get(name);
        if (scores == null || scores.isEmpty()) return 0.0;
        int sum = 0;
        for (int s : scores) {
            sum += s;
        }
        return (double) sum / scores.size();
    }

    public List<String> getAboveAverage(int threshold) {
        List<String> result = new ArrayList<>();
        for (Map.Entry<String, List<Integer>> entry
                : grades.entrySet()) {
            double avg = getAverage(entry.getKey());
            if (avg >= threshold) {
                result.add(entry.getKey());
            }
        }
        return result;
    }

    public List<Map.Entry<String, Double>> getTopN(int n) {
        // 名前と平均点のペアを作成
        List<Map.Entry<String, Double>> avgList
            = new ArrayList<>();
        for (String name : grades.keySet()) {
            avgList.add(Map.entry(name, getAverage(name)));
        }
        // 平均点の降順でソート
        avgList.sort(Comparator.comparingDouble(
            (Map.Entry<String, Double> e) -> e.getValue()
        ).reversed());
        // 上位N件を返す
        return avgList.subList(0, Math.min(n, avgList.size()));
    }

    public void printAll() {
        for (Map.Entry<String, List<Integer>> entry
                : grades.entrySet()) {
            String name = entry.getKey();
            List<Integer> scores = entry.getValue();
            System.out.printf("%s: %s 平均: %.1f%n",
                name, scores, getAverage(name));
        }
    }

    public static void main(String[] args) {
        GradeManager gm = new GradeManager();

        // データ登録
        int[] tanaka = {85, 90, 78};
        int[] suzuki = {92, 88, 95};
        int[] sato   = {70, 65, 80};
        int[] yamada = {88, 92, 85};

        for (int s : tanaka) gm.addScore("田中", s);
        for (int s : suzuki) gm.addScore("鈴木", s);
        for (int s : sato)   gm.addScore("佐藤", s);
        for (int s : yamada) gm.addScore("山田", s);

        System.out.println("=== 成績一覧 ===");
        gm.printAll();

        System.out.println("--- 平均80点以上 ---");
        for (String name : gm.getAboveAverage(80)) {
            System.out.printf("%s: %.1f%n",
                name, gm.getAverage(name));
        }

        System.out.println("--- トップ2 ---");
        List<Map.Entry<String, Double>> top
            = gm.getTopN(2);
        for (int i = 0; i < top.size(); i++) {
            System.out.printf("%d位: %s (%.1f)%n",
                i + 1, top.get(i).getKey(),
                top.get(i).getValue());
        }
    }
}
解説
  • computeIfAbsent() はキーが存在しない場合のみラムダを実行して値を生成します。putIfAbsent() との違いは、値の生成を遅延できる点です。
  • Map.entrySet() は Set<Map.Entry<K,V>> を返します。拡張for文でキーと値を同時に取得する際に使います。
  • Comparator.comparingDouble().reversed() で降順ソートを実現しています。ラムダ式の型推論が効かない場合は、引数に型を明示する必要があります。
  • subList(fromIndex, toIndex) は元のリストのビュー（参照）を返します。範囲外アクセスを防ぐため Math.min(n, size) でガードしています。

import java.time.DayOfWeek;
import java.time.Duration;
import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
import java.util.ArrayList;
import java.util.Comparator;

public class ScheduleManager {

    static class Event {
        String name;
        LocalDateTime start;
        LocalDateTime end;

        Event(String name,
              LocalDateTime start, LocalDateTime end) {
            this.name = name;
            this.start = start;
            this.end = end;
        }
    }

    static class Scheduler {
        private ArrayList<Event> events = new ArrayList<>();
        private DateTimeFormatter fmt
            = DateTimeFormatter.ofPattern("MM/dd HH:mm");

        static boolean isOverlapping(Event a, Event b) {
            return a.start.isBefore(b.end)
                && b.start.isBefore(a.end);
        }

        public boolean addEvent(Event newEvent) {
            for (Event existing : events) {
                if (isOverlapping(existing, newEvent)) {
                    System.out.println("重複エラー: "
                        + newEvent.name + " は "
                        + existing.name + " と重複しています");
                    return false;
                }
            }
            events.add(newEvent);
            System.out.printf("追加: %s (%s-%s)%n",
                newEvent.name,
                newEvent.start.format(fmt),
                newEvent.end.format(
                    DateTimeFormatter.ofPattern("HH:mm")));
            return true;
        }

        public void printSchedule() {
            events.sort(Comparator.comparing(e -> e.start));
            for (int i = 0; i < events.size(); i++) {
                Event e = events.get(i);
                long minutes = Duration.between(
                    e.start, e.end).toMinutes();
                System.out.printf(
                    "%d. %-8s %s - %s (%d分)%n",
                    i + 1, e.name,
                    e.start.format(fmt),
                    e.end.format(DateTimeFormatter
                        .ofPattern("HH:mm")),
                    minutes);
            }
        }

        static int countBusinessDays(
                LocalDate from, LocalDate to) {
            int count = 0;
            LocalDate current = from;
            while (!current.isAfter(to)) {
                DayOfWeek dow = current.getDayOfWeek();
                if (dow != DayOfWeek.SATURDAY
                        && dow != DayOfWeek.SUNDAY) {
                    count++;
                }
                current = current.plusDays(1);
            }
            return count;
        }
    }

    public static void main(String[] args) {
        Scheduler scheduler = new Scheduler();

        // 基準日: 2026/3/10 (火曜日)
        LocalDate baseDate = LocalDate.of(2026, 3, 10);

        System.out.println("=== イベント追加 ===");
        scheduler.addEvent(new Event("朝会",
            baseDate.atTime(9, 0),
            baseDate.atTime(10, 0)));
        scheduler.addEvent(new Event("企画会議",
            baseDate.atTime(14, 0),
            baseDate.atTime(15, 30)));
        scheduler.addEvent(new Event("ランチ",
            baseDate.atTime(12, 0),
            baseDate.atTime(13, 0)));
        // 重複テスト
        scheduler.addEvent(new Event("進捗報告",
            baseDate.atTime(9, 30),
            baseDate.atTime(10, 30)));

        System.out.println("=== スケジュール (開始時間順) ===");
        scheduler.printSchedule();

        System.out.println("=== 営業日計算 ===");
        LocalDate from = LocalDate.of(2026, 3, 1);
        LocalDate to = LocalDate.of(2026, 3, 15);
        int bizDays = Scheduler.countBusinessDays(from, to);
        System.out.printf("%s 〜 %s: 営業日 %d日%n",
            from, to, bizDays);
    }
}
解説
  • LocalDateTime はイミュータブルです。plusDays() などの操作メソッドは新しいオブジェクトを返し、元のオブジェクトは変更されません。JavaSilverではこの特性がよく問われます。
  • 時間重複の判定: A.start < B.end && B.start < A.end は区間の重なりを判定する定番パターンです。端点が一致する場合（10:00終了と10:00開始）は重複とみなしません。
  • DayOfWeek は enum で、MONDAY〜SUNDAY の7つの定数を持ちます。getValue() で1（月曜）〜7（日曜）の数値を取得できます。
  • Duration は時間ベースの期間（時・分・秒）、Period は日付ベースの期間（年・月・日）です。Duration.between() で2つの時刻間の差を計算できます。
  • DateTimeFormatter.ofPattern() でカスタム書式を定義できます。MM（月）、dd（日）、HH（24時間制の時）、mm（分）が基本パターンです。
