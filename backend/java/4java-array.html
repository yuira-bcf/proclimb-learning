Section 4-1

配列

配列は同じ型のデータを複数まとめて扱うための基本的なデータ構造です。ループと組み合わせることで、大量のデータを効率的に処理できます。後に学ぶArrayListなどのコレクションも、内部では配列を使用しています。配列の仕組みを理解することは、パフォーマンスを意識したプログラミングの基礎となります。

public class ArrayDemo {
    public static void main(String[] args) {
        // 配列の宣言
        int[] numbers;
        String[] names;

        // 配列の生成
        numbers = new int[5];  // 要素数5の配列

        // 宣言と生成を同時に
        int[] scores = new int[3];

        // 初期化子を使った初期化
        int[] values = {1, 2, 3, 4, 5};
        int[] data = new int[]{10, 20, 30};

        // 配列の要素にアクセス
        values[0] = 100;  // 最初の要素に代入
        int first = values[0];  // 最初の要素を取得

        // 配列の長さ
        int length = values.length;  // 5

        System.out.println("first: " + first);
        System.out.println("length: " + length);
    }
}
menu_book 構文説明

配列の宣言

型名[] 配列名;

配列を宣言します。この時点ではまだ配列は生成されていません（参照変数のみ）。

配列の生成

配列名 = new 型名[要素数];

newキーワードで指定した要素数の配列を生成します。要素は型に応じたデフォルト値で初期化されます。

配列初期化子

型名[] 配列名 = {値1, 値2, 値3};

波括弧内に初期値を列挙して配列を初期化します。要素数は自動的に決定されます。

配列要素へのアクセス

配列名[インデックス]

角括弧内にインデックス（0から始まる番号）を指定して要素にアクセスします。lengthプロパティで配列の長さを取得できます。

warning 配列の初期値

配列をnewで生成すると、要素は自動的にデフォルト値で初期化されます（int:0、boolean:false、参照型:null）。

chapter02/MultiArrayDemo.java Java 
public class MultiArrayDemo {
    public static void main(String[] args) {
        // 2次元配列の宣言と生成
        int[][] matrix = new int[3][4];  // 3行4列

        // 初期化子を使った初期化
        int[][] grid = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // 要素へのアクセス
        matrix[0][0] = 10;  // 0行0列目
        int value = grid[1][2];  // 1行2列目（値は6）

        // ジャグ配列（各行の長さが異なる配列）
        int[][] jagged = new int[3][];
        jagged[0] = new int[2];
        jagged[1] = new int[4];
        jagged[2] = new int[3];

        System.out.println("matrix[0][0]: " + matrix[0][0]);
        System.out.println("grid[1][2]: " + value);
        System.out.println("jagged行数: " + jagged.length);
    }
}
menu_book 構文説明

2次元配列の宣言と生成

型名[][] 配列名 = new 型名[行数][列数];

行と列を持つ配列（配列の配列）を生成します。表やマトリックス形式のデータを扱う場合に使用します。

2次元配列の要素アクセス

配列名[行インデックス][列インデックス]

2つのインデックスを指定して要素にアクセスします。どちらも0から始まります。

ジャグ配列

型名[][] 配列名 = new 型名[行数][];

各行の長さが異なる配列です。行数のみ指定し、各行の配列は後から個別に生成します。

Section 4-2

コマンドライン引数の利用

コマンドライン引数を使うと、プログラム実行時に外部から値を渡せます。設定ファイルのパスやモード切り替えなど、同じプログラムを異なる条件で動かす場合に便利です。バッチ処理やシェルスクリプトとの連携でよく使われます。

mainメソッドの引数String[] argsでコマンドライン引数を受け取れます。

public class CommandLineDemo {
    public static void main(String[] args) {
        // 引数の数を確認
        System.out.println("引数の数: " + args.length);

        // 各引数を表示
        for (int i = 0; i < args.length; i++) {
            System.out.println("args[" + i + "] = " + args[i]);
        }
    }
}
menu_book 構文説明

コマンドライン引数の受け取り

public static void main(String[] args)

argsはString型の配列で、プログラム実行時に渡された引数を格納します。args.lengthで引数の個数、args[0]で最初の引数にアクセスできます。引数がない場合、argsは空の配列（長さ0）になります。

実行例 Terminal 
$ java CommandLineDemo Hello World 123
引数の数: 3
args[0] = Hello
args[1] = World
args[2] = 123
Exercise

4章 練習問題

Q1

次のコードの実行結果として正しいものはどれですか？

int[] arr = new int[3];
System.out.println(arr[0]);
  1. null
  2. 0
  3. コンパイルエラー
  4. ArrayIndexOutOfBoundsException
Q2

次のコードの実行結果として正しいものはどれですか？

int[] arr = {10, 20, 30};
System.out.println(arr[arr.length - 1]);
  1. 10
  2. 20
  3. 30
  4. ArrayIndexOutOfBoundsException
Q3

次の2次元配列の宣言で、コンパイルエラーになるものはどれですか？

  1. int[][] a = new int[3][4];
  2. int[][] b = new int[3][];
  3. int[][] c = new int[][4];
  4. int[][] d = {{1,2},{3,4}};
Answer

4章 解説・解答

Q1の解答: B

int型配列をnewで生成すると、各要素はデフォルト値の0で初期化されます。参照型の場合はnull、booleanの場合はfalseがデフォルト値になります。

Q2の解答: C

arr.lengthは3なので、arr[3 - 1] → arr[2]となり、値は30です。配列のインデックスは0から始まるため、最後の要素にアクセスするにはlength - 1を使います。

Q3の解答: C

2次元配列では行数（1次元目）を省略できません。new int[][4]は行数が指定されていないためコンパイルエラーです。new int[3][]のように列数のみ省略することは可能です（ジャグ配列）。
