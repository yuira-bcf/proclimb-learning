Section 10-1

コレクションフレームワーク

コレクションは、複数のオブジェクトを格納・操作するためのデータ構造です。配列と違い、サイズを動的に変更でき、便利なメソッドが豊富に用意されています。List、Set、Mapは実務で最も頻繁に使うデータ構造であり、使いこなせることが必須スキルです。

10-1-1 ArrayList

ArrayListは可変長のリストで、要素の追加・取得・削除が簡単にできます。List<型>のようにジェネリクスで格納する型を指定し、型安全にデータを管理します。

chapter10/ListDemo.javaJava
import java.util.ArrayList;
import java.util.List;

public class ListDemo {
    public static void main(String[] args) {
        List<String> list = new ArrayList<>();
        list.add("A");
        list.add("B");
        System.out.println("get(0): " + list.get(0));     // A
        System.out.println("size: " + list.size());        // 2
        list.remove(0);    // "A"を削除
        System.out.println("削除後: " + list);             // [B]
    }
}

ArrayList（List<型>）

List<型> 変数名 = new ArrayList<>();

ArrayListはjava.util.Listインタフェースの代表的な実装クラスです。ジェネリクスで格納する要素の型を指定し、add()で要素を追加、get(index)で取得、remove(index)で削除、size()でサイズを確認できます。

10-1-2 HashMap

HashMapは「キーと値のペア」でデータを管理するデータ構造です。Map<キーの型, 値の型>のようにジェネリクスで型を指定し、キーから高速に値を検索できます。

chapter10/MapDemo.javaJava
import java.util.HashMap;
import java.util.Map;

public class MapDemo {
    public static void main(String[] args) {
        Map<String, Integer> map = new HashMap<>();
        map.put("Alice", 25);
        map.put("Bob", 30);
        System.out.println("get Alice: " + map.get("Alice"));           // 25
        System.out.println("containsKey Bob: " + map.containsKey("Bob")); // true
        System.out.println("map: " + map);                                // {Alice=25, Bob=30}
    }
}
menu_book 構文説明

HashMap（Map<K, V>）

Map<キーの型, 値の型> 変数名 = new HashMap<>();

HashMapはjava.util.Mapインタフェースの代表的な実装クラスです。put(key, value)でキーと値のペアを格納し、get(key)でキーから値を取得、containsKey(key)でキーの存在を確認できます。

work実際のユースケース
  • ユーザー一覧の管理：List<User> でデータベースから取得したユーザーを格納し、拡張 for 文でループ処理します。Spring Boot のサービス層で最も頻繁に使うパターンです。

  • 重複排除：Set<String> でタグやカテゴリの一覧を管理すると、同じ値が自動的に排除されます。ユーザーの権限一覧（ROLE_ADMIN, ROLE_USER）の管理にも活用されます。
Exercise

10章 練習問題

Q1

ArrayListについて正しい説明はどれですか？

  1. サイズは固定で変更できない
  2. ジェネリクスで格納する型を指定できる
  3. キーと値のペアでデータを管理する
  4. 重複する要素を自動的に排除する
Answer

10章 解説・解答

Q1の解答: B

ArrayListはジェネリクス（List<String>など）で格納する型を指定できます。サイズは動的に変更可能（Aは誤り）、キーと値のペアはHashMap（Cは誤り）、重複排除はHashSet（Dは誤り）の特徴です。
