Java基礎

Javaプログラミングの基礎からオブジェクト指向、例外処理まで体系的に学べるチュートリアルです。

1
Javaの概要と環境構築

Javaの歴史と特徴、JDK/JRE/JVMの関係、開発環境のセットアップを学びます。

歴史 JVM 環境構築
2
コンパイル・パッケージ・インポート

javac/javaコマンドの使い方、パッケージ構成、import文を学びます。

javac package import
3
変数・リテラル・データ型

変数の宣言、プリミティブ型、リテラル、型推論（var）を学びます。

変数 リテラル var
4
配列

1次元配列・多次元配列の宣言と操作、コマンドライン引数を学びます。

配列 多次元 args
5
文字列の操作

Stringクラスのメソッド、テキストブロック、StringBuilderの使い方を学びます。

String StringBuilder テキストブロック
6
演算子とデータの比較

算術・比較・論理演算子、基本型と参照型の比較方法を学びます。

演算子 比較 equals
7
型変換とラッパークラス

暗黙の型変換、キャスト、ラッパークラス、ボクシング/アンボクシングを学びます。

キャスト ラッパー ボクシング
8
条件分岐

if-else文、switch文、switch式（Java 14以降）による条件分岐を学びます。

if-else switch switch式
9
繰り返しと制御文

for文、while文、拡張for文、break/continueによるループ制御を学びます。

for while break
10
クラスの基本

クラスの定義、インスタンスの生成、アクセス修飾子、オーバーロードを学びます。

class new アクセス修飾子
11
コンストラクタとstatic

コンストラクタによる初期化、static変数/メソッド、参照のコピーを学びます。

コンストラクタ static 参照型
12
継承

クラスの継承、オーバーライド、コンストラクタチェーン、Objectクラスを学びます。

extends override super
13
抽象クラスとインタフェース

抽象クラス・抽象メソッド、インタフェースの作成と実装、defaultメソッドを学びます。

abstract interface default
14
ポリモーフィズム

アップキャスト・ダウンキャスト、instanceofによる型判定、ポリモーフィズムを学びます。

キャスト instanceof 多態性
15
コレクションフレームワーク

ArrayList、HashMapなど、動的なデータ構造の使い方を学びます。

ArrayList HashMap ジェネリクス
16
例外処理

try-catch文、throwsによる例外宣言、カスタム例外の作成方法を学びます。

try-catch throws カスタム例外
17
Mavenの基礎

Mavenによるビルド自動化、pom.xmlの設定、依存関係管理を学びます。

Maven pom.xml 依存関係
