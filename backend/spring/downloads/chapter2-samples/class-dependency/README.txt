=============================================
  クラス依存のサンプルコード
=============================================

【概要】
クラス依存（実装依存）の問題点を示すサンプルです。
クラスBからクラスCに差し替える場合、利用側のコードで
複数箇所の修正が必要になります。

【ファイル説明】
- B.java      : 使われる側のクラス（変更前）
- C.java      : 使われる側のクラス（変更後）
- A_Before.java : 使う側のクラス（Bを使用）
- A_After.java  : 使う側のクラス（Cを使用）

【実行方法】
1. すべてのファイルをコンパイル
   javac *.java

2. 変更前の実行
   java A_Before

3. 変更後の実行
   java A_After

【ポイント】
A_Before.java と A_After.java を比較すると、
型名(B→C)と変数名(b→c)の2箇所を変更する必要があります。
利用箇所が多いほど修正量が増大します。
