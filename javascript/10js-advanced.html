Section 2

オブジェクト（Object）とは？

「名前（キー）と値（バリュー）のセット」

オブジェクトは、（キー）と呼ばれる名前と、それに対応する値（バリュー）をセットで管理します。ひとつひとつのデータに、名札が付いているイメージですね！

名前と値の間は「:（コロン）」で区切り、複数のデータを入れるときは「,（カンマ）」で区切ります。全体は { }（波かっこ）で囲みます。

オブジェクトは、「1つのものに関する複数の情報を、名前で整理してまとめる」ための仕切りのある整理箱。

たとえば、1人のユーザー情報をまとめたい場合

JavaScript 
let user = {
  name: "たろう",
  age: 18,
  city: "東京"
};

取り出し方

JavaScript 
console.log(user.name);  // たろう
console.log(user.age);   // 18
console.log(user["city"]); // 東京
  • console.log(user.age); ← ドット記法
  • console.log(user["city"]); ← ブラケット記法

どちらの方法でも良く、変更・追加も簡単

JavaScript 
user.age = 19;          // 変更
user.email = "taro@test.com"; // 追加

配列 × オブジェクト（最強の組み合わせ）

JavaScript 
let users = [
  { name: "たろう", age: 18 },
  { name: "はなこ", age: 22 },
  { name: "さくら", age: 20 }
];

users.forEach((u) => console.log(u.name + "（" + u.age + "歳）"));

結果：

たろう（18歳）
はなこ（22歳）
さくら（20歳）

1人分の情報（名前と年齢）を オブジェクト { } でまとめ、それを 配列 [ ] に並べて複数人を一度に管理しています。

構造イメージ

users
 └─ 0: { name: "たろう", age: 18 }
 └─ 1: { name: "はなこ", age: 22 }
 └─ 2: { name: "さくら", age: 20 }

配列の中に、オブジェクトが入っているイメージ

Section 3

localStorage（ローカルストレージ）

ブラウザの中にデータを保存しておける「小さな倉庫」！

  • データはブラウザを閉じても残る
  • ユーザー設定やToDoリストの保存に使える
  • 文字列しか保存できない（JSONに変換する）

基本操作

操作 コード 説明
保存 localStorage.setItem("key", "value") データを保存
取得 localStorage.getItem("key") データを読み出す
削除 localStorage.removeItem("key") 指定データを削除
全削除 localStorage.clear() すべて削除

ファイル：Chapter_10-3 \ index.html

例：名前を保存する

HTML 
<input id="name">
<button onclick="save()">保存</button>
<button onclick="load()">読み込み</button>

<p id="output"></p>

<script>
  function save() {
    const n = document.getElementById("name").value;
    localStorage.setItem("username", n);
  }

  function load() {
    const n = localStorage.getItem("username");
    document.getElementById("output").textContent = "ようこそ、" + n + "さん！";
  }
</script>

結果：

  • （名前を入力して「保存」ボタン押下）名前がブラウザに保存される。
  • ページの再読み込み後「読み込み」ボタンを押すと、保存された名前が表示されます。

ブラウザを閉じても、再読み込みしてもデータが残る！

JSON（オブジェクトを保存する裏ワザ）

JavaScript 
let user = { name: "さくら", age: 20 };
localStorage.setItem("user", JSON.stringify(user));

let data = JSON.parse(localStorage.getItem("user"));
console.log(data.name); // さくら

結果：さくら（コンソール上）

JSON は「JavaScript Object Notation」＝データを文字列として表すための形式です。

Section 4

window / document の活用

オブジェクト 説明 例
window ブラウザ全体 window.alert("こんにちは！")
document HTML全体 document.title = "新しいタイトル"
location ページのURL情報 location.href
navigator ブラウザやOS情報 navigator.userAgent
history ページ履歴 history.back()（戻る）

例：ページ移動ボタン

HTML 
<button onclick="goGoogle()">Googleへ</button>
<script>
  function goGoogle() {
    location.href = "https://www.google.com";
  }
</script>

結果：ボタンを押すと、googleページが表示される

locationは「window」オブジェクトの一部です。正式には window.location と書きます（window. は省略可能）

Section 5

実践：メモ保存アプリ

ファイル：Chapter_10-5 \ index.html

コード例

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>メモ保存アプリ</title>
  <style>
    textarea { width: 90%; height: 150px; }
    button { margin: 5px; }
  </style>
</head>
<body>
  <h1>メモ帳</h1>
  <textarea id="memo"></textarea><br>
  <button onclick="saveMemo()">保存</button>
  <button onclick="loadMemo()">読み込み</button>
  <button onclick="clearMemo()">削除</button>

  <script>
    function saveMemo() {
      const text = document.getElementById("memo").value;
      localStorage.setItem("mymemo", text);
      alert("保存しました！");
    }

    function loadMemo() {
      const text = localStorage.getItem("mymemo");
      document.getElementById("memo").value = text || "";
    }

    function clearMemo() {
      localStorage.removeItem("mymemo");
      document.getElementById("memo").value = "";
      alert("削除しました！");
    }

    // ページ読み込み時に自動で読み込み
    window.addEventListener("load", loadMemo);
  </script>
</body>
</html>

ポイント

  • localStorage.setItem() でメモを保存
  • localStorage.getItem() でメモを読み込み
  • localStorage.removeItem() でメモを削除
  • window.addEventListener("load", loadMemo) でページ読み込み時に自動読み込み
