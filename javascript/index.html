JavaScript入門

Webページに動きをつけ、インタラクティブな体験を作り出すプログラミング言語を学びましょう。基礎から実践的なアプリケーション開発まで、段階的に習得できます。

1
JavaScriptの世界へようこそ

JavaScriptの歴史、開発環境の準備、初めてのコード実行まで。プログラミングの第一歩を踏み出します。

歴史 開発環境 DevTools
2
JavaScript基礎文法

変数、データ型、演算子など、プログラミングの基本要素を学びます。電卓アプリを作成します。

変数 データ型 演算子
3
条件分岐と繰り返し

if文、switch文、for/whileループ、配列の基礎を学び、おみくじアプリを作成します。

if-else ループ 配列
4
関数

関数宣言、アロー関数、スコープ、コールバックを学び、カラーピッカーを作成します。

function アロー関数 スコープ
5
DOM操作

HTML要素の取得、作成、変更、削除を学び、ToDoリストアプリを作成します。

querySelector createElement classList
6
イベント処理

イベントリスナー、イベントオブジェクト、委譲を学び、ダークモード切替を実装します。

addEventListener イベント委譲 LocalStorage
7
オブジェクトと配列の応用

オブジェクト操作、配列メソッド、分割代入、JSONを学び、商品リストを作成します。

map/filter 分割代入 JSON
8
フォーム処理とバリデーション

フォーム要素、バリデーション、正規表現を学び、会員登録フォームを作成します。

バリデーション 正規表現 フォームイベント
9
非同期処理とAPI連携

Promise、async/await、Fetch APIを学び、天気予報アプリを作成します。

Promise async/await Fetch API
10
アニメーション

CSSトランジション、Intersection Observer、スクロールアニメーションを学びます。

transition Observer スクロール
11
モダンJavaScriptとモジュール

ES Modules、モジュール分割パターンを学び、モジュール版ToDoアプリを作成します。

import/export モジュール設計 単一責任
12
総合演習 - ポートフォリオサイト

学んだ知識を総動員して、本格的なポートフォリオサイトを構築します。

ローダー ギャラリー API連携
