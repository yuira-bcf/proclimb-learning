Section 2

スムーズスクロール

ページ内リンクをクリックしたとき、スルスルと滑らかにスクロールする機能

ファイル：Chapter_11-2 \ index.html

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>スムーズスクロール</title>
  <style>
    section { height: 100vh; padding: 20px; }
    #section1 { background: #ffcccc; }
    #section2 { background: #ccffcc; }
    #section3 { background: #ccccff; }
    nav { position: fixed; top: 0; background: white; padding: 10px; }
    nav a { margin: 0 10px; }
  </style>
</head>
<body>
  <nav>
    <a href="#section1">セクション1</a>
    <a href="#section2">セクション2</a>
    <a href="#section3">セクション3</a>
  </nav>

  <section id="section1"><h2>セクション1</h2></section>
  <section id="section2"><h2>セクション2</h2></section>
  <section id="section3"><h2>セクション3</h2></section>

  <script>
    const links = document.querySelectorAll('nav a');

    links.forEach((link) => {
      link.addEventListener('click', (e) => {
        e.preventDefault();
        const targetId = link.getAttribute('href');
        const target = document.querySelector(targetId);
        target.scrollIntoView({ behavior: 'smooth' });
      });
    });
  </script>
</body>
</html>

ポイント

  • e.preventDefault() - デフォルトのジャンプ動作を止める
  • getAttribute('href') - リンク先のIDを取得
  • scrollIntoView({ behavior: 'smooth' }) - 滑らかにスクロール
Section 3

ハンバーガーメニュー

スマホサイトでよく見る三本線のメニューボタン

ファイル：Chapter_11-3 \ index.html

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>ハンバーガーメニュー</title>
  <style>
    * { margin: 0; padding: 0; box-sizing: border-box; }

    .hamburger {
      width: 30px;
      height: 25px;
      position: fixed;
      top: 20px;
      right: 20px;
      cursor: pointer;
      z-index: 100;
    }

    .hamburger span {
      display: block;
      width: 100%;
      height: 3px;
      background: #333;
      margin-bottom: 8px;
      transition: 0.3s;
    }

    .hamburger.active span:nth-child(1) {
      transform: rotate(45deg) translate(8px, 8px);
    }
    .hamburger.active span:nth-child(2) {
      opacity: 0;
    }
    .hamburger.active span:nth-child(3) {
      transform: rotate(-45deg) translate(8px, -8px);
    }

    .menu {
      position: fixed;
      top: 0;
      right: -250px;
      width: 250px;
      height: 100vh;
      background: #333;
      transition: 0.3s;
      padding-top: 80px;
    }

    .menu.active {
      right: 0;
    }

    .menu a {
      display: block;
      color: white;
      padding: 15px 20px;
      text-decoration: none;
    }
  </style>
</head>
<body>
  <div class="hamburger" id="hamburger">
    <span></span>
    <span></span>
    <span></span>
  </div>

  <nav class="menu" id="menu">
    <a href="#">ホーム</a>
    <a href="#">サービス</a>
    <a href="#">会社概要</a>
    <a href="#">お問い合わせ</a>
  </nav>

  <h1 style="padding: 100px 20px;">ハンバーガーメニュー</h1>

  <script>
    const hamburger = document.getElementById('hamburger');
    const menu = document.getElementById('menu');

    hamburger.addEventListener('click', () => {
      hamburger.classList.toggle('active');
      menu.classList.toggle('active');
    });
  </script>
</body>
</html>

ポイント

  • classList.toggle('active') - クラスの追加/削除を切り替え
  • CSSのtransitionでアニメーション効果
  • transform: rotate()で三本線を×に変形
Section 4

モーダルウィンドウ

画面中央にポップアップで表示されるダイアログ

ファイル：Chapter_11-4 \ index.html

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>モーダルウィンドウ</title>
  <style>
    .overlay {
      display: none;
      position: fixed;
      top: 0;
      left: 0;
      width: 100%;
      height: 100%;
      background: rgba(0, 0, 0, 0.5);
    }

    .overlay.active {
      display: flex;
      justify-content: center;
      align-items: center;
    }

    .modal {
      background: white;
      padding: 30px;
      border-radius: 10px;
      max-width: 400px;
      text-align: center;
    }

    .modal h2 { margin-bottom: 15px; }
    .modal p { margin-bottom: 20px; color: #666; }

    .btn {
      padding: 10px 30px;
      border: none;
      border-radius: 5px;
      cursor: pointer;
      font-size: 16px;
    }

    .btn-open { background: #007bff; color: white; }
    .btn-close { background: #6c757d; color: white; }
  </style>
</head>
<body>
  <div style="padding: 50px;">
    <button class="btn btn-open" id="openBtn">モーダルを開く</button>
  </div>

  <div class="overlay" id="overlay">
    <div class="modal">
      <h2>お知らせ</h2>
      <p>これはモーダルウィンドウです。背景をクリックしても閉じます。</p>
      <button class="btn btn-close" id="closeBtn">閉じる</button>
    </div>
  </div>

  <script>
    const openBtn = document.getElementById('openBtn');
    const closeBtn = document.getElementById('closeBtn');
    const overlay = document.getElementById('overlay');

    openBtn.addEventListener('click', () => {
      overlay.classList.add('active');
    });

    closeBtn.addEventListener('click', () => {
      overlay.classList.remove('active');
    });

    // 背景クリックで閉じる
    overlay.addEventListener('click', (e) => {
      if (e.target === overlay) {
        overlay.classList.remove('active');
      }
    });
  </script>
</body>
</html>

ポイント

  • display: noneとdisplay: flexで表示切り替え
  • e.target === overlayで背景クリックを判定
  • rgba(0, 0, 0, 0.5)で半透明の背景
Section 5

画像スライダー

画像を左右にスライドして切り替える機能

ファイル：Chapter_11-5 \ index.html

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>画像スライダー</title>
  <style>
    .slider {
      width: 400px;
      margin: 50px auto;
      text-align: center;
    }

    .slider img {
      width: 100%;
      height: 300px;
      object-fit: cover;
      border-radius: 10px;
    }

    .controls {
      margin-top: 15px;
    }

    .controls button {
      padding: 10px 20px;
      margin: 0 5px;
      border: none;
      background: #333;
      color: white;
      border-radius: 5px;
      cursor: pointer;
    }

    .dots {
      margin-top: 10px;
    }

    .dot {
      display: inline-block;
      width: 12px;
      height: 12px;
      background: #ccc;
      border-radius: 50%;
      margin: 0 5px;
      cursor: pointer;
    }

    .dot.active {
      background: #333;
    }
  </style>
</head>
<body>
  <div class="slider">
    <img id="slideImg" src="https://picsum.photos/400/300?random=1" alt="スライド">

    <div class="controls">
      <button id="prevBtn">← 前へ</button>
      <button id="nextBtn">次へ →</button>
    </div>

    <div class="dots" id="dots"></div>
  </div>

  <script>
    const images = [
      'https://picsum.photos/400/300?random=1',
      'https://picsum.photos/400/300?random=2',
      'https://picsum.photos/400/300?random=3',
      'https://picsum.photos/400/300?random=4'
    ];

    let currentIndex = 0;
    const slideImg = document.getElementById('slideImg');
    const dotsContainer = document.getElementById('dots');

    // ドットを生成
    images.forEach((_, i) => {
      const dot = document.createElement('span');
      dot.classList.add('dot');
      if (i === 0) dot.classList.add('active');
      dot.addEventListener('click', () => goToSlide(i));
      dotsContainer.appendChild(dot);
    });

    function updateSlide() {
      slideImg.src = images[currentIndex];
      // ドットの更新
      document.querySelectorAll('.dot').forEach((dot, i) => {
        dot.classList.toggle('active', i === currentIndex);
      });
    }

    function goToSlide(index) {
      currentIndex = index;
      updateSlide();
    }

    document.getElementById('prevBtn').addEventListener('click', () => {
      currentIndex = (currentIndex - 1 + images.length) % images.length;
      updateSlide();
    });

    document.getElementById('nextBtn').addEventListener('click', () => {
      currentIndex = (currentIndex + 1) % images.length;
      updateSlide();
    });
  </script>
</body>
</html>

ポイント

  • % images.length - 配列の範囲内でループ
  • forEachでドットを動的に生成
  • classList.toggle('active', 条件) - 条件に応じてクラス切り替え
Section 6

フォームバリデーション

入力内容をチェックしてエラーメッセージを表示

ファイル：Chapter_11-6 \ index.html

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>フォームバリデーション</title>
  <style>
    .form-group {
      margin-bottom: 20px;
      max-width: 400px;
    }

    label {
      display: block;
      margin-bottom: 5px;
      font-weight: bold;
    }

    input {
      width: 100%;
      padding: 10px;
      border: 2px solid #ddd;
      border-radius: 5px;
      font-size: 16px;
    }

    input.error {
      border-color: #dc3545;
    }

    input.success {
      border-color: #28a745;
    }

    .error-message {
      color: #dc3545;
      font-size: 14px;
      margin-top: 5px;
    }

    button {
      padding: 12px 30px;
      background: #007bff;
      color: white;
      border: none;
      border-radius: 5px;
      font-size: 16px;
      cursor: pointer;
    }
  </style>
</head>
<body>
  <div style="padding: 50px;">
    <h1>お問い合わせフォーム</h1>

    <form id="contactForm">
      <div class="form-group">
        <label>お名前</label>
        <input type="text" id="name">
        <div class="error-message" id="nameError"></div>
      </div>

      <div class="form-group">
        <label>メールアドレス</label>
        <input type="text" id="email">
        <div class="error-message" id="emailError"></div>
      </div>

      <div class="form-group">
        <label>電話番号</label>
        <input type="text" id="tel">
        <div class="error-message" id="telError"></div>
      </div>

      <button type="submit">送信</button>
    </form>
  </div>

  <script>
    const form = document.getElementById('contactForm');

    form.addEventListener('submit', (e) => {
      e.preventDefault();
      let isValid = true;

      // 名前のチェック
      const name = document.getElementById('name');
      const nameError = document.getElementById('nameError');
      if (name.value.trim() === '') {
        nameError.textContent = 'お名前を入力してください';
        name.classList.add('error');
        name.classList.remove('success');
        isValid = false;
      } else {
        nameError.textContent = '';
        name.classList.remove('error');
        name.classList.add('success');
      }

      // メールのチェック
      const email = document.getElementById('email');
      const emailError = document.getElementById('emailError');
      const emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
      if (!emailPattern.test(email.value)) {
        emailError.textContent = '正しいメールアドレスを入力してください';
        email.classList.add('error');
        email.classList.remove('success');
        isValid = false;
      } else {
        emailError.textContent = '';
        email.classList.remove('error');
        email.classList.add('success');
      }

      // 電話番号のチェック
      const tel = document.getElementById('tel');
      const telError = document.getElementById('telError');
      const telPattern = /^[0-9-]+$/;
      if (tel.value !== '' && !telPattern.test(tel.value)) {
        telError.textContent = '数字とハイフンのみ入力してください';
        tel.classList.add('error');
        tel.classList.remove('success');
        isValid = false;
      } else {
        telError.textContent = '';
        tel.classList.remove('error');
        if (tel.value !== '') tel.classList.add('success');
      }

      if (isValid) {
        alert('送信完了！');
      }
    });
  </script>
</body>
</html>

ポイント

  • e.preventDefault() - フォームの送信を止める
  • trim() - 前後の空白を除去
  • /正規表現/.test(値) - パターンマッチングでチェック
Section 7

スクロールアニメーション

スクロールすると要素がフェードインして登場する効果

ファイル：Chapter_11-7 \ index.html

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>スクロールアニメーション</title>
  <style>
    * { margin: 0; padding: 0; box-sizing: border-box; }

    body {
      font-family: sans-serif;
      line-height: 1.8;
    }

    section {
      padding: 100px 50px;
      text-align: center;
    }

    section:nth-child(odd) {
      background: #f5f5f5;
    }

    .fade-in {
      opacity: 0;
      transform: translateY(30px);
      transition: opacity 0.6s, transform 0.6s;
    }

    .fade-in.visible {
      opacity: 1;
      transform: translateY(0);
    }

    h2 { margin-bottom: 20px; }
    p { max-width: 600px; margin: 0 auto; color: #666; }
  </style>
</head>
<body>
  <section>
    <div class="fade-in">
      <h2>セクション 1</h2>
      <p>スクロールすると下のセクションがフェードインします。</p>
    </div>
  </section>

  <section>
    <div class="fade-in">
      <h2>セクション 2</h2>
      <p>このように、スクロールに応じてコンテンツを表示できます。</p>
    </div>
  </section>

  <section>
    <div class="fade-in">
      <h2>セクション 3</h2>
      <p>Webサイトの演出によく使われるテクニックです。</p>
    </div>
  </section>

  <section>
    <div class="fade-in">
      <h2>セクション 4</h2>
      <p>最後のセクションです。お疲れ様でした！</p>
    </div>
  </section>

  <script>
    const fadeElements = document.querySelectorAll('.fade-in');

    function checkScroll() {
      fadeElements.forEach((el) => {
        const rect = el.getBoundingClientRect();
        const windowHeight = window.innerHeight;

        // 要素が画面の80%の位置に来たら表示
        if (rect.top < windowHeight * 0.8) {
          el.classList.add('visible');
        }
      });
    }

    // スクロール時にチェック
    window.addEventListener('scroll', checkScroll);

    // ページ読み込み時にもチェック
    window.addEventListener('load', checkScroll);
  </script>
</body>
</html>

ポイント

  • getBoundingClientRect() - 要素の位置を取得
  • window.innerHeight - ウィンドウの高さ
  • rect.top < windowHeight * 0.8 - 80%の位置で発火
