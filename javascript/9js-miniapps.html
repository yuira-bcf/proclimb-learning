Section 1

デジタル時計アプリ

リアルタイムで時刻を表示するアプリ。JavaScriptの「時間オブジェクト（Date）」を使います。

ファイル：Chapter_9-1_DigitalClock \ index.html

コード例

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>デジタル時計</title>
  <style>
    body {
      text-align: center;
      font-size: 48px;
      font-family: monospace;
      background: black;
      color: lime;
    }
  </style>
</head>
<body>
  <div id="clock">00:00:00</div>

  <script>
    function showClock() {
      const now = new Date();
      const h = String(now.getHours()).padStart(2, "0");
      const m = String(now.getMinutes()).padStart(2, "0");
      const s = String(now.getSeconds()).padStart(2, "0");
      document.getElementById("clock").textContent = `${h}:${m}:${s}`;
    }

    setInterval(showClock, 1000); // 1秒ごとに実行
  </script>
</body>
</html>

ポイント

  • new Date()：現在の時刻を取得する、JavaScriptで用意されている構文
  • padStart(2, "0")：2桁に揃える、かつ取得した時間が1桁しかない場合は先頭に0を追加して2桁にする。（例：9時 → 09）
  • setInterval(showClock, 1000)：1000ミリ秒（1秒）ごとに、showClock関数を呼び続ける
Section 2

簡単な電卓アプリ

2つの数字を入力して「＋−×÷」の計算をするアプリ。

ファイル：Chapter_9-2_Calculator \ index.html

コード例

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>簡単電卓</title>
</head>
<body>
  <h1>電卓アプリ</h1>

  <input type="number" id="num1" placeholder="数字1">
  <input type="number" id="num2" placeholder="数字2">
  <button onclick="calc('+')">＋</button>
  <button onclick="calc('-')">−</button>
  <button onclick="calc('*')">×</button>
  <button onclick="calc('/')">÷</button>

  <h2 id="result"></h2>

  <script>
    function calc(op) {
      const a = Number(document.getElementById("num1").value);
      const b = Number(document.getElementById("num2").value);
      let r;

      if (op === "+") r = a + b;
      else if (op === "-") r = a - b;
      else if (op === "*") r = a * b;
      else if (op === "/") r = b !== 0 ? a / b : "エラー（0で割れません）";

      document.getElementById("result").textContent = "結果：" + r;
    }
  </script>
</body>
</html>

ポイント

  • Number()で文字を数値に変換
  • ifで計算の種類を切り替える
  • 0除算エラーを条件で防止
三項演算子

r = b !== 0 ? a / b : "エラー（0で割れません）";

  1. b は 0 ではない（条件）
  2. true なら、a / b の処理を実行
  3. false なら、"エラー（0で割れません）" を表示

以下のif文と同じ意味になる省略記法：

if (b !== 0) {
    r = a / b;
} else {
    r = "エラー（0で割れません）";
}
Section 3

おみくじアプリ

ボタンを押すとランダムに結果が出る「おみくじ」。

ファイル：Chapter_8-3_Omikuji \ index.html

コード例

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>おみくじ</title>
</head>
<body style="text-align:center;">
  <h1>今日の運勢は？</h1>
  <button onclick="draw()">おみくじを引く</button>
  <h2 id="result"></h2>

  <script>
    function draw() {
      const results = ["大吉", "中吉", "小吉", "凶"];
      const index = Math.floor(Math.random() * results.length);
      document.getElementById("result").textContent = results[index];
    }
  </script>
</body>
</html>

ポイント

  • results.length：配列の中の要素の数を返す（4）
  • Math.random()：0〜1の乱数を出す
    • Math.random() * results.length で 配列の数をかけるので、0~1 * 4 = 0 ~4 のランダムな数が出る
  • Math.floor()：小数点を切り捨て
    • Math.floor(Math.random() * results.length) で、1.23456... のような数字になっても、1 になる
  • 配列（今回はresults）を使って選択肢を用意
    • results[index] で、indexには 0~4 のいずれかの値が入る
Section 4

ToDoリストアプリ

タスクを入力して、ボタンで追加／削除できるアプリ。

ファイル：Chapter_9-4_ToDoList \ index.html

コード例

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>ToDoリスト</title>
  <style>
    li { cursor: pointer; }
    li.done { text-decoration: line-through; color: gray; }
  </style>
</head>
<body>
  <h1>ToDoリスト</h1>
  <input id="task" placeholder="やることを入力">
  <button id="addBtn">追加</button>
  <ul id="list"></ul>

  <script>
    document.getElementById("addBtn").addEventListener("click", () => {
      const taskText = document.getElementById("task").value.trim();
      if (taskText === "") return;

      const li = document.createElement("li");
      li.textContent = taskText;

      li.addEventListener("click", () => {
        li.classList.toggle("done");
      });

      document.getElementById("list").appendChild(li);
      document.getElementById("task").value = "";
    });
  </script>
</body>
</html>

ポイント

  • .trim()：要素の先頭と末尾の空白を除去する関数
    • if (taskText === "") return; で、trim()の結果、""（空文字）になる場合は、何もせず処理を終える
  • classList.toggle()：クラス「done」の付け外し（ON/OFFスイッチの役割）
    • li.done { text-decoration: line-through; color: gray; } で、doneクラスが付与されたら打消し線と、色をグレーにする
  • appendChild(li)：HTMLに li に入っている要素をリストとして追加
  • document.getElementById("task").value = "";：最後に入力フォームに入っていた文字を""(空文字)にして、入力前の状態に戻しておく
