Section 1-1

JavaScriptとは

JavaScriptは、Webページに動きやインタラクティブな機能を追加するためのプログラミング言語です。世界中で最も使われているプログラミング言語の一つで、Web開発には欠かせない存在です。

JavaScriptの歴史

JavaScriptは1995年、Netscape社のブレンダン・アイクによってわずか10日間で開発されました。当初は「LiveScript」という名前でしたが、当時人気だったJava言語にあやかって「JavaScript」に改名されました。

JavaとJavaScriptは別物です！

名前が似ていますが、JavaとJavaScriptは全く異なるプログラミング言語です。「メロン」と「メロンパン」くらい違います。混同しないよう注意しましょう。

ECMAScriptとの関係

JavaScriptの標準仕様はECMAScript（エクマスクリプト）と呼ばれます。Ecma Internationalという標準化団体が策定しており、各ブラウザはこの仕様に基づいてJavaScriptエンジンを実装しています。

バージョン リリース年 主な追加機能
ES5 2009 strict mode, JSON, forEach
ES6 (ES2015) 2015 let/const, アロー関数, class, Promise, モジュール
ES2017 2017 async/await
ES2020 2020 オプショナルチェーン(?.), Null合体演算子(??)
Section 1-2

JavaScriptでできること

JavaScriptは当初ブラウザでのみ動作する言語でしたが、現在ではサーバー、モバイルアプリ、デスクトップアプリなど、あらゆる場面で活躍しています。

Webページの動的な変更

HTMLやCSSを操作して、ページの見た目や内容をリアルタイムで変更できます。

🎨

スタイルの変更

ボタンクリックで色を変えたり、スクロールに応じてヘッダーを固定したりできます。

✏️

コンテンツの更新

新しい要素を追加したり、テキストを書き換えたりできます。

🎭

アニメーション

フェードイン、スライド、回転などの動きを実装できます。

ユーザーインタラクション

ユーザーの操作（クリック、入力、スクロールなど）に反応して、様々な処理を実行できます。

chat プロンプト 
## 機能
ボタンクリックでアラート表示、フォーム入力でコンソール出力

## 実装要件
- `addEventListener`でclickイベントを登録
- `addEventListener`でinputイベントを登録
- `alert()`でメッセージを表示
- `console.log()`で入力値を出力

## 使用するAPI
- document.querySelector()
- addEventListener()
- alert()
- console.log()

インタラクションの例
JavaScript 
// ボタンがクリックされたら実行
button.addEventListener('click', function() {
  alert('ボタンがクリックされました！');
});

// フォームに入力されたら実行
input.addEventListener('input', function(e) {
  console.log('入力値:', e.target.value);
});

開発環境の準備

JavaScript開発を始めるために必要なツールをインストールしましょう。基本的にはテキストエディタとブラウザがあれば開発を始められます。

インストール

拡張機能名 説明

console.log() でデバッグ

console.log()は、変数の値やプログラムの実行状況を確認するために使用します。JavaScriptコード内に記述すると、その内容がConsoleに出力されます。

chat プロンプト 
## 機能
`console.log()`による各種データ型の出力

## 実装要件
- 文字列リテラルの出力
- 変数の値を出力
- 計算結果の出力
- オブジェクトの出力

## 使用するAPI
- console.log()

## 出力するデータ型
- String（文字列）
- Number（数値）
- Object（オブジェクト）
console.log() の使い方
JavaScript 
// 文字列を出力
console.log("プログラムが開始されました");

// 変数の値を確認
let name = "田中";
console.log("名前:", name);  // → 名前: 田中

// 計算結果を確認
let price = 1000;
let tax = price * 0.1;
console.log("税額:", tax);    // → 税額: 100

// オブジェクトを出力
let user = { name: "田中", age: 25 };
console.log(user);  // → {name: "田中", age: 25}
出力結果

console.log()

console.log(出力したい値);

開発者ツールのコンソールにメッセージや変数の値を出力します。デバッグ時に変数の中身を確認したり、プログラムの実行状況を追跡するために使用します。複数の値をカンマで区切って同時に出力することもできます。

warning console.log() は開発用です

本番環境（公開するWebサイト）にはconsole.log()を残さないようにしましょう。ユーザーには見えませんが、セキュリティ上のリスクになることがあります。
