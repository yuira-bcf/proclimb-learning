Section 1

DOM（ドム）とは？

意味：Document Object Model（ドキュメント・オブジェクト・モデル）

ちょっと難しそうですが、簡単に言うと…
JavaScriptからHTMLを見たり、変えたりできる"地図"のような仕組みです。

JavaScriptがなぜWebページに動きを与えられるのか。それは、Webページの構造であるHTMLをブラウザが読み込んだとき、ブラウザはJavaScriptが読込可能なdocumentというオブジェクトを生成します。そのdocumentオブジェクトの中から、HTMLのタグなどの要素を探して拾い、その要素に直接プログラミングとして、処理を実行しているからです。

たとえば、こんなHTMLがあったとしましょう。

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
</head>
<body>
  <h1 id="title">DOMの説明</h1>
  <ul>
    <li class="list">HTMLとは</li>
    <li class="list">JavaScriptとは</li>
    <li class="list">DOMとは</li>
  </ul>
</body>
</html>

このHTMLをブラウザが読み込んだとき、ブラウザは以下のようなdocumentオブジェクトを生成します。

DOMツリー構造

document
 └ <html>
     ├ <head>
     └ <body>
          ├ <h1 id="title">DOMの説明</h1>
          └ <ul>
               ├ <li class="list">HTMLとは</li>
               ├ <li class="list">JavaScriptとは</li>
               └ <li class="list">DOMとは</li>

JavaScriptはこのdocumentオブジェクトを参照し、その下のbodyなどをツリー構造として把握します。様々なメソッドを介して、このツリーを伝って要素を探索し、いろんな操作をします。

Section 2

要素を取得する（探す）

このdocumentオブジェクトからJavaScriptがHTMLの要素を探す（取得する）には以下のようなメソッドをします。

メソッド 意味 使い方の例
getElementById() id名で探す document.getElementById("title")
getElementsByClassName() クラス名で探す（複数） document.getElementsByClassName("item")
getElementsByTagName() タグ名で探す（複数） document.getElementsByTagName("p")
querySelector() CSSセレクタで探す（最初1つ） document.querySelector(".item")
querySelectorAll() CSSセレクタで探す（全部） document.querySelectorAll("p.item")
POINT

要素を取得する場合、必ず最初は「document」を指定します。とても長いメソッド名ですが、一語一句覚えなくても大丈夫。IdやClassNameなど、まずは代表的な文言で、何をしたいかを把握していきましょう。

ファイル：Chapter_7-2 \ index.html

例1：idで要素を取得する

HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>DOM操作</title>
</head>
<body>
  <h1 id="title">こんにちは！</h1>

  <script>
    let element = document.getElementById("title")
    console.log(element);
  </script>
</body>
</html>

コンソール結果：<h1 id="title">こんにちは</h1>

idで、変数elemnetの中に、h1タグの要素が取得出来ていることがわかります。

例2：取得した要素のテキストを変更する

こうして取得した要素に対して、状態を変化させるメソッドを使うことで、JavaScriptはWebページをいろいろ変化させることが可能です。

.textContent を使う（タグのテキストを変更する）

先ほどのコードの下に、以下を書いてみましょう。

JavaScript 
element.textContent = "こんばんは";

ブラウザ表示結果：こんばんは

HTMLタグの中の「テキスト部分」が変わりました。こんなふうに、JavaScriptはHTMLを特定して、その要素を直接変更することができます。

Section 3

HTMLタグを変更・追加する

ファイル：Chapter_7-2 \ index.htmlの内容に追記していこう。

例3：HTMLごと変更（innerHTML）

HTML 
<p id="msg">こんにちは！</p>
<script>
  document.getElementById("msg").innerHTML = "<strong>こんにちは！</strong> JavaScriptです。";
</script>

innerHTML は、追加したテキストにHTMLタグが含まれていると、HTMLのタグの機能を持った状態で変更できます。

POINT

.textContentでは、HTMLではなく、単純な文章として解釈されてしまう。

document.getElementById("msg").textContent = "<strong>こんにちは！</strong> JavaScriptです。";

結果：<strong>こんにちは！</strong> JavaScriptです。

例4：要素を追加する（createElement, appendChild）

HTML 
<ul id="list"></ul>

<script>
  let list = document.getElementById("list");
  let newItem = document.createElement("li");
  newItem.textContent = "新しい項目";
  list.appendChild(newItem);
</script>

ブラウザ表示結果：・新しい項目

HTML結果：

<ul>
  <li>新しい項目</li>
</ul>

index.htmlに記述されていない要素を、JavaScriptでファイルを変更することなく、リアルタイムで要素を作成して追加することができます。

  • createElement() → 新しいタグを作る
  • appendChild() → 作ったタグを子要素として追加する

処理の流れ

  1. document.getElementById("list"); で、id="list"の要素を取得
  2. document.createElement("li"); で、<li>タグを新規作成（まだ中身のテキストは何もない）して、newItemに入れる
  3. newItem.textContent = "新しい項目"; で、<li></li>の間にテキストを挿入（これで<li>新しい項目</li>の出来上がり）
  4. list.appendChild(newItem); で、<ul>の直下に<li>新しい項目</li>を子要素として追加
Section 4

スタイルを変える（CSSを操作）

JavaScriptではHTMLだけではなく、取得した要素のCSSを変更することもできます。

例5：文字色や背景を変える

HTML 
<p id="text">色が変わるよ！</p>

<script>
  let text = document.getElementById("text");
  text.style.color = "red";
  text.style.backgroundColor = "yellow";
</script>

結果：赤文字＋黄色背景

.styleはCSSに変更を加えるメソッド。それ以降のメソッドは、具体的な変更内容を指定するメソッドが続きます。

※「.style.プロパティ名」のプロパティ名は、 CSS の 「-」部分を キャメルケース に変える必要があります。

CSS の指定方法 JavaScriptの.styleでの指定方法 意味
background-color backgroundColor 背景色を変える
font-size fontSize 文字の大きさを変える
text-align textAlign 文字の配置を変える
Section 5

classListを使ってデザイン切り替え

HTMLのクラス（class="●●"）は、重複可能な識別名をタグに付与します。これによって、複数の要素にCSSを当てたり、JavaScriptで一括で要素を操作することができます。

classListメソッドは、HTMLタグにクラスを付与したり、外したりできるメソッドです。

例6：クリックでクラスを切り替え

HTML 
<style>
  .active {
    color: white;
    background: red;
  }
</style>

<button id="btn">クリックで変化</button>

<script>
  let btn = document.getElementById("btn");
  btn.addEventListener("click", () => {
    btn.classList.toggle("active");
  });
</script>
ユーザー操作 ブラウザ表示 HTMLの中身
1度目のクリック ボタンが赤くなる <button id="btn" class="active">
2度目のクリック ボタンが元の色に戻る <button id="btn">

classList.toggle("クラス名")
要素にイベントが発生するたびに、指定のクラスを付けたり外したりすることが可能
※イベントについては第8章でくわしく説明

Section 6

複数要素をまとめて操作する

HTML 
<p class="item">りんご</p>
<p class="item">みかん</p>
<p class="item">ぶどう</p>

<script>
  let items = document.querySelectorAll(".item");

  items.forEach((i) => {
    i.style.color = "blue";
  });
</script>

document.querySelectorAll() で同じクラスの要素をまとめて取得できます。まとめて取得された各itemクラスには、配列で取得されます。それにforEach()を使用することで、取得した要素すべてに順番に変更を加えることができます。

結果：3つの文字がすべて青色になります

Section 7

値の取得（入力フォームなど）

HTML 
<input type="text" id="name">
<button id="showBtn">表示</button>
<p id="output"></p>

<script>
  document.getElementById("showBtn").addEventListener("click", showName);
  function showName() {
    let name = document.getElementById("name").value;
    document.getElementById("output").textContent = "こんにちは、" + name + "さん！";
  }
</script>

.valueメソッドを使うと、入力フォームなどの入力した値などが取得できます。それをテキストが空っぽのpタグに、JavaScriptで追加する、という挙動になっています。

ちょっとした応用

JavaScriptで関数を作っておくと、クリックイベントに引数として渡すことが出来ます。その結果、クリックイベントの記述が短くスッキリします。

document.getElementById("showBtn").addEventListener("click", showName);
この一行でOK！

Section 8

要素を削除する

HTML 
<ul id="delList">
  <li>消える予定の項目</li>
  <li>消える予定の項目</li>
  <li>消える予定の項目</li>
</ul>

<script>
  let list = document.getElementById("delList");
  list.remove(); // その要素を削除
</script>

<ul id="delList">は消さずに残して、子要素（<li>）だけ消したい場合：

JavaScript 
list.innerHTML = "";

JavaScriptのコードをコメントアウトしてみて、動きを確認もしてみよう。

Section 9

DOM操作の基本パターンまとめ

操作 コード 説明
要素を取得 document.getElementById("id") 特定の要素を取る
テキストを変える .textContent = "文字" テキストを変更
HTMLを変える .innerHTML = "<b>HTML</b>" HTMLタグが機能するテキストに変換
スタイル変更 .style.color = "red" CSS変更
クラス追加削除 .classList.add("a"), .remove(), .toggle() クラスの付け外しによるスタイル切替
新しい要素を作る document.createElement("p") 新しいタグを作る
要素を追加 .appendChild(要素) 既存タグにテキストや要素を追加
要素を削除 .remove() 消す
値を取得 .value 入力内容を読む

練習問題

ファイル：Chapter_7-10 \ index.htmlに書いてみよう

問1

id="msg" の中身「ここが変わる」を、JavaScriptを使って「JavaScript 最高！」に書き換えよう。

問2

ボタンをクリックしたら背景色を「lightblue」に変える。

問3

入力欄に名前を入れ、「表示」ボタンで「こんにちは、○○さん！」と表示。

問4

リストに新しい <li> 要素を追加する。

問5

クリックするたびに文字の色を赤→黒→赤…と切り替える。

解答例を見る

問1

<p id="msg">ここが変わる</p>
<script>
  document.getElementById("msg").textContent = "JavaScript最高！";
</script>

問2

<button id="btn">色を変える</button>
<script>
  document.getElementById("btn").addEventListener("click", () => {
    document.body.style.backgroundColor = "lightblue";
  });
</script>

問3

<input id="name"><button id="showBtn">表示</button><p id="out"></p>
<script>
  document.getElementById("showBtn").addEventListener("click", show);
  function show() {
    let n = document.getElementById("name").value;
    document.getElementById("out").textContent = "こんにちは、" + n + "さん！";
  }
</script>

問4

<ul id="list"></ul>
<script>
  let li = document.createElement("li");
  li.textContent = "新しい項目";
  document.getElementById("list").appendChild(li);
</script>

問5

<p id="txt">クリックで色変更</p>
<script>
  document.getElementById("txt").addEventListener("click", toggleColor);
  let red = false;
  function toggleColor() {
    red = !red;
    document.getElementById("txt").style.color = red ? "red" : "black";
  }
</script>
まとめ

第7章のまとめ

  • DOMはHTMLをJavaScriptから操作するための仕組み
  • getElementById()などで要素を取得
  • textContentでテキスト変更、innerHTMLでHTML変更
  • .styleでCSSを動的に変更
  • classListでクラスの付け外し
  • createElementとappendChildで要素を追加
  • .valueで入力フォームの値を取得
