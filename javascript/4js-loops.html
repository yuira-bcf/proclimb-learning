Section 1

「繰り返し」ってなに？

人間がやると大変だけど、コンピュータが得意なことがあります。それが――同じ作業を正確に、ひたすらに繰り返すこと！

たとえば…

  • 1〜10までの数字を順番に表示する
  • 10個の商品を一覧に出す
  • 5回「Hello!」と書く

「同じ処理を何度も繰り返す」 例として、1 行ずつ書くとこうなります。

JavaScript 
console.log("Hello!");
console.log("Hello!");
console.log("Hello!");
console.log("Hello!");
console.log("Hello!");

「これをもう100回増やして」なんて言われたら大変ですし、100回書いてからHelloをGood moning に変えておいて、って言われたらもう最悪です。書くのも、メンテナンスするのも大変なんです。

そこで使うのが3大原則最後の原則、「繰り返し文（ループ）」です。

JavaScriptでは主に以下の2つのループ文があります：

文の種類 読み方 使い道
for 文 フォー文 決まった回数を繰り返したいとき
while 文 ホワイル文 条件が満たされるまで繰り返したいとき（主に回数が決まっていないとき）
Section 2

for文の基本形

JavaScript 
for (初期値; 条件; 更新) {
  繰り返す処理;
}

意味：「初期値からスタートして、条件がtrueの間、処理を繰り返す。1回ごとに更新を行う。」

デベロッパーツールのコンソールに書いてみよう↓↓

例1：1〜5まで表示

JavaScript 
for (let i = 1; i <= 5; i++) {
  console.log(i);
}

実行結果：

1
2
3
4
5

解説：

部分 意味
let i = 1; 最初の数（初期値）を設定（最初の1回しか実行しない）
i <= 5; 繰り返す条件（変数i が5以下の間）
i++ 処理が終わる度に、i に +1 する（更新：インクリメントと呼ばれる）
{ ... } 繰り返す処理のかたまり

もう少しコードに合わせて説明

for (①let i = 1; ②i <= 5; ④i++) { ③console.log(i); }

  1. ①：初期値を設定。for文を読み込んだら最初に1回だけ実行（変数 i が作られ、1が代入される）
  2. ②：条件式で比較する（変数iはこの時点で1、5以下のためtrue）
  3. ③：条件がtrueなので、ここの処理を実行（コンソールに1が表示）
  4. ④：すべての処理が終了したら、最後に更新処理を行う（変数iの値に+1を実行、この時点でiは2になる）
  5. ⑤：②に戻って再度条件比較をする（④でiは2になっているが、5以下のためtrue）
  6. ⑥：②～④を、条件がfalseになるまで繰り返す
  7. ⑦：iが更新後に6になったとき、②の比較後にfalseになり、処理が実行されず繰り返しが終了する

複雑に見えますが、②～④をぐるぐるしている、という感じです。①だけは最初の一回だけしか実行されません。

Section 3

for文で応用してみよう

例2：メッセージを5回出す

JavaScript 
for (let i = 1; i <= 5; i++) {
  console.log(i + "回目：こんにちは！");
}

結果：

1回目：こんにちは！
2回目：こんにちは！
3回目：こんにちは！
4回目：こんにちは！
5回目：こんにちは！

例3：合計を計算する

JavaScript 
let sum = 0;

for (let i = 1; i <= 10; i++) {
  sum = sum + i;
}

console.log("合計は " + sum);

結果：合計は 55

i++ の意味

i++ は「iに1を足す」という意味で、i はカウンタ変数などと呼ばれます。この++は「インクリメント」と呼ばれる省略記法です。逆に--という、1ずつ減らすものもあり、「デクリメント」と呼ばれます。

インクリメントと同じ意味の書き方：i = i + 1;

Section 4

while文（条件がtrueの間くり返す）

JavaScript 
while (条件) {
  繰り返す処理;
}

for文は「回数が決まっている」繰り返し、while文は「条件が満たされるまで続く」繰り返しです。

例4：パスワードが合致するまで繰り返し入力を求められる

JavaScript 
let pass = "1234";
let input = prompt("パスワードを入力してください");

while (input != pass) {　// input が passと一致「しない」ときtrue
  input = prompt("パスワードが違います。もう一度入力してください");
}

console.log("ログイン成功しました");

prompt()は、ブラウザ上のポップアップで、ユーザーに入力を求めるメソッドです。

上記ではユーザーの入力が、あらかじめ変数passに入れている文字列と合致しない限り、パスワードの入力を促され続ける繰り返し処理となっています。

これは1回で入れることもあれば、何回も間違える可能性もあります。このように、繰り返す回数が定かでないときには、while文を使います。

Section 5

無限ループに注意！

たとえばもし、下記のようなコードにしてしまうと…

JavaScript 
let i = 1;
while (i <= 10) {
  console.log(i);
  i = i // +i を書き忘れた！！
}
警告

iが永遠に10になることがなく、終わらない「無限ループ」になります。「無限ループ」はブラウザやパソコンをフリーズさせる原因になります。繰り返し処理では、必ず変数を変化させましょう。

※ パソコンがフリーズしたりするので、実行しないでください

Section 6

for文とwhile文の使い分け

状況 おすすめの文 理由
回数が決まっている for 文 繰り返し回数・増減・条件を1行にまとめられる
終了回数があいまい while 文 状況に応じて柔軟に繰り返しを続けられる

例5：while文でカウントダウン

while文ではfor文と同じようにプログラムを組むこともできます。

JavaScript 
let count = 3;

while (count > 0) {
  console.log(count);
  count--;
}

console.log("スタート！");

結果：

3
2
1
スタート！
  1. 事前に初期設定の変数を用意
  2. 条件式を記述
  3. 処理の最後に更新値を記述

これでfor文と同じ流れを実現できます。

Section 7

繰り返し＋if文で条件を加える

JavaScript 
for (let i = 1; i <= 10; i++) {
  if (i % 2 == 0) {
    console.log(i + " は偶数です");
  }
}

% は「割った余り」を求める演算子です。i を2 で割って余りが0 なら「偶数」と言えます。偶数・奇数の判定をする定番の書き方です。

結果：

2 は偶数です
4 は偶数です
6 は偶数です
8 は偶数です
10 は偶数です
Section 8

ネスト（入れ子）ループ

ループの中にループを入れることもできます。

例7：2重ループで九九を全部計算！

JavaScript 
for (let i = 1; i <= 9; i++) {
  for (let j = 1; j <= 9; j++) {
    console.log(i + " × " + j + " = " + (i * j));
  }
}

結果：

1 × 1 = 1
1 × 2 = 2
...
9 × 9 = 81

81 行の結果が出ます！（9 × 9）

解説

  1. 最初のfor文でi =1となります
  2. i <= 9 で条件が判定されます（1回目はi=1なのでtrue）
  3. trueのとき、実行される処理はまたfor文です。jに1が入ります。（この時点でi=1, j=1）
  4. j <= 9 で条件が判定されます（1回目はj=1なのでtrue）
  5. trueの処理として、コンソールにiとjを掛け算した結果と文字列を表示します。
  6. 内側のfor文の処理が一度終わったので、更新処理としてjに1が足されます（この時点でi=1, j=2）
  7. ④～⑥を、jが9になるまで繰り返します。
  8. jが10になったら条件NGなので、i++で、iが2になります（この時点でi=2, j=10）
  9. ③から行われ、またjに1が入ります（この時点でi=2, j=1）
  10. ②～⑨を、iが10になるまで繰り返します。
  11. iが10になったら条件NGとなるので、すべての処理が終了します。
Section 9

break と continue

ループを途中で止めたり、スキップしたりできます。

命令 意味
break ループを完全に終了する
continue その回だけスキップして次へ進む

例8：breakで止める

JavaScript 
for (let i = 1; i <= 10; i++) {
  if (i === 5) break;
  console.log(i);
}

結果：

1
2
3
4

（iが5になったで繰り返しを強制終了）

例9：continueでスキップ

JavaScript 
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue;
  console.log(i);
}

結果：

1
2
4
5

（iが3のときだけ、処理をスキップ）

Section 10

実践課題：★を並べてみよう！

問題：★ を5個横に並べて表示してみましょう。

ヒント：文字列連結を使います

JavaScript 
let stars = "";
for (let i = 1; i <= 5; i++) {
  // ここに「stars に ★ を足す」処理を書く
}
console.log(stars);

結果：★★★★★

Section 11

ここまでのポイントまとめ

ポイント 内容
for文 決まった回数の繰り返し
while 文 回数の決まっていない繰り返し
i++ / -- i を1 ずつ増やす / 減らす（インクリメント/デクリメント）
break / continue 終了 / スキップ
ifとの組み合わせ 条件付きのループ

まとめ

第4章のまとめ

  • for文は決まった回数を繰り返す（初期値; 条件; 更新）
  • while文は条件が満たされるまで繰り返す
  • i++はインクリメント（+1）、i--はデクリメント（-1）
  • 無限ループに注意（必ず終了条件を設定）
  • breakでループを終了、continueでスキップ
  • ループの中にif文やループを入れられる（ネスト）
