Git & GitHub完全ガイド

バージョン管理の基礎からGitHub活用、チーム開発ワークフロー、GitHub Actions・Codespaces、VSCode MCP連携まで。現場で使えるGitスキルを15章で体系的に学べるチュートリアルです。

1
バージョン管理入門

バージョン管理の必要性、集中型と分散型の違い、Gitが選ばれる理由と基本用語を学びます。

バージョン管理 集中型vs分散型 基本用語
2
Gitの導入と初期設定

OS別のGitインストール、git configによる初期設定、エディタ設定と確認方法を学びます。

インストール git config 初期設定
3
基本操作（init, add, commit）

リポジトリ初期化、ファイルの追跡、コミット、ステータス確認、3つのエリアの関係を学びます。

git init git add git commit
4
履歴の確認と探索

git logの表示オプション、コミット詳細、差分確認（git diff）、変更履歴の追跡を学びます。

git log git diff git blame
5
変更の取り消しテクニック

ステージング取り消し、ワーキングツリー復元、コミット修正、revert、resetの使い分けを学びます。

git restore git revert git reset
6
ブランチの基礎と運用

ブランチの概念、作成・切り替え、削除、命名規則、活用パターンを学びます。

git branch git switch 命名規則
7
マージとリベース

Fast-forwardマージ、3-wayマージ、リベースの仕組み、使い分けの判断基準を学びます。

git merge git rebase Fast-forward
8
GitHubの基礎と設定

GitHubアカウント作成、SSH鍵設定、リポジトリ作成・削除、画面の見方を学びます。

アカウント作成 SSH設定 リポジトリ管理
9
リモート操作とコラボレーション

リモート追加・確認、push/pull/clone/fetch操作、コラボレーター招待と権限管理を学びます。

git push git pull コラボレーター
10
プルリクエストとIssue管理

PR作成手順、コードレビュー、Issue活用、ラベル・マイルストーン、PRとIssueの連携を学びます。

Pull Request Issue コードレビュー
11
コンフリクト解決テクニック

コンフリクトの発生条件、解決手順、VSCodeでの解決方法、予防のベストプラクティスを学びます。

コンフリクト 解決手順 VSCode活用
12
チーム開発フローと.gitignore

Git Flow、GitHub Flow、.gitignore設定、チーム開発のベストプラクティスを学びます。

Git Flow GitHub Flow .gitignore
13
GitHubの活用機能

GitHub Pages、Actions（CI/CD）、Codespaces、Wiki、Projects、Discussionsの活用方法を学びます。

GitHub Pages Actions Codespaces
14
実践：チームプロジェクト演習

チームポートフォリオサイトを題材に、ブランチ戦略・PR・コードレビューを実践演習します。

実践演習 チーム開発 コードレビュー
15
VSCode GitHub MCP活用

Model Context Protocolの概要、VSCodeでのGitHub MCP設定、AIを使ったGit操作の自動化を学びます。

MCP VSCode AI連携
