Docker構築入門

コンテナ技術の基礎からDockerfile、Docker Compose、ボリューム・ネットワーク、開発環境構築、AI・MCP活用まで。現場で即戦力になるDocker スキルを10章で体系的に学べるチュートリアルです。

1
コンテナ技術入門

Docker概要、仮想マシンとコンテナの違い、コンテナが選ばれる理由と基本用語を学びます。

コンテナ VM vs コンテナ 基本用語

Docker Desktopの導入

Windows/macOS/LinuxへのDocker Desktopインストール、初期設定、動作確認を学びます。

インストール Docker Desktop 動作確認
3
イメージとコンテナの基礎

docker pull/run/stop/rm/ps/imagesの基本コマンドとイメージ・コンテナの関係を学びます。

docker run docker ps ライフサイクル
4
Dockerfileの書き方

FROM/RUN/COPY/CMD/EXPOSE命令、マルチステージビルド、.dockerignoreを学びます。

Dockerfile docker build レイヤー
5
Docker Composeの基礎

docker-compose.ymlの書き方、複数コンテナの起動・停止・管理方法を学びます。

docker compose YAML マルチコンテナ
6
ボリュームとネットワーク

データの永続化（ボリューム・バインドマウント）、コンテナ間通信、ネットワーク設定を学びます。

Volume Network 永続化
7
実践：Webアプリのコンテナ化

Node.js/Nginxアプリケーションのコンテナ化、本番向けイメージの最適化を実践します。

Node.js Nginx 最適化
8
Docker Hubとイメージ管理

Docker Hubへのpush/pull、タグ戦略、プライベートリポジトリ、GHCR活用を学びます。

Docker Hub タグ GHCR
9
開発環境の構築

Java/Spring Boot開発環境のDocker化、DB連携、ホットリロード設定を学びます。

Java Spring Boot DB連携
10
Docker × AI・MCP活用

MCPサーバーのDocker実行、GitHub MCP、Claude Code連携、セキュリティのベストプラクティスを学びます。

MCP AI連携 セキュリティ
