GitHub Actions入門

CI/CDの基礎概念からワークフロー構築、テスト自動化、自動デプロイ、キャッシュ・アーティファクト、セキュリティまで。現場で即戦力になるGitHub Actionsスキルを10章で体系的に学べるチュートリアルです。

1
CI/CDとGitHub Actions入門

CI/CDの概念、GitHub Actionsの特徴、他ツール（Jenkins, CircleCI等）との比較を学びます。

CI/CD 自動化 概要
2
ワークフローの基本構造

YAMLの基本構文、workflow/job/stepの階層構造、初めてのワークフロー作成を学びます。

YAML workflow 構造
3
トリガーとイベント

push、pull_request、schedule、workflow_dispatch等のイベントトリガーを学びます。

on push schedule
4
ジョブとランナー

jobs設定、runs-on、ジョブ間依存（needs）、マトリックスビルド、サービスコンテナを学びます。

jobs runs-on matrix
5
ステップとアクション活用

uses、run、with、actions/checkout、actions/setup-node等の公式アクションを学びます。

uses Marketplace setup
6
環境変数とシークレット

env、secrets、GITHUB_TOKEN、vars、Environments、保護ルールを学びます。

env secrets GITHUB_TOKEN
7
キャッシュとアーティファクト

actions/cache、upload/download-artifact、ジョブ間データ共有、ビルド高速化を学びます。

cache artifact 高速化
8
CI実践：テスト自動化

ユニットテスト、Lint、コードカバレッジ、ステータスバッジ、PRチェックを実践します。

テスト Lint カバレッジ
9
CD実践：自動デプロイ

GitHub Pages、Docker Hub、AWS/GCP/Azure連携、リリース自動化を実践します。

Pages Docker Hub デプロイ
10
カスタムアクションとセキュリティ

Composite Action、Reusable Workflows、OIDC認証、Dependabot、セキュリティベストプラクティスを学びます。

カスタム Reusable セキュリティ
