軽量なCRUD対応グリッドコンポーネント

4-3-1 info jsGridの主な機能

jsGridは、CRUD操作に特化した軽量なjQueryグリッドプラグインです。インライン編集やフィルタリングが標準搭載されています。

add_circle インライン追加
edit インライン編集
delete 行削除
filter_alt カラムフィルター
sort ソート機能
api Ajax/API連携

4-3-2 table_chart CRUD対応グリッド

tips_and_updates 上部のフィルター行で検索、「+」ボタンで新規追加、各行の編集・削除ボタンでCRUD操作ができます

4-3-5 code 初期化コード

jsGridの基本的な初期化方法：

// CDN読み込み
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/jsgrid/1.5.3/jsgrid.min.css">
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/jsgrid/1.5.3/jsgrid-theme.min.css">
<script src="https://cdnjs.cloudflare.com/ajax/libs/jsgrid/1.5.3/jsgrid.min.js"></script>

// CRUD対応グリッドの初期化
$("#jsGrid").jsGrid({
  width: "100%",
  height: "400px",
  inserting: true,
  editing: true,
  sorting: true,
  paging: true,
  filtering: true,
  data: clients,
  fields: [
    { name: "name", type: "text", width: 150 },
    { name: "age", type: "number", width: 50 },
    { type: "control" }
  ]
});
