固定フッターレイアウト

フッターを画面下部に固定し、常に表示されるレイアウトです。著作権情報やバージョン情報などを常に表示できます。

  • フッターが常に画面下部に表示
  • 著作権やバージョン情報を常に確認可能
  • クイックアクションをフッターに配置可能
  • コンテンツ量に関わらずフッターが見える

6-5-1レイアウトの特徴

固定フッターレイアウトは、重要な情報やアクションを常に画面下部に表示したい場合に適しています。

主なユースケース:

  • 著作権・法的情報の常時表示
  • バージョン情報の表示
  • クイックアクションボタンの配置
  • ページナビゲーションの固定表示

6-5-2実装のポイント

このレイアウトを実装する際の注意点:

  • position: fixed でフッターを固定
  • コンテンツ下部にpadding-bottomを追加
  • サイドバーの下端をフッター上に設定
  • z-indexでフッターを適切な位置に

6-5-3スクロールデモ

下にスクロールしても、フッターは固定されたままです。
