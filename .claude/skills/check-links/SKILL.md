# /check-links - リンク整合性検証

教材内の全HTMLファイルのリンク（href, src）が正しいファイルを参照しているか検証します。

## 使い方
```
/check-links              # 全教材を検証
/check-links [パス]       # 指定ディレクトリのみ検証
```

例: `/check-links software-design/`

## 処理手順

1. 対象ディレクトリの全HTMLファイルを収集する
2. 各HTMLファイルから以下のリンクを抽出する:
   - `<a href="...">` のhref属性
   - `<link href="...">` のhref属性
   - `<script src="...">` のsrc属性
   - `<img src="...">` のsrc属性
3. 各リンクを検証する:
   - 相対パスのリンク: ファイルの存在を確認
   - `#` アンカー: 同一ファイル内のid存在を確認
   - 外部URL: スキップ（ログに記録）
4. 結果をテーブル形式で出力する:
   - OK: 正常なリンク数
   - BROKEN: リンク切れの一覧（ファイルパス + 行番号 + リンク先）
   - EXTERNAL: 外部リンクの一覧

## 出力形式
```
## リンク検証結果
- 検証ファイル数: XX
- 正常リンク: XX
- リンク切れ: XX
- 外部リンク: XX（検証スキップ）

### リンク切れ一覧
| ファイル | 行 | リンク先 |
|---------|-----|---------|
| ... | ... | ... |
```
