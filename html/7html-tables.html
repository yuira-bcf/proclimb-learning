Introduction

テーブルタグの基本

データをわかりやすく整理したいときに使うのが <table>タグ！時間割や料金表など、
きれいに並んだ表を作ることができます。ここでは、まず表の基本構造をマスターしましょう。

Section 7-1

表の基本構造

表は、いくつかのタグを組み合わせて作ります。

タグ 役割 意味
<table> 表の全体を囲む 「ここから表ですよ」と知らせる
<tr> 行を作る Table Row（行）
<th> 見出しセルを作る Table Header（見出し）
<td> データセルを作る Table Data（中身）

POINT

表はプログラミング系の世界で「table(テーブル)」と呼びます。
行は「row（ロウ）」、列は「column（カラム）」と呼びます。
Excelなどの表計算ソフトの表現に基づいており、
HTMLでExcelのあの表みたいなものを作りたいときに、<table>タグを使用します。

HTML 
<table>
  <tr>
    <th>項目</th>
    <th>値段</th>
  </tr>
  <tr>
    <td>りんご</td>
    <td>100円</td>
  </tr>
  <tr>
    <td>みかん</td>
    <td>80円</td>
  </tr>
</table>

各タグの関係はこんな感じ

構造 
<table>　　 ← 表全体を意味する親要素
    <tr>　　    ← table row（行）、このタグのセットの数だけ行が増える
        <th> or <td>　　 ← table heading（見出しセル） or table description（普通のセル）、この数だけ列が増える
    </tr>
</table>

POINT

  • <tr> は "行（row）" を増やすたびに追加
  • <th> は "見出しセル"、<td> は "中身のセル"
  • <table> の中に <tr> があり、さらにその中に <th> や <td> が入る構造

ワンポイントアドバイス

表を作るときは、何行で何列の表を作りたいのかを決めてから、
必要な数の<tr>と<td>を先に用意してあげてから書くと、混乱しにくくなります。

Section 7-2

表をつくってみよう！

基本の書き方

HTML 
<table border="1">
  <tr>
    <th>商品名</th>
    <th>価格</th>
  </tr>
  <tr>
    <td>アップルジュース</td>
    <td>120円</td>
  </tr>
  <tr>
    <td>オレンジジュース</td>
    <td>150円</td>
  </tr>
</table>
  • <tr> のセットが3つ → 3行
  • <th> or <td>の数が2つずつ → 2列

※ <td>の数の分だけ列が作られるので、各<tr>の中の <td>の数がズレると崩れた表になってしまうので注意

Section 7-3

セルをつなげてみよう（結合）

表の中で、セルを横や縦につなげて1つにまとめたいときがあります。
Excelでもセルの結合をしますよね、それと同じです。
HTMLでは、<td> に属性として以下を設定してあげると、結合が可能になります。

  • 横方向 → colspan（カラムスパン）
  • 縦方向 → rowspan（ロースパン）

例：横に2つのセルをつなげる

HTML 
<table border="1">
  <tr>
    <th colspan="2">今日のおすすめメニュー</th>
  </tr>
  <tr>
    <td>トマトパスタ</td>
    <td>サラダ付き 900円</td>
  </tr>
  <tr>
    <td>ビーフカレー</td>
    <td>ドリンク付き 850円</td>
  </tr>
</table>

ポイント！

colspan="2" はセルを横に2つ分つなげる ⇒ 列を2つ繋げる
この場合、横に繋げる<tr>の中の<td>の数を、1つ減らします。

縦方向にセルをつなげよう

セルを上下に結合したいときは、属性 rowspan（ロースパン） を使います！
これによって、セルを縦に結合＝行を結合することが出来ます。

使い方の例

HTML 
<td rowspan="2">スープ付き</td>

こう書くと、次の<tr>にあるセル（<td>1つ分）と縦に結合され、1つの長いセルとして表示されます

ポイント！

  • 横方向は colspan、縦方向は rowspan！
  • 数字は「いくつ分つなげるか」を指定（例：rowspan="3"）
  • rowspanの場合は、結合した数の分だけ次の<tr>の中の、<td>をひとつ減らします

3行目と4行目を1つに結合

HTML 
<table border="1">
  <tr>
    <th>セル1</th>
    <th>セル2</th>
  </tr>
  <tr>
    <td rowspan="2">セル3＋5</td>
    <td>セル4</td>
  </tr>
  <tr>
    <td>セル6</td>
  </tr>
</table>

3行目と4行目のセルが1つにつながって、縦長のセルになりました。
