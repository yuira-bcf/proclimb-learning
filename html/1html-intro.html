Introduction

HTMLとは何か？

HTML（エイチティーエムエル）は、Hyper Text Markup Language の略。Webページの中身を作るための「設計図」のような言語です。

たとえば、

  • どこが見出しなのか
  • どこに画像を表示するのか
  • どんなリンクがあるのか

といった情報を、タグ（< >で囲まれた命令）を使ってコンピューターに伝えます

Section 1-1

タグのイメージ

要素 HTMLタグ例 意味
見出し <h1>猫の実態</h1> ページのタイトル
段落 <p>今日は一日よく寝ています。</p> 本文
画像 <img src="neko.jpg" alt="眠る猫"> 写真表示
リンク <a href="about.html">続きを読む</a> 他ページへのリンク

HTMLは、このようなタグを組み合わせて構造を作ることでページを形にします

POINT

HTMLは「見た目」ではなく「意味」を作る言語です。
見た目を整えるのはCSS、動きをつけるのはJavaScript。
まずは「内容を正しく伝えるタグ」を覚えましょう！

Section 1-2

HTMLファイルを作ろう

Step 1：テキストエディターを使ってファイルを作成

HTMLを書くには「テキストエディター」を使います。
VS Codeを起動

  1. 上のメニューから「ファイル → 新しいファイル」
  2. 保存するときは「index.html」として保存

POINT

保存場所は「academia\src\HTML\project_1_1-2\ 」

Step 2：HTMLを書いてみよう

HTML 
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="UTF-8">
    <title>猫の一日</title>
    <meta name="description" content="日々の猫の暮らしを紹介します。">
  </head>
  <body>
    <h1>猫の一日</h1>
    <p>今日もよく寝ています。</p>
  </body>
</html>

Step3 : ブラウザーで開いてみよう

保存したファイルをダブルクリック！
ブラウザーが立ち上がって、自分で作ったページが表示されます

Section 1-3

HTMLファイルの骨組み

HTMLは大きく分けて「見えない部分」と「見える部分」で構成されています。

HTML構造 
ーーーー以下、見えない部分ーーーー
<!doctype html>　　← HTMLの宣言
<html lang="ja">　　← 文書全体＋言語指定
  <head>　　← ページの 設定情報
    <meta charset="UTF-8">　　← 文字コード
    <title>ページタイトル</title>　　← ページのタイトル
    <meta name="description" content="検索用の説明文">　　→ 検索時に表示される文章
  </head>
ーーーー以下、見える部分ーーーー
  <body>　　← ここから画面に表示されるという意味のタグ
    <h1>猫の一日</h1>　　← 見出しの文章
  </body>
</html>

各タグの役割

タグ 役割
<!doctype html> HTMLのバージョン宣言（これがあると最新版HTMLとして認識）
<html lang="ja"> ページの言語を日本語に設定
<head> ページの設定情報をまとめる（見えない部分）
<meta charset="UTF-8"> 文字化けを防ぐ設定
<title> タブや検索結果に表示されるタイトル
<meta name="description"> 検索結果に表示されるページ説明文
<body> 実際に画面に表示されるコンテンツ

ポイント整理

  • head＝裏方（そのページの設定情報）
  • body＝主役（そのページの、目で見える範囲）

HTMLはこの2つのエリアで構成されています。
この骨組みを覚えれば、どんなWebページも理解しやすくなります

まとめ

第1章のまとめ

  • HTMLはWebページの「骨組み」を作る言語
  • タグ（< >）を使ってコンピュータに意味を伝える
  • headは設定情報（見えない部分）
  • bodyは画面に表示されるコンテンツ
  • 見た目はCSS、動きはJavaScriptで作る
