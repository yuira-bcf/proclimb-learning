見出しをつけよう

文章を「章立て」する

Section 3-1

見出しタグ <h1>〜<h6>

Webページの文章を「章立て」するのが見出しタグ。
<h1> は Heading 1（＝見出し1） の意味で、数字が大きくなるほど小さな見出しになります。

たとえば：

HTML 
<h1>1番大きな見出し</h1>
<h2>2番目の見出し</h2>
<h3>3番目の見出し</h3>
<h4>4番目の見出し</h4>
<h5>5番目の見出し</h5>
<h6>6番目の見出し</h6>

ポイント！

  • <h1> はページのタイトルとして1ページに1つだけという推奨ルールがあります
  • <h2>〜<h6> は各テーマごとの章、と考えると良い
Section 3-2

見出しタグを使う"順番"を守ろう！

HTMLでは、見出しをつけるときに使うタグが<h1>〜<h6>まで6種類あります。
数字が小さいほど大きな見出し、数字が大きいほど小さな見出しです

タグ 役割 例
<h1> ページ全体のタイトル（1ページに1つ！） トップタイトル
<h2> 大きな章タイトル セクションの見出し
<h3> h2の中の小見出し（章の中の項目） 内容の細分化
<h4>〜<h6> ひとつ上の中のさらに小さい見出し 補足説明など

階層のイメージ

構造 
<body>
  ├<h1>
  ├<h2>
  │  ├<h3>
  │  └<h3>
  │      └<h4>
  └<h2>
      └<h3>

見出しタグは、階層の順に使うのがルールです。<h1> → <h2> → <h3> と数字を順番に下げて使いましょう。見た目を大きくしたいからといって <h1>など を使うのはNG！（見た目の大きさはCSSで変えます）

Section 3-3

<h1>はページのタイトルにひとつだけ！

<h1>はそのページで一番重要な見出し。
原則、1ページに1つまでしか使わないことがルールとなっています。

例：

  • 「猫の実態」 → ページ全体のテーマ（<h1>）
  • 「猫の趣味は？」「好きな食べ物は？」 → 大見出し（<h2>）
  • 「昼寝が好き」「カリカリが好き」 → 小見出し（<h3>）

実際のコード例

HTML 
<h1>猫の実態</h1>

<h2>猫の趣味は？</h2>
<h3>昼寝が大好き！</h3>
<h3>ボール遊びも楽しい</h3>

<h2>好きな食べ物は？</h2>
<h3>やっぱりカリカリ</h3>
<h3>猫缶もお気に入り</h3>

見出しの関係を"本の章構成"で考えよう

  • <h1> → 本のタイトル
  • <h2> → 各章のタイトル
  • <h3> → 章の中の小見出し
  • <h4>〜<h6> → 細かい補足説明

見出しタグを正しく使うと、ページの"構造"がわかりやすくなります
検索エンジン（Googleなど）も「どんな内容のページか」を理解しやすくなります！

まとめ

第3章のまとめ

  • <h1>はページに1つだけ！
  • <h2>〜<h6>は順番に使う
  • 見た目を変えるのはCSSで！
  • 構造を意識すると、検索にも強いページになる
