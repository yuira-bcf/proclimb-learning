Section 3-1

CSSの書き方と適用方法

CSS（Cascading Style Sheets）は、HTMLで作成した構造に対して、見た目を指定するための言語です。3つの方法でHTMLに適用できます。

- コメントの書き方を含める

## CSSの構造
- セレクタ: スタイルを適用する対象
- プロパティ: 何を変更するか
- 値: どのように変更するか
CSSの構文 CSS

    プロパティ: 値;
    プロパティ: 値;
}

/* 例 */

    color: blue;
    font-size: 24px;
}

プロパティ 何を変更するか（色、大きさなど）
値 どのように変更するか

1.2 CSSの3つの適用方法

chat プロンプト 
## 機能
CSSの3つの適用方法を説明

## 実装要件
- 外部CSS（linkタグで読み込み）
- 内部CSS（styleタグで記述）
- インラインCSS（style属性で記述）

## 推奨される方法
- 外部CSS: 管理しやすく再利用可能
- 内部CSS: 1ページ限定のスタイル
- インラインCSS: 動的な変更時のみ

1. 外部CSS（推奨）

index.html HTML 
<head>
    <link rel="stylesheet" href="css/style.css">
</head>

2. 内部CSS

headタグ内に記述 HTML 
<head>
    <style>
        h1 {
            color: blue;
        }
    </style>
</head>

3. インラインCSS

要素のstyle属性 HTML 
<h1 style="color: blue; font-size: 24px;">見出し</h1>
外部CSSが推奨される理由
  • HTMLとCSSを分離でき、管理しやすい
  • 複数のHTMLファイルで共有できる
  • キャッシュが効いてページ読み込みが速くなる
Section 3-2

セレクタの種類

セレクタは、スタイルを適用する対象を指定するための仕組みです。様々な種類のセレクタを組み合わせることで、細かく対象を指定できます。

2.1 基本セレクタ

chat プロンプト 
## 機能
CSSの基本セレクタを説明

## 実装要件
- 要素セレクタ（タグ名で指定）
- クラスセレクタ（.クラス名）
- IDセレクタ（#ID名）
- 全称セレクタ（*）

## 使用するCSS
- 要素名: p, h1, div など
- .クラス名: 複数の要素に適用
- #ID名: 特定の1つの要素に適用
- *: すべての要素に適用
基本セレクタ
CSS 
/* 要素セレクタ */
p { color: black; }

/* クラスセレクタ */
.highlight { background: yellow; }

/* IDセレクタ */
#header { height: 60px; }

/* 全称セレクタ（全要素） */
* { margin: 0; padding: 0; }
プレビュー

要素セレクタ: p { color: black; }

クラスセレクタ: .highlight { background: yellow; }

IDセレクタ: #header { height: 60px; }
menu_book 構文説明

CSSセレクタ（要素、クラス、ID）

要素名 { ... } /* 要素セレクタ: すべての該当要素 */ .クラス名 { ... } /* クラスセレクタ: class属性が一致する要素 */ #ID名 { ... } /* IDセレクタ: id属性が一致する要素 */ * { ... } /* 全称セレクタ: すべての要素 */

要素セレクタはタグ名で指定し、クラスセレクタは.（ドット）で始まり、IDセレクタは#（シャープ）で始まります。クラスは複数の要素に使え、IDはページ内で一意でなければなりません。

プロパティと値（property: value;）

セレクタ { プロパティ: 値; プロパティ: 値; }

CSSの基本構文です。セレクタで対象を指定し、波括弧{}内にプロパティ（何を）と値（どのように）のペアを記述します。各宣言はセミコロン;で終わります。

## 機能

## 実装要件
- 子孫セレクタ（スペース）
- 子セレクタ（>）
- 隣接セレクタ（+）
- 一般兄弟セレクタ（~）

## 使用するCSS記法
- A B: Aの中のすべてのB
- A > B: Aの直下のB

- A ~ B: Aより後のすべてのB
- A, B: AとBの両方
結合セレクタ
CSS

/* 子セレクタ（直下のみ） */

/* 隣接セレクタ（直後の兄弟） */
h2 + p { font-size: 18px; }

/* 一般兄弟セレクタ（後の兄弟すべて） */
h2 ~ p { margin-left: 20px; }

/* グループセレクタ */
h1, h2, h3 { font-family: serif; }

2.6 詳細度（優先順位）

同じ要素に複数のスタイルが指定された場合、詳細度が高いセレクタが優先されます。

セレクタ 詳細度
インラインスタイル 1000
IDセレクタ (#id) 100
クラス、属性、疑似クラス 10
要素、疑似要素 1
!importantは最後の手段

!importantを使うとすべてを上書きできますが、保守性が悪くなるため、使用は避けましょう。

## 実装要件

menu_book 構文説明

色指定（hex, rgb, hsl）

color: red; /* カラーネーム */ color: #ff0000; /* HEX（16進数） */ color: #f00; /* HEX短縮形 */ color: rgb(255, 0, 0); /* RGB（0-255） */ color: rgba(255, 0, 0, 0.5); /* RGBA（透明度付き） */ color: hsl(0, 100%, 50%); /* HSL（色相, 彩度, 明度） */

CSSでは複数の方法で色を指定できます。HEXは#に続く6桁の16進数（RRGGBBの順）、rgb()は赤・緑・青を0-255で指定、hsl()は色相(0-360)・彩度(%)・明度(%)で指定します。rgba()とhsla()では4番目の値で透明度（0-1）を指定できます。

透明度の指定

RGBA、HSLA以外にも、opacityプロパティで要素全体の透明度を指定できます。
opacity: 0.5; は要素全体が50%透明になります。

ボックスモデル

すべてのHTML要素は「ボックス」として扱われます。ボックスモデルを理解することは、レイアウトを作る上で非常に重要です。

/* 全要素に適用（リセットCSS） */
*, *::before, *::after {

}

※ border-box を使うと、padding を追加しても要素の全体幅が変わらないため、レイアウトが崩れにくくなります。

border-boxが便利な理由

width: 300pxと指定した場合、paddingやborderを追加しても、要素の幅は300pxのままになります。計算が簡単になり、レイアウトが崩れにくくなります。

Exercise

3章 練習問題

Q1

CSSの詳細度（優先順位）が最も高いのはどれですか？

  1. 要素セレクタ（p, div）
  2. クラスセレクタ（.class）
  3. IDセレクタ（#id）
  4. 属性セレクタ（[type="text"]）
Q2

padding: 10px 20px 30px; の各値が適用される方向として正しいものはどれですか？

  1. 上10px、右20px、下30px、左20px
  2. 上10px、左右20px、下30px
  3. 上下10px、左右20px、下30px
  4. 上10px、右30px、下20px、左30px
Q3

要素を完全な円形にするために必要なCSSプロパティはどれですか？

  1. border-radius: 100px;
  2. border-radius: 50%;
  3. border-radius: circle;
  4. shape: circle;
Answer

3章 解説・解答

Q1の解答: C

詳細度は IDセレクタ(100) > クラス・属性セレクタ(10) > 要素セレクタ(1) の順です。インラインスタイルはさらに高い1000となります。

Q2の解答: B

3つの値を指定した場合、「上、左右、下」の順で適用されます。つまり上10px、左右20px、下30pxです。

Q3の解答: B

border-radius: 50%;を指定すると、正方形の要素は完全な円になります。100pxなどの固定値だと、要素のサイズによっては円にならない場合があります。
