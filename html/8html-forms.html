Introduction

フォームとは

お問い合わせや予約フォームなど、ユーザーが情報を入力できる場所がフォーム（form）です。

Section 8-1

複数のパーツを組み合わせて作る

アカウント登録などを思い出してください。
名前やメールアドレス、パスワードなどを入力して、最後に送信ボタンを押しますよね？
あれはHTMLの<form>タグで作られています。

<form>タグはフォーム全体を意味する親要素で、
その中の入力フォームなどが<input type="●●">や<textarea>、<select>などの子要素という構造です。

たとえばこんなフォーム

入力項目 HTMLタグ
名前 <input type="text">
メールアドレス <input type="email">
メッセージ <textarea>
送信ボタン <input type="submit">

基本の構文

HTML 
<form action="send.php" method="post" name="contactForm">
  <p><input type="text" name="username" placeholder="お名前"></p>
  <p><input type="email" name="email" placeholder="メールアドレス"></p>
  <p><textarea name="message" placeholder="メッセージ"></textarea></p>
  <p><input type="submit" value="送信"></p>
</form>

<form>の必須属性の説明

属性名 役割 説明
action データの送信先 入力内容をどこへ送るか（例：サーバーのURLやファイル名）
method 送信方法 getまたはpostを指定
name フォームの名前 送信データを識別するための名前

POINT

  • action は「宛先」 ⇒ 「どこに送るか」
  • method は「どうやって送るか」
  • name は「送信データの識別名」

この3つをセットで覚えよう

get と postの違い

  • get - 簡易的な送信。主に検索に使われ、URLに送信内容が見える形で送信される
  • post - 登録や複雑な照合を行う検索などの送信。URLなどには表示されない
Section 8-2

フォームの中身を作ってみよう！

フォームは <form> タグで囲んだ中に、入力欄やボタンなどのパーツを入れて作ります。
ここでは、実際に使われるパーツを1つずつ見ていきましょう。

<form> タグの基本構造

HTML 
<form action="example.php" method="post" name="contact-form">
  <!-- ここにフォームの内容が入ります -->
</form>

これだけでフォームの"枠"が完成！　この中に入力欄や送信ボタンを入れていきます。
次にフォームの中身を実際に作っていってみましょう。

フォームで使う基本パーツ

フォームの中で入力エリアやボタンを作るには、主に <input> タグを使います。

1行テキスト入力欄：<input type="text">

HTML 
<label>名前：</label>
<input type="text">
属性 役割 意味
type 入力の種類を指定（必須） type="text" は1行入力欄
name データの識別名（ほぼ必須） name="username" は、この入力データは「username」という名前で送られる、という意味
<label> フォームの見出し（あった方が良い） これは何の入力フォームか、を示すのに<label>タグを使用します

POINT

各入力フォームは、主に type属性を変更して表現します（一部違うのもある）

Section 8-3

入力欄にあらかじめ文字を表示するには

このフォームには、このように書いてほしい……
と、ユーザーに促したい場合は、placeholder（プレースホルダー）属性を使って、うっすら文字を表示します。

使い方の例

HTML 
<input type="text" placeholder="お名前">

ユーザーが文字を入力すると、placeholderの表示は自動的に消えます。

Section 8-4

いろいろな種類の1行入力欄

ただの1行の入力フォームだけでなく、HTMLではいろんな意味を持たせた1行入力フォームを
typeによってさまざま表現できます。

代表的な1行フォーム

HTML 
<input type="text" placeholder="お名前">
<input type="email" placeholder="メールアドレス">
<input type="password" placeholder="半角英数字8桁以上">
属性値（type） 内容 特徴
text ただのテキスト入力欄 なんでも普通に入力できる
email メールアドレス入力用 @や@以降に. が入っていないと送信できない
password パスワード向け入力用 入力した文字がすべて●で表示される

他にもurlやsearchなど、たくさんあるので、使うときに調べてみましょう。

Section 8-5

ラジオボタンを作ろう！ <input type="radio">

複数選択肢の中から、どれかひとつだけを選択させたいときに使うフォームです。
◉ ← のようなチェックマーク、見たことありません？ あれがラジオボタンです。

主な属性の役割

属性名 内容 説明
name グループ名（必須） 同じnameのものがグループになる
value 値（必須） データとして送られる値
checked 初期状態（必須ではない） ページを開いたときに、最初からチェックを入れさせる

サンプルコード

HTML 
<p>好きなドリンクを選んでください：</p>
<input type="radio" name="drink" value="coffee" checked> コーヒー
<input type="radio" name="drink" value="tea"> 紅茶
<input type="radio" name="drink" value="juice"> ジュース

POINT

同じnameのものの中から1つだけ選択できるようになります。
nameを間違えてしまうと、複数選択できるような見た目になってしまうので注意！

Section 8-6

チェックボックスを作ろう！ <input type="checkbox">

ラジオボタンと違って、複数選択できるものをチェックボックスと呼びます。
書き方の違いは、type="checkbox"の部分だけ、あとはすべて同じです。

書き方例：

HTML 
<p>好きなフルーツを選んでください：</p>
<input type="checkbox" name="fruit" value="apple" checked> りんご
<input type="checkbox" name="fruit" value="banana"> バナナ
<input type="checkbox" name="fruit" value="orange"> みかん
<input type="checkbox" name="fruit" value="grape"> ぶどう
Section 8-7

送信ボタンを作ろう！ <input type="submit">

フォームは、送信されなければ意味がありません。
送信とは、入力内容を「サーバに送信する」という意味になります。
<form action="●●">と、送信先を指定していますが、それはサーバに●●のHTML情報をください！
とリクエストをするという意味です。その際に、フォームの入力データもリクエストの中に混ぜて送っています。

この「送信」自体の役割を実行するのがtype="submit"となります。

基本の書き方

HTML 
<input type="submit" value="メッセージを送る">

valueに記述した内容で、ボタンが作成されます。これをクリックするだけで、サーバにデータが送信されます。

POINT

type="submit"は<form> ～ </form>で囲まれた中にある<input type="●●">の入力されたデータを一度にすべて送信します。
同じページのフォームでも、別の<form>に囲まれた入力データは送信されません。
また、同じ<form>の中に複数のtype="submit"があっても、送信される内容は変わりません。

ボタンを画像にしたい場合

もしオリジナルデザインのボタンを使いたいときは、
<input type="image"> を使って画像をボタンにすることもできます。

書き方の例

HTML 
<input type="image" src="btn.png" alt="送信ボタン">

上記の例だとボタンが画像になっていますが、
これをクリックしてもtype="submit"と同じ働きをします。

属性名 内容 説明
name ボタンの名前 フォーム識別用
src 使用する画像ファイルの場所 ファイルパスを指定
alt 代替テキスト 画像が表示されない時に表示される説明文
タグ／属性 内容 ポイント
<input type="submit"> 送信ボタンを作る クリックでフォーム送信
<input type="image"> 画像をボタンにする オリジナルデザインに便利
value ボタンの文字を変える 「送信」「検索」「予約」などに変更OK！
alt 代替テキスト 画像が見えない時でも安心
Section 8-8

セレクトボックスを作ろう！ <select> + <option>

セレクトボックスとは、プルダウンメニュー、ドロップダウンリスト、アコーディオンリストなどと呼ばれる
選択式の入力フォームです。
生年月日や都道府県の選択時に、縦にずらっと出てくる、アレです。

選択項目が多いとき、画面に表示させると大変なものなどはこのセレクトボックスを使います。

<select>タグの役割と属性

セレクトボックスは、親要素と子要素で分かれています。
<select>タグが親要素、選択肢をまとめる"外側の箱"のようなものです。
<option>タグが子要素で、ここに書いた内容ひとつひとつが選択肢となる項目になります。

<option>タグの役割と属性

属性名 内容 説明
value 送信される値 実際にサーバーに送られるデータ
selected 初期選択状態にする 最初から選ばれた状態にする場合に指定

コード例：好きなドリンクを選ぶフォーム

HTML 
<p>好きなドリンクを選んでください：</p>
<select name="drink">
  <option value="coffee" selected>コーヒー</option>
  <option value="tea">紅茶</option>
  <option value="juice">オレンジジュース</option>
  <option value="water">ミネラルウォーター</option>
</select>

クリックするとメニューが開き、好きな飲み物を選べます

Section 8-9

複数行の入力欄を作ろう！ <textarea>タグ

問い合わせの詳細内容記入や、備考欄など、複数行で入力したいときに<textarea>（テキストエリア）タグを使います。
これは<input>タグではなく、</textarea>の終了タグも必要となります。

基本の書き方

HTML 
<textarea name="message">ここにメッセージを入力</textarea>

<textarea> で囲んだ部分が「入力欄」として表示されます。
ブラウザー上では枠付きの大きなボックスとして表示され、複数行の文字入力が可能になります！

placeholder を使う方が、ユーザーにとって「何を書けばいいか」が分かりやすいです。
また、右下の角をドラッグすると入力欄を拡げることも出来ます。

Section 8-10

ユーザビリティの高いフォームにする

フォームを作るとき、入力しやすさってとても大切です。
見た目だけでなく、「押しやすい」「分かりやすい」を意識することで、ユーザーにやさしいフォームが作れます

フォームのラベルを連動させる <label>タグ

ラベル（label）とは、入力欄の見出しになるテキストのこと。
ただ書くだけでは、構造としてフォームの見出しである、という意味にしかなりません。
ですが、ひと手間加えてあげると便利なフォームになります。

たとえば…

ラジオボタンやチェックボックスを押すとき、見出しのテキストをクリックすると
それだけでチェック機能が働くフォームを見たことありませんか？

それは<label>タグの連動機能を上手く使っている証拠です。

コード例：好きな動物を選ぼう

HTML 
<input type="checkbox" name="animal" value="dog" id="dog">
<label for="dog">犬</label>

<input type="checkbox" name="animal" value="cat" id="cat">
<label for="cat">猫</label>

<input type="checkbox" name="animal" value="rabbit" id="rabbit">
<label for="rabbit">うさぎ</label>

上記のフォームでは、見出しをクリックするだけで各項目にチェックが入ります。
この機能を使うには、for属性とid属性を使用します。
<label>と<input>のそれぞれにforとidを「同じ名前」で指定してあげると実現できます。

注意点

id はファイル内で重複しない値にすることがルールです。
同じidを2つ以上使うと、JavaScriptやCSSでエラーが発生したりします。

まとめ

第8章のまとめ

  • <form>でフォーム全体を囲む（action, method, name属性）
  • <input type="text">で1行入力欄
  • <input type="radio">でラジオボタン（1つだけ選択）
  • <input type="checkbox">でチェックボックス（複数選択可）
  • <input type="submit">で送信ボタン
  • <select> + <option>でセレクトボックス
  • <textarea>で複数行入力欄
  • <label>のfor属性とid属性で連動させる
