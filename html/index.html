HTML/CSS入門

Webサイト制作の基礎から実践的なレイアウト、外部サービス連携、Tailwind CSSまで体系的に学べるチュートリアルです。

1
Web制作の準備

Web開発を始めるための環境構築とツールの準備について学びます。VS Codeのインストールから拡張機能の設定まで。

VS Code ブラウザ 拡張機能
2
HTMLの基本

HTMLの基本構造、要素、属性について学び、Webページの骨格を作成します。セマンティックなマークアップも習得。

タグ 属性 フォーム セマンティック
3
CSSの基本

CSSの基本文法、セレクタ、ボックスモデルについて学びます。色やフォントの設定方法も解説。

セレクタ 色・フォント ボックスモデル
4
レイアウトの基本

FlexboxとCSS Gridを使った現代的なレイアウト手法を学びます。レスポンシブデザインの基礎も。

Flexbox CSS Grid レスポンシブ
5
シングルカラムサイト

ヘッダー、メインコンテンツ、フッターで構成されるシンプルなサイトを制作します。

ヘッダー フッター ナビゲーション
6
2カラムサイト

サイドバーを持つ2カラムレイアウトのサイトを制作します。ブログ形式のレイアウトを実践。

サイドバー メインコンテンツ ブログ形式
7
タイル型サイト

CSS Gridを使ったタイル型のギャラリーサイトを制作します。カードUIとホバー効果も実装。

カードUI グリッドレイアウト ホバー効果
8
外部サービス連携

Google Maps、YouTube、SNS連携などの外部サービスをWebサイトに組み込みます。

Google Maps YouTube OGP
9
Tailwind CSS

ユーティリティファーストのCSSフレームワーク、Tailwind CSSを学びます。効率的なスタイリング手法を習得。

ユーティリティクラス レスポンシブ 状態変化
