Section 6-1

2カラムレイアウトとは

2カラムレイアウトは、メインコンテンツとサイドバーを並べて配置するレイアウトです。ブログやニュースサイトでよく使われます。

1.1 2カラムの特徴

パターン 特徴 用途
左メイン・右サイドバー 最も一般的なパターン ブログ、ニュースサイト
右メイン・左サイドバー ナビゲーションを目立たせる ECサイト、ポータルサイト

1.2 基本構造

chat プロンプト 
## 機能
2カラムレイアウトの基本構造を作成

## 実装要件
- CSS Gridで2カラムレイアウトを実装
- メインコンテンツは可変幅（1fr）
- サイドバーは固定幅（300px）
- 列間の間隔を設定（gap）
- min-width: 0でグリッド内のオーバーフロー防止

## 使用するCSS
- display: grid / grid-template-columns: 1fr 300px
- gap / max-width / margin: 0 auto
2カラムの基本構造 HTML 
<div class="layout">
    <main class="main-content">
        <!-- メインコンテンツ -->
    </main>

    <aside class="sidebar">
        <!-- サイドバー -->
    </aside>
</div>
CSS CSS 
.layout {
    max-width: 1200px;
    margin: 0 auto;
    padding: 40px 20px;
    display: grid;
    grid-template-columns: 1fr 300px;
    gap: 40px;
}

.main-content {
    min-width: 0;  /* Grid内でのオーバーフロー防止 */
}

.sidebar {
    /* サイドバー固定幅 */
}

2.2 CSS Gridでの実装

chat プロンプト 
## 機能
CSS Gridで2カラムレイアウトを実装（推奨）

## 実装要件
- grid-template-columns: 1fr 300pxで可変幅と固定幅を組み合わせ

    margin: 0 auto;
    padding: 40px 20px;
}

/* 比率指定 */

    grid-template-columns: 7fr 3fr;

/* 左サイドバー */
.layout-sidebar-left {

}

menu_book 構文説明

サイドバーレイアウト（Grid）

grid-template-columns: 1fr 300px;

1frはメインコンテンツが残りのスペースを占有し、300pxでサイドバーを固定幅にします。左右の順序を入れ替えることで左サイドバーも実現できます。

サイドバーレイアウト（Flexbox）

display: flex; .main { flex: 1; } .sidebar { flex: 0 0 300px; }

flex: 1でメインが伸縮し、flex: 0 0 300pxでサイドバーを固定幅にします。min-width: 0を指定しないとコンテンツがはみ出す場合があります。

レスポンシブ対応

5.1 モバイル時の1カラム化

chat プロンプト 
## 機能
2カラムレイアウトのレスポンシブ対応

## 実装要件
- モバイル（デフォルト）：1カラム表示（display: block）
- サイドバーをメインコンテンツの下に配置
- タブレット（768px以上）：2カラム（1fr 280px）
- PC（1024px以上）：2カラム（1fr 300px）
- 記事カードは600px以下で画像を上に配置

## 使用するCSS
- display: block / display: grid
- @media (min-width: 768px) / @media (min-width: 1024px)
- @media (max-width: 600px)
レスポンシブ対応 CSS
CSS 
/* モバイルファースト */
.layout {
    display: block;  /* 1カラム */
    padding: 20px;
}

.sidebar {
    margin-top: 40px;
}

/* タブレット以上で2カラム */
@media (min-width: 768px) {
    .layout {
        display: grid;
        grid-template-columns: 1fr 280px;
        gap: 30px;
        padding: 40px 20px;
    }

    .sidebar {
        margin-top: 0;
    }
}

/* PC */
@media (min-width: 1024px) {
    .layout {
        grid-template-columns: 1fr 300px;
        gap: 40px;
    }
}

/* 記事カードのレスポンシブ */
@media (max-width: 600px) {
    .post-card-link {
        grid-template-columns: 1fr;
    }

    .post-card-image {
        height: 180px;
    }
}

menu_book 構文説明

メディアクエリ（@media）

@media (条件) { スタイル }

画面サイズやデバイス特性に応じてスタイルを切り替えます。min-widthは「指定値以上」、max-widthは「指定値以下」の場合に適用されます。

モバイルファーストの書き方

/* 基本スタイル（モバイル用） */ @media (min-width: 768px) { /* タブレット用 */ } @media (min-width: 1024px) { /* PC用 */ }

まずモバイル向けの基本スタイルを書き、min-widthで大きな画面用のスタイルを追加していきます。コードの重複が少なく、メンテナンスしやすい手法です。

レスポンシブブレークポイント

480px（スマートフォン大）/ 768px（タブレット）/ 1024px（PC小）/ 1280px（PC大）

一般的なブレークポイントの目安です。プロジェクトや対象デバイスに応じて調整します。768pxと1024pxが最もよく使用されます。

5.2 サイドバーの固定（スティッキー）

chat プロンプト 
## 機能
スクロール時にサイドバーを固定（スティッキー）

## 実装要件
- position: stickyでスクロール時に固定

## 使用するCSS
- position: sticky / top
- height: fit-content

        position: sticky;
        top: 90px;  /* ヘッダー高さ + 余白 */
        height: fit-content;
    }
}

Exercise

6章 練習問題

Q1

2カラムレイアウトで右サイドバーを300px固定にするCSS Gridの記述はどれですか？

  1. grid-template-columns: 300px 1fr;
  2. grid-template-columns: 1fr 300px;
  3. grid-template-columns: auto 300px;
  4. grid-columns: main 300px;
Answer

6章 解説・解答

Q1の解答: B

grid-template-columns: 1fr 300px;で、左側が残りスペース、右側が300px固定になります。HTMLの順序どおり、左がメイン、右がサイドバーになります。
