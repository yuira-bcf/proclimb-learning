display: flex

display: flex | inline-flex;

要素をフレックスコンテナにします。子要素は自動的にフレックスアイテムとなり、横並び（デフォルト）で配置されます。inline-flexはインライン要素として扱われます。

justify-content

justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;

主軸方向（デフォルトでは横方向）のアイテム配置を制御します。space-betweenは両端揃え、space-aroundは各要素の周りに均等な間隔、space-evenlyはすべての間隔が均等になります。

align-items

align-items: stretch | flex-start | flex-end | center | baseline;

交差軸方向（デフォルトでは縦方向）のアイテム配置を制御します。stretchは高さを揃え、centerは中央揃え、baselineはテキストのベースラインで揃えます。

flex-direction

flex-direction: row | row-reverse | column | column-reverse;

主軸の方向を設定します。rowは横方向（左から右）、columnは縦方向（上から下）、reverseを付けると逆方向になります。

gap

gap: 値; または gap: 行の間隔 列の間隔;

フレックスアイテム間の間隔を設定します。row-gapとcolumn-gapで個別に指定することもできます。marginと異なり、外側には間隔が付きません。

CSS Gridの基本

CSS Gridは、2次元（行と列の両方）のレイアウトに最適な方法です。複雑なレイアウトも直感的に作成できます。

chat プロンプト

CSS Gridコンテナを有効化

## 実装要件
- display: gridでグリッドコンテナを作成
- 列の幅を定義（固定幅とfr単位）
- 行の高さを定義（autoとfr単位）
- グリッドアイテム間の間隔を設定

## 使用するCSS
- display: grid
- grid-template-columns / grid-template-rows / gap
Gridの開始

    display: grid;

    /* 列の定義 */
    grid-template-columns: 200px 1fr 200px;

    /* 行の定義 */
    grid-template-rows: auto 1fr auto;

    /* 間隔 */
    gap: 20px;
}

menu_book 構文説明

display: grid

display: grid | inline-grid;

要素をグリッドコンテナにします。子要素は自動的にグリッドアイテムとなり、行と列で構成される2次元レイアウトを作成できます。

grid-template-columns

grid-template-columns: 値1 値2 値3 ...;

列の数と幅を定義します。pxで固定幅、frで残りスペースの比率、autoでコンテンツに応じた幅を指定できます。repeat(3, 1fr)で繰り返しパターンも可能です。

grid-template-rows

grid-template-rows: 値1 値2 値3 ...;

行の数と高さを定義します。autoは内容に応じた高さ、frは残りスペースの比率、minmax(min, max)で最小・最大値を指定できます。

grid-gap（gap）

gap: 値; または gap: 行の間隔 列の間隔;

グリッドアイテム間の間隔を設定します。row-gapとcolumn-gapで個別に指定することもできます。現在はgapプロパティの使用が推奨されています。

menu_book 構文説明

position: static

position: static;

デフォルト値です。要素は通常のドキュメントフローに従って配置されます。top、right、bottom、leftプロパティは効果がありません。

position: relative

position: relative; top: 値; left: 値;

要素は通常の位置を基準にして相対的に配置されます。元の位置のスペースは保持されたままです。子要素のabsolute配置の基準点としてもよく使われます。

position: absolute

position: absolute; top: 値; right: 値; bottom: 値; left: 値;

要素はドキュメントフローから外れ、最も近いposition: relative/absolute/fixedを持つ祖先要素を基準に配置されます。基準がなければviewportが基準になります。

position: fixed

position: fixed; top: 値; left: 値;

要素はビューポート（画面）を基準に固定されます。スクロールしても位置が変わりません。固定ヘッダーやナビゲーションによく使用されます。

position: sticky

position: sticky; top: 値;

スクロールに応じてrelativeとfixedを切り替えます。指定した位置（top等）に達するまでは通常配置、達した後は固定されます。親要素の範囲内でのみ有効です。

        grid-template-columns: repeat(2, 1fr);  /* タブレット：2列 */
    }
}

@media (min-width: 1024px) {
    .card-grid {
        grid-template-columns: repeat(3, 1fr);  /* PC：3列 */
    }
}

4章 練習問題

Q1

Flexboxで要素を横方向に中央揃えするプロパティはどれですか？

  1. align-items: center;
  2. justify-content: center;
  3. text-align: center;
  4. margin: 0 auto;
Q2

CSS Gridで3列均等のレイアウトを作るプロパティはどれですか？

  1. grid-columns: 3;
  2. grid-template-columns: repeat(3, 1fr);
  3. columns: 3;
  4. grid-template: 3 columns;
Q3

モバイルファーストでレスポンシブデザインを書く場合、正しいメディアクエリはどれですか？

  1. @media (max-width: 768px)
  2. @media (min-width: 768px)
  3. @media screen and (width: 768px)
  4. @media mobile
Answer

4章 解説・解答

Q1の解答: B

justify-contentは主軸方向（デフォルトでは横方向）の配置を制御します。align-itemsは交差軸（縦方向）の配置です。

Q2の解答: B

grid-template-columns: repeat(3, 1fr);で3列均等のグリッドを作成できます。repeat()関数で繰り返し、1frで均等分割されます。

Q3の解答: B

モバイルファーストでは、基本スタイルをモバイル用に書き、min-widthを使って大きな画面用のスタイルを追加していきます。
