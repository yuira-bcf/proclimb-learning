Introduction

タグで囲むが基本ルール

ブラウザはタグ（< と > で囲む記号）で囲まれたものを、HTMLの文章として読み取ります。

基本構文 
<タグ名> コンテンツ内容 </タグ名>
  • <タグ名> … 開始タグ
  • </タグ名> … 終了（閉じ）タグ

このセットでできたひとかたまりを 要素（element） と呼びます。
※ 終了タグの必要がないタグもいくつか存在します。書きながら覚えていきましょう。

Section 2-1

タグを書くときのミニルール

半角英数字で書くこと！　全角だとブラウザーが読めません。

例 
良い：<p>Webサイト制作</p>
悪い：＜ｐ＞Webサイト制作＜／ｐ＞　←全角で書いているので×

半角小文字で統一！　HTML5では半角小文字が基本。

例 
良い：<h1>
悪い：<H1>　←大文字で書いても認識はするけど、推奨されない

分かりやすく改行とインデント

HTML 
<div>
      <p>今日も良い日</p>　←改行されているので、<div>に含まれたpタグ、というのがわかりやすい
</div>
Section 2-2

タグの中にタグを入れる（入れ子）

HTMLでは、あるタグの中にさらに別のタグを入れられます。これを「入れ子構造（ネスト）」といいます。
ただし、最後に開いたものから順に閉じる！ これが大原則です

HTML 
<!-- OK -->
<p>おすすめは <strong>濃いめのコーヒー</strong> です。</p>

<!-- NG -->
<p>おすすめは <strong>濃いめのコーヒー</p></strong>
Section 2-3

タグに情報を追加する「属性」

タグには、追加情報（属性）をつけることができます。
属性は 開始タグの中 に書いて、名前="値" の形で指定します。

HTML 
<a href="about.html">Aboutページへ</a>
  • <a> … リンクを作るタグ
  • href= … リンク先を指定します、という意味の属性
  • "about.html" … リンク先のURLやパス情報(値)

POINT

タグは「意味を持つ箱」。
属性はその箱に「詳細な設定を加えるオプション」。

どちらも正しく組み合わせることで、Webページの内容がしっかりブラウザに伝わるようになります。

Section 2-4

ぜんぶ合わせた最小サンプル

HTML 
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="utf-8">
    <title>はじめてのHTML</title>
    <meta name="description" content="HTMLの基本形を学ぼう。">
  </head>
  <body>
    <h1>見出し（h1タグ）</h1>
    <p>これは段落（pタグ）です。<strong>強調（strongタグ）</strong>もできます。</p>
    <a href="about.html">くわしく見る（aタグ）</a>
  </body>
</html>
