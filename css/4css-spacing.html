Introduction

余白を理解しよう

余白を上手に活用することで、Webページ全体のバランスや見やすさが格段に上がります。
ここでは外余白のmarginと、内余白のpaddingを使っていきます。

Section 4-1

余白ってなに？

おさらいとして、要素の領域図を再度確認しましょう。

Section 4-2

ボックスの構造

要素の構造は、外から順に……
margin → border → padding → content

名前 説明
margin 要素の枠線より外側のすきま。隣の要素までの距離になります
border 内余白を囲んでいる枠線。ここが境界線
padding 要素と枠線の間の内側のすきま
content 要素。文字や画像など

POINT

要素の大きさ（幅と高さ）には「要素」「内余白」「枠線」が含まれます。
背景色（background-color）は「padding」までは塗られますが、「margin」部分には塗られません。

Section 4-3

外側の余白：margin プロパティ

margin（マージン）は、要素の外のスキマを作ります。
隣の要素と距離を取ったり、中央に配置したりするのに使います。

基本の書き方

CSS 
margin: 10px;    			 /* すべての方向に10pxの余白 */
margin: 10px 20px;   		/* 上下10px、左右20px */
margin: 10px 20px 30px; 		/* 上10px、左右20px、下30px */
margin: 10px 20px 30px 40px;	/* 上→右→下→左（時計回り） */

個別に指定したいとき

CSS 
margin-top: 20px;		 /* 上の余白 */
margin-right: 10px;  		/* 右の余白 */
margin-bottom: 30px; 		/* 下の余白 */
margin-left: 50px;    		/* 左の余白 */

ちょっと便利な使い方

中央寄せ：横幅を決めて margin: 0 auto; と書くと、要素が中央に配置されます。

フォルダ構成

フォルダ構成 
Chapter4_1-4
└── index.html
└── style.css
HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8" />
  <title>marginのサンプル</title>
  <link rel="stylesheet" href="style.css" /></head>
<body>
  <div class="box">こんにちは！私はmarginで外のスキマを作ります。</div>
</body>
</html>
CSS 
body {
  background: #f0f8ff;
  font-family: sans-serif;
}

.box {
  background: #4fa4ff;
  color: white;
  padding: 10px;
  margin: 30px 0 0 100px; /* 上30px、左100pxにスキマ */
  width: 250px;
  border-radius: 8px;
}

結果：上に30px、左に100pxの余白がついて、箱が右下にずれます。

POINT！

隣り合う要素同士のmarginは、加算ではなく「相殺」となります。
要素Aの下に100px、要素Bの上に150pxのmarginを指定しても、要素間の距離は250pxにならず、
重なった100pxは相殺され、残り50pxがプラスされます。
結果、要素Aと要素Bの外側の余白は、どちらから見ても150pxとなります。

Section 4-4

内側の余白：padding プロパティ

padding（パディング）は、要素の内側のクッションを作ります。
文字と枠がピッタリくっつかないように調整します。

基本の書き方

CSS 
padding: 20px;          		 /* すべての方向に20px */
padding: 10px 20px;      		/* 上下10px、左右20px */
padding: 10px 20px 30px 40px; 	/* 上→右→下→左（時計回り） */

POINT

paddingには相殺などの概念がありません

フォルダ構成

フォルダ構成 
Chapter4_2-2
 └── index.html
 └── style.css
HTML 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8" />
  <title>paddingのサンプル</title>
  <link rel="stylesheet" href="style.css" />
</head>
<body>
  <div class="box">こんにちは！私はpaddingで内側に余裕を作ります。</div>
</body>
</html>
CSS 
body {
  background: #fffbea;
  font-family: sans-serif;
}

.box {
  background: #ffb84f;
  color: #333;
  padding: 40px;       /* 内側の余白を40pxに */
  border-radius: 8px;
}

結果：ボックス内のテキストまわりに40pxの余裕ができて、読みやすくなります。
