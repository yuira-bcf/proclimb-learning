Section 1

CSSとは

CSS（シー・エス・エス）とは、Cascading Style Sheets（カスケーディング・スタイル・シート）の略称です。
Webページの見た目（デザイン）を整えるための言語であり、
HTMLで作られた文書のレイアウトや色、文字の大きさ、背景などを指定できます。

HTMLはページの「構造」や「内容（テキストや画像）」を定義する言語。
CSSはそのHTML要素に装飾や配置などの"スタイル"を付ける役割を持っています。
JavaScriptは、HTML/CSSを操作して、リアルタイムにWebページに動きを付ける言語です。

Section 2

CSSを使わないと、どうなるのか

HTMLのみでWebページを作ると、見た目が無機質で無骨な印象のページにしかなりません。
ユーザーにとっては「使いにくいページ」となってしまいます。

Webページの目的は、集客や認知、成約、購入などです。
それを「見て」「感じて」「購入ボタンを押す」のは、最後は心、人です。

人にとって使いやすいページにする。良い体験が出来るページにする。
それを左右するのが、CSSとなります。

Section 3

CSS適用方法

CSSは、HTMLに読み込まれることで、その装飾がブラウザに反映されます。
全部で3つの適用方法があるので、順番に説明していきます。

3-1. CSS専用ファイルを読み込んで反映（外部スタイル）

index.htmlというHTMLファイルがあったら、CSSを「style.css」というファイルを作ることが一般的です。
そのstyle.cssをHTMLで読み込ませて、スタイルを反映させる方法が、実務で最も多い手法となります。

フォルダ構成のイメージ

フォルダ構成 
Chapter1_3-1
 ├── index.html
 └── CSS
　　　└── style.css
HTML（index.html） 
<!doctype html>
<html lang="ja">
<head>
    <meta charset="utf-8">
    <meta name="description" content="カフェメニュー紹介ページ">
    <title>Café Latte｜メニュー</title>

    <!-- CSSファイルの読み込み -->
    <link rel="stylesheet" href="css/style.css">
</head>

<body>
  <h1>ようこそ Café Latte へ</h1>
  <p>季節限定のメニューをご紹介します。</p>
</body>
</html>

CSSファイルの読込は、上記のコードの部分で行われています。

HTML 
<link rel="stylesheet" href="css/style.css">

HTMLでは、外部のファイルやウェブ上のリソースを適用させるために、
この<link>タグを<head>タグ内に記述して利用・反映します。
CSSが外部ファイルとして存在する場合、上記方法で読み取ってもらえます。

POINT

「rel="stylesheet"」は、「これから読み込むファイルはCSSですよ」と表しなさい。
href の中には CSSファイルまでの道順（ファイルパス） を正確に書きます。
index.html と同じ階層なら href="style.css"、
フォルダ内なら href="css/style.css" のように書き分けましょう。

実際のCSSファイルの中身（style.css）

CSS（style.css） 
/* style.css */
body {
  background-color: #fdf6ec;
  font-family: "Hiragino Sans", "Helvetica", sans-serif;
}

h1 {
  color: #6b3e26;
  text-align: center;
}

p {
  color: #444;
  line-height: 1.8;
  text-align: center;
}

CSSの構成・書き方は後述します。
上記では、<body>の背景をベージュに、<h1>はブラウンで中央寄せ、<p>はグレーで中央寄せ
などのスタイルを指定しています。

表示結果

Webサイトは1ページと限りません。ほとんどの場合で複数ページが存在します。
複数の.htmlファイルに適用するCSSをまとめてstyle.cssに書いておける、管理が出来るので
この外部スタイルでの反映方法が最もポピュラーです。

3-2. HTML内で<style>タグに記述して読込（内部スタイル）

HTMLの<head>内に、ページの設定情報として、<style>タグを記述します。
<style></style>の中には、CSSファイルに書く内容と、同じものを記述することができます。

フォルダ構成のイメージ

フォルダ構成

  <title>Café Latte｜季節限定メニュー</title>

   <!-- ここにCSSを直接書く！ -->
  <style>

    background-color: #fff9f3;

      "Hiragino Kaku Gothic ProN",
      "Yu Gothic UI",

       text-align: center;
    }

      color: #555;
       text-align: center;

<body>

</body>

内部スタイルは、そのページでのみ反映されます。
外部スタイルとは別に、そのページだけ別のスタイルを当てたい、などというときに使います。

3-3. HTMLのタグの属性にstyle="〇〇" を記述して読込（インラインスタイル）

タグの中に属性として、直接styleを適用させる方法です。
要素ひとつひとつに適用していく方法のため、数が多いと大変で基本的には推奨されません。

3つのCSS適用方法：まとめ

方法 記述場所 特徴
① 外部スタイルシート 別ファイル（.css） 複数ページに一括適用できる（推奨）★基本はコレを使う
② 内部スタイルシート <style>タグ内 ページごとのデザイン調整に便利
③ インラインスタイル HTMLのタグの中に直接書く 部分的な変更に使用（あまりよくない）

外部スタイルシートを使うことを、まずは前提としておきましょう。

Section 4

CSS適用の優先順位

外部スタイルや内部スタイル、インライン。
この3つの適用方法で、たまに同じ要素にそれぞれ違うスタイルを当てる場合があります。
そのような状態のことを「競合」すると言います。

競合しても、正しく表示されるように、CSSではどの適用が優先されるという順位が決まっています。

① CSSの適用による優先順位

インライン ＞ 内部スタイル＞ 外部スタイル

より詳細な指定が優先されるよう設計されています。

② 詳細度による優先順位

もうひとつ、同じ適用スタイルの中でも、タグ名やクラス、idなどの指定方法に優先順位があります。
それを「詳細度」と呼ばれるものです。

#id名 ＞ .class名 ＞ タグ名

指定方法 詳細度の持ち点
#id 100点
.class 10点
タグ名 1点

競合した場合は、上記の詳細度の点数の合計が高いものが優先される
※ 合計点数が同じ場合は、最後に書かれたものが優先される

フォルダ構成のイメージ

フォルダ構成 
Chapter1_4-1
└── index.html
HTML（index.html） 
<!doctype html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <meta name="description" content="カフェの季節限定メニューページ">
  <title>Café Latte｜季節限定メニュー</title>
  <style>
    #lead {
      color: blue;
    }
    .note {
      color: green;
    }
    p {
      color: tomato;
    }
  </style>
</head>

<body>
  <p id="lead" class="note" >
    本日のおすすめ：キャラメルラテ
  </p>
</body>
</html>

表示結果

idの優先度が最も高いので、後に.noteやpで指定していても、
#leadに書かれたものが最も優先されます。

※ クラスやidについては後述します。

Section 5

CSSの基本の書き方を身につけよう

5-1.CSSの基本文法

CSSは次の3つの要素で構成されています

CSS 
セレクタ {
  プロパティ: 値;
}
用語 意味 例
セレクタ（selector） どの部分にデザインを適用するか h1, .menu, #header
プロパティ（property） 何を変えるか（色・大きさなど） color, font-size
値（value） どう変えるか（具体的な内容） red, 20px

5-2.フォルダ構成

フォルダ構成 
Chapter1_5-2
 ├── index.html
 └── style.css
CSS（style.css） 
h1 {
  color: #6b3e26;
  font-size: 28px;
  text-align: center;
}
指定セレクタ プロパティ（意味） 値（意味）
<h1>タグ（タグ名で指定） color（文字の色） #6b3e26（茶系）
font-size（文字の大きさ） 28px（28というピクセル単位の大きさ）
text-align（1行の中の文字の位置） center（中央寄せ）
HTML（index.html） 
<!doctype html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <meta name="description" content="Café Latteの季節メニュー紹介ページ">
  <title>Café Latte｜メニュー</title>
   <!-- CSSファイルを読み込む -->
  <link rel="stylesheet" href="style.css">
</head>

<body>
  <h1>Café Latteへようこそ！</h1>
</body>
</html>

Section 6

セレクタとは

セレクタでは、「タグ名」や「クラス名」「id名」で指定することができます。

セレクタの種類 書き方 説明
要素セレクタ h1, p, a HTMLタグ名を指定
クラスセレクタ .menu, .title 同じデザインを複数の要素に適用
IDセレクタ #header, #footer ページ内で一意の要素に適用

例：カフェメニューを装飾

クラスとid

HTMLの属性として、classやidを付与することができます。
これを利用することで、同じタグでもそれぞれ違うスタイルを適用することができます。
classは重複可能で、同じクラス名を別のタグに付与できます。
idはそのページの中では、同じ名前を使うことは出来ません。
idは必ず一意の値（ユニーク）でなければなりません。

Section 7

プロパティと値

プロパティは、指定されたセレクタの要素の「何を変えるのか」、値は「どう変えるのか」を指定します。

例：

CSS 
p {
  color: #333;        /* 文字の色をグレーに */
  font-size: 18px;    /* 文字サイズを18pxに */
  line-height: 1.6;   /* 行間を広げに */
}

7-1. 1行ごとに

「プロパティ: 値;」の形式で書くのがルールです！
「:（コロン）」と「;（セミコロン）」は必須です。忘れないようにしましょう。

7-2.CSSを書くときのルール

ルール 内容
① 半角英数字で書く 全角や日本語は使いません。必ず半角で！
② 小文字で統一する CSSは大文字小文字を区別しないが、読みやすさのために小文字が基本。
③ 整の区切りで改行・スペースはある程度入れる 見やすくなります。
④ 改行・スペースは自由 英文と一緒で、単語の途中には入れられません。

・1行で書く例

CSS 
h1 { color: #6b3e26; font-size: 28px; }

1行でまとめてもOKですが、少し読みにくいです。
読みやすくするために改行して書くのが一般的です。

CSS 
h1 {
  color: #6b3e26;
  font-size: 28px;
}

・複数のプロパティを指定するとき
1つの要素に複数のプロパティを適用するときは、単純に書き足していきます。

CSS 
h1 {
  color: #6b3e26;
  font-size: 28px;
  text-align: center;
}
Section 8

要素の中の要素を指定

タグの中に、同じタグが複数入っている場合があります（<ul>の中の<li>など）
その中でも、特定のタグにだけスタイルを適用したい場合に、指定する方法があります。

例：カフェメニューの中の小タイトルだけ色を変える

8-1.フォルダ構成

フォルダ構成 
Chapter1_8-1
 ├── index.html
 └── style.css
HTML（index.html） 
<!doctype html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="description" content="Café Latteのメニュー紹介ページ">
   <title>Café Latte｜メニュー紹介</title>

  <!-- CSSファイルを読み込む -->
  <link rel="stylesheet" href="style.css">
</head>
<body>
<div class="menu">
  <p>ブレンドコーヒー</p>
  <p>本日のケーキ</p>
</div>
<p>営業時間：10:00〜19:00</p>
</body>
</html>
CSS（style.css） 
.menu p {
    color: #6b3e26;
    font-weight: bold;
    /* メニュー名の文字だけ太字のブラウン */
}

POINT

.menu p のように、半角スペースで区切ると、「.menu の中にある <p>」を指定する意味になる
.menu で囲まれていない <p> には影響しません。
