Introduction

グリッドレイアウトとは

FlexBoxの他に、もうひとつ横並びを実現できるプロパティがあります。
それがグリッドレイアウトとよばれる方法です。

Section 7-1

CSSグリッドとは？

CSSグリッド（Grid Layout）は、ページの中に見えないマス目（グリッド）を作り、
要素をタイル状にきれいに並べるための仕組みです。

レイアウト 特徴
Flexbox 1列や1行の並びに強い
Grid 2次元（タイル状）に並べるのが得意

つまり、「縦×横のレイアウトをまとめてコントロールできる」超便利機能です！

Section 7-2

グリッドの基本構造

Flexboxと同じく、「親」と「子」の関係で考えます。

名称 役割
Gridコンテナ（親） 全体を包む箱（マス目の外枠）
Gridアイテム（子） 中に並ぶ各ブロック（マスの中身）

h1やp、divなどは最初から画面横いっぱいの領域を持っているため、
この段階ではあまり見分けは付きません

Section 7-3

列（横方向）をつくる：grid-template-columns

grid-template-columns は子要素に対して、
「何列に分けるか」「列の幅はどれくらいか」を決めます。

たとえば横に3つ並べたいときはこう書きます

CSS 
grid-template-columns: 200px 200px 200px;

これは、200pxの要素を、縦3列で並べる、という意味の指定になります。

フォルダ構成

フォルダ構成 
Chapter7_1-3
 └── index.html
 └── style.css

▶index.html ※7_1-2と同じ

CSS 
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px; /* 3列 × 幅200px */

  background: #B7F7FF;
}
.item {
  background: #0bd;
  color: #fff;
  padding: 10px;
}

結果：アイテムが「1行に3つ×2段」でタイル状に並びます。

Section 7-4

余白：gap プロパティ

これまでの要素はすべてくっついているため、非常に見ずらいですよね。
各要素同士の隙間を空けたいときは、gapプロパティで指定します。

フォルダ構成

フォルダ構成 
Chapter7_1-4
└── index.html
└── style.css

▶index.html ※7_1-2と同じ

CSS 
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
  gap: 10px; /* 各マスのすきまを10pxに！ */

   background: #B7F7FF;
}
.item {
  background: #0bd;
  color: #fff;
  padding: 10px;
}

結果：アイテムの間に10pxの余白ができて、ぐっと見やすくなります

Section 7-5

自動で広がる：fr 単位を使おう

これまで要素の幅をpx単位で固定指定していましたが、子要素ひとつひとつが綺麗に等分した幅にするのを
指定することも出来ます。それがfr（フラ）という単位です

CSS 
grid-template-columns: 1fr 1fr 1fr;

「1fr」は"1等分"という意味。
1fr 1fr 1fr とすれば、3列で1:1:1という意味になります。

フォルダ構成

フォルダ構成 
Chapter7_1-5
 └── index.html
 └── style.css

▶index.html ※7_1-2と同じ

CSS 
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;

background: #B7F7FF;
}
.item {
  background: #0bd;
  color: #fff;
  padding: 10px;
}

結果：ウィンドウを広げるとボックスが広がり、狭めると自動で縮む！
レスポンシブデザインの第一歩です

POINT

frの前に指定している数字は対比の倍率指定となります。
1fr 2fr 3fr などと指定すると、3列で1:2:3という幅でそれぞれの子要素に違う幅で指定できます。

Section 7-6

行（縦方向）を整える：grid-template-rows

FlexBox同様に、グリッドレイアウトでも縦の要素を整えることが可能です。

フォルダ構成

フォルダ構成 
Chapter7_1-6
 └── index.html
 └── style.css

▶index.html ※7_1-2と同じ

CSS 
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 横3列 */
  grid-template-rows: 200px 200px;    /* 縦2行（高さ200pxずつ） */
  gap: 10px;
background: #B7F7FF;
}
.item {
  background: #0bd;
  color: #fff;
  padding: 10px;
}

結果：3列×2行＝6つのボックスがきれいなタイル状に並びます

まとめ

第7章のまとめ

プロパティ 意味 例
display: grid; グリッドを使う宣言 -
grid-template-columns 列の幅・数を指定 200px 200px 200px
grid-template-rows 行の高さを指定 200px 200px
gap すきまの幅を指定 gap: 10px;
fr 画面に合わせて自動調整 1fr 1fr 1fr

便利な応用

grid-template-columns: repeat(3, 1fr); は、1fr を 3回繰り返す。
grid-template-columns: 1fr 1fr 1fr; と同じ意味。
同じ値のみを指定するならこの記法が楽。
見た目がスッキリ、わかりやすい！
