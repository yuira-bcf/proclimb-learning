Section 02-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

Step 1で作成したSpring Bootプロジェクトに、データベース機能を追加してください。

【技術スタック】
- H2 Database（組み込みDB）
- MyBatis（ORマッパー）

【テーブル設計】
テーブル名: todos
- id: BIGINT, PRIMARY KEY, AUTO_INCREMENT
- title: VARCHAR(100), NOT NULL
- completed: BOOLEAN, DEFAULT FALSE
- created_at: TIMESTAMP, DEFAULT CURRENT_TIMESTAMP

【作成するファイル】
1. Todo.java - Entityクラス（Lombokを使用）
2. schema.sql - テーブル作成SQL
3. data.sql - 初期データ（3件程度のサンプル）
4. application.properties - H2データベース設定を追加

【ファイル配置】
src/main/java/com/example/todo/entity/Todo.java
src/main/resources/schema.sql
src/main/resources/data.sql

【追加する依存関係】
- mybatis-spring-boot-starter
- h2database

pom.xmlの依存関係追加と各ファイルのコードを実装してください。
lightbulb ポイント

H2コンソール（http://localhost:8080/h2-console）で直接SQLを実行してテーブルの確認ができます。
