あなたはJavaプログラマーです。

ToDoアプリに一括削除機能を追加してください。

【要件】
- 各ToDoにチェックボックスを追加
- ヘッダーに「全選択」チェックボックス
- 選択中の件数をリアルタイム表示
- 「一括削除」ボタンで選択したToDoを削除
- 削除前に確認ダイアログを表示

【実装箇所】

■ list.html
- 各行にチェックボックス追加
  <input type="checkbox" class="todo-checkbox" th:value="${todo.id}">
- ヘッダーに全選択チェックボックス
- 選択件数表示エリア
- 一括削除ボタン（選択0件時は無効化）

■ bulk-delete.js（新規作成）
```javascript
// 全選択/全解除
function toggleAll(checked) {
    document.querySelectorAll('.todo-checkbox')
        .forEach(cb => cb.checked = checked);
    updateSelectedCount();
}

// 選択件数更新
function updateSelectedCount() {
    const count = document.querySelectorAll('.todo-checkbox:checked').length;
    document.getElementById('selectedCount').textContent = count;
    document.getElementById('bulkDeleteBtn').disabled = count === 0;
}

// 一括削除実行
function bulkDelete() {
    const ids = Array.from(document.querySelectorAll('.todo-checkbox:checked'))
        .map(cb => cb.value);
    if (confirm(`${ids.length}件のToDoを削除しますか？`)) {
        // フォームで送信 or fetch API
    }
}
```

■ TodoController.java
- @PostMapping("/bulk-delete")
- @RequestParam List<Integer> ids で複数ID受け取り
- サービス経由で削除実行

■ TodoService.java
- deleteByIds(List<Integer> ids) メソッド追加
- 空リストの場合は何もしない

■ TodoMapper.xml

- 選択行のハイライト表示
