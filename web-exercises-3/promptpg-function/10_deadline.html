Section 10-2

成果物

この機能で作成・更新するファイルの一覧です。

ファイル 変更内容

期限表示のルール

  • 期限切れ（過去日） - 赤背景 + 「期限切れ」表示
  • 本日期限 - オレンジ背景 + 「本日期限」表示
  • 期限3日以内 - 黄色背景 + 「あと○日」表示
  • 期限4日以上先 - 通常表示 + 日付のみ
  • 期限未設定 - 「-」表示
Section 10-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

ToDoアプリに期限設定機能を追加してください。

【要件】
- 期限は日付のみ（時刻なし）
- 期限は任意入力（未設定可）
- 期限切れのタスクを赤く警告表示
- 期限間近（3日以内）を黄色で警告
- 期限でのソート機能

【実装箇所】

■ Todo.java
- deadline (LocalDate) フィールド追加
- @DateTimeFormat(pattern = "yyyy-MM-dd") アノテーション
- isOverdue() メソッド - 期限切れ判定
- getDaysUntilDeadline() メソッド - 残り日数計算

■ schema.sql
- deadline DATE カラム追加（NULL許容）

■ TodoMapper.xml
- INSERT/UPDATE文にdeadlineを追加
- 期限順ソートのORDER BY対応
- 期限切れ一覧取得クエリ（WHERE deadline < CURRENT_DATE）

■ form.html
- <input type="date" th:field="*{deadline}">
- @DateTimeFormatでフォーマット指定
- min属性で過去日入力を制限（任意）

■ list.html
- 期限表示: th:text="${#temporals.format(todo.deadline, 'yyyy/MM/dd')}"
- 期限切れ判定: th:if="${todo.deadline != null and todo.deadline.isBefore(#temporals.createToday())}"
- 期限間近判定: 残り日数が0〜3日の場合
- 条件に応じたクラス付与（deadline-overdue, deadline-soon等）

■ style.css
- .deadline-overdue { background: #fef2f2; color: #dc2626; border-left: 4px solid #dc2626; }
- .deadline-today { background: #fff7ed; color: #ea580c; }
- .deadline-soon { background: #fffbeb; color: #d97706; }

■ Controller
- 期限順ソートのパラメータ対応

必要なコードを提示してください。
