Section 06-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

ToDoアプリに削除確認ダイアログを実装してください。

【要件】
- 削除ボタンクリック時に確認ダイアログを表示
- 「本当に削除しますか？」というメッセージ
- 削除対象のタイトルを表示
- OKで削除実行、キャンセルで中止

【実装パターン2つ】

■ パターン1: シンプルなconfirm()
1. list.html の削除ボタン
   - onsubmit="return confirmDelete('タイトル名')"
   - JavaScriptでconfirm()を呼び出し
   - falseを返すとフォーム送信を中止

■ パターン2: カスタムモーダル（推奨）
1. モーダルHTML
   - オーバーレイ + モーダルボックス
   - タイトル、メッセージ、ボタン（削除/キャンセル）

2. モーダルCSS
   - 中央配置
   - オーバーレイで背景を暗く
   - アニメーション付き表示

3. JavaScript
   - モーダルの表示/非表示制御
   - 削除対象のIDとタイトルを動的に設定
   - 削除実行時にフォームをsubmit
