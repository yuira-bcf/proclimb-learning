Section 24-1

工程概要

ToDo作成時や期限が近づいた際にメールで通知を送信します。

学習テーマ

メール送信

主な技術

JavaMailSender / Thymeleaf Template

application.propertiesにSMTPサーバーの設定を追加

メール送信サービス

JavaMailSenderを使用したメール送信機能の実装

HTMLメールテンプレート

Thymeleafで動的なHTMLメールを生成

期限リマインダー

@Scheduledで毎日期限チェックしてメール送信

Section 24-2

成果物

この機能で作成・更新するファイルの一覧です。

ファイル 変更内容
application.properties SMTPサーバー設定
MailService.java メール送信サービス
todo-created.html ToDo作成通知メールテンプレート
deadline-reminder.html 期限リマインダーメールテンプレート
DeadlineNotificationJob.java 期限通知のスケジュールジョブ

メール送信の主要コンポーネント

  • JavaMailSender - Springのメール送信インターフェース
  • MimeMessageHelper - HTMLメール、添付ファイルのヘルパー
  • TemplateEngine - Thymeleafでメール本文を生成
  • @Async - 非同期でメールを送信
Section 24-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

Spring Boot 3.xのToDoアプリにメール送信機能を追加してください。

【現在の状態】
- Spring Boot 3.x + MyBatis + Thymeleafで構築されたToDoアプリ
- ユーザーにはemailフィールドがある
- メール送信機能は未実装

【実装要件】
1. SMTP設定
   - application.propertiesにSMTPサーバー設定
   - Gmail SMTPを使用する例
   - 認証情報は環境変数から取得

2. メール送信サービス（MailService.java）
   - JavaMailSenderを使用
   - プレーンテキストメール送信
   - HTMLメール送信（Thymeleafテンプレート使用）
   - 添付ファイル付きメール送信
   - @Asyncで非同期送信

3. メールテンプレート
   - todo-created.html（ToDo作成通知）
   - deadline-reminder.html（期限リマインダー）
   - Thymeleafで動的にコンテンツを生成

4. 使用例
   - ToDo作成時に作成通知メールを送信
   - 期限が近いToDoのリマインダーを送信

5. スケジュール処理
   - @Scheduledで毎日期限チェック
   - 期限が1日前のToDoをリストアップしてメール送信

【技術スタック】
- Spring Boot 3.x
- spring-boot-starter-mail
- JavaMailSender / MimeMessageHelper
- Thymeleaf（メールテンプレート）

必要なコードの変更点を提示してください。
lightbulb ポイント

開発環境ではMailHogなどのメールキャプチャツールを使用すると、実際にメールを送信せずにテストできます。
