Section 01-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

Spring Bootを使用して「Hello! Spring Boot」と表示するWebアプリケーションを作成してください。

【技術スタック】
- Java 17以上
- Spring Boot 3.x
- Thymeleaf（テンプレートエンジン）
- Maven

【要件】
1. http://localhost:8080 にアクセスすると「Hello! Spring Boot」と表示される
2. Thymeleafテンプレートを使用してHTMLを表示する
3. Controllerクラスを作成してリクエストを処理する

【作成するファイル】
1. pom.xml - ビルド設定ファイル
2. TodoApplication.java - メインクラス
3. HelloController.java - Controllerクラス
4. index.html - テンプレートファイル
5. application.properties - 設定ファイル

【ファイル構成】
src/main/java/com/example/todo/
├── TodoApplication.java
└── controller/
    └── HelloController.java

src/main/resources/
├── application.properties
└── templates/
    └── index.html

各ファイルのコードを実装してください。
