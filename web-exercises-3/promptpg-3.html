Section 03-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

Step 2で作成したTodoエンティティに対するRepository層（データアクセス層）を作成してください。

【技術スタック】
- MyBatis
- XMLマッピング方式

【Todoエンティティ】
- id: Long（主キー）
- title: String
- completed: Boolean
- createdAt: LocalDateTime

【作成するメソッド】
1. findAll() - 全件取得（作成日時の降順）
2. findById(Long id) - ID指定取得
3. insert(Todo todo) - 新規登録（useGeneratedKeysでID取得）
4. update(Todo todo) - 更新（title, completedのみ）
5. delete(Long id) - 削除

【作成するファイル】
1. TodoMapper.java - Mapperインターフェース（@Mapperアノテーション付き）
2. TodoMapper.xml - SQLマッピング定義

【ファイル配置】
src/main/java/com/example/todo/mapper/TodoMapper.java
src/main/resources/mapper/TodoMapper.xml

【application.propertiesへの追加設定】
mybatis.mapper-locations=classpath:mapper/*.xml
mybatis.configuration.map-underscore-to-camel-case=true

各ファイルのコードを実装してください。
lightbulb ポイント

map-underscore-to-camel-caseを有効にすると、created_at → createdAt のような変換が自動で行われます。
