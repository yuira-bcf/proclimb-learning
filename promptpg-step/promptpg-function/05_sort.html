Section 05-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

サンプルソースコード

TodoMapper.xml（動的ORDER BY） 
<select id="findAllWithSort" resultType="Todo">
    SELECT * FROM todos
    <choose>
        <when test="sortBy == 'priority'">
            ORDER BY priority ${sortOrder}
        </when>
        <when test="sortBy == 'title'">
            ORDER BY title ${sortOrder}
        </when>
        <when test="sortBy == 'deadline'">
            ORDER BY deadline ${sortOrder}
        </when>
        <otherwise>
            ORDER BY created_at DESC
        </otherwise>
    </choose>
</select>
TodoController.java 
@GetMapping("/todos")
public String list(
        @RequestParam(defaultValue = "created_at") String sortBy,
        @RequestParam(defaultValue = "DESC") String sortOrder,
        Model model) {

    List<Todo> todos = todoService.findAllWithSort(sortBy, sortOrder);
    model.addAttribute("todos", todos);
    model.addAttribute("sortBy", sortBy);
    model.addAttribute("sortOrder", sortOrder);
    return "todos/list";
}

menu_book コード解説

  • <choose>タグ: Javaのswitch文に相当。条件に応じてSQL文を切り替え
  • ${sortOrder}: 文字列として直接SQL文に挿入（ASC/DESCのみ許可）
  • <otherwise>: デフォルトのソート条件（作成日の降順）
  • セキュリティ: <choose>でカラム名を限定し、SQLインジェクションを防止

【実装内容】
1. TodoMapper.java
   - List findAllWithSort(@Param("sortBy") String sortBy, @Param("sortOrder") String sortOrder);

2. TodoMapper.xml
   - タグで動的ORDER BYを実装
   - SQLインジェクション対策として、許可されたカラム名のみを使用

3. TodoController.java
   - @RequestParam でソート条件を受け取り
   - デフォルト値を設定（sortBy="created_at", sortOrder="DESC"）

4. list.html
   - ソート選択のドロップダウンを追加
   - 現在のソート条件を保持
   - ソート変更時に自動的にフォーム送信

【セキュリティ注意点】
- ソートカラム名は${}ではなく、で安全に指定
- ユーザー入力を直接SQLに使用しない

【SQL例】

  
    ORDER BY priority ${sortOrder}
  
  
    ORDER BY title ${sortOrder}
  
  
    ORDER BY created_at DESC
