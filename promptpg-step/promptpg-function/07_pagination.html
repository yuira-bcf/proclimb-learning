Pageオブジェクトの主要メソッド

  • getContent() - 現在ページのデータリスト
  • getTotalPages() - 総ページ数
  • getTotalElements() - 総データ件数
  • getNumber() - 現在のページ番号（0始まり）
  • hasNext() / hasPrevious() - 次/前ページの有無
Section 07-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

サンプルソースコード

TodoController.java 
@GetMapping("/todos")
public String list(
        @PageableDefault(size = 10, sort = "createdAt",
                         direction = Sort.Direction.DESC) Pageable pageable,
        Model model) {

    Page<Todo> todoPage = todoService.findAll(pageable);

    model.addAttribute("todos", todoPage.getContent());
    model.addAttribute("page", todoPage);
    return "todos/list";
}
TodoMapper.xml 
<select id="findAllWithPagination" resultType="Todo">
    SELECT * FROM todos
    ORDER BY created_at DESC
    LIMIT #{size} OFFSET #{offset}
</select>

<select id="countAll" resultType="long">
    SELECT COUNT(*) FROM todos
</select>

menu_book コード解説

  • @PageableDefault: ページサイズ、ソート条件のデフォルト値を設定
  • Page<Todo>: データリスト + ページング情報を含むオブジェクト
  • LIMIT #{size}: 1ページあたりの件数を指定
  • OFFSET #{offset}: 取得開始位置（page × size）

auto_awesome AIプロンプト

あなたはJavaプログラマーです。

ToDoアプリにページネーション機能を追加してください。

【要件】
- 1ページあたり10件表示
- 「前へ」「次へ」ボタンを配置
- ページ番号リンクを表示（1, 2, 3...）
- 現在のページをハイライト表示
- 総件数と表示範囲を表示（例：全50件中 1-10件）

【実装箇所】

■ TodoController.java
- @PageableDefault(size = 10, sort = "createdAt", direction = DESC)
- Pageableパラメータを受け取る
- Page<Todo>をモデルに追加

■ TodoMapper.xml（MyBatis使用時）
- SELECT文にLIMIT #{size} OFFSET #{offset} を追加
- 総件数取得用のCOUNT(*)クエリを追加

■ TodoRepository.java
- Page<Todo> findAll(Pageable pageable) メソッド追加
- 検索+ページングの複合メソッド追加

■ list.html（Thymeleaf）
- ページャーUI（前へ/次へ/ページ番号）
- 現在ページにactiveクラスを付与
- 件数表示「全{total}件中 {start}-{end}件を表示」

【注意点】
- Spring Dataのページ番号は0始まり（表示時は+1）
- 検索条件がある場合はページングと併用できるように
- 総ページ数が多い場合は省略表示を検討

必要なコードを提示してください。
