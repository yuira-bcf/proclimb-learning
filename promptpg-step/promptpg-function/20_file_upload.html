Section 20-1

工程概要

ToDoに画像やファイルを添付できる機能を実装し、ダウンロード・削除も行えるようにします。

学習テーマ

画像・ファイルの添付機能

主な技術

MultipartFile / ResponseEntity / Resource

関連レイヤー

Controller / Service / Repository / View

この機能で行うこと

settings

ファイルアップロード設定

application.propertiesでサイズ制限

data_object

添付ファイルエンティティの作成

メタ情報をDBに保存

cloud_upload

FileStorageServiceの実装

UUID形式で安全に保存

download

ダウンロード・削除機能

ResponseEntityでファイル返却

Section 20-2

成果物

この機能で作成・更新するファイルの一覧です。

ファイル 変更内容
application.properties ファイルサイズ上限、保存先設定
TodoAttachment.java 添付ファイルエンティティ
FileStorageService.java ファイル保存・取得サービス
TodoController.java アップロード・ダウンロードエンドポイント
AttachmentMapper.xml 添付ファイルCRUD
attach.html アップロードフォーム

セキュリティ上の注意点

  • ファイル名サニタイズ - パストラバーサル対策（..を含むパスを拒否）
  • ファイル種類制限 - 許可する拡張子・MIME型を制限
  • ファイルサイズ上限 - application.propertiesで設定
  • 保存ディレクトリ - Webから直接アクセスできない場所に配置
  • UUID形式のファイル名 - 元のファイル名は露出させない
Section 20-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

Spring Boot 3.xのToDoアプリにファイル添付機能を追加してください。

【要件】
- MultipartFileを使用してファイルをアップロード
- application.propertiesでファイルサイズ上限（10MB）を設定
- TodoAttachmentエンティティで添付ファイルのメタ情報を管理
- FileStorageServiceでファイル保存（UUID形式のファイル名に変換）
- ファイル名のサニタイズ（パストラバーサル対策）
- ダウンロード機能（Content-Dispositionヘッダーでファイル名を指定）
- 添付ファイルの削除機能

【実装箇所】

■ application.properties
- spring.servlet.multipart.max-file-size=10MB
- spring.servlet.multipart.max-request-size=10MB
- app.upload.dir=./uploads

■ TodoAttachment.java
- id, todoId, originalFilename, storedFilename
- contentType, fileSize, uploadedAt
- getter/setter

■ FileStorageService.java
- store(Integer todoId, MultipartFile file) - UUID形式でファイル保存
- StringUtils.cleanPath()でファイル名サニタイズ
- ".."を含むパスは拒否
- DBにメタ情報を保存

■ TodoController.java
- @PostMapping("/{id}/attach") - アップロード処理
- @GetMapping("/attachments/{attachmentId}/download") - ダウンロード
- ResponseEntity<Resource>でファイルを返却
- Content-Dispositionヘッダーでファイル名指定（UTF-8エンコード）

■ AttachmentMapper.xml
- insert, findByTodoId, findById, delete

■ attach.html
- enctype="multipart/form-data"フォーム
- ファイル選択input type="file"

【技術スタック】
- Spring Boot 3.x
- MultipartFile
- MyBatis
- Thymeleaf
