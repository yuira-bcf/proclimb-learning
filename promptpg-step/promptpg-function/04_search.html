Section 04-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

ToDoアプリにキーワード検索機能を実装してください。

【検索仕様】
- 検索対象: title と description
- 検索方式: 部分一致（LIKE検索）
- 大文字小文字: 区別しない
- キーワード未入力時: 全件表示

【実装内容】
1. TodoMapper.java
   - List searchByKeyword(@Param("keyword") String keyword);

2. TodoMapper.xml
   - MyBatisの動的SQLを使用
   - タグでキーワードの有無を判定
   - CONCAT('%', #{keyword}, '%') でLIKE検索
   - LOWER()関数で大文字小文字を統一

3. TodoService.java
   - 検索メソッドを追加

4. TodoController.java
   - @RequestParam("keyword") で検索キーワードを受け取り
   - Modelに検索キーワードを保持（フォームに表示するため）

5. list.html
   - 検索フォームを一覧の上に追加
   - 検索ボタンとクリアボタン
   - 検索後もキーワードを入力欄に保持

【SQL例】
SELECT * FROM todos
WHERE LOWER(title) LIKE LOWER(CONCAT('%', #{keyword}, '%'))
   OR LOWER(description) LIKE LOWER(CONCAT('%', #{keyword}, '%'))

必要なコードの変更点を提示してください。
lightbulb ポイント

SQLインジェクション対策として、MyBatisの#{keyword}を使用します。${}は使用しないでください。
