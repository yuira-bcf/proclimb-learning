AOPアノテーションとPointcut式

  • @Before - メソッド実行前に処理
  • @After - メソッド実行後に処理（成功/失敗に関わらず）
  • @AfterReturning - メソッドが正常に終了した後に処理
  • @AfterThrowing - 例外が発生した場合に処理
  • @Around - メソッドの前後を完全に制御
  • execution(* com.example.service.*.*(..)) - serviceパッケージの全メソッド
Section 18-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

Spring Boot 3.xのToDoアプリにAOP（アスペクト指向プログラミング）でログ出力機能を追加してください。

【要件】
- spring-boot-starter-aopの依存関係を追加
- LoggingAspectクラスを作成し、Serviceクラスの全メソッドを対象にログ出力
- @Before: メソッド実行前に「メソッド名」と「引数」をログ出力
- @AfterReturning: メソッド正常終了後に「メソッド名」と「戻り値」をログ出力
- @AfterThrowing: 例外発生時に「メソッド名」と「エラーメッセージ」をログ出力
- PerformanceAspectクラスを作成し、@Aroundで実行時間を計測

【実装箇所】

■ pom.xml
- spring-boot-starter-aopの依存関係を追加

■ LoggingAspect.java
- @Aspect、@Componentを付与
- @Pointcut("execution(* com.example.service.*.*(..))")でServiceを対象
- @Before: joinPoint.getSignature().getName()でメソッド名取得
- @AfterReturning: returning = "result"で戻り値取得
- @AfterThrowing: throwing = "ex"で例外取得

■ PerformanceAspect.java
- @Around("execution(* com.example.service.*.*(..))")
- System.currentTimeMillis()で開始/終了時間を取得
- joinPoint.proceed()で実際のメソッドを実行
- 実行時間をログ出力

【技術スタック】
- Spring Boot 3.x
- Spring AOP
- SLF4J（ロギング）
