Section 12-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

ToDoアプリにCSVエクスポート機能を追加してください。

【要件】
- 一覧画面に「CSVダウンロード」ボタンを配置
- 全ToDoデータをCSV形式でダウンロード
- Excelで開けるようにBOM付きUTF-8で出力
- ファイル名に日付を含める（todo_export_20250120.csv）

【CSVカラム】
ID, タイトル, 説明, 優先度, カテゴリ, 期限, ステータス, 作成日

【実装箇所】

■ TodoController.java
```java
@GetMapping("/export/csv")
public void exportCsv(HttpServletResponse response) throws IOException {
    // Content-Type設定
    response.setContentType("text/csv; charset=UTF-8");

    // ファイル名設定（日付付き）
    String filename = "todo_export_" + LocalDate.now().format(DateTimeFormatter.ofPattern("yyyyMMdd")) + ".csv";
    response.setHeader("Content-Disposition", "attachment; filename=\"" + filename + "\"");

    // BOM付きUTF-8で出力
    try (PrintWriter writer = response.getWriter()) {
        writer.write('\uFEFF'); // BOM
        csvExportService.writeCsv(writer, todoService.findAll());
    }
}
```

【注意点】
- CSVの値にカンマが含まれる場合はダブルクォートで囲む
- ダブルクォート自体はエスケープ（""）
- BOMがないとExcelで日本語が文字化けする
