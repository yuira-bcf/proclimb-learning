Section 08-3

生成プロンプト

以下のプロンプトをAIアシスタント（Claude、ChatGPT等）に入力することで、実装コードを生成できます。

smart_toy AIへの入力プロンプト

あなたはJavaプログラマーです。

ToDoアプリに優先度設定機能を追加してください。

【要件】
- 優先度は「高」「中」「低」の3段階
- 新規作成時と編集時に優先度を選択可能
- 一覧画面で優先度をバッジ表示（色分け）
- デフォルト値は「中」

【実装箇所】

■ Priority.java（新規作成）
- Enumで HIGH, MEDIUM, LOW を定義
- displayName（表示名）とcolor（カラーコード）を持つ
- 例: HIGH("高", "#ef4444")

■ Todo.java
- Priority priority フィールドを追加
- デフォルト値は Priority.MEDIUM

■ schema.sql
- priority VARCHAR(10) カラムを追加
- DEFAULT 'MEDIUM' を設定

■ TodoMapper.xml
- INSERT/UPDATE文にpriorityを追加
- priorityでの絞り込み・ソートに対応

■ form.html
- select要素でPriority一覧をth:eachで表示
- 編集時は現在の値を選択状態にする

■ list.html
- 優先度に応じたバッジを表示
- th:classappendで色分けクラスを付与

■ style.css
- .priority-high { background: #fef2f2; color: #dc2626; }
- .priority-medium { background: #fffbeb; color: #d97706; }
- .priority-low { background: #f0fdf4; color: #16a34a; }

必要なコードを提示してください。
lightbulb ポイント

Enumにメソッドを追加することで、表示名やカラーコードを一元管理できます。DBにはEnum名（文字列）を保存します。
