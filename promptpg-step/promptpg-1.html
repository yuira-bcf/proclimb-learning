Section 1-1

Spring Initializrでプロジェクト作成

Spring Initializrを使用して、Spring Bootプロジェクトを作成します。必要な依存関係を選択し、開発環境を構築しましょう。

smart_toy AIプロンプト

Spring Initializrで以下の設定でプロジェクトを作成する手順を教えてください。

- Project: Maven
- Language: Java
- Spring Boot: 3.x（最新安定版）
- Group: com.example
- Artifact: todo
- Packaging: Jar
- Java: 17

依存関係:
- Spring Web
- Thymeleaf
- Spring Data JPA
- H2 Database
- Lombok
- Spring Boot DevTools

プロジェクト作成後のIDEへのインポート方法も含めてください。

作成手順

language

1. Spring Initializrにアクセス

https://start.spring.io を開く

settings

2. 設定を入力

Project、Language、Spring Bootバージョン、メタデータを設定

  • Project: Maven
  • Language: Java
  • Spring Bootバージョン: 3.5.x
  • メタデータを設定: 変更無し
add_circle

3. 依存関係を追加

ADD DEPENDENCIESボタンから必要なライブラリを選択

  • Spring Web
  • Thymeleaf
  • Spring Data JPA
  • H2 Database
  • Lombok
  • Spring Boot DevTools
download

4. ダウンロード

GENERATEボタンでZIPファイルをダウンロード

Section 1-2

プロジェクト構造の確認

ダウンロードしたプロジェクトをIDEで開き、ディレクトリ構造を確認します。

VSCodeでのプロジェクト設定

folder_zip

1. ZIPファイルを解凍

ダウンロードしたZIPファイルを任意のフォルダに解凍

folder_open

2. VSCodeでフォルダを開く

「ファイル」→「フォルダーを開く」で解凍したフォルダを選択

extension

3. 拡張機能のインストール

以下の拡張機能をインストール（推奨）

  • Extension Pack for Java
  • Spring Boot Extension Pack
  • Lombok Annotations Support
sync

4. プロジェクトの同期

右下に表示される通知から「Import Java Projects」を選択し、依存関係を読み込む

プロジェクト構造
todo/
├── src/
│   ├── main/
│   │   ├── java/com/example/todo/
│   │   │   └── TodoApplication.java    ← エントリーポイント
│   │   └── resources/
│   │       ├── static/                  ← 静的ファイル（CSS, JS, 画像）
│   │       ├── templates/               ← Thymeleafテンプレート
│   │       └── application.properties   ← 設定ファイル
│   └── test/
└── pom.xml                              ← Maven設定
Section 1-3

Hello World の表示

最初のControllerを作成し、ブラウザに「Hello! Spring Boot」と表示させます。

smart_toy AIプロンプト

Spring Bootで「Hello! Spring Boot」と表示するシンプルなControllerを作成してください。

要件:
- URLは /hello
- Thymeleafテンプレートを使用
- メッセージは動的にControllerから渡す

Controller、テンプレート、必要な設定をすべて含めてください。
check_circle 確認ポイント

アプリケーションを起動し、http://localhost:8080/hello にアクセスして「Hello! Spring Boot」が表示されれば成功です。
