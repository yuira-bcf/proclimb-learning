プロンプト例 
あなたはJavaプログラマーです。

Spring Bootを使用して「Hello! Spring Boot」と表示するWebアプリケーションを作成してください。

【技術スタック】
- Java 17以上
- Spring Boot 3.x
- Thymeleaf（テンプレートエンジン）
- Maven または Gradle

【要件】
1. http://localhost:8080 にアクセスすると「Hello! Spring Boot」と表示される
2. Thymeleafテンプレートを使用してHTMLを表示する
3. Controllerクラスを作成してリクエストを処理する

【作成するファイル】
1. pom.xml - ビルド設定ファイル
2. TodoApplication.java - メインクラス
3. HelloController.java - Controllerクラス
4. index.html - テンプレートファイル
5. application.properties - 設定ファイル

【ファイル構成】
pom.xml

src/main/java/com/example/todo/
├── TodoApplication.java
└── controller/
    └── HelloController.java

src/main/resources/
├── application.properties
└── templates/
    └── index.html

各ファイルのコードを実装してください。

・パッケージ名 → com.example.todo（デフォルトでOK）
・CSSスタイル → なし（シンプルに始める）
・ポート番号 → 8080（デフォルトでOK）

Column Spring Initializrについて

Spring BootプロジェクトはSpring Initializr（https://start.spring.io）で簡単に作成できます。

  1. Project: Maven を選択
  2. Language: Java を選択
  3. Spring Boot: 3.x.x（最新安定版）を選択
  4. Dependencies: Spring Web と Thymeleaf を追加
  5. Generate ボタンでダウンロード

AIにプロジェクトを作成してもらう場合は、このプロンプトを使用してください。
