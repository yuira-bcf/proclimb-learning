Section 4-1

H2データベースの設定

H2データベースは、開発時に便利なインメモリデータベースです。設定が簡単で、すぐに使い始めることができます。

smart_toy AIプロンプト

Spring BootでH2データベースを設定してください。

要件:
- application.propertiesに必要な設定を追加
- H2コンソールを有効化（/h2-console）
- データベース名: tododb
- ユーザー名: sa
- パスワード: なし

開発時と本番環境での使い分けについても説明してください。
application.properties（H2設定例）
# H2 Database
spring.datasource.url=jdbc:h2:mem:tododb
spring.datasource.driverClassName=org.h2.Driver
spring.datasource.username=sa
spring.datasource.password=

# H2 Console
spring.h2.console.enabled=true
spring.h2.console.path=/h2-console

# JPA
spring.jpa.hibernate.ddl-auto=create-drop
spring.jpa.show-sql=true
Section 4-2

MySQLへの切り替え

本番環境ではMySQLなどの永続化データベースを使用します。設定の切り替え方法を学びます。

smart_toy AIプロンプト

Spring BootでMySQLに接続するための設定を教えてください。

要件:
- application.propertiesの設定
- pom.xmlに追加する依存関係
- MySQLでのデータベース作成コマンド
- 接続テストの方法

データベース名: tododb
ユーザー名: root
パスワード: password
ポート: 3306
H2
  • 設定が簡単
  • インメモリで高速
  • 開発・テスト向け
MySQL
  • データが永続化
  • 本番環境向け
  • 別途インストール必要
